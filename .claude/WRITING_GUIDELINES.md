# 콘텐츠 작성 가이드라인

이 문서는 edu-guide 프로젝트의 모든 콘텐츠 작성 시 반드시 준수해야 하는 규칙입니다.

---

## 1. 법적 준수 사항 (최우선)

### ⛔ 절대 금지 사항

**구체적 금액 정보 작성 금지:**
- ❌ "시간당 5만원"
- ❌ "월 40만원"
- ❌ "연간 500-1000만원대"
- ❌ "등록금 300만원"
- ❌ 모든 형태의 구체적 숫자 + 원/만원/천원

**과장 표현 자제:**
- ❌ "반드시", "절대", "100%", "무조건"
- ❌ "최고의", "최상의", "완벽한"
- ❌ 단언하는 표현들

### ✅ 대신 사용 가능한 표현

**금액 관련:**
- ✅ "상대적으로 저렴한 편"
- ✅ "일반고보다 높은 편"
- ✅ "경제적 부담이 있음"
- ✅ 비율 표현: "1:1의 60-70% 수준"
- ✅ "다양함", "시간당 비용과 횟수에 따라 다양함"
- ✅ "XX원" (변수 표현으로 예시를 들 때만)

**권유 표현:**
- ✅ "추천합니다", "좋습니다", "효과적입니다"
- ✅ "~할 수 있습니다", "~하는 편입니다"
- ✅ "대부분", "일반적으로", "보통"

## 2. CTA (Call-to-Action) 사용 규칙

### 사용 가능한 Shortcodes

프로젝트에서 사용 가능한 CTA shortcode는 다음과 같습니다:

#### A. 과외 관련 콘텐츠
```markdown
{{< cta-tutoring >}}
```
- **용도**: 과외 관련 모든 글 (중등과외, 고등과외 등)
- **위치**: 본문 중간 1회 + 글 끝 1회 (총 2회 권장)

#### B. 학습 전략/상담 콘텐츠
```markdown
{{< cta-consultation message="맞춤 학습 전략이 필요하신가요?" >}}
```
- **용도**: 학습법, 내신 관리, 자기주도학습 등
- **위치**: 본문 중간 1회
- **파라미터**: message는 상황에 맞게 변경 가능

#### C. Dual CTA (카톡 + 전화상담)
```markdown
{{< cta-dual type="inline" >}}
```
- **용도**: 본문 중간에 삽입
- **위치**: 본문 50% 지점 추천

```markdown
{{< cta-dual type="final" >}}
```
- **용도**: 글 마지막에 삽입
- **위치**: 본문 끝, 관련 글 링크 위

#### D. 최종 상담 CTA
```markdown
{{< cta-consultation-final >}}
```
- **용도**: 학습 전략 글의 마지막
- **위치**: 글 끝

### CTA 배치 가이드

**표준 구조 (과외 관련):**
```markdown
[본문 서론]

[본문 전개 40%]

{{< cta-tutoring >}}

[본문 전개 60%]

{{< cta-tutoring >}}
```

**표준 구조 (학습 전략):**
```markdown
[본문 서론]

{{< cta-consultation message="맞춤 전략이 필요하신가요?" >}}

[본문 전개 60%]

**관련 글:**
```

### ⚠️ 주의사항

- 너무 이른 시점(서론 직후)에는 삽입 지양

## 3. 이미지 사용 규칙

### Featured Image (대표 이미지)

모든 글의 frontmatter에는 반드시 `featured_image`를 포함해야 합니다.

**올바른 형식:**
```yaml
featured_image: "https://images.unsplash.com/photo-{IMAGE_ID}?w=1200&h=630&fit=crop"
```

### ✅ 권장 사항

1. **Unsplash 사용**: `https://images.unsplash.com/`
2. **일관된 크기**: `w=1200&h=630` (고정)
3. **fit=crop**: 반드시 포함
4. **검증된 이미지**: Unsplash에서 실제 존재하는 이미지 ID 사용

### ❌ 피해야 할 실수

- ❌ 로컬 경로 사용: `/images/photo.jpg`
- ❌ 깨진 URL
- ❌ 크기 파라미터 누락
- ❌ 존재하지 않는 이미지 ID

### 이미지 검색 방법

1. [Unsplash](https://unsplash.com/)에서 키워드로 검색
2. 이미지 선택 후 URL에서 photo ID 확인
3. 형식에 맞게 작성

**예시:**
```
Unsplash URL: https://unsplash.com/photos/abc123def456
↓
featured_image: "https://images.unsplash.com/photo-abc123def456?w=1200&h=630&fit=crop"
```

### 테마별 추천 키워드

- 과외/학습: student studying, education, tutoring, learning
- 시험/내신: exam, test, assessment
- 고등학교: high school, classroom
- 독서/어휘: books, reading, library
- 계획/관리: planning, notebook, calendar

## 5. 글 작성 프로세스

### 글 작성 전 체크리스트

- [ ] featured_image URL 검증 (Unsplash 실제 이미지)
- [ ] 카테고리가 올바른지 확인
- [ ] 태그 5-7개 선정
- [ ] 구체적 금액 언급 없는지 확인

### 글 작성 중

- [ ] CTA 위치 적절한지 확인
- [ ] 과장 표현 사용 안 했는지 확인
- [ ] FAQ 최소 5개 이상 작성
- [ ] 체크리스트/정리 섹션 포함

### 글 작성 후 검증

```bash
# 금액 표현 검사
grep -E "[0-9]+ ?만원|[0-9]+ ?천원|시간당.*[0-9]|월.*[0-9]+만" {파일명}

# 과장 표현 검사
grep -E "반드시|절대|100%|무조건|최고의|완벽한" {파일명}

# CTA 포함 확인
grep "{{< cta-" {파일명}

# 이미지 URL 확인
grep "featured_image" {파일명}
```

## 7. 자주하는 실수 체크리스트

### ❌ 하지 말아야 할 것

- [ ] 구체적 금액 언급 (X만원, X천원)
- [ ] 과장된 단언 ("반드시", "100%", "절대")

- [ ] 깨진 이미지 URL
- [ ] 로컬 이미지 경로 사용
- [ ] FAQ 없음
- [ ] 관련 글 링크 없음
- [ ] 너무 짧은 글 (500줄 미만)

### ✅ 반드시 해야 할 것

- [ ] 800-1000줄 분량 (중등 기준)
- [ ] CTA 2-3회 적절히 배치
- [ ] FAQ 5개 이상
- [ ] 체크리스트/실행 가이드 포함
- [ ] Unsplash 검증된 이미지
- [ ] 관련 글 링크 3개 이상
- [ ] 법적 준수 (금액/과장 표현 없음)
