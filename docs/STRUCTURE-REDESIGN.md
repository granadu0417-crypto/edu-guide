## 🔗 내부 링크 전략

### 정보성 → 전환 유도

**예시 1**: 학습 가이드 글
```
"중학생 영어 문법 정리" (정보성 콘텐츠)
    ↓
[관련 글]
- "영어 학원 vs 과외, 어떤 게 나을까?" (비교)
- "강남구 영어 학원 추천 가이드" (지역)
    ↓
[CTA]
"우리 아이에게 맞는 영어 학습법 무료 상담 받기 →"
```

**예시 2**: 지역 정보 글
```
"마포구 교육 환경 분석" (지역 정보)
    ↓
[관련 글]
- "마포구 수학 학원 찾는 법" (구체적)
- "학원 선택 체크리스트" (가이드)
    ↓
[CTA]
"마포구 맞춤 학원/과외 추천 받기 →"
```

## 📱 CTA (Call-to-Action) 배치

### 1. Floating Button (모든 페이지)
```
┌─────────────────┐
│ 💬 무료 학습 상담 │  ← 우하단 고정
└─────────────────┘
   ↓ 클릭 시
┌─────────────────┐
│ ☎️ 전화 상담     │
│ 💬 카카오톡 상담 │
│ 📝 폼 작성하기   │
└─────────────────┘
```

### 2. 사이드바 CTA
```
┌─────────────────────┐
│ 🎓 무료 학습 진단    │
│                     │
│ ✓ 학습 고민 상담     │
│ ✓ 맞춤 학원/과외 추천│
│ ✓ 3분 간단 신청     │
│                     │
│ [지금 신청하기 →]    │
└─────────────────────┘
```

### 3. 콘텐츠 내 자연스러운 CTA
```
글 중간 또는 끝:

┌────────────────────────────────┐
│ 💡 이런 고민 있으신가요?        │
│                                │
│ ✓ 우리 아이에게 맞는 학습법은?  │

│                                │
│ [무료로 전문가 상담 받기 →]     │
└────────────────────────────────┘
```

## 🛠️ Hugo 구현 방법

### 1. 섹션 디렉토리 구조
```
content/
├── elementary/     (기존)
├── middle/         (기존)
├── high/           (기존)
├── subjects/       (신규 - 과목별)
│   ├── korean/
│   ├── english/
│   ├── math/
│   ├── science/
│   └── social/
├── tutoring/       (신규 - 학원/과외)
│   ├── guide/
│   ├── academy/
│   └── private/
├── local/          (강화 - 지역별)
│   ├── seoul/
│   │   ├── gangnam/
│   │   ├── seocho/
│   │   └── ...
│   └── gyeonggi/
│       ├── bundang/
│       └── ...
├── consultation/   (신규 - 상담)
│   └── _index.md
└── education-info/ (신규 - 교육정보)
    ├── admission/
    └── system/
```

### 2. 메뉴 설정 (config.toml)
```toml
[menu]
  [[menu.main]]
    name = "학습 가이드"
    identifier = "learning"
    weight = 1

  [[menu.main]]
    name = "초등학생"
    parent = "learning"
    url = "/elementary/"
    weight = 1

  [[menu.main]]
    name = "중학생"
    parent = "learning"
    url = "/middle/"
    weight = 2

  [[menu.main]]
    name = "고등학생"
    parent = "learning"
    url = "/high/"
    weight = 3

  [[menu.main]]
    name = "과목별"
    identifier = "subjects"
    weight = 2

  [[menu.main]]
    name = "국어"
    parent = "subjects"
    url = "/subjects/korean/"
    weight = 1

  # ... (나머지 과목들)

  [[menu.main]]
    name = "학원·과외"
    identifier = "tutoring"
    weight = 3

  # ... (하위 메뉴들)

  [[menu.main]]
    name = "무료 상담"
    url = "/consultation/"
    weight = 5
```
