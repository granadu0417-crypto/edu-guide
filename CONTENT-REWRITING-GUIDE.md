## 📝 리라이팅 규칙

### 1. 본문 리라이팅 원칙

- **같은 의미, 다른 표현**: 원본의 핵심 정보는 유지하되 문장 구조와 어휘를 변경
- **자연스러운 문체**: 딱딱한 설명문이 아닌, 학부모에게 말하듯 친근한 어조
- **구체적인 예시 추가**: 추상적인 설명보다 구체적인 상황/예시 제시
- **학년별 특성 반영**: 해당 학년의 특징과 고민을 반영

### 2. 문장 변형 예시

| 원본 | 리라이팅 버전 1 | 리라이팅 버전 2 |
|------|----------------|----------------|
| "영어 과외를 통해 실력을 향상시킬 수 있습니다" | "1:1 맞춤 수업으로 우리 아이 영어 실력이 눈에 띄게 달라집니다" | "선생님과 함께하는 영어 공부, 아이가 먼저 '재밌다'고 말합니다" |
| "체계적인 커리큘럼을 제공합니다" | "아이 수준에 딱 맞춘 단계별 학습 로드맵을 설계해드립니다" | "우리 아이만을 위한 맞춤 학습 계획표, 함께 만들어갑니다" |

### 3. 섹션별 리라이팅 포인트

각 섹션은 다음 요소를 포함해야 합니다:

1. **도입부**: 학부모의 고민/걱정에 공감
2. **본문**: 해결책 제시 (구체적이고 실용적으로)
3. **연한 상아색 박스**: 해당 섹션 내용을 바탕으로 한 홍보 문구
4. **마무리**: 행동 유도 (CTA와 연결)

## 🟡 연한 상아색 박스 규칙

### 스타일 코드

```html
<div style="background-color: #FDF8F0; border-left: 3px solid #d4a574; padding: 18px; margin: 15px 0; font-size: 0.95em;">
<strong>이렇게 수업합니다!</strong><br>
[위 섹션 내용을 참고한 홍보 문구 - 2~3문장]
</div>
```

### 작성 원칙

1. **위치**: 각 주요 섹션(h2 또는 h3) 끝에 배치
2. **개수**: 글 하나당 6~10개 권장
3. **문구 길이**: 2~3문장 (너무 길지 않게)
4. **시작**: 항상 `<strong>이렇게 수업합니다!</strong><br>`로 시작
5. **내용**: 바로 위 섹션 내용과 연관된 서비스 홍보

### 문구 작성 예시

| 섹션 주제 | 상아색 박스 문구 |
|-----------|------------------|
| 영어 회화 수업 | 원어민 발음과 자연스러운 표현! 아이가 직접 말해보며 자신감을 키웁니다. 매 수업 스피킹 연습 포함! |
| 문법 학습 | 어려운 문법도 쉽게! 아이 눈높이에 맞춘 설명으로 헷갈리던 문법이 정리됩니다. |
| 시험 대비 | 내신부터 수능까지! 기출문제 분석과 취약점 집중 보완으로 목표 점수를 달성합니다. |
| 학습 습관 | 공부 습관이 달라집니다! 매일 꾸준히 하는 습관, 선생님과 함께 만들어갑니다. |

### 주의사항

- 같은 글 내에서 상아색 박스 문구가 중복되지 않도록 주의
- 각 박스는 해당 섹션의 내용을 반영해야 함
- 과장된 표현 지양 (예: "100% 성적 향상 보장" X)
- **금액 정보는 상아색 박스에 넣지 않음** - 본문에 삽입

## 🔘 CTA 버튼 규칙

### 사용할 스타일

- **`type="final"`만 사용** (보라색 그라데이션 스타일)
- `type="inline"` 사용 안 함 (궁금한 점이 있으신가요? 스타일 ❌)

### CTA 코드

```hugo
{{</* cta-dual type="final" */>}}
```

### 배치 규칙

1. **글 중단에 1개만 배치** (예: "선생님 선택" 섹션 뒤, 글 전체의 중간 정도)
2. single.html 템플릿에서 자동으로 하단 CTA 배너가 추가됨
3. 총 2개의 보라색 CTA가 표시됨 (마크다운 중단 1개 + 템플릿 하단 1개)
