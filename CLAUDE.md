# 콘텐츠 리라이팅 가이드라인

이 문서는 edu-guide 프로젝트의 콘텐츠 리라이팅 작업에 대한 완전한 가이드입니다.
**새로운 세션에서도 반드시 이 문서를 먼저 읽고 작업을 진행하세요.**

---

## 1. 리라이팅의 핵심 의도

### 절대 잊지 말아야 할 것
- **템플릿 복사 + 지역명만 변경 = 잘못된 방식**
- **같은 정보라도 표현을 다르게 = 올바른 방식**

### 목표
각 파일이 마치 **다른 사람이 쓴 블로그 글**처럼 보이게 하는 것.
- 같은 주제, 같은 정보를 다루더라도
- 문장 표현, 비유, 설명 순서, 서두/마무리가 다르게
- SEO에서 중복 콘텐츠로 인식되지 않도록

### 예시: 같은 내용의 다른 표현
```
# "고등학교 수학이 어렵다"는 내용을 다르게 표현하는 방법

버전 A: "고등학교에 올라가면 수학이 확 어려워집니다."
버전 B: "중학교 때 수학 잘했는데, 고등학교 가니 갑자기 안 되네요."
버전 C: "수학 때문에 머리 아프시죠? 고등학교 수학은 차원이 다릅니다."
버전 D: "고등학교 수학, 왜 이렇게 어려운 걸까요?"
버전 E: "많은 학생들이 고등학교에 올라와서 수학의 벽에 부딪힙니다."
```

## 2. 고정 정보 (절대 변경 불가)

### 2-1. 수학 과외 비용
```
**고1~2**는 주1회 기준 25만원에서 35만원, 주2회 기준 42만원에서 58만원 선입니다.

**고3**은 주1회 기준 30만원에서 42만원, 주2회 기준 50만원에서 70만원이 일반적입니다.
```

### 2-2. 영어 과외 비용
```
**고1~2**는 주1회 기준 22만원에서 32만원, 주2회 기준 38만원에서 52만원 선입니다.

**고3**은 주1회 기준 28만원에서 38만원, 주2회 기준 45만원에서 62만원이 일반적입니다.
```

### 2-3. 지역별 학교 정보
각 지역에 해당하는 학교는 정확히 유지해야 함:
- **종로구**: 경복고, 동성고, 중앙고
- **강남구**: 휘문고, 단대부고, 중동고, 숙명여고 등 (파일별로 확인 필요)
- **기타 지역**: 해당 파일의 기존 학교 정보 유지

## 3. 변경 가능/변경해야 하는 요소

### 3-1. 서두 (첫 문장/문단)
파일마다 다르게 시작해야 함:
```
# 예시들 (수학 과외 글)
- "고등학교 수학, 중학교 때와 완전히 다르네요."
- "수학 성적이 갑자기 떨어졌나요?"
- "고등학교에 올라온 후 수학이 어려워졌다는 말, 정말 많이 듣습니다."
- "내 아이 수학, 왜 이렇게 안 오르는 걸까요?"
- "수포자가 되기 전에, 지금 바로 잡아야 합니다."
```

### 3-2. 섹션 제목
같은 내용이라도 제목을 다르게:
```
# "고등학교 수학의 특징" 섹션 제목 변형
- 고등학교 수학, 무엇이 달라질까요?
- 고등 수학, 이게 왜 이렇게 어려운 거죠?
- 중학교와 다른 고등학교 수학의 특징
- 고등학교 수학이 어려운 진짜 이유
```

### 3-3. 설명 방식
같은 정보를 다른 방식으로 설명:
```
# "킬러 문항"에 대한 설명 변형

버전 A: "킬러 문항(21번, 29번, 30번)은 수능에서 가장 어려운 문제입니다."
버전 B: "수능 수학에서 21번, 29번, 30번은 '킬러'라고 불립니다. 이 세 문제가 1등급과 2등급을 가릅니다."
버전 C: "많은 학생들이 킬러 문항에서 무너집니다. 21번, 29번, 30번 - 이 세 문제만 풀 수 있다면 1등급입니다."
```

### 3-4. 아이보리 박스 내용
7개의 아이보리 박스 내용도 파일마다 조금씩 다르게:
```
# 첫 번째 박스 (진단) 변형

버전 A: "첫 수업에서 학생의 수학 실력을 정확하게 진단합니다."
버전 B: "수업 시작 전, 학생이 어디서 막히는지 정확히 파악합니다."
버전 C: "무작정 진도를 나가지 않습니다. 먼저 학생의 현재 실력을 꼼꼼히 체크합니다."
```

### 3-5. FAQ 질문/답변 표현
같은 질문이라도 표현을 다르게:
```
# "내신과 수능 중 어떤 것을 먼저?" 변형

질문 A: "내신과 수능 중 어떤 것을 먼저 준비해야 하나요?"
질문 B: "내신이 중요한가요, 수능이 중요한가요?"
질문 C: "내신과 수능, 어디에 더 집중해야 할까요?"
```

### 3-6. 마무리 문단
파일마다 다른 마무리:
```
버전 A: "종로구 삼청동 고등학생 수학, 지금 시작하세요."
버전 B: "지금이 바로 시작할 때입니다. 삼청동 고등학생 여러분의 수학 실력 향상을 응원합니다."
버전 C: "수학은 시작이 반입니다. 오늘 상담받고, 내일부터 달라지세요."
```

## 5. OLD 포맷 vs NEW 포맷 비교

### OLD 포맷 특징 (변환 전)
- 약 275~300줄
- 이모지 사용 (🎯📚💡🔥✅ 등)
- 불릿 포인트 많이 사용
- `{{< cta-dual type="middle" >}}` 사용
- `{{< cta-dual type="final" >}}` 사용
- "관련 가이드" 섹션 있음
- --- 구분선 있음

### NEW 포맷 특징 (변환 후)
- 약 160~165줄
- 이모지 없음
- 불릿 없음 (문단 형태)
- 중간 CTA 없음
- `{{< cta-dual type="final" >}}` 만 사용
- "관련 가이드" 섹션 없음
- 구분선 없음
- 7개의 아이보리 박스

```bash

Read: content/high/[파일명1].md
Read: content/high/[파일명2].md
```

### 7-2. 포맷 확인
- 이모지 있으면 → OLD 포맷 → 리라이팅 필요
- 아이보리 박스 있으면 → NEW 포맷 → 이미 완료

### 7-3. 리라이팅 시 체크리스트

- [ ] 설명 방식/비유가 다른가?
- [ ] 아이보리 박스 내용이 다른가?
- [ ] 마무리가 다른가?
- [ ] 고정 정보(비용, 학교)는 정확한가?
- [ ] NEW 포맷 규칙을 모두 지켰는가?

### 7-4. 커밋 & 푸시

```bash
git add -A && git commit -m "콘텐츠 리라이팅: [지역] [학교급] [과목] 가이드 N개 파일"
```

## 8. 표현 변형 아이디어 모음

### 8-1. 서두 변형
```
# 수학
- "고등학교 수학, 중학교 때와 완전히 다르네요."
- "수학 성적이 갑자기 떨어졌나요? 고등학교 수학은 다릅니다."
- "내 아이가 수포자가 될까 봐 걱정되시나요?"
- "수학만 잘해도 대학이 달라집니다."
- "고등학교 수학, 어디서부터 잡아야 할지 모르겠다면"

# 영어
- "고등학교 영어, 중학교 때와 완전히 다르네요."
- "영어 성적이 오르지 않아 고민이신가요?"
- "수능 영어 1등급, 생각보다 어렵지 않습니다."
- "영어는 꾸준함이 답입니다. 지금 시작하세요."
- "고등학교 영어, 막막하다면 함께 시작해요."
```

### 8-2. "왜 과외가 필요한가" 설명 변형
```
버전 A: "학원은 진도만 나가지만, 과외는 학생 이해도에 맞춥니다."
버전 B: "대형 학원에서 못 따라가는 학생이 과외에서 성적이 오릅니다."
버전 C: "1:1로 약점을 정확히 파악하고 집중 보완하는 것이 과외의 강점입니다."
버전 D: "학원에서 질문 한 번 못 하고 나오는 학생, 과외에서는 다릅니다."
```

### 8-3. 학교 설명 변형 (같은 학교, 다른 표현)
```
# 경복고 설명 변형
버전 A: "경복고는 종로구에 위치한 명문 고등학교입니다."
버전 B: "명문 경복고는 난이도 높은 시험으로 유명합니다."
버전 C: "경복고 시험은 만만치 않습니다. 심화 문제가 쏟아집니다."
```

## 9. 주의사항

### 9-1. 표현만 다르게, 정보는 정확하게
- 비용, 학교명, 과목 특성 등 **사실 정보는 정확히** 유지
- **표현과 문장 구조만** 다르게

### 9-2. 자연스러움 유지
- 억지로 다르게 쓰려다 어색해지지 않도록
- 실제 블로그 글처럼 자연스럽게

### 9-3. 일관성 유지
- 같은 파일 내에서는 톤앤매너 일관성 유지
- 파일 간에는 다양성 추구
