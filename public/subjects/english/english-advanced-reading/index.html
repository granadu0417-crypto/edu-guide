1. 고급 독해란?

1.1 정의와 수준

고급 독해의 특징

  • 복잡한 구문
  • 추상적 개념
  • 함축적 의미
  • 논리적 전개
  • 배경 지식 필요

기본 독해와의 차이

  • 기본: 문장 이해, 직접적 정보
  • 고급: 구조 분석, 추론, 비판

필요한 시기

  • 고등학교 2학년 이상
  • 수능 준비
  • 대학 입학 후

1.2 중요성

수능

  • 고난도 지문
  • 변별력 문제
  • 만점을 위한 필수

대학 영어

  • 전공 서적
  • 논문 읽기
  • 학술적 글쓰기

실용

  • 영어 원서
  • 뉴스, 칼럼
  • 전문 자료

1.3 어려운 이유

문제점

  • 어휘 수준 높음
  • 구문 복잡
  • 배경 지식 부족
  • 추상적 개념
  • 논리 파악 어려움

극복 방법

  • 체계적 구문 분석
  • 배경 지식 확충
  • 다양한 지문 경험
  • 논리 훈련

2. 복잡한 구문 분석

2.1 긴 문장 끊어 읽기

명사절

  • That/Whether/If절
  • 간접의문문
  • 주어, 목적어, 보어로

예시

  • That he succeeded surprised everyone.
  • [That he succeeded] (주어절) / surprised (동사) / everyone (목적어)

관계절

  • 관계대명사
  • 관계부사
  • 선행사 수식

접속사절

  • 시간, 이유, 조건, 양보
  • 부사 역할

분사구문

  • ~ing / ~ed
  • 시간, 이유, 조건
  • 접속사 생략형

2.2 도치

부정어 도치

  • Never, Seldom, Rarely
  • 부정어 + 조동사 + 주어

예시

  • Never have I seen such a beautiful place.

Only 도치

  • Only then did he understand.

장소 부사구 도치

  • Here comes the bus.

2.3 생략

접속사 뒤 생략

  • When (you are) young, …
  • If (it is) possible, …

관계대명사 생략

  • The book (that) I bought

비교 구문 생략

  • He is taller than I (am tall).

2.4 삽입

콤마 사이

  • He, I believe, will come.

대시 사이

  • The plan — we all agreed — was perfect.

괄호

  • The result (see Table 1) shows…

FAQ

Q1. 고급 독해는 언제부터 시작하나요?

A1. 고1 겨울방학이 적기입니다.

  • 기본 독해 완성 후
  • 수능 대비 시작
  • 충분한 시간 확보

Q2. 복잡한 구문을 못 읽겠어요.

A2. 끊어 읽기부터 시작하세요.

  • 절 구분
  • 수식 관계 파악
  • 핵심 문장 성분 찾기
  • 반복 연습

Q3. 배경 지식이 부족해요.

A3. 다양한 독서를 하세요.

  • 교양서 읽기
  • 신문 칼럼
  • 다큐멘터리
  • 타 과목 공부

Q4. 문학 지문이 어려워요.

A4. 함축적 의미를 찾으세요.

  • 상징
  • 은유
  • 행간의 의미
  • 작품 배경 공부

Q5. 논리를 못 따라가겠어요.

A5. 주장과 근거를 구분하세요.

  • 필자 주장 찾기
  • 근거 확인
  • 연결 고리 파악
  • 논리 오류 찾기

Q6. 시간이 부족해요.

A6. 속도 연습을 하세요.

  • 시간 재며 읽기
  • 스키밍 연습
  • 핵심만 파악
  • 불필요한 반복 읽기 줄이기

Q7. 어휘가 너무 어려워요.

A7. 문맥으로 추론하세요.

  • 앞뒤 맥락
  • 어원 활용
  • 고급 어휘 학습
  • 어휘 노트

Q8. 비판적 독해가 뭔가요?

A8. 의심하고 평가하는 것입니다.

  • 무조건 믿지 않기
  • 근거 확인
  • 논리 오류 찾기
