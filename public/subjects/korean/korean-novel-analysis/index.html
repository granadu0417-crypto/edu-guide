5. 시점 분석

1인칭 관찰자 시점

  • “나"가 옆에서 관찰
  • 제한적 정보

3인칭 전지적 시점

  • 서술자가 모든 것 앎
  • 객관적 서술
  • 심리 묘사 자유

3인칭 관찰자 시점

  • 겉모습만 관찰
  • 객관적
  • 심리 직접 모름

5.2 시점의 효과

1인칭의 효과

  • 진솔함
  • 공감
  • 한계: 제한적 정보

3인칭 전지적의 효과

  • 폭넓은 시야
  • 다양한 인물 심리
  • 한계: 거리감

3인칭 관찰자의 효과

  • 객관성
  • 긴장감
  • 한계: 심리 파악 어려움

5.3 시점 파악하기

서술자 확인

  • 1인칭: “나”, “우리”
  • 3인칭: 이름, “그”, “그녀”

정보의 범위

  • 전지적: 모든 것 앎
  • 제한적: 일부만 앎

FAQ

Q1. 소설을 어떻게 읽어야 하나요?

A1. 두 번 읽으세요.

  1. 처음: 전체 줄거리 파악
  2. 다시: 꼼꼼히 분석
  • 메모하며 읽기
  • 질문하며 읽기

Q2. 주제를 어떻게 찾나요?

A2. 여러 단서를 종합하세요.

  • 제목
  • 반복되는 말·장면
  • 인물의 깨달음
  • 갈등과 해결
  • 결말의 의미

Q3. 시점을 어떻게 구분하나요?

A3. 서술자를 확인하세요.

  • “나”: 1인칭
  • 이름, “그”: 3인칭
  • 모든 것 아는가? → 전지적
  • 겉모습만? → 관찰자

Q4. 인물 성격을 어떻게 파악하나요?

A4. 말과 행동을 주목하세요.

  • 대사
  • 행동
  • 다른 인물의 평가
  • 심리 묘사
  • 갈등 상황 대처

Q5. 고전소설이 어려워요.

A5. 배경지식을 먼저 익히세요.

  • 시대적 배경
  • 어휘 정리
  • 현대어 번역본
  • 선생님 도움

Q6. 소설 분석이 재미없어요.

A6. 흥미로운 작품부터 시작하세요.

  • 관심 분야
  • 쉬운 작품
  • 영화·드라마 원작 → 점차 확대

Q7. 감상문을 어떻게 쓰나요?

A7. 자신의 생각을 담으세요.

  • 줄거리 요약 X
  • 인상 깊은 부분
  • 공감/비판
  • 삶과 연결

Q8. 수능 소설은 어떻게 대비하나요?

A8. 다양한 작품 읽고 문제 풀기.

  • 기출 문제 분석
  • 시간 재며 연습
  • 선지 분석 능력
  • 근거 찾기 훈련

Q9. 소설 분석 과외가 필요한가요?

A9. 심화 학습에 효과적입니다.

  • 작품 해설
  • 분석 방법 지도
  • 감상문 첨삭
  • 필독서 선정

Q10. 얼마나 읽어야 하나요?

A10. 꾸준히 읽는 것이 중요합니다.

  • 주 1-2권
  • 다양한 장르
  • 메모하며 읽기
  • 토론·감상 나누기
