2. 보고서 구성

2.1 표지

필수 항목

  • 실험 제목
  • 실험 날짜
  • 학년, 반, 번호, 이름
  • 공동 실험자 (있는 경우)

제목 작성법

  • 명확하고 구체적으로
  • 실험 내용 반영
  • 간결하게

2.2 실험 목적

의미

  • 왜 이 실험을 하는가?
  • 무엇을 알아보려 하는가?

작성 요령

  • 한두 문장으로
  • “~을 알아본다”
  • “~의 관계를 확인한다”
  • 명확하게

예시

  • “물의 온도에 따른 용해도 변화를 알아본다”
  • “산과 염기의 중화 반응을 관찰한다”

2.3 실험 원리

포함 내용

  • 관련 과학 개념
  • 법칙, 이론
  • 공식

작성 요령

  • 교과서 내용 참고
  • 이해한 대로 표현
  • 그림, 도표 활용

깊이

  • 중학생: 기본 개념
  • 고등학생: 심화 원리
  • 수준에 맞게

2.4 실험 재료 및 기구

나열 방식

  • 모든 재료 기록
  • 수량 명시
  • 규격 (필요시)

예시

  • 비커 (500mL) 1개
  • 온도계 1개
  • 소금 50g
  • 증류수 200mL

2.5 실험 방법

작성 원칙

  • 순서대로
  • 단계별로 번호
  • 명령형 또는 평서형

상세도

  • 다른 사람이 재현 가능하도록
  • 구체적 수치
  • 주의 사항

예시

  1. 비커에 물 200mL를 넣는다.
  2. 온도계로 물의 온도를 측정한다.
  3. 소금 10g을 넣고 저어준다.
  4. 모두 녹을 때까지의 시간을 기록한다.

그림

  • 실험 장치도
  • 간단하고 명확하게
  • 라벨 표시

2.6 실험 결과

데이터 기록

  • 관찰한 것 그대로
  • 수치 정확히
  • 단위 명시

표 작성

  • 깔끔하게 정리
  • 제목 명시
  • 항목 명확히

그래프

  • 적절한 종류 선택
  • 축 이름, 단위
  • 제목
  • 범례

사진

  • 중요 장면 촬영
  • 선명하게
  • 설명 추가

2.7 결론

내용

  • 목적에 대한 답
  • 결과 요약
  • 발견한 사실

작성법

  • 간결하게
  • 데이터 근거

3. 실험 설계

3.1 탐구 문제 설정

좋은 탐구 문제

  • 관찰에서 출발
  • 궁금한 것
  • 실험 가능한 것
  • 변인 명확

예시

  • “온도가 효소 활성에 영향을 줄까?”
  • “빛의 색깔이 광합성 속도에 영향을 줄까?”

3.2 가설 설정

가설이란?

  • 예상되는 답
  • 근거 있는 추측

작성법

  • “~하면 ~할 것이다”
  • “~일수록 ~할 것이다”

예시

  • “온도가 높을수록 효소 활성이 증가할 것이다”

3.3 변인 설정

독립 변인

  • 의도적으로 변화시키는 것
  • 원인

종속 변인

  • 측정하는 것
  • 결과

통제 변인

  • 일정하게 유지
  • 공정한 실험

예시: 온도와 용해도

  • 독립: 온도
  • 종속: 용해도
  • 통제: 물의 양, 소금 종류, 저어주는 방법

3.4 실험 설계

대조 실험

  • 비교 대상
  • 차이 확인

반복 실험

  • 여러 번 측정
  • 평균값
  • 신뢰도 향상

정량적 측정

  • 수치화
  • 객관적 비교

FAQ

Q1. 실험 보고서를 어디서부터 시작하나요?

A1. 실험 목적부터 명확히 하세요.

  1. 무엇을 알아보려 하는가?
  2. 그에 맞는 실험 설계
  3. 순서대로 작성

Q2. 결과와 결론의 차이는?

A2. 결과는 사실, 결론은 해석입니다.

  • 결과: 관찰, 측정한 데이터
  • 결론: 그 데이터가 의미하는 것

Q3. 고찰을 어떻게 써야 하나요?

A3. 다음을 포함하세요.

  • 왜 그런 결과가 나왔나?
  • 원리와 연결
  • 오차 분석
  • 개선 방안
  • 추가 탐구

Q4. 예상과 다른 결과가 나왔어요.

A4. 솔직하게 기록하고 분석하세요.

  • 그대로 기록
  • 원인 추정
  • 오차 가능성
  • 재실험 제안

Q5. 그래프를 어떻게 그리나요?

A5. 적절한 종류를 선택하세요.

  • 변화 추세: 꺾은선
  • 비교: 막대
  • 상관관계: 산점도
  • 제목, 축 이름, 단위 필수

Q6. 오차를 줄이려면?
