# 영어 형용사·부사 완벽 정복

형용사와 부사가 헷갈리나요?

"형용사와 부사를 어떻게 구별하나요?"
"-ly를 언제 붙이나요?"
"good과 well의 차이가 뭔가요?"

형용사·부사는 **역할과 위치 이해**로 반드시 정복할 수 있습니다.

## 형용사 vs 부사

### 형용사 (Adjective)

**명사를 수식**

```
역할: 명사 꾸밈
위치: 명사 앞, be동사 뒤

a beautiful girl (아름다운 소녀)
The book is interesting. (책이 재미있다)
```

### 부사 (Adverb)

**동사/형용사/부사/문장 수식**

```
역할: 동사, 형용사, 부사, 문장 꾸밈
형태: 대부분 -ly

He runs fast. (빨리 달린다 - 동사 수식)
She is very tall. (매우 크다 - 형용사 수식)
He runs very fast. (매우 빨리 - 부사 수식)
Fortunately, he passed. (다행히 - 문장 수식)
```

## 형용사의 용법

### 한정적 용법

**명사 앞에서 수식**

```
a good student (좋은 학생)
a beautiful flower (아름다운 꽃)
an interesting book (재미있는 책)
```

### 서술적 용법

**be동사/감각동사 뒤**

```
be동사:
She is beautiful. (그녀는 아름답다)
The book is interesting. (책이 재미있다)

감각동사:
It looks good. (좋아 보인다)
It smells nice. (좋은 냄새가 난다)
It sounds great. (좋게 들린다)
It tastes sweet. (달콤한 맛이 난다)
It feels soft. (부드럽게 느껴진다)
```

### 수량 형용사

**many, much, some, any, few, little**

```
many books (많은 책들)
much water (많은 물)
some money (약간의 돈)
```

## 부사의 용법

### 동사 수식

**방법, 정도 표현**

```
He runs fast. (빨리 달린다)
She speaks slowly. (천천히 말한다)
They work hard. (열심히 일한다)
```

### 형용사 수식

**정도 표현**

```
very tall (매우 큰)
too expensive (너무 비싼)
quite interesting (꽤 재미있는)
really good (정말 좋은)
```

### 부사 수식

**정도 표현**

```
very quickly (매우 빠르게)
too slowly (너무 천천히)
quite well (꽤 잘)
```

### 문장 전체 수식

**화자의 태도**

```
Fortunately, he passed the exam.
(다행히, 그는 시험에 합격했다)

Unfortunately, it rained.
(불행히도, 비가 왔다)

Actually, I don't know.
(사실, 나는 모른다)
```

## 형용사 → 부사 변화

### 규칙 변화: -ly

**형용사 + ly:**

```
slow → slowly (천천히)
quick → quickly (빠르게)
careful → carefully (조심스럽게)
beautiful → beautifully (아름답게)
```

**-y → -ily:**

```
happy → happily (행복하게)
easy → easily (쉽게)
heavy → heavily (무겁게)
```

**-le → -ly:**

```
simple → simply (간단하게)
possible → possibly (가능하게)
```

**-ic → -ically:**

```
basic → basically (기본적으로)
dramatic → dramatically (극적으로)
```

### 불규칙 변화

**형태 변화:**

```
good → well (잘)
fast → fast (빠르게 - 동일)
hard → hard (열심히 - 동일)
late → late (늦게 - 동일)
early → early (일찍 - 동일)
```

**주의: 의미 변화:**

```
hard (열심히) vs hardly (거의 ~않다)
late (늦게) vs lately (최근에)
near (가까이) vs nearly (거의)
high (높이) vs highly (매우)
```

## 형용사·부사 위치

### 형용사 위치

**1. 명사 앞:**

```
a beautiful girl
an interesting book
```

**2. be동사/감각동사 뒤:**

```
She is beautiful.
It looks good.
```

**3. something/anything 뒤:**

```
something important (중요한 것)
anything special (특별한 것)
nothing new (새로운 것 없음)
```

### 부사 위치

**1. 일반 동사 뒤:**

```
He runs fast.
She speaks English well.
```

**2. 형용사/부사 앞:**

```
very tall
really quickly
```

**3. 문장 처음/끝:**

```
Fortunately, he passed.
He passed the exam easily.
```

**4. be동사/조동사 뒤, 일반동사 앞:**

```
She is always late. (be동사 뒤)
He can never understand. (조동사 뒤)
She usually gets up early. (일반동사 앞)
```

## 빈도부사

### 위치

**be동사 뒤, 일반동사 앞, 조동사 뒤**

```
always (항상 - 100%)
usually (보통 - 80%)
often (자주 - 60%)
sometimes (때때로 - 40%)
seldom (드물게 - 20%)
never (절대 ~않다 - 0%)
```

**예문:**

```
I am always tired. (be동사 뒤)
I always get up early. (일반동사 앞)
I can always help you. (조동사 뒤)
```

## 혼동하기 쉬운 형용사·부사

### good vs well

**good: 형용사**

```
He is a good student. (좋은 학생)
This book is good. (이 책은 좋다)
```

**well: 부사 (건강한)**

```
He speaks English well. (영어를 잘한다)
She is well. (그녀는 건강하다)
```

### hard vs hardly

**hard: 열심히 (부사), 어려운 (형용사)**

```
He works hard. (열심히 일한다)
This is a hard question. (어려운 질문)
```

**hardly: 거의 ~않다**

```
I can hardly see it.
(그것을 거의 볼 수 없다)

He hardly studies.
(그는 거의 공부하지 않는다)
```

### late vs lately

**late: 늦게**

```
He came late. (늦게 왔다)
Don't be late. (늦지 마)
```

**lately: 최근에 (= recently)**

```
I haven't seen him lately.
(최근에 그를 못 봤다)
```

### near vs nearly

**near: 가까이**

```
Come near. (가까이 와)
He lives near here. (근처에 산다)
```

**nearly: 거의 (= almost)**

```
It's nearly 5 o'clock.
(거의 5시다)

I nearly fell.
(하마터면 넘어질 뻔했다)
```

### high vs highly

**high: 높이**

```
The bird flew high. (새가 높이 날았다)
Jump high. (높이 뛰어)
```

**highly: 매우 (추상적)**

```
I highly recommend it.
(매우 추천한다)

He is highly respected.
(매우 존경받는다)
```

## enough 위치

### 형용사/부사 뒤

```
형용사 + enough:
He is old enough to drive.
(그는 운전할 만큼 충분히 나이가 들었다)

부사 + enough:
He runs fast enough.
(그는 충분히 빨리 달린다)
```

### 명사 앞/뒤

```
enough + 명사:
I have enough money.

명사 + enough:
I have money enough. (덜 일반적)
```

## too vs very

### very: 매우 (중립)

```
She is very tall.
(그녀는 매우 크다 - 사실 진술)
```

### too: 너무 (부정적)

```
She is too tall.
(그녀는 너무 크다 - 문제가 있음)

It's too expensive.
(너무 비싸다 - 살 수 없음)
```

## -ly 없는 부사

### fast, hard, late, early

**형용사와 형태 동일:**

```
형용사:
a fast car (빠른 차)
a hard question (어려운 질문)

부사:
He runs fast. (빨리 달린다)
She works hard. (열심히 일한다)
```

## 감각동사 + 형용사

### look, sound, smell, taste, feel

**감각동사 + 형용사:**

```
○ She looks beautiful. (아름다워 보인다)
× She looks beautifully.

○ It sounds good. (좋게 들린다)
○ It smells nice. (좋은 냄새가 난다)
○ It tastes sweet. (달콤하다)
○ It feels soft. (부드럽다)
```

## 자주 묻는 질문

**Q1. 형용사와 부사를 어떻게 구별하나요?**

A. 역할로 구별합니다.
형용사: 명사 수식 (beautiful girl)
부사: 동사/형용사/부사 수식 (run fast, very tall)
대부분 부사는 -ly가 붙습니다.

**Q2. -ly를 언제 붙이나요?**

A. 형용사를 부사로 만들 때 붙입니다.
slow → slowly, quick → quickly
하지만 fast, hard, late는 그대로 씁니다.
good만 well로 불규칙 변화합니다.

**Q3. good과 well의 차이가 뭔가요?**

A. good은 형용사, well은 부사입니다.
"He is a good student."(좋은 - 형용사)
"He speaks well."(잘 - 부사)
단, "She is well."은 "건강하다"는 뜻입니다.

**Q4. hard와 hardly의 차이는요?**

A. 의미가 완전히 다릅니다.
hard: 열심히, 어렵게
"He works hard."(열심히 일한다)
hardly: 거의 ~않다
"He hardly works."(거의 일하지 않는다)

**Q5. 감각동사 뒤에는 왜 형용사를 쓰나요?**

A. 감각동사(look, sound, smell, taste, feel)는 주어의 상태를 설명하기 때문입니다.
"She looks beautiful."(그녀가 아름답다)
"It sounds good."(그것이 좋다)
동사가 아닌 주어를 수식하므로 형용사를 씁니다.

**Q6. 빈도부사는 어디에 두나요?**

A. be동사 뒤, 일반동사 앞, 조동사 뒤입니다.
"I am always tired."(be동사 뒤)
"I always study."(일반동사 앞)
"I can always help."(조동사 뒤)

**Q7. enough는 어디에 위치하나요?**

A. 형용사/부사 뒤, 명사 앞입니다.
"old enough"(형용사 뒤)
"fast enough"(부사 뒤)
"enough money"(명사 앞)

**Q8. late와 lately의 차이가 뭔가요?**

A. 의미가 다릅니다.
late: 늦게
"He came late."(늦게 왔다)
lately: 최근에 (= recently)
"I haven't seen him lately."(최근에 못 봤다)

**Q9. too와 very의 차이는요?**

A. 뉘앙스가 다릅니다.
very: 매우 (중립적)
"very tall"(매우 크다 - 사실)
too: 너무 (부정적)
"too tall"(너무 크다 - 문제)

**Q10. 1:1 과외가 도움이 되나요?**

A. 네, 매우 효과적입니다.
형용사·부사는 위치와 형태가 헷갈려서
혼자 공부하면 실수가 많습니다.
1:1 과외에서는 감각동사, 빈도부사 같은
까다로운 부분을 실전 예문으로 연습하므로
빠르게 정복할 수 있습니다.
