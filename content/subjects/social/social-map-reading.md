## 2. 지도의 기본 요소

### 2.1 제목

**역할**
- 지도 주제
- 내용 파악
- 가장 먼저 확인

**확인 사항**
- 무엇에 관한 지도?
- 어디의 지도?
- 언제의 지도?

### 2.2 방위

**방향 표시**
- 북쪽 화살표 (N)
- 없으면 위쪽이 북
- 동서남북 파악

**중요성**
- 위치 관계
- 지역 비교
- 이동 방향

### 2.3 축척

**정의**
- 실제 거리 : 지도 거리
- 비율 표시

**종류**
- 수치 축척: 1:50,000
- 문자 축척: "1cm = 500m"
- 선 축척: 눈금자 형태

**대축척 vs 소축척**
- 대축척: 비율 크다 (1:5,000), 좁은 지역, 자세히
- 소축척: 비율 작다 (1:1,000,000), 넓은 지역, 간략히

**축척 계산**
- 실제 거리 = 지도 거리 × 축척의 분모
- 지도 거리 = 실제 거리 ÷ 축척의 분모

### 2.4 범례

**정의**
- 기호 설명
- 색깔 의미
- 패턴 해석

**역할**
- 지도 읽기 필수
- 정확한 이해
- 오해 방지

**확인**
- 모르는 기호
- 색깔별 의미
- 패턴 구분

## 3. 지형도

### 3.1 지형도란?

**정의**
- 지표면 지형 표현
- 등고선 사용
- 높이 정보

**특징**
- 매우 자세함
- 대축척
- 실생활 활용

### 3.2 등고선

**정의**
- 같은 높이를 연결한 선
- 해발 고도
- 일정 간격 (10m, 20m 등)

**주곡선과 간곡선**
- 주곡선: 굵은 선, 큰 간격
- 간곡선: 가는 선, 작은 간격
- 계곡선: 더 굵은 선 (100m 등)

**등고선의 특징**
- 하나의 선으로 이어짐
- 교차하지 않음
- 간격 좁음 = 경사 급
- 간격 넓음 = 경사 완만

### 3.3 지형 판독

**산**
- 등고선 동심원
- 중심이 높음
- 정상 표시

**계곡**
- 등고선이 산 쪽으로 휨
- 하천 흐름
- V자 모양

**능선**
- 등고선이 낮은 쪽으로 볼록
- 산의 줄기
- 역V자 모양

**분지**
- 사방이 산으로 둘러싸임
- 중간이 낮음

**고개 (안부)**
- 두 산 사이
- 상대적으로 낮음
- 교통로

**절벽**
- 등고선 매우 밀집
- 거의 겹침
- 경사 매우 급

### 3.4 지형도 기호

**자연 지형**
- 하천: 파란 선
- 호수: 파란 면
- 산림: 녹색

**인문 지형**
- 도로: 색깔별 구분 (고속도로, 국도 등)
- 철도: 점선 또는 실선
- 건물: 검은 사각형
- 논: 초록 선
- 밭: 갈색 선

**주요 시설**
- 학교: ⊕
- 사찰: ⊚
- 교회: +
- 우체국: 〒
- 관공서: ⊗

## FAQ

### Q1. 지도를 어디서부터 봐야 하나요?

**A1.** 제목, 방위, 축척, 범례 순으로 보세요.
1. 제목: 무엇에 관한 지도?
2. 방위: 방향 파악
3. 축척: 넓이 추정
4. 범례: 기호 이해

### Q2. 등고선이 헷갈려요.

**A2.** 간격에 주목하세요.
- 간격 좁음 = 경사 급 (산, 절벽)
- 간격 넓음 = 경사 완만 (평야)
- 산 쪽으로 휨 = 계곡
- 낮은 쪽으로 볼록 = 능선

### Q3. 축척 계산이 어려워요.

**A3.** 단위를 맞추세요.
- 지도 cm → 실제 cm → km 변환
- 예: 1:50,000에서 2cm = 2×50,000cm = 1,000m = 1km

### Q4. 통계 지도는 어떻게 읽나요?

**A4.** 범례가 핵심입니다.
- 색깔별 의미 확인
- 기호별 의미 확인
- 분포, 집중 지역 파악
- 원인 추론

### Q5. 시험에서 시간이 부족해요.

**A5.** 문제부터 읽고 필요한 부분만 보세요.
- 문제 파악
- 관련 정보만 찾기
- 빠르게 제거법
- 확실한 답부터

### Q6. 지도가 복잡해요.

**A6.** 단순화해서 보세요.
- 핵심만 파악
- 큰 틀 먼저
- 세부는 필요시
- 메모 활용

### Q7. 위치를 어떻게 설명하나요?

**A7.** 상대적 위치가 쉽습니다.
- "서울의 남쪽"
- "해안에 인접"
- "~와 ~ 사이"
- 필요시 경위도

### Q8. 실생활에서 활용하려면?

**A8.** 디지털 지도를 많이 사용하세요.
- 길 찾기
- 여행 계획
- 주변 탐색
- 습관화

### Q9. 과외가 필요한가요?

**A9.** 체계적 학습에 큰 도움이 됩니다.
- 읽기 방법 지도
- 다양한 유형 연습
- 공간 감각 훈련
- 즉각 피드백

### Q10. 지도 실력 향상에 도움 되나요?

**A10.** 지리 학습의 핵심입니다.
- 모든 지리 문제에 지도
- 공간 이해력
- 자료 해석 능력
- 실생활 유용
