# 공간도형과 좌표 완벽 가이드

공간을 수학적으로 이해하고 싶으신가요?

"3차원이 상상이 안 돼요."
"공간벡터가 어려워요."
"정사영이 뭔지 모르겠어요."

**공간 감각**을 키우면 공간도형도 쉽습니다.

## 공간도형 기초

### 2차원 vs 3차원

**차원**

```
평면 (2D):
x, y 좌표

공간 (3D):
x, y, z 좌표

차이:
깊이 추가

→ 확장
```

### 공간좌표

**3성분**

```
점 P(x, y, z):
x: 가로
y: 세로
z: 높이

원점:
O(0, 0, 0)

→ 위치 표현
```

### 거리

**3차원**

```
두 점 A(x₁,y₁,z₁), B(x₂,y₂,z₂):

거리:
√{(x₂-x₁)² + (y₂-y₁)² + (z₂-z₁)²}

→ 피타고라스
```

## 공간도형

### 다면체

**면으로 둘러싸임**

```
정다면체:
정사면체 (4면)
정육면체 (6면)
정팔면체 (8면)
정십이면체 (12면)
정이십면체 (20면)

→ 5가지만
```

### 회전체

**회전**

```
원기둥:
직사각형 회전

원뿔:
직각삼각형 회전

구:
반원 회전

→ 대칭
```

### 부피

**공식**

```
정육면체: a³
구: (4/3)πr³
원기둥: πr²h
원뿔: (1/3)πr²h

→ 암기
```

### 겉넓이

**표면적**

```
정육면체: 6a²
구: 4πr²
원기둥: 2πr(r+h)
원뿔: πr(r+l)

→ 전개도
```

## 공간벡터

### 정의

**3성분**

```
벡터 a⃗ = (a₁, a₂, a₃)

크기:
|a⃗| = √(a₁² + a₂² + a₃²)

→ 방향량
```

### 연산

**덧셈·뺄셈·스칼라배**

```
a⃗ + b⃗ = (a₁+b₁, a₂+b₂, a₃+b₃)
a⃗ - b⃗ = (a₁-b₁, a₂-b₂, a₃-b₃)
ka⃗ = (ka₁, ka₂, ka₃)

→ 성분별
```

### 내적

**스칼라 곱**

```
a⃗·b⃗ = a₁b₁ + a₂b₂ + a₃b₃
     = |a⃗||b⃗|cosθ

수직:
a⃗·b⃗ = 0

각도:
cosθ = (a⃗·b⃗)/(|a⃗||b⃗|)

→ 각도 관련
```

### 외적

**벡터 곱**

```
a⃗ × b⃗:
a⃗, b⃗에 모두 수직

크기:
|a⃗ × b⃗| = |a⃗||b⃗|sinθ

활용:
법선벡터

→ 3차원 특유
```

## 직선과 평면

### 직선의 방정식

**매개변수**

```
점 A, 방향벡터 d⃗:
OP⃗ = OA⃗ + td⃗

성분:
x = x₀ + at
y = y₀ + bt
z = z₀ + ct

→ 매개변수 표현
```

### 평면의 방정식

**법선벡터**

```
점 A(a,b,c), 법선 n⃗=(A,B,C):

방정식:
A(x-a) + B(y-b) + C(z-c) = 0

일반형:
Ax + By + Cz + D = 0

→ 수직 방향
```

### 위치 관계

**직선과 평면**

```
평행:
d⃗·n⃗ = 0

수직:
d⃗ = kn⃗

교점:
대입해서 풀기

→ 내적 활용
```

## 거리

### 점과 평면

**수직 거리**

```
점 P(x₀,y₀,z₀)
평면 Ax+By+Cz+D=0

거리:
|Ax₀+By₀+Cz₀+D|/√(A²+B²+C²)

→ 공식
```

### 두 평면 사이

**평행할 때**

```
평면 사이 거리:
한 평면 위의 점에서
다른 평면까지 거리

→ 점-평면 활용
```

## 정사영

### 개념

**그림자**

```
정의:
점을 평면에 수직으로 내림

활용:
거리 계산
넓이 계산

→ 투영
```

### 길이

**삼각함수**

```
선분 AB를 평면에 정사영:

길이:
|AB|cosθ

θ: AB와 평면이 이루는 각

→ cos 곱
```

### 넓이

**평면도형**

```
도형 넓이 S를 정사영:

넓이:
Scosθ

θ: 두 평면이 이루는 각

→ cos 곱
```

## 구

### 방정식

**표준형**

```
중심 (a,b,c), 반지름 r:
(x-a)² + (y-b)² + (z-c)² = r²

일반형:
x² + y² + z² + Ax + By + Cz + D = 0

→ 평방완성
```

### 구와 평면

**위치 관계**

```
d: 중심에서 평면까지 거리

d < r: 교선(원)
d = r: 접함
d > r: 만나지 않음

→ 거리 비교
```

## 자주 묻는 질문

**Q1. 공간을 어떻게 상상하나요?**

A. 처음에는 그림을 많이 그려보세요.
실제 모형을 만들어보는 것도 좋습니다.

**Q2. 공간벡터가 어려워요.**

A. 평면벡터와 원리는 같습니다.
성분이 하나 더 늘어난 것뿐입니다.

**Q3. 외적은 왜 배우나요?**

A. 두 벡터에 수직인 벡터를 구하기 위해서입니다.
평면의 법선벡터를 찾을 때 유용합니다.

**Q4. 정사영이 뭔가요?**

A. 그림자를 생각하면 됩니다.
점이나 도형을 평면에 수직으로 내린 것입니다.

**Q5. 거리 공식을 외워야 하나요?**

A. 점-평면 거리는 외우는 게 좋습니다.
자주 사용됩니다.

**Q6. 평면의 방정식은 어떻게 구하나요?**

A. 법선벡터를 찾는 게 핵심입니다.
외적을 활용하면 쉽습니다.

**Q7. 공간도형은 왜 어려운가요?**

A. 3차원을 2차원 종이에 그려야 하기 때문입니다.
공간 감각을 키우는 연습이 필요합니다.

**Q8. 좌표를 어떻게 설정하나요?**

A. 대칭적이고 계산하기 쉽게 설정하세요.
원점을 적절히 선택하는 게 중요합니다.

**Q9. 공간도형 문제 풀이 순서는?**

A. 그림 그리기 → 좌표 설정 → 벡터 이용 → 계산
순서가 중요합니다.

**Q10. 1:1 과외가 도움이 되나요?**

A. 네, 매우 효과적입니다.
공간도형은 시각화가 중요한데
개인 지도로 그림 그리는 법과
공간 감각을 효과적으로 배울 수 있습니다.

복잡한 공간 문제도 풀 수 있습니다.
