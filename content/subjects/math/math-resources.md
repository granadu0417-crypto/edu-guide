# 수학 학습 자료 총정리 완벽 가이드

수학 학습에는 좋은 자료가 필수입니다. 교재부터 온라인 강의, 앱까지 모든 자료를 총정리하고 효과적인 활용법을 알려드립니다.

## 필수 교재

### 초등 수학

**개념서:**
- **디딤돌 초등수학**: 기본 개념
- **최상위 수학**: 심화
- **큐브 수학**: 사고력

**문제집:**
- **기본**: 백점 수학, 우등생 해법
- **심화**: 최고수준, 상위권 수학

**추천 조합:**
- 기본: 디딤돌 + 백점
- 심화: 최상위 + 최고수준

### 중등 수학

**개념서:**
- **개념원리**: 가장 기본
- **숨마쿰라우데**: 자세한 설명
- **완자**: 교과서 충실

**문제집:**
- **기본**: 오투, 일품
- **심화**: 블랙라벨, 최고득점

**내신:**
- 학교 기출문제집
- 자이스토리 (유형)

**추천 조합:**
- 내신: 개념원리 + 학교 기출
- 실력: 숨마쿰라우데 + 블랙라벨

### 고등 수학

**개념서:**
- **수학의정석**: 절대 강자
- **개념원리**: 쉬운 설명
- **수학의바이블**: 중간

**문제집:**
- **기본**: 쎈, RPM
- **심화**: 블랙라벨, N제
- **수능**: 마더텅, 자이스토리

**실전:**
- **모의고사**: 각 출판사 N제
- **기출**: 평가원 기출 (필수)

**추천 조합:**
- 개념: 수학의정석
- 유형: 쎈
- 심화: N제
- 기출: 마더텅

## 효과적인 활용법

### 교재 선택

**원칙:**
- 한 번에 1-2권
- 끝까지 완주
- 수준에 맞게

**순서:**
1. 개념서 1회독
2. 기본 문제집
3. 심화 (필요시)

**회독:**
- 1회독: 전체 파악
- 2회독: 틀린 것
- 3회독: 완벽히

### 인강 활용

**시청:**
- 1.3-1.5배속
- 메모하며
- 예제 따라 풀기

- 당일 노트 정리
- 3일 후 재시청
- 문제 풀이

**주의:**
- 강의만 듣지 말기
- 반드시 문제 풀기
- 손으로 쓰기

- 복습용

**주의:**

## FAQ

### Q1. 교재 몇 권이 적당한가요?

**동시에 1-2권!**

**이유:**
- 집중력
- 완주 중요
- 깊이 > 넓이

**순서:**
개념 → 기본 → 심화

### Q2. 비싼 교재가 더 좋나요?

**아닙니다!**

**중요한 것:**
- 본인 수준
- 설명 방식
- 완주 가능성

**가성비:** 개념원리, 쎈

### Q3. 인강 vs 교재, 뭐가 나을까요?

**둘 다 필요!**

**비율:**
- 인강: 30%
- 교재: 70%

**역할:**
- 인강: 개념 이해
- 교재: 문제 풀이

### Q4. 무료 vs 유료, 차이가 크나요?

**있지만 절대적은 아닙니다.**

**무료로 충분:**
- 자기주도 가능
- 기본~중급 목표

**유료 필요:**
- 동기 부여 필요
- 고급 목표
- 빠른 이해

### Q5. 앱으로만 공부 가능한가요?

**불가능합니다!**

**앱의 역할:**
- 보조 도구 (20%)
- 복습 (30%)
- 흥미 (50%)

**메인:** 교재 + 인강

### Q6. 교재 중고로 사도 되나요?

**괜찮습니다!**

**주의:**
- 답지 확인
- 최신판 (3년 이내)
- 필기 많으면 패스

**추천:** 개념서는 새것

### Q7. 학년 올라가면 교재 바꿔야 하나요?

**네!**

**이유:**
- 교육과정 변경
- 난이도 상승
- 새로운 유형

**예외:** 수학의정석 (계속 사용)

### Q8. 외국 교재도 필요한가요?

**선택입니다.**

**필요한 경우:**
- 해외 대학 목표
- 올림피아드 준비
- 영어 수학 학습

**추천:**
- Art of Problem Solving
- Singapore Math

### Q9. 독학 vs 과외, 자료 차이는?

**독학:**
- 시중 교재
- 온라인 자료
- 스스로 선택

**과외:**
- 선생님 추천 교재
- 자체 제작 자료
- 맞춤 자료

**과외 장점:** 효율적, 맞춤

### Q10. 1:1 과외 시 어떤 교재를 사용하나요?

**과외 교재 선택:**

**개념:**
- 수학의정석 (고등)
- 개념원리 (중등)
- 학생 수준 맞춤

**문제:**
- 기본: 쎈, RPM
- 심화: 블랙라벨, N제
- 기출: 평가원, 교육청

**특별 자료:**
- 선생님 자체 제작
- 유형별 문제 모음
- 오답 분석 자료
- 맞춤 프린트

**장점:**
- 불필요한 부분 스킵
- 약점만 집중
- 효율 3배
- 시간 절약

**추천:**
- 기본 교재 1-2권
- 선생님 자료 병행
- 학교 교재 활용

**1:1 과외의 강점:**
교재를 **선택**하는 것이 아니라, 학생에게 **맞는** 교재와 자료를 **조합**하여 최적의 학습 효과를 냅니다!

수학 자료는 많지만, 중요한 것은 **선택**과 **활용**입니다. 전문 1:1 과외로 자신에게 맞는 최적의 학습 자료와 방법을 찾으시기 바랍니다!
