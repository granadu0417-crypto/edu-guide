## 수학 문제집 선택 완벽 가이드

어떤 문제집을 골라야 할까요?

"문제집이 너무 많아요."
"어떤 걸 사야 하나요?"
"제 수준에 맞는 게 뭐죠?"

**올바른 문제집 선택**이 실력 향상의 지름길입니다.

## 문제집의 종류

### 유형별

**목적**

```
개념서:
이론 중심
예제 상세

유형서:
문제 유형별 정리
패턴 학습

실전서:
기출 문제
실전 대비

심화서:
고난도 문제
최상위용

→ 목적에 맞게
```

## 수준별 선택

### 기초 (60점 이하)

**개념 완성**

```
필요:
- 쉬운 개념서
- 기본 문제 많음
- 상세한 해설

목표:
개념 이해
기본 문제 풀이

→ 탄탄한 기초
```

### 중급 (60-80점)

**유형 학습**

```
필요:
- 유형별 문제집
- 다양한 문제
- 단계별 난이도

목표:
유형 익히기
속도 향상

→ 패턴 암기
```

### 고급 (80-100점)

**심화**

```
필요:
- 고난도 문제집
- 기출 문제
- 킬러 대비

목표:
만점
실수 zero

→ 완성
```

## 시기별 선택

### 학기 초

**예습**

```
선택:
개념 중심
기본 문제

목적:
새 단원 학습

→ 이해 중심
```

### 학기 중

**정리**

```
선택:
유형별 문제집
학교 문제집

목적:
내신 대비

→ 문제 중심
```

### 시험 직전

**실전**

```
선택:
기출 문제
실전 모의고사

목적:
실전 감각

→ 연습
```

### 방학

**집중**

```
선택:
두꺼운 문제집
심화 문제집

목적:
실력 향상

→ 체계적
```

## 목적별 선택

### 내신

**학교 맞춤**

```
선택:
- 유형별
- 내신 대비
- 학교 프린트

주의:
학교 난이도

→ 학교 기준
```

### 수능

**표준화**

```
선택:
- 기출 문제집
- 실전 모의고사
- EBS 연계

필수:
평가원 기출

→ 기출이 왕
```

### 선행

**예습**

```
선택:
- 쉬운 개념서
- 기본 문제

주의:
속도보다 이해

→ 개념 위주
```

## 문제집 선택 기준

### 해설

**중요**

```
확인:
- 상세한가?
- 이해 가능한가?
- 다른 풀이도?

이유:
혼자 공부 시 필수

→ 해설이 핵심
```

### 문제 수

**적당히**

```
너무 많음:
부담
완성 어려움

너무 적음:
연습 부족

적정:
300-500문제

→ 균형
```

### 난이도

**단계적**

```
구성:
기본 → 유형 → 심화

비율:
4:4:2

→ 점진적
```

## 추천 전략

### 기본 세트

**필수**

```
1권: 개념서 (한 권)
2권: 유형서 (한 권)
3권: 실전서 (수능/내신)

총 3권:
충분

→ 적당히
```

### 많이 vs 집중

**전략**

```
× 많이 사기
○ 한 권 완벽히

원칙:
한 권을 3회 이상

→ 완성도
```

## 활용법

### 1회독

**전체 파악**

```
방법:
- 모든 문제
- 표시하며
- 빠르게

목표:
전체 유형 파악

→ 큰 그림
```

### 2회독

**약점 집중**

```
방법:
- 틀린 문제
- 어려운 문제
- 천천히

목표:
완전 이해

→ 보완
```

### 3회독

**완성**

```
방법:
- 다시 틀린 문제
- 시간 재며

목표:
자동화

→ 숙달
```

기호:
○ 맞음

× 틀림
×× 다시 틀림

× 이상만

→ 효율
```

```

## 자주 묻는 질문

**Q1. 문제집을 몇 권 사야 하나요?**

A. 개념서 1권, 유형서 1권, 실전서 1권이면 충분합니다.
많이 사는 것보다 한 권을 완벽히 하는 게 중요합니다.

**Q2. 어떤 문제집이 좋나요?**

A. 자신의 수준에 맞는 것이 가장 좋습니다.
서점에서 직접 보고 해설이 자세한 것을 고르세요.

**Q3. 문제집을 다 풀어야 하나요?**

A. 네, 가능하면 다 푸는 게 좋습니다.
하지만 너무 쉽거나 중복되는 것은 건너뛰어도 됩니다.

**Q4. 해설이 이해 안 되면요?**

A. 그 문제집은 자신에게 어려운 것입니다.
쉬운 문제집으로 바꾸거나, 선생님께 질문하세요.

**Q5. 한 권을 몇 번 풀어야 하나요?**

A. 최소 3번을 권장합니다.
1회독: 전체, 2회독: 틀린 것, 3회독: 다시 틀린 것

**Q6. 문제집을 바꿔도 되나요?**

A. 진도가 절반 이상 나갔다면 완성하는 게 좋습니다.
처음이라면 자신에게 맞는 것으로 바꾸세요.

**Q7. 기출 문제집은 필수인가요?**

A. 수능 준비라면 필수입니다.
최근 5개년 기출은 반드시 풀어보세요.

**Q8. 오답 노트를 꼭 만들어야 하나요?**

A. 문제집에 직접 표시하고 다시 푸는 것도 좋습니다.
따로 만들면 더 체계적이지만 필수는 아닙니다.

**Q9. 온라인 문제도 괜찮나요?**

A. 보조 자료로는 좋습니다.
하지만 주 교재는 종이 문제집을 권장합니다.

**Q10. 1:1 과외에서 문제집을 추천해주나요?**

A. 네, 매우 효과적입니다.
개인 수준을 정확히 파악하여
맞춤 문제집을 추천하고
효과적인 활용법도 지도합니다.

## 마무리하며

문제집은 많이 푸는 것보다
한 권을 완벽히 하는 것이 중요합니다.
자신의 수준에 맞는 문제집을 선택하여
체계적으로 학습하세요.
