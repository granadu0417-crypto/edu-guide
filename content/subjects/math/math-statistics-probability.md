## 2. 경우의 수

### 2.1 합의 법칙

**정의**

두 사건 A, B가 동시에 일어나지 않을 때:
- A가 일어나는 경우의 수: m
- B가 일어나는 경우의 수: n
- A 또는 B가 일어나는 경우의 수: m + n

**예시 1**
- 주사위를 던져 홀수 또는 4의 배수가 나오는 경우?
- 홀수: 1, 3, 5 → 3가지
- 4의 배수: 4 → 1가지
- 합: 3 + 1 = 4가지

**예시 2**
- A반 5명, B반 6명 중 1명을 뽑는 경우?
- A반: 5가지
- B반: 6가지
- 합: 5 + 6 = 11가지

### 2.2 곱의 법칙

**정의**

사건 A, B가 차례로 일어날 때:
- A가 일어나는 경우의 수: m
- 각 A에 대해 B가 일어나는 경우의 수: n
- A와 B가 모두 일어나는 경우의 수: m × n

**예시 1**
- 주사위 2개를 던지는 경우?
- 첫 번째: 6가지
- 두 번째: 6가지
- 곱: 6 × 6 = 36가지

**예시 2**
- 서울에서 부산 가는 길: 3가지
- 부산에서 제주 가는 길: 2가지
- 서울→부산→제주: 3 × 2 = 6가지

### 2.3 중복 허용과 불허

**중복 허용**

같은 것을 여러 번 선택 가능
- n개 중 r개 뽑기 (중복 허용): nʳ

**예시**
- 주사위 3번: 6³ = 216가지

**중복 불허**

한 번 선택한 것은 다시 선택 불가
- n개 중 r개 뽑기 (중복 불허): P(n,r)

**예시**
- 10명 중 반장, 부반장: P(10,2) = 10×9 = 90가지

## 3. 순열 (Permutation)

### 3.1 순열의 정의

**서로 다른 n개 중 r개를 뽑아 순서대로 나열**

P(n, r) = n! / (n-r)!
        = n × (n-1) × ... × (n-r+1)

**계승 (Factorial)**
n! = n × (n-1) × ... × 2 × 1
0! = 1 (정의)

**예시**
- 5! = 5 × 4 × 3 × 2 × 1 = 120
- 3! = 3 × 2 × 1 = 6

### 3.2 순열 계산

**P(n, r) 계산**

**예시 1**
- P(5, 2) = 5!/(5-2)! = 5!/3! = 5×4 = 20

**예시 2**
- 10명 중 3명을 뽑아 1등, 2등, 3등 정하기?
- P(10, 3) = 10×9×8 = 720가지

**예시 3**
- 5명을 일렬로 세우기?
- P(5, 5) = 5! = 120가지

### 3.3 같은 것이 있는 순열

**n개 중 같은 것이 a개, b개, ... 있을 때**

순열 수 = n! / (a! × b! × ...)

**예시 1**
- MISSISSIPPI의 순열?
- 총 11개: M(1), I(4), S(4), P(2)
- 11! / (1! × 4! × 4! × 2!)
- = 34,650

**예시 2**
- AABBC의 순열?
- 총 5개: A(2), B(2), C(1)
- 5! / (2! × 2! × 1!)
- = 120 / 4 = 30

### 3.4 원순열

**n개를 원형으로 배열**

원순열 = (n-1)!

**이유**: 회전해도 같은 것으로 봄

**예시 1**
- 5명이 원탁에 앉기?
- (5-1)! = 4! = 24가지

**예시 2**
- 6개의 구슬로 목걸이?
- 목걸이는 뒤집어도 같음
- (6-1)! / 2 = 5! / 2 = 60가지

## 4. 조합 (Combination)

### 4.1 조합의 정의

**서로 다른 n개 중 r개를 뽑기 (순서 무관)**

C(n, r) = n! / [r! × (n-r)!]
        = P(n, r) / r!

**표기**
- C(n, r) = ₙCᵣ = (n choose r)

**예시**
- C(5, 2) = 5! / (2! × 3!)
- = (5 × 4) / (2 × 1)
- = 10

### 4.2 조합의 성질

**대칭성**

C(n, r) = C(n, n-r)

**예시**
- C(10, 3) = C(10, 7) = 120

**파스칼의 삼각형**

C(n, r) = C(n-1, r-1) + C(n-1, r)

```
        1
      1   1
    1   2   1
  1   3   3   1
1   4   6   4   1
```

### 4.3 조합 계산

**예시 1**
- 10명 중 3명을 뽑기?
- C(10, 3) = 10! / (3! × 7!)
- = (10 × 9 × 8) / (3 × 2 × 1)
- = 120

**예시 2**
- 남학생 5명, 여학생 4명 중 남 2명, 여 2명 뽑기?
- C(5, 2) × C(4, 2)
- = 10 × 6 = 60가지

**예시 3**
- 10개 중 적어도 1개 뽑기?
- 전체 - 0개
- 2¹⁰ - 1 = 1024 - 1 = 1023
- 또는 C(10,1) + C(10,2) + ... + C(10,10)

### 4.4 중복조합

**중복을 허용하여 뽑기**

H(n, r) = C(n+r-1, r)

**예시 1**
- 사과, 배, 감 중 5개 뽑기 (중복 허용)?
- H(3, 5) = C(3+5-1, 5)
- = C(7, 5) = C(7, 2) = 21가지

**예시 2**
- x + y + z = 10의 음이 아닌 정수해?
- H(3, 10) = C(12, 10) = C(12, 2) = 66가지

## 5. 확률 (Probability)

### 5.1 확률의 정의

**수학적 확률**

P(A) = (A가 일어나는 경우의 수) / (전체 경우의 수)

**성질**
- 0 ≤ P(A) ≤ 1
- P(전체) = 1
- P(공집합) = 0

**예시**
- 주사위를 던져 홀수가 나올 확률?
- 홀수: 1, 3, 5 → 3가지
- 전체: 6가지
- P = 3/6 = 1/2

### 5.2 확률의 덧셈정리

**두 사건 A, B**

P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

**배반사건** (동시에 일어나지 않음)

P(A ∩ B) = 0이면
P(A ∪ B) = P(A) + P(B)

**예시**
- 주사위: 짝수 또는 3의 배수?
- 짝수: 2, 4, 6 → P = 3/6
- 3의 배수: 3, 6 → P = 2/6
- 교집합: 6 → P = 1/6
- P(짝수 ∪ 3의 배수) = 3/6 + 2/6 - 1/6 = 4/6 = 2/3

### 5.3 확률의 곱셈정리

**독립사건**

두 사건이 서로 영향을 주지 않으면:

P(A ∩ B) = P(A) × P(B)

**예시**
- 동전 2번: 모두 앞면?
- P(앞) = 1/2
- P(앞 ∩ 앞) = 1/2 × 1/2 = 1/4

**종속사건 (조건부 확률)**

A가 일어난 조건에서 B가 일어날 확률:

P(B|A) = P(A ∩ B) / P(A)

P(A ∩ B) = P(A) × P(B|A)

**예시**
- 52장 카드에서 2장: 모두 하트?
- P(1번째 하트) = 13/52 = 1/4
- P(2번째 하트 | 1번째 하트) = 12/51
- P(모두 하트) = (13/52) × (12/51) = 1/17

### 5.4 여사건

**A의 여사건 Aᶜ**

P(Aᶜ) = 1 - P(A)

**활용**: "적어도 1개"는 여사건 이용

**예시**
- 주사위 3번: 적어도 1번 6이 나올 확률?
- 여사건: 한 번도 6이 안 나옴
- P(6 안 나옴) = (5/6)³ = 125/216
- P(적어도 1번) = 1 - 125/216 = 91/216

### 5.5 확률의 곱셈정리 활용

**예시 1: 불량품**
- 100개 중 불량품 5개
- 2개 뽑아 모두 정상일 확률?
- P = C(95, 2) / C(100, 2)
- = (95×94)/(100×99) × (1/2)/(1/2)
- = (95×94)/(100×99)
- ≈ 0.903

**예시 2: 복권**
- 당첨 확률 1/100인 복권 3장?
- 적어도 1장 당첨 확률?
- P(모두 낙첨) = (99/100)³
- P(적어도 1장) = 1 - (99/100)³ ≈ 0.0297

## 6. 통계 (Statistics)

### 6.1 평균 (Mean)

**산술평균**

x̄ = (x₁ + x₂ + ... + xₙ) / n
  = Σxᵢ / n

**예시**
- 점수: 80, 70, 90, 60, 85
- 평균 = (80+70+90+60+85) / 5 = 77

**가중평균**

x̄ = (w₁x₁ + w₂x₂ + ... + wₙxₙ) / (w₁ + w₂ + ... + wₙ)

**예시**
- 중간 40%, 기말 60%
- 중간 80점, 기말 70점
- 평균 = (40×80 + 60×70) / 100 = 74점

### 6.2 분산과 표준편차

**분산 (Variance)**

σ² = Σ(xᵢ - x̄)² / n

**표준편차 (Standard Deviation)**

σ = √(분산)

**의미**
- 데이터가 평균에서 얼마나 퍼져 있는지
- 작을수록 집중, 클수록 분산

**예시**
- 데이터: 2, 4, 6
- 평균: 4
- 분산: [(2-4)² + (4-4)² + (6-4)²] / 3
- = (4 + 0 + 4) / 3 = 8/3
- 표준편차: √(8/3) ≈ 1.63

### 6.3 분산 계산 공식

**편차 제곱의 평균**

σ² = Σ(xᵢ - x̄)² / n
   = (Σxᵢ² / n) - x̄²
   = E[X²] - (E[X])²

**활용**: 두 번째 공식이 계산 편함

**예시**
- 데이터: 2, 4, 6
- Σxᵢ² = 4 + 16 + 36 = 56
- Σxᵢ²/n = 56/3
- x̄² = 16
- 분산 = 56/3 - 16 = 8/3

### 6.4 확률변수

**이산 확률변수**

정수 값만 가지는 변수

**확률질량함수**
- P(X = xᵢ) = pᵢ
- Σpᵢ = 1

**기댓값 (평균)**
E[X] = Σxᵢpᵢ

**분산**
V[X] = E[(X-E[X])²]
     = E[X²] - (E[X])²

**예시**
- 주사위 한 번
- E[X] = 1×(1/6) + 2×(1/6) + ... + 6×(1/6)
- = (1+2+3+4+5+6) / 6 = 3.5

### 6.5 이항분포

**정의**

성공 확률 p인 시행을 n번 반복할 때, 성공 횟수 X:

P(X = k) = C(n, k) × pᵏ × (1-p)ⁿ⁻ᵏ

표기: X ~ B(n, p)

**기댓값과 분산**
- E[X] = np
- V[X] = np(1-p)

**예시**
- 주사위 10번, 6이 나오는 횟수 X?
- X ~ B(10, 1/6)
- P(X = 2) = C(10, 2) × (1/6)² × (5/6)⁸
- E[X] = 10 × (1/6) = 5/3 ≈ 1.67
- V[X] = 10 × (1/6) × (5/6) = 25/18

### 6.6 정규분포

**표준정규분포**

N(0, 1): 평균 0, 표준편차 1

**일반 정규분포**

X ~ N(μ, σ²)
- 평균 μ, 분산 σ²

**표준화**

Z = (X - μ) / σ
Z ~ N(0, 1)

**68-95-99.7 규칙**
- μ ± σ: 68%
- μ ± 2σ: 95%
- μ ± 3σ: 99.7%

## 7. 확률과 통계 응용

### 7.1 기댓값의 성질

**선형성**

E[aX + b] = aE[X] + b

E[X + Y] = E[X] + E[Y] (항상 성립)

**독립이면**

E[XY] = E[X] × E[Y]

### 7.2 분산의 성질

**변환**

V[aX + b] = a²V[X]

**독립이면**

V[X + Y] = V[X] + V[Y]
V[X - Y] = V[X] + V[Y] (빼기도 더하기!)

### 7.3 확률분포표

**이산확률분포 예시**

| X | 1 | 2 | 3 | 4 |
|---|---|---|---|---|
| P(X) | 0.1 | 0.3 | 0.4 | 0.2 |

- E[X] = 1×0.1 + 2×0.3 + 3×0.4 + 4×0.2 = 2.7
- E[X²] = 1²×0.1 + 2²×0.3 + 3²×0.4 + 4²×0.2 = 8.3
- V[X] = 8.3 - 2.7² = 1.01

- 통계: 평균? 분산?

## 10. FAQ

### Q1. 확률과 통계가 왜 중요한가요?

**실생활과 직결됩니다.**

- 날씨 예보
- 보험료 계산
- 주식 투자
- 데이터 분석

**수능에서**
- 선택 과목 (미적분보다 쉬움)
- 실수하기 쉬움
- 꼼꼼함이 중요

### Q2. 순열과 조합의 차이는?

**순서가 중요한가?**

- **순열**: 순서 O (123 ≠ 321)
- **조합**: 순서 X (123 = 321)

**예시**
- 반장, 부반장: 순열 (순서 중요)
- 대표 2명: 조합 (순서 무관)

### Q3. 독립과 종속의 차이는?

**영향을 주는가?**

- **독립**: 영향 X
  - 동전 2번
  - P(A ∩ B) = P(A) × P(B)

- **종속**: 영향 O
  - 카드 2장 (복원 안 함)
  - P(A ∩ B) = P(A) × P(B|A)

### Q4. 조건부 확률은 언제 쓰나요?

**"~일 때" 조건이 있으면**

P(B|A) = P(A ∩ B) / P(A)

**예시**
- A가 일어났다는 조건에서 B가 일어날 확률
- "비가 올 때 우산 가져갈 확률"

### Q5. 여사건은 언제 쓰나요?

**"적어도 ~"는 여사건**

P(적어도 1개) = 1 - P(0개)

**예시**
- 적어도 1번 6: 1 - P(6 안 나옴)
- 계산이 훨씬 간단!

### Q6. 분산은 왜 계산하나요?

**데이터의 퍼진 정도**

- 작으면: 집중 (일정함)
- 크면: 분산 (불규칙)

**예시**
- 시험 점수 분산 작음 = 모두 비슷
- 시험 점수 분산 큼 = 편차 큼

### Q7. 이항분포는 어디에 쓰나요?

**같은 시행을 여러 번**

- 동전 n번
- 주사위 n번
- 성공/실패 시행

**공식**
- P(X = k) = C(n, k) pᵏ (1-p)ⁿ⁻ᵏ
- E[X] = np

### Q8. 정규분포는 왜 중요한가요?

**자연 현상의 대부분**

- 키, 몸무게
- 시험 점수
- 측정 오차

**중심극한정리**
- 표본이 크면 정규분포에 가까워짐

### Q9. 수능 확률과 통계 준비법은?

**3단계 전략**

**1단계**: 개념 완벽 이해
- 순열·조합 구분
- 확률 공식 암기
- 확률분포 이해

**2단계**: 계산 연습
- 빠르고 정확하게
- 실수 줄이기

**3단계**: 기출 반복
- 유형 익히기
- 시간 관리

### Q10. 1:1 과외가 효과적인가요?

**확률과 통계는 1:1이 효과적입니다.**

**이유**:
- 개념 헷갈림 많음
- 실수 교정 필요
- 문제 유형별 접근
- 즉각 피드백

**효과**:
- 빠른 이해
- 실수 감소
- 고득점 달성
