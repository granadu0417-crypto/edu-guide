**"함수가 뭔가요?"** 수학의 핵심, 함수를 제대로 이해하면 수학이 쉬워집니다.

함수는 **변수 간의 관계**를 나타내는 수학의 기본 개념으로, 중학교부터 대학까지 모든 수학의 기초가 됩니다.

## 1. 함수의 기본 개념

### 1.1 함수란?

**정의**
- 두 집합 X, Y에서
- X의 각 원소에 Y의 원소가 **하나씩** 대응
- f: X → Y

**핵심 용어**
- 정의역(Domain): 입력값 집합 X
- 공역(Codomain): 출력값 집합 Y
- 치역(Range): 실제 대응된 값들
- 함숫값: f(x)

### 1.2 함수의 조건

**함수가 되려면**
1. 정의역의 **모든 원소**에 대응
2. 각 원소는 **하나**의 값에만 대응

**함수가 아닌 경우**
- 일부 원소에 대응 없음
- 한 원소가 두 개 이상에 대응
- 일대다 관계

### 1.3 함수의 표현

**다양한 표현법**
- 식: f(x) = 2x + 1
- 그래프: 좌표평면
- 표: 입력-출력 표
- 말: "두 배하고 1을 더한다"

**대응 관계**
- 일대일 대응
- 다대일 대응
- 항등함수
- 상수함수

## 2. 함수의 종류

### 2.1 일차함수

**정의**
- f(x) = ax + b (a ≠ 0)
- 직선 그래프

**성질**
- 기울기: a
- y절편: b
- 증가/감소: a의 부호

**그래프**
- 직선
- 한 점과 기울기로 결정
- 두 점으로 결정

**활용**
- 비례관계
- 속도-시간
- 요금 계산

### 2.2 이차함수

**정의**
- f(x) = ax² + bx + c (a ≠ 0)
- 포물선 그래프

**표준형**
- f(x) = a(x - p)² + q
- 꼭짓점: (p, q)
- 축: x = p

**성질**
- 최댓값/최솟값
- 대칭축
- 판별식 D

**활용**
- 포물선 운동
- 최대·최소 문제
- 넓이·부피

### 2.3 유리함수

**정의**
- f(x) = k/x (k ≠ 0)
- 쌍곡선 그래프

**성질**
- 점근선: x축, y축
- 제1, 3사분면 또는 제2, 4사분면
- 반비례 관계

**활용**
- 속력-시간
- 농도 문제
- 일의 양

### 2.4 무리함수

**정의**
- f(x) = √x
- 제곱근 포함

**성질**
- 정의역: x ≥ 0
- 치역: y ≥ 0
- 증가함수

**활용**
- 거리 계산
- 기하 문제

### 2.5 지수·로그함수

**지수함수**
- f(x) = aˣ (a > 0, a ≠ 1)
- 폭발적 증가/감소

**로그함수**
- f(x) = log_a x
- 지수함수의 역함수

**활용**
- 인구 증가
- 복리 계산
- pH, 지진 규모

### 2.6 삼각함수

**기본 함수**
- f(x) = sin x
- f(x) = cos x
- f(x) = tan x

**성질**
- 주기함수
- 진폭, 주기, 위상

**활용**
- 파동
- 회전 운동
- 진동

## 3. 함수의 그래프

### 3.1 그래프 그리기

**기본 단계**
1. 정의역 확인
2. 특수값 계산 (절편, 꼭짓점)
3. 증가·감소 파악
4. 대칭성 확인
5. 점근선 찾기
6. 점 찍고 연결

**체크포인트**
- y절편: f(0)
- x절편: f(x) = 0
- 꼭짓점
- 변곡점
- 점근선

### 3.2 그래프 변환

**평행이동**
- f(x) → f(x - a): 오른쪽으로 a
- f(x) → f(x) + b: 위로 b

**대칭이동**
- f(x) → -f(x): x축 대칭
- f(x) → f(-x): y축 대칭

**확대·축소**
- f(x) → af(x): y축 방향으로 a배
- f(x) → f(ax): x축 방향으로 1/a배

**복합 변환**
- 순서 중요
- 여러 변환 조합

### 3.3 그래프 해석

**읽어내야 할 정보**
- 증가·감소 구간
- 최댓값·최솟값
- 함수의 성질
- 대칭성
- 주기성

**활용**
- 실생활 그래프
- 통계 그래프
- 과학 데이터

## 4. 함수의 성질

### 4.1 증가와 감소

**증가함수**
- x₁ < x₂ ⇒ f(x₁) < f(x₂)
- 오른쪽 위로 상승

**감소함수**
- x₁ < x₂ ⇒ f(x₁) > f(x₂)
- 오른쪽 아래로 하강

**구간별 증감**
- 증가 구간
- 감소 구간
- 변곡점

### 4.2 최댓값과 최솟값

**정의**
- 최댓값: 가장 큰 함숫값
- 최솟값: 가장 작은 함숫값

**찾는 방법**
- 그래프 이용
- 미분 이용 (고등)
- 정의역 끝점 확인

**활용**
- 최적화 문제
- 경제학 응용

### 4.3 대칭성

**우함수**
- f(-x) = f(x)
- y축 대칭

**기함수**
- f(-x) = -f(x)
- 원점 대칭

**활용**
- 그래프 그리기 간편
- 적분 계산

### 4.4 주기성

**주기함수**
- f(x + p) = f(x)
- 주기: p

**대표 예**
- 삼각함수
- 톱니파
- 계단함수

## 5. 함수의 연산

### 5.1 사칙연산

**덧셈**
- (f + g)(x) = f(x) + g(x)

**뺄셈**
- (f - g)(x) = f(x) - g(x)

**곱셈**
- (fg)(x) = f(x) × g(x)

**나눗셈**
- (f/g)(x) = f(x) / g(x) (g(x) ≠ 0)

### 5.2 합성함수

**정의**
- (g ∘ f)(x) = g(f(x))
- f를 먼저, g를 나중에

**성질**
- 결합법칙 성립
- 교환법칙 불성립
- (g ∘ f) ≠ (f ∘ g)

**활용**
- 복잡한 함수 분해
- 미분·적분

### 5.3 역함수

**정의**
- f(a) = b ⇔ f⁻¹(b) = a
- 정의역과 치역 교환

**존재 조건**
- 일대일 대응
- 정의역의 모든 원소에 대응

**그래프**
- y = x에 대칭
- 원래 함수와 대칭

**활용**
- 방정식 풀이
- 역변환

## FAQ

### Q1. 함수가 왜 중요한가요?

**A1.** 수학의 거의 모든 분야에 등장합니다.
- 중학교부터 대학까지
- 물리, 경제, 생물 등에 응용
- 실생활 문제 해결

### Q2. 함수 그래프를 어떻게 잘 그리나요?

**A2.** 체계적으로 접근하세요.
1. 정의역 확인
2. 특수점 찾기 (절편, 꼭짓점)
3. 증감 파악
4. 점 찍고 연결

### Q3. 함수와 방정식의 차이는?

**A3.** 관점이 다릅니다.
- 함수: 대응 관계
- 방정식: 특정 값을 만족하는 x
- 함수의 그래프와 x축의 교점 → 방정식의 해

### Q4. 이차함수가 어려워요.

**A4.** 단계별로 연습하세요.
1. 표준형 변환 연습
2. 그래프 그리기
3. 최대·최소 문제
4. 판별식 활용

### Q5. 함수의 증가·감소를 어떻게 판단하나요?

**A5.** 그래프나 식으로 판단합니다.
- 그래프: 오른쪽으로 갈 때 상승/하강
- 식: x가 커질 때 f(x)의 변화
- 미분 이용 (고등)

### Q6. 합성함수가 헷갈려요.

**A6.** 순서를 명확히 하세요.
- (g ∘ f)(x) = g(f(x))
- f를 먼저 계산, 그 결과에 g 적용
- 안에서 바깥으로

### Q7. 역함수는 언제 배우나요?

**A7.** 고등학교 수학입니다.
- 수학 I에서 정식 학습
- 일대일 대응 필요
- 그래프: y=x 대칭

### Q8. 수능에서 함수가 많이 나오나요?

**A8.** 거의 모든 문제에 등장합니다.
- 직접 출제: 함수 자체
- 간접 출제: 방정식, 부등식, 미적분
- 고득점의 핵심

### Q9. 과외가 꼭 필요한가요?

**A9.** 선택사항이지만 효과적입니다.
- 독학 가능: 기본 개념, 표준 문제
- 과외 도움: 오개념 교정, 고난도 문제, 효율적 학습

### Q10. 함수 공부 시간은 얼마나?

**A10.** 매일 꾸준히 하는 것이 중요합니다.
- 개념 학습: 주 2-3회, 회당 1시간
- 문제 풀이: 매일 30분-1시간
- 복습: 주말 2시간
