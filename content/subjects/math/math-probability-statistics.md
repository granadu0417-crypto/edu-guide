## 경우의 수 심화

### 순열과 조합

**기본**

```
순열 (순서 O):
nPr = n!/(n-r)!

조합 (순서 X):
nCr = n!/r!(n-r)!

구분:
"배열" → 순열
"선택" → 조합

→ 핵심 개념
```

### 중복순열과 중복조합

**중복 허용**

```
중복순열:
nΠr = n^r

중복조합:
nHr = n+r-1Cr

예:
중복순열: 비밀번호
중복조합: 방정식 해

→ 응용
```

### 분할

**나누기**

```
유형:
- 같은 것 포함 순열
- 집합 분할
- 자연수 분할

주의:
구별 여부

→ 복잡
```

### 경우의 수 전략

**체계적**

```
원칙:
1. 경우 나누기
2. 빠짐없이 세기
3. 중복 제거

도구:
- 표
- 나무그림
- 벤다이어그램

→ 시각화
```

## 확률 심화

### 조건부확률

**핵심**

```
정의:
P(A|B) = P(A∩B)/P(B)

의미:
B가 일어났을 때 A의 확률

활용:
순차적 사건

→ 매우 중요
```

### 독립과 종속

**관계**

```
독립:
P(A∩B) = P(A)P(B)

종속:
독립 아님

확인:
곱셈 확인

→ 구분
```

### 베이즈 정리

**역확률**

```
P(B|A) = P(A|B)P(B)/P(A)

의미:
원인의 확률

활용:
질병 진단 등

→ 고급
```

## 확률분포

### 이산확률분포

**개별 값**

```
확률변수:
X = 1, 2, 3, ...

확률분포:
각 값의 확률

성질:
합 = 1

→ 기본
```

### 기댓값과 분산

**중심과 퍼짐**

```
기댓값:
E(X) = Σ xP(x)

분산:
V(X) = E(X²) - {E(X)}²

표준편차:
σ = √V(X)

→ 특성값
```

### 이항분포

**베르누이**

```
B(n, p):
n번 시행, 성공 확률 p

확률:
P(X=r) = nCr p^r (1-p)^(n-r)

기댓값: E(X) = np
분산: V(X) = np(1-p)

→ 가장 중요
```

### 정규분포

**종 모양**

```
N(m, σ²):
평균 m, 표준편차 σ

특징:
- 좌우 대칭
- m 중심 종 모양
- 68-95-99.7 규칙

→ 자연 현상
```

### 표준정규분포

**표준화**

```
N(0, 1):
평균 0, 표준편차 1

표준화:
Z = (X - m)/σ

표:
확률 찾기

→ 변환
```

## 통계적 추정

### 모집단과 표본

**개념**

```
모집단:
전체

표본:
일부

목적:
표본으로 모집단 추정

→ 추론
```

### 표본평균

**분포**

```
성질:
E(X̄) = m
V(X̄) = σ²/n

중심극한정리:
n이 크면 정규분포

→ 중요 정리
```

### 신뢰구간

**구간 추정**

```
95% 신뢰구간:
X̄ ± 1.96σ/√n

의미:
모평균이 이 안에 95% 확률

→ 범위
```

## 자주 묻는 질문

**Q1. 확률과 통계가 왜 중요한가요?**

A. 실생활에 가장 많이 쓰이는 수학입니다.
데이터 시대에 필수 소양입니다.

**Q2. 경우의 수가 헷갈려요.**

A. 체계적으로 세는 연습을 하세요.
표나 나무그림을 활용하세요.

**Q3. 순열과 조합을 어떻게 구분하나요?**

A. 순서가 중요하면 순열,
선택만 하면 조합입니다.

**Q4. 조건부확률이 어려워요.**

A. 표나 그림을 그려보세요.
조건이 주어진 상황을 시각화하세요.

**Q5. 정규분포는 왜 중요한가요?**

A. 자연 현상 대부분이 정규분포를 따르기 때문입니다.
통계의 기초입니다.

**Q6. 표준화는 왜 하나요?**

A. 표를 사용하기 위해서입니다.
모든 정규분포를 하나로 통일합니다.

**Q7. 신뢰구간의 의미는?**

A. 모평균이 이 구간에 있을 확률이 95%라는 뜻입니다.
구간 추정의 핵심입니다.

**Q8. 가설검정은 언제 쓰나요?**

A. 어떤 주장이 통계적으로 유의미한지 판단할 때 사용합니다.
과학 연구에서 필수입니다.

**Q9. 확률과 통계를 잘하려면?**

A. 개념을 정확히 이해하고,
다양한 유형의 문제를 풀어보세요.

**Q10. 1:1 과외가 도움이 되나요?**

A. 네, 매우 효과적입니다.
확률과 통계는 개념 이해가 중요하고
유형이 다양해
개인 맞춤 지도가 큰 도움이 됩니다.
