# 수학 공부법

수학을 정복하기 위한 체계적인 학습 방법을 제공합니다.

## 주요 학습 영역

### 개념 이해
- 수학 개념 완벽히 이해하기
- 공식 유도 과정 알기
- 개념 간 연결고리 찾기

### 문제 풀이 전략
- 문제 유형별 접근법
- 실수 줄이는 방법
- 시간 관리 전략

### 도형과 함수
- 도형 문제 정복하기
- 함수 개념 마스터
- 그래프 해석 능력

### 수학 문장제
- 문장제 읽는 법
- 식 세우기 연습
- 단위 확인 습관

## 학년별 수학 학습

- **초등학생**: 연산 완성, 문장제 익히기
- **중학생**: 도형/함수 기초, 문제 풀이 연습
- **고등학생**: 미적분, 확률통계, 수능 대비
