# 수학 학습장애 극복 완벽 가이드

수학이 유독 어려운 학생들이 있습니다. 이는 게으름이 아니라 학습 특성의 차이일 수 있습니다. 이 글에서는 수학 학습장애를 이해하고 극복하는 방법을 알려드립니다.

## 수학 학습장애란?

### 난산증 (Dyscalculia)

**정의:**
- 수를 이해하고 계산하는 데 어려움
- 뇌의 수 처리 영역 차이
- 지능과 무관
- 인구의 3-7%

**증상:**
- 숫자 개념 부족
- 계산 매우 느림
- 수 순서 헷갈림
- 시계 읽기 어려움
- 거스름돈 계산 못함

**진단:**
- 전문가 검사 필요
- 교육청 학습클리닉
- 병원 정신과

### 수포자 (수학 포기자)

**정의:**
- 학습장애는 아니지만
- 누적된 실패 경험
- 수학 자신감 상실
- 포기 상태

**원인:**
- 기초 구멍
- 반복된 실패
- 부정적 피드백
- 잘못된 학습법

**차이:**
- 난산증: 뇌 구조적 문제
- 수포자: 심리적 문제

## 수학 학습장애 자가 진단

### 체크리스트

**기본 개념:**
- [ ] 숫자 크기 비교 어려움
- [ ] 수직선 이해 못함
- [ ] 자릿수 개념 없음
- [ ] 분수 개념 전혀 모름

**계산:**
- [ ] 덧셈/뺄셈 매우 느림
- [ ] 구구단 외워도 못 씀
- [ ] 계산 실수 매우 많음
- [ ] 암산 거의 불가능

**문제 해결:**
- [ ] 문장제 이해 못함
- [ ] 문제 풀이 전략 없음
- [ ] 공식 적용 못함
- [ ] 논리적 사고 약함

**일상생활:**
- [ ] 시간 개념 약함
- [ ] 돈 계산 어려움
- [ ] 거리/무게 감각 없음

**5개 이상:** 전문가 상담 권장
**10개 이상:** 난산증 가능성

### 전문 기관

**무료:**
- 교육청 학습클리닉
- Wee센터
- 학교 상담실

**유료:**
- 아동병원 정신과
- 심리 검사 센터
- 학습 클리닉

## 맞춤 학습 전략

### 시각적 학습

**구체물 활용:**
- 숫자 블록
- 주판
- 수막대
- 동전/지폐

**그림으로:**
- 모든 문제 그림
- 수직선 활용
- 도표 만들기
- 색깔 구분

**예시:**
```
3 + 4 = ?

● ● ●  +  ● ● ● ●  =  ● ● ● ● ● ● ●

시각적으로 보면 이해 쉬움!
```

### 단계별 학습

**초급:**
1. 수 개념 (1-10)
2. 덧셈/뺄셈 (한 자리)
3. 곱셈 기초

**중급:**
4. 두 자리 계산
5. 구구단
6. 분수 기초

**고급:**
7. 문장제
8. 복잡한 계산
9. 응용

**원칙:**
- 한 단계 완벽히
- 서두르지 않기
- 반복 연습
- 성공 경험

### 반복과 연습

**소량 반복:**
- 하루 10분씩
- 3-5문제만
- 매일 꾸준히

**다양한 방법:**
- 쓰기

- 게임

**게임화:**
