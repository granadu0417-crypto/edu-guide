## 2. 논리적 추론

### 2.1 연역 추론

**정의**
- 일반 → 특수
- 원리 적용
- 필연적 결론

**수학에서**
- 정의, 공리, 정리
- 증명
- 공식 적용

**예시**
- "모든 짝수는 2로 나누어진다" (일반)
- "6은 짝수다" (특수)
- "따라서 6은 2로 나누어진다" (결론)

**연습**
- 정의로부터 결론 유도
- 공식 적용 과정 설명
- 증명 따라가기

### 2.2 귀납 추론

**정의**
- 특수 → 일반
- 패턴 발견
- 일반화

**수학에서**
- 규칙 찾기
- 추측하기
- 귀납적 증명

**예시**
- 1+2=3 (홀수)
- 3+2=5 (홀수)
- 5+2=7 (홀수)
- → "홀수 + 짝수 = 홀수"

**주의**
- 반례 확인
- 항상 참은 아님
- 증명 필요

### 2.3 유추

**정의**
- 유사한 것으로부터 추론
- 관계 찾기

**수학에서**
- 비슷한 문제
- 낮은 차원에서 생각
- 구조적 유사성

**예시**
- 직선에서의 개념 → 평면으로
- 2차원 → 3차원
- 대수 ↔ 기하

## 3. 문제 해결 전략

### 3.1 Polya의 4단계

**1단계: 이해하기**
- 무엇을 구하는가?
- 주어진 것은?
- 조건은?
- 가능한가?

- 전에 본 문제?

- 어떤 방법?
- 보조 문제?

**3단계: 계획 실행**

- 올바른가?

**4단계: 반성하기**
