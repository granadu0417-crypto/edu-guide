## 1. 대수의 중요성

## 2. 방정식

### 2.1 일차방정식

**정의**

ax + b = 0 (a ≠ 0)

**풀이**
1. 이항: 상수항을 우변으로
2. 계산: x = -b/a

**예시**
- 3x + 6 = 0
- 3x = -6
- x = -2

**응용 문제**

**나이 문제**
- "아버지 나이는 아들의 3배보다 5살 많다"
- 설정: 아들 x살, 아버지 3x+5살

**속력 문제**
- "2시간 동안 80km 갔다"
- 설정: 속력 x km/h, 2x = 80

### 2.2 연립방정식

**정의**

두 개 이상의 방정식을 동시에 만족하는 해:
- ax + by = c
- dx + ey = f

**풀이법 1: 대입법**

1. 한 방정식에서 x를 y로 표현
2. 다른 방정식에 대입
3. y 계산
4. x 계산

**예시**
- x + y = 5 ... (1)
- 2x - y = 1 ... (2)

(1)에서 x = 5 - y
(2)에 대입: 2(5-y) - y = 1
10 - 2y - y = 1
-3y = -9
y = 3, x = 2

**풀이법 2: 가감법**

1. 계수를 같게 만들기
2. 더하거나 빼기
3. 한 미지수 소거
4. 다른 미지수 계산

**예시**
- 2x + 3y = 13 ... (1)
- 3x - 3y = 2 ... (2)

(1)+(2): 5x = 15, x = 3
y = 7/3

**응용 문제**

**농도 문제**
- "5% 소금물 x g과 10% 소금물 y g을 섞어 8% 300g"
- x + y = 300
- 0.05x + 0.10y = 0.08×300

### 2.3 이차방정식

**정의**

ax² + bx + c = 0 (a ≠ 0)

**풀이법 1: 인수분해**

(x - α)(x - β) = 0
x = α 또는 x = β

**예시**
- x² - 5x + 6 = 0
- (x-2)(x-3) = 0
- x = 2 또는 x = 3

**풀이법 2: 근의 공식**

x = [-b ± √(b²-4ac)] / 2a

**판별식 D = b² - 4ac**
- D > 0: 서로 다른 두 실근
- D = 0: 중근 (한 개)
- D < 0: 허근 (실근 없음)

**예시**
- x² - 4x + 1 = 0
- a=1, b=-4, c=1
- D = 16 - 4 = 12 > 0
- x = [4 ± √12] / 2 = [4 ± 2√3] / 2 = 2 ± √3

**풀이법 3: 완전제곱식**

x² + 2bx = (x+b)² - b²

**예시**
- x² + 6x + 5 = 0
- x² + 6x = -5
- (x+3)² - 9 = -5
- (x+3)² = 4
- x + 3 = ±2
- x = -1 또는 x = -5

### 2.4 근과 계수의 관계

**이차방정식 ax² + bx + c = 0의 두 근 α, β**

**합**: α + β = -b/a
**곱**: αβ = c/a

**활용**
- 두 근의 합과 곱을 알면 방정식 복원
- 예: 두 근의 합 5, 곱 6
- x² - 5x + 6 = 0

**변형 문제**
- α² + β² = (α+β)² - 2αβ
- 1/α + 1/β = (α+β) / αβ

### 2.5 고차방정식

**삼차방정식**

ax³ + bx² + cx + d = 0

**풀이**
1. 인수정리: f(a) = 0이면 (x-a)가 인수
2. 조립제법으로 인수 찾기
3. 이차방정식으로 변환

**예시**
- x³ - 6x² + 11x - 6 = 0
- f(1) = 1-6+11-6 = 0 → (x-1)이 인수
- (x-1)(x²-5x+6) = 0
- (x-1)(x-2)(x-3) = 0
- x = 1, 2, 3

**사차방정식**

**대칭형**
- x⁴ + ax³ + bx² + ax + 1 = 0
- x²으로 나눈 후 t = x + 1/x 치환

**상반방정식**
- 계수가 대칭

## 3. 부등식

### 3.1 일차부등식

**정의**

ax + b > 0, ax + b < 0

**풀이**
- 일차방정식과 동일
- **음수로 나누면 부등호 방향 바뀜**

**예시**
- 3x - 6 > 0
- 3x > 6
- x > 2

- -2x + 4 < 0
- -2x < -4
- x > 2 (부등호 방향 변경!)

### 3.2 이차부등식

**정의**

ax² + bx + c > 0 또는 < 0

**풀이**

1. 좌변을 인수분해 (또는 근의 공식)
2. 그래프 그리기
3. 범위 결정

**예시 1**: x² - 5x + 6 > 0
- (x-2)(x-3) > 0
- 그래프: 아래로 볼록 (a=1>0)
- 근: x = 2, 3
- 해: x < 2 또는 x > 3

**예시 2**: x² - 5x + 6 < 0
- (x-2)(x-3) < 0
- 해: 2 < x < 3

**판별식과 해**

| D의 부호 | ax² + bx + c > 0 | ax² + bx + c < 0 |
|----------|------------------|------------------|
| D > 0 | x < α 또는 x > β | α < x < β |
| D = 0 | x ≠ α (모든 x) | 해 없음 |
| D < 0 | 모든 x | 해 없음 |

(단, a > 0이고 α < β일 때)

### 3.3 연립부등식

**두 개 이상의 부등식을 동시에 만족**

**예시**
- x - 3 > 0 ... (1)
- 2x - 5 < 7 ... (2)

(1): x > 3
(2): 2x < 12, x < 6

**교집합**: 3 < x < 6

**그래프로 이해**
- 수직선에 표시
- 겹치는 부분이 해

### 3.4 절댓값 부등식

**정의**

|x| < a ⇔ -a < x < a
|x| > a ⇔ x < -a 또는 x > a

**예시 1**: |x - 2| < 3
- -3 < x - 2 < 3
- -1 < x < 5

**예시 2**: |x + 1| > 2
- x + 1 < -2 또는 x + 1 > 2
- x < -3 또는 x > 1

**복잡한 절댓값**
- 절댓값 안이 0이 되는 점 찾기
- 구간별로 경우 나누기
- 각 구간에서 풀기

### 3.5 고차 부등식

**인수분해 가능한 경우**

**예시**: (x-1)(x-2)(x-3) > 0
- 근: 1, 2, 3
- 구간: x<1, 1<x<2, 2<x<3, x>3
- 부호 판정:
  - x<1: (-)(-)(-)  = (-)
  - 1<x<2: (+)(-)(-)  = (+)
  - 2<x<3: (+)(+)(-)  = (-)
  - x>3: (+)(+)(+)  = (+)
- 해: 1 < x < 2 또는 x > 3

**그래프 활용**
- 최고차 계수가 양수면 ∩∩∩ 모양
- 근의 좌우에서 부호 변화

## 4. 함수

### 4.1 함수의 개념

**정의**

집합 X의 각 원소 x에 집합 Y의 원소 y를 대응
- **일대일 대응**: 하나의 x에 하나의 y
- 표기: f(x), y = f(x)

**함수의 조건**
1. 정의역의 모든 원소가 대응
2. 하나의 x에 하나의 y만 대응

**예시**
- f(x) = 2x + 1
- x = 1 → f(1) = 3
- x = 2 → f(2) = 5

### 4.2 일차함수

**정의**

f(x) = ax + b (a ≠ 0)

**그래프**: 직선
- **기울기**: a
- **y절편**: b
- **x절편**: -b/a

**기울기의 의미**
- a > 0: 증가 (오른쪽 위)
- a < 0: 감소 (오른쪽 아래)
- |a|가 클수록: 기울기 가파름

**두 점을 지나는 직선**
- 점 (x₁, y₁), (x₂, y₂)
- 기울기: a = (y₂-y₁) / (x₂-x₁)
- 방정식: y - y₁ = a(x - x₁)

**예시**
- 점 (1, 3), (3, 7)
- a = (7-3)/(3-1) = 2
- y - 3 = 2(x-1)
- y = 2x + 1

### 4.3 이차함수

**정의**

f(x) = ax² + bx + c (a ≠ 0)

**표준형**: f(x) = a(x-p)² + q
- **꼭짓점**: (p, q)
- **축**: x = p

**그래프**: 포물선
- a > 0: 아래로 볼록 (U자)
- a < 0: 위로 볼록 (∩자)
- |a|가 클수록: 폭 좁음

**꼭짓점 구하기**

**방법 1**: 완전제곱식
- f(x) = x² - 4x + 5
- = (x² - 4x + 4) + 1
- = (x-2)² + 1
- 꼭짓점: (2, 1)

**방법 2**: 공식
- 축: x = -b/2a
- 꼭짓점: (-b/2a, f(-b/2a))

### 4.4 이차함수의 최댓값·최솟값

**꼭짓점이 최대 또는 최소**

- a > 0 (아래 볼록): 최솟값 q (최댓값 없음)
- a < 0 (위로 볼록): 최댓값 q (최솟값 없음)

**구간에서의 최댓값·최솟값**

**경우 1**: 축이 구간 안에 있을 때
- 축에서 최대/최소
- 끝점에서 반대쪽

**경우 2**: 축이 구간 밖에 있을 때
- 끝점에서 최대와 최소

**예시**
- f(x) = -(x-2)² + 5 (꼭짓점 (2,5), 위로 볼록)
- 구간 [0, 3]에서 최댓값과 최솟값?
- 축 x=2가 구간 [0,3] 안에 있음
- 최댓값: f(2) = 5 (꼭짓점)
- 최솟값: f(0) = 1 또는 f(3) = 4 중 작은 값 = f(0) = 1

### 4.5 이차함수와 이차방정식

**그래프와 x축의 교점**

f(x) = ax² + bx + c = 0의 해 = 그래프의 x절편

**판별식 D = b² - 4ac**
- D > 0: 두 점에서 교차
- D = 0: 한 점에서 접함 (x축에 접함)
- D < 0: 교점 없음 (x축과 만나지 않음)

**이차함수의 부호**

f(x) = a(x-α)(x-β)
- α < β일 때
- f(x) > 0: x < α 또는 x > β (a>0)
- f(x) < 0: α < x < β (a>0)

### 4.6 다항함수

**정의**

f(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀

**차수**: n

**삼차함수**

f(x) = ax³ + bx² + cx + d

**그래프**: S자 곡선
- a > 0: 왼쪽 아래 → 오른쪽 위
- a < 0: 왼쪽 위 → 오른쪽 아래

**극값**
- 극댓값, 극솟값 존재 (미분으로 구함)

**사차함수**

f(x) = ax⁴ + bx³ + cx² + dx + e

**그래프**: W자 또는 M자
- a > 0: W자 (양 끝이 위)
- a < 0: M자 (양 끝이 아래)

## 5. 인수분해

### 5.1 기본 공식

**공통인수**
- ab + ac = a(b + c)

**합차공식**
- a² - b² = (a+b)(a-b)

**완전제곱식**
- a² + 2ab + b² = (a+b)²
- a² - 2ab + b² = (a-b)²

**예시**
- x² + 6x + 9 = (x+3)²
- x² - 10x + 25 = (x-5)²

### 5.2 이차식 인수분해

**x² + (a+b)x + ab = (x+a)(x+b)**

합이 계수, 곱이 상수항인 두 수 찾기

**예시**
- x² + 5x + 6
- 합 5, 곱 6인 두 수: 2, 3
- (x+2)(x+3)

**acx² + (ad+bc)x + bd = (ax+b)(cx+d)**

**예시**
- 2x² + 7x + 3
- 2×3 = 6, 합 7인 두 수: 1, 6
- 2x² + x + 6x + 3
- x(2x+1) + 3(2x+1)
- (2x+1)(x+3)

### 5.3 고차식 인수분해

**인수정리**

f(a) = 0이면 (x-a)가 f(x)의 인수

**예시**
- f(x) = x³ - 7x + 6
- f(1) = 1 - 7 + 6 = 0
- (x-1)이 인수
- 조립제법: (x-1)(x²+x-6)
- (x-1)(x+3)(x-2)

**치환**

복잡한 식을 간단히

**예시**
- x⁴ + x² - 2
- t = x² 치환
- t² + t - 2 = (t+2)(t-1)
- (x²+2)(x²-1)
- (x²+2)(x+1)(x-1)

## 6. 그래프와 변환

### 6.1 평행이동

**y = f(x)의 그래프를 평행이동**

- x축 방향으로 a만큼: y = f(x-a)
- y축 방향으로 b만큼: y = f(x) + b
- 둘 다: y = f(x-a) + b

**예시**
- y = x²를 오른쪽 2, 위 3 이동
- y = (x-2)² + 3

### 6.2 대칭이동

**대칭**

- x축 대칭: y = -f(x)
- y축 대칭: y = f(-x)
- 원점 대칭: y = -f(-x)

**예시**
- y = x² - 4x + 3을 y축 대칭
- y = (-x)² - 4(-x) + 3
- y = x² + 4x + 3

### 6.3 확대·축소

**y = af(x)**

- a > 1: y축 방향 확대
- 0 < a < 1: y축 방향 축소

**y = f(ax)**

- a > 1: x축 방향 축소
- 0 < a < 1: x축 방향 확대

## 7. 방정식·부등식 응용

### 7.1 도형과 방정식

**직선의 방정식**
- 기울기 a, y절편 b: y = ax + b
- 두 점 통과: 기울기 구한 후 대입

**원의 방정식**
- 중심 (a, b), 반지름 r: (x-a)² + (y-b)² = r²

### 7.2 최대·최소 문제

**이차함수의 최대·최소**
- 넓이, 부피, 거리 등의 최댓값·최솟값
- 이차함수로 나타낸 후 꼭짓점

**예시**
- 둘레 20인 직사각형의 최대 넓이?
- 가로 x, 세로 10-x
- 넓이 S = x(10-x) = -x² + 10x
- S = -(x-5)² + 25
- 최댓값: 25 (x=5일 때)

### 7.3 속력·거리 문제

**기본 공식**
- 거리 = 속력 × 시간

**강 문제**
- 정지수면 속력 a, 강 속력 b
- 내려갈 때: a+b
- 올라갈 때: a-b

### 7.4 농도·혼합 문제

**농도 공식**
- 농도 = (소금 무게) / (소금물 무게) × 100%

**혼합**
- a% x g + b% y g = c% (x+y) g
- 0.01ax + 0.01by = 0.01c(x+y)

## 10. FAQ

### Q1. 대수가 왜 중요한가요?

**모든 수학의 기초입니다.**

대수 없이는:
- 미적분 불가능
- 확률통계 어려움
- 고등 수학 막힘

**실생활 활용**:
- 문제 해결 능력
- 논리적 사고
- 분석력

### Q2. 방정식을 빨리 푸는 방법은?

**기본 공식을 완벽히 외우세요.**

필수 공식:
- 인수분해 공식
- 근의 공식
- 완전제곱식

**연습**:
- 반복 연습으로 속도 향상
- 패턴 인식
- 계산 능력

### Q3. 이차함수 그래프를 잘 그리려면?

**핵심 요소를 파악하세요.**

**1단계**: 포물선 방향 (a의 부호)
**2단계**: 꼭짓점 찾기
**3단계**: x절편 (있다면)
**4단계**: y절편
**5단계**: 그래프 그리기

**연습**: 많이 그려보는 것이 최선

### Q4. 인수분해가 안 될 때는?

**여러 방법을 시도하세요.**

**1순위**: 공통인수
**2순위**: 합차공식, 완전제곱
**3순위**: 십자가 인수분해
**4순위**: 인수정리 (고차)
**5순위**: 치환

**안 되면**: 근의 공식 사용

### Q5. 절댓값 문제는 어떻게 푸나요?

**경우를 나누세요.**

**1단계**: 절댓값 안이 0이 되는 점 찾기
**2단계**: 수직선에 표시
**3단계**: 구간별로 절댓값 풀기
**4단계**: 각 구간에서 방정식/부등식 풀기
**5단계**: 해를 모아서 정리

### Q6. 응용 문제가 어려워요.

**문제를 식으로 바꾸는 연습을 하세요.**

**1단계**: 무엇을 구하는지 파악
**2단계**: 미지수 설정
**3단계**: 조건을 식으로
**4단계**: 방정식 세우기
**5단계**: 풀고 검증

**많이 풀어보기**: 유형별 연습

### Q7. 계산 실수가 많아요.

**실수 패턴을 찾으세요.**

**자주 하는 실수**:
- 부호 실수
- 분수 계산 오류
- 괄호 풀 때 실수
- 이항 시 부호 바뀜

**대책**:
- 천천히 정확하게
- 검산 습관
- 중간 과정 쓰기

### Q8. 수능 대수는 어떻게 준비하나요?

**기본 개념부터 탄탄히 하세요.**

**공통 과목**:
- 다항식, 방정식, 함수
- 기출 문제 반복

**선택 과목**:
- 미적분: 대수 활용 필수
- 기하: 대수와 연결
- 확률과 통계: 대수 기초

**실전 연습**: 시간 재고 풀기

### Q9. 고차방정식은 어떻게 푸나요?

**인수정리를 활용하세요.**

**1단계**: 상수항의 약수를 f(x)에 대입
**2단계**: f(a) = 0이 되는 a 찾기
**3단계**: (x-a)로 조립제법
**4단계**: 나머지 방정식 풀기

**연습**: 조립제법 숙달

### Q10. 1:1 과외가 필요한가요?

**개념이 약하면 강력 추천합니다.**

**1:1 과외의 장점**:
- 개념 완벽 이해
- 즉각 질문 해결
- 맞춤 학습
- 실수 교정

**학원과 병행**: 가능하다면 시너지 효과
