## 1. 서술형 문제란?

### 1.1 서술형의 의미

**정의**
- 풀이 과정을 서술
- 답만이 아닌 과정 평가
- 사고 과정 확인

**객관식과의 차이**
- 객관식: 답만
- 서술형: 과정+답
- 부분 점수 가능

### 1.2 중요성

**학습 효과**
- 깊이 있는 이해
- 논리적 사고
- 표현 능력

**평가 비중**
- 내신 30-50%
- 수능 단답형
- 대학 논술

### 1.3 어려운 이유

**문제점**
- 문장 이해 어려움
- 과정 서술 익숙하지 않음
- 시간 부족
- 실수 불안

**극복 방법**
- 체계적 훈련
- 연습 반복
- 전략 수립

## 2. 문장제 해결 전략

### 2.1 문제 읽기

**천천히 정확하게**
- 한 번에 이해하려 하지 말기
- 여러 번 읽기
- 중요한 부분 표시

**핵심 파악**
- 무엇을 구하는가?
- 주어진 것은?
- 조건은?

### 2.2 조건 정리

**표로 정리**
- 주어진 것
- 구하는 것
- 조건

**그림 그리기**
- 도형 문제
- 관계도
- 시각화

**기호 정하기**
- 미지수 설정
- 일관성 유지
- 명확한 표기

### 2.3 전략 수립

**문제 유형 파악**
- 방정식/부등식
- 함수
- 도형
- 확률/통계

**적절한 방법 선택**
- 식 세우기
- 그림 활용
- 논리적 추론

**단계 계획**
1. 먼저 할 것
2. 다음 할 것
3. 마지막 확인

## 3. 풀이 과정 작성

### 3.1 기본 원칙

**단계별로**
- 한 단계씩
- 논리적 순서
- 생략하지 않기

**명확하게**
- 정확한 수학 용어
- 기호 정확히
- 읽기 쉽게

**정확하게**
- 계산 실수 주의
- 검산
- 답 확인

### 3.2 서술 형식

**"~라고 하자"**
- 미지수 설정
- "x라고 하자"
- 명확히 정의

**"~이므로"**
- 이유 제시
- 논리적 연결
- 근거 명시

**"따라서"**
- 결론 도출
- 답 제시
- 단위 확인

### 3.3 점수 받는 서술

**채점 기준 이해**
- 식 세우기 (30%)
- 풀이 과정 (50%)
- 답 (20%)

**부분 점수 전략**
- 식이라도 세우기
- 중간 과정 자세히
- 마지막까지 포기 않기

**감점 요인**
- 논리 비약
- 설명 부족
- 단위 누락
- 답 표시 없음

### 4.3 도형

**공식 적용**

### 4.4 확률/통계

**확률 계산**

## 6. 시간 관리

### 6.1 문제별 시간 배분

**난이도 파악**
- 쉬운 것부터
- 어려운 것 나중에
- 배점 고려

**시간 제한**
- 한 문제 최대 시간
- 넘기면 다음으로
- 나중에 다시

### 6.2 효율적 풀이

**빠른 계산**
- 암산 활용
- 계산기 (허용시)
- 간단히 정리

**불필요한 과정 생략**
- 핵심만
- 하지만 논리 비약 주의

### 6.3 검산 시간 확보

**마지막 5-10분**
- 답 확인
- 빠진 것 없나
- 단위, 표기

## FAQ

### Q1. 서술형을 어떻게 시작하나요?

**A1.** 조건을 정리하세요.
1. 주어진 것 나열
2. 구하는 것 명확히
3. 미지수 설정
4. 식 세우기

### Q2. 풀이를 얼마나 자세히 써야 하나요?

**A2.** 남이 봐도 이해되게.
- 논리 비약 없이
- 각 단계 설명
- 왜 그렇게 했는지

### Q3. 시간이 부족해요.

**A3.** 연습으로 속도를 높이세요.
- 쉬운 것부터
- 불필요한 과정 최소화
- 반복 연습

### Q4. 계산 실수가 많아요.

**A4.** 천천히 정확하게.
- 급하지 않게
- 한 줄씩 확인
- 검산 습관

### Q5. 답은 맞는데 과정에서 감점당해요.

**A5.** 서술을 더 자세히 하세요.
- "~이므로" 사용
- 각 단계 설명
- 근거 명시

### Q6. 문장제가 이해 안 돼요.

**A6.** 천천히 여러 번 읽으세요.
- 한 문장씩
- 그림 그리기
- 표로 정리

### Q7. 부분 점수를 어떻게 받나요?

**A7.** 과정을 최대한 쓰세요.
- 식만이라도
- 중간까지라도
- 마지막까지 포기 않기

### Q8. 모범 답안처럼 써야 하나요?

**A8.** 논리만 맞으면 됩니다.
- 정답은 여러 가지
- 자신의 방법도 OK
- 논리적이면 인정

### Q9. 서술형 과외가 필요한가요?

**A9.** 효율적 향상에 도움됩니다.
- 채점 기준 학습
- 즉각 피드백
- 표현 연습
- 시간 관리

### Q10. 얼마나 연습해야 하나요?

**A10.** 매일 조금씩이 중요합니다.
- 매일 2-3문제
- 다양한 유형
- 오답 정리
- 모의고사 주기적으로
