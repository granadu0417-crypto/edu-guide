- 외각의 성질
- 중선, 중점연결정리

### 2.2 사각형

**종류**
- 정사각형

**성질**
- 내각의 합: 360°
- 대각선의 성질
- 각 사각형의 특징

**넓이**
- 직사각형: 가로 × 세로
- 평행사변형: 밑변 × 높이

### 2.3 원

**기본 성질**
- 중심, 반지름, 지름

**정리**

## 4. 좌표기하

### 4.1 좌표평면

**좌표계**
- x축, y축
- 원점, 사분면
- 점의 좌표 (x, y)

**거리 공식**
- 두 점 A(x₁,y₁), B(x₂,y₂)
- AB = √[(x₂-x₁)² + (y₂-y₁)²]

**내분점·외분점**
- 선분을 m:n으로 나누는 점
- 내분점: [(nx₁+mx₂)/(m+n), (ny₁+my₂)/(m+n)]

### 4.2 직선의 방정식

**기울기**
- 기울기 m = (y₂-y₁)/(x₂-x₁)
- 평행: 기울기 같음
- 수직: 기울기의 곱 -1

**방정식**
- 기울기-절편: y = mx + b
- 점-기울기: y - y₁ = m(x - x₁)
- 두 점을 지나는 직선

**위치관계**
- 평행
- 수직
- 교점

### 4.3 원의 방정식

**표준형**
- (x-a)² + (y-b)² = r²
- 중심: (a, b)
- 반지름: r

**일반형**
- x² + y² + Ax + By + C = 0
- 표준형으로 변환

**직선과 원**
- 교점 개수
- 접선의 방정식

### 4.4 도형의 이동

**평행이동**
- (x, y) → (x+a, y+b)

**대칭이동**
- x축 대칭: (x, y) → (x, -y)
- y축 대칭: (x, y) → (-x, y)
- 원점 대칭: (x, y) → (-x, -y)
- y=x 대칭: (x, y) → (y, x)

**회전이동**
- 원점 중심 θ만큼 회전

## 5. 벡터

### 5.1 벡터의 기본

**정의**
- 크기와 방향을 가진 양
- 화살표로 표현
- 벡터 a = (a₁, a₂)

**성분**
- 위치벡터
- 방향벡터
- 크기(norm): |a| = √(a₁² + a₂²)

**연산**
- 덧셈: a + b
- 뺄셈: a - b
- 스칼라곱: ka

### 5.2 내적

**정의**
- a · b = |a||b|cosθ
- a · b = a₁b₁ + a₂b₂

**성질**
- 교환법칙
- 분배법칙
- a · b = 0 ⇔ a ⊥ b

**활용**
- 각도 계산
- 정사영
- 일의 양

### 5.3 벡터의 응용
