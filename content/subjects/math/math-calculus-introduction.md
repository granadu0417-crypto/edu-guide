## 1. 미적분의 중요성

### 1.1 왜 미적분인가

**변화를 다루는 수학**

미적분의 핵심:
- **순간 변화율**: 속도, 가속도
- **누적**: 거리, 넓이, 부피
- **최적화**: 최댓값, 최솟값
- **예측**: 함수의 행동

**뉴턴과 라이프니츠**
- 17세기 독립적으로 발견
- 과학 혁명의 토대
- 현대 과학의 언어

### 1.2 실생활 응용

**물리학**
- 운동: 속도, 가속도
- 파동: 진동, 음파, 빛
- 전자기학: 전기장, 자기장

**공학**
- 건축: 구조 최적화
- 전자: 회로 설계
- 기계: 효율 계산

**경제학**
- 비용 최소화
- 수익 최대화
- 수요·공급 분석

**생물학**
- 개체군 증가
- 질병 전파
- 약물 농도

### 1.3 대학 수학과의 연결

**필수 선수 과목**

| 전공 | 필요한 미적분 | 난이도 |
|------|---------------|--------|
| 수학·물리 | 고급 미적분, 미분방정식 | ★★★ |
| 공학 | 다변수 미적분, 선형대수 | ★★★ |
| 경제·경영 | 미분, 편미분 | ★★☆ |
| 생명과학 | 기본 미적분 | ★☆☆ |

**고교 미적분이 약하면**
- 대학 진도 따라가기 어려움
- 재수강률 높음
- 전공 변경 고려

## 2. 극한 (Limit)

### 2.1 극한의 개념

**정의**

x가 a에 한없이 가까워질 때, f(x)가 L에 가까워지면:

lim(x→a) f(x) = L

**의미**
- x = a에서의 값이 아님
- x가 a에 "접근"할 때의 경향

**예시**
- lim(x→2) (x²-4)/(x-2)
- x=2일 때 0/0 (정의 안 됨)
- 하지만 x→2일 때?
- (x²-4)/(x-2) = (x+2)(x-2)/(x-2) = x+2
- x→2일 때 x+2 → 4
- lim(x→2) (x²-4)/(x-2) = 4

### 2.2 극한의 성질

**극한의 기본 성질**

lim(x→a) [f(x) + g(x)] = lim(x→a) f(x) + lim(x→a) g(x)

lim(x→a) [f(x) · g(x)] = lim(x→a) f(x) · lim(x→a) g(x)

lim(x→a) [f(x) / g(x)] = lim(x→a) f(x) / lim(x→a) g(x)
(단, lim(x→a) g(x) ≠ 0)

**상수배**
lim(x→a) [c · f(x)] = c · lim(x→a) f(x)

### 2.3 극한값 계산

**치환**

분자와 분모를 약분

**예시**
- lim(x→3) (x²-9)/(x-3)
- = lim(x→3) (x+3)(x-3)/(x-3)
- = lim(x→3) (x+3) = 6

**유리화**

근호 제거

**예시**
- lim(x→0) (√(x+1) - 1)/x
- 분자 분모에 √(x+1) + 1 곱하기
- = lim(x→0) [(x+1) - 1] / [x(√(x+1)+1)]
- = lim(x→0) x / [x(√(x+1)+1)]
- = lim(x→0) 1 / (√(x+1)+1) = 1/2

**최고차 항**

무한대로 갈 때

**예시**
- lim(x→∞) (3x²+2x+1)/(x²-5)
- 분자 분모를 x²으로 나누기
- = lim(x→∞) (3 + 2/x + 1/x²) / (1 - 5/x²)
- = 3/1 = 3

### 2.4 좌극한과 우극한

**정의**

- **좌극한**: x가 a보다 작은 쪽에서 접근
  lim(x→a⁻) f(x)

- **우극한**: x가 a보다 큰 쪽에서 접근
  lim(x→a⁺) f(x)

**극한의 존재 조건**

lim(x→a) f(x)가 존재 ⇔ lim(x→a⁻) f(x) = lim(x→a⁺) f(x)

**예시**

f(x) = { x+1  (x < 2)
       { 2x   (x ≥ 2)

- lim(x→2⁻) f(x) = 2+1 = 3
- lim(x→2⁺) f(x) = 2×2 = 4
- 좌극한 ≠ 우극한 → **극한 존재하지 않음**

### 2.5 함수의 연속

**정의**

함수 f(x)가 x=a에서 연속 ⇔

1. f(a)가 정의됨
2. lim(x→a) f(x)가 존재
3. lim(x→a) f(x) = f(a)

**불연속 종류**

**제거 가능한 불연속**
- 극한은 존재하지만 f(a) 정의 안 됨
- 또는 f(a) ≠ lim(x→a) f(x)

**뛰어넘는 불연속**
- 좌극한 ≠ 우극한

**무한 불연속**
- 극한이 무한대

## 3. 미분 (Differentiation)

### 3.1 미분계수

**정의**

f'(a) = lim(h→0) [f(a+h) - f(a)] / h

**의미**
- x=a에서의 순간 변화율
- 그래프의 접선 기울기

**예시**

f(x) = x²의 x=2에서 미분계수?

f'(2) = lim(h→0) [(2+h)² - 4] / h
     = lim(h→0) [4+4h+h² - 4] / h
     = lim(h→0) [4h+h²] / h
     = lim(h→0) (4+h) = 4

### 3.2 도함수

**정의**

f'(x) = lim(h→0) [f(x+h) - f(x)] / h

**표기**
- f'(x)
- df/dx
- dy/dx

**예시**

f(x) = x³의 도함수?

f'(x) = lim(h→0) [(x+h)³ - x³] / h
     = lim(h→0) [x³+3x²h+3xh²+h³ - x³] / h
     = lim(h→0) [3x²h+3xh²+h³] / h
     = lim(h→0) (3x²+3xh+h²)
     = 3x²

### 3.3 미분 공식

**기본 공식**

| 함수 | 도함수 |
|------|--------|
| f(x) = c | f'(x) = 0 |
| f(x) = xⁿ | f'(x) = nxⁿ⁻¹ |
| f(x) = eˣ | f'(x) = eˣ |
| f(x) = ln x | f'(x) = 1/x |
| f(x) = sin x | f'(x) = cos x |
| f(x) = cos x | f'(x) = -sin x |

**예시**
- f(x) = x⁵ → f'(x) = 5x⁴
- f(x) = x⁻² → f'(x) = -2x⁻³
- f(x) = √x = x^(1/2) → f'(x) = (1/2)x^(-1/2) = 1/(2√x)

### 3.4 미분 법칙

**합의 미분**

[f(x) + g(x)]' = f'(x) + g'(x)

**상수배**

[c·f(x)]' = c·f'(x)

**곱의 미분**

[f(x)·g(x)]' = f'(x)·g(x) + f(x)·g'(x)

**예시**
- f(x) = x²·sin x
- f'(x) = 2x·sin x + x²·cos x

**몫의 미분**

[f(x)/g(x)]' = [f'(x)·g(x) - f(x)·g'(x)] / [g(x)]²

**예시**
- f(x) = x/cos x
- f'(x) = [1·cos x - x·(-sin x)] / cos²x
- = (cos x + x sin x) / cos²x

### 3.5 합성함수의 미분

**연쇄법칙 (Chain Rule)**

f(g(x))' = f'(g(x)) · g'(x)

또는 dy/dx = dy/du · du/dx

**예시 1**
- y = (x²+1)³
- u = x²+1로 치환
- y = u³, u = x²+1
- dy/du = 3u², du/dx = 2x
- dy/dx = 3u² · 2x = 3(x²+1)² · 2x = 6x(x²+1)²

**예시 2**
- y = sin(3x)
- u = 3x로 치환
- dy/du = cos u, du/dx = 3
- dy/dx = cos(3x) · 3 = 3cos(3x)

## 4. 미분의 응용

### 4.1 접선의 방정식

**x=a에서 접선**

기울기: f'(a)
지나는 점: (a, f(a))

방정식: y - f(a) = f'(a)(x - a)

**예시**
- f(x) = x² - 3x + 2, x=1에서 접선?
- f(1) = 1 - 3 + 2 = 0
- f'(x) = 2x - 3
- f'(1) = 2 - 3 = -1
- 접선: y - 0 = -1(x - 1)
- y = -x + 1

### 4.2 함수의 증가와 감소

**판정**

- f'(x) > 0 → f(x) 증가
- f'(x) < 0 → f(x) 감소
- f'(x) = 0 → 극값 후보

**예시**
- f(x) = x³ - 3x²
- f'(x) = 3x² - 6x = 3x(x-2)
- f'(x) = 0일 때 x = 0, 2
- 구간별 부호:
  - x<0: f'(x) > 0 (증가)
  - 0<x<2: f'(x) < 0 (감소)
  - x>2: f'(x) > 0 (증가)

### 4.3 극값

**정의**

- **극댓값**: f'(a) = 0이고 좌우에서 부호가 + → -
- **극솟값**: f'(a) = 0이고 좌우에서 부호가 - → +

**찾는 법**

1. f'(x) = 0인 점 찾기
2. 좌우에서 부호 판정
3. 극댓값 또는 극솟값 결정

**예시**
- f(x) = x³ - 3x²
- f'(x) = 3x(x-2)
- x = 0: 증가→감소 → 극댓값 f(0) = 0
- x = 2: 감소→증가 → 극솟값 f(2) = -4

### 4.4 최댓값과 최솟값

**닫힌 구간 [a, b]에서**

1. 극값 후보: f'(x) = 0인 점
2. 끝점: a, b
3. 모두 계산하여 비교

**예시**
- f(x) = x³ - 3x², [0, 3]에서 최댓값·최솟값?
- f'(x) = 3x(x-2) = 0 → x = 0, 2
- f(0) = 0
- f(2) = -4
- f(3) = 27 - 27 = 0
- 최댓값: f(3) = 0
- 최솟값: f(2) = -4

### 4.5 속도와 가속도

**위치·속도·가속도**

- 위치: s(t)
- 속도: v(t) = s'(t)
- 가속도: a(t) = v'(t) = s''(t)

**예시**
- s(t) = t³ - 6t² + 9t
- v(t) = s'(t) = 3t² - 12t + 9 = 3(t-1)(t-3)
- a(t) = v'(t) = 6t - 12 = 6(t-2)

**분석**
- v(t) = 0일 때: t = 1, 3 (정지)
- a(2) = 0 (속도 변화 없음)

## 5. 적분 (Integration)

### 5.1 부정적분

**정의**

F'(x) = f(x)일 때, F(x)를 f(x)의 부정적분 또는 원시함수

표기: ∫f(x)dx = F(x) + C

**예시**
- ∫x²dx = (1/3)x³ + C
- 검증: d/dx[(1/3)x³ + C] = x²

### 5.2 적분 공식

**기본 공식**

| 함수 | 부정적분 |
|------|----------|
| ∫xⁿdx | (1/(n+1))xⁿ⁺¹ + C (n≠-1) |
| ∫(1/x)dx | ln|x| + C |
| ∫eˣdx | eˣ + C |
| ∫sin x dx | -cos x + C |
| ∫cos x dx | sin x + C |

**예시**
- ∫x⁴dx = (1/5)x⁵ + C
- ∫(1/x³)dx = ∫x⁻³dx = (-1/2)x⁻² + C

### 5.3 적분 법칙

**합의 적분**

∫[f(x) + g(x)]dx = ∫f(x)dx + ∫g(x)dx

**상수배**

∫c·f(x)dx = c·∫f(x)dx

**예시**
- ∫(3x² + 2x)dx
- = 3∫x²dx + 2∫xdx
- = 3·(1/3)x³ + 2·(1/2)x²+ C
- = x³ + x² + C

### 5.4 치환적분

**공식**

∫f(g(x))·g'(x)dx = ∫f(u)du (u = g(x))

**예시 1**
- ∫2x(x²+1)³dx
- u = x²+1로 치환
- du/dx = 2x → du = 2x dx
- ∫u³du = (1/4)u⁴ + C
- = (1/4)(x²+1)⁴ + C

**예시 2**
- ∫sin(3x)dx
- u = 3x, du = 3dx
- ∫sin u · (1/3)du
- = (-1/3)cos u + C
- = (-1/3)cos(3x) + C

### 5.5 정적분

**정의**

∫[a→b] f(x)dx = F(b) - F(a)

(F는 f의 부정적분)

**예시**
- ∫[1→3] x²dx
- = [(1/3)x³]₁³
- = (1/3)·27 - (1/3)·1
- = 9 - 1/3 = 26/3

### 5.6 정적분의 성질

**구간의 분할**

∫[a→c] f(x)dx = ∫[a→b] f(x)dx + ∫[b→c] f(x)dx

**상하 바꾸기**

∫[a→b] f(x)dx = -∫[b→a] f(x)dx

**같은 점**

∫[a→a] f(x)dx = 0

## 6. 적분의 응용

### 6.1 넓이

**곡선과 x축 사이**

y = f(x), x=a, x=b, x축으로 둘러싸인 넓이:

S = ∫[a→b] |f(x)|dx

**주의**: f(x) < 0이면 음수 → 절댓값 필요

**예시**
- y = x², x=0, x=2, x축
- S = ∫[0→2] x²dx
- = [(1/3)x³]₀²
- = 8/3

### 6.2 두 곡선 사이의 넓이

**y = f(x)와 y = g(x) 사이 (f(x) ≥ g(x))**

S = ∫[a→b] [f(x) - g(x)]dx

**예시**
- y = x², y = 2x
- 교점: x² = 2x → x(x-2) = 0 → x = 0, 2
- S = ∫[0→2] (2x - x²)dx
- = [x² - (1/3)x³]₀²
- = 4 - 8/3 = 4/3

### 6.3 부피

**회전체**

y = f(x)를 x축 중심으로 회전:

V = π∫[a→b] [f(x)]²dx

**예시**
- y = x, x=0, x=1을 x축 중심 회전
- V = π∫[0→1] x²dx
- = π[(1/3)x³]₀¹
- = π/3

### 6.4 속도와 거리

**속도 함수 v(t)가 주어졌을 때**

거리 = ∫[a→b] |v(t)|dt

변위 = ∫[a→b] v(t)dt

**차이**
- 거리: 실제 이동한 총 길이
- 변위: 최종 위치 - 초기 위치

## 7. 미분과 적분의 관계

### 7.1 미적분학의 기본정리

**제1 기본정리**

F(x) = ∫[a→x] f(t)dt이면

F'(x) = f(x)

**의미**: 정적분의 미분 = 원래 함수

**제2 기본정리**

∫[a→b] f(x)dx = F(b) - F(a)

(F'(x) = f(x))

**의미**: 미분과 적분은 역연산

### 7.2 응용

**변화량 계산**

f(b) - f(a) = ∫[a→b] f'(x)dx

**예시**
- 속도 v(t) → 위치 변화
- s(b) - s(a) = ∫[a→b] v(t)dt

## 8. 학년별 미적분 학습

**시험 대비**
- 개념 문제 70%

- 응용 문제 10%

**학습 내용**

**학습 전략**

**시험 대비**
- 개념 50%
- 계산 30%
- 응용 20%

### 8.3 수능 준비

**공통 과목 (수학I, II)**
- 기본 미분·적분

**전략**
- 기출 10개년 반복

- 계산 속도 향상
- 실수 줄이기

## 9. 1:1 과외의 장점

### 9.1 개념 완벽 이해

**추상적 → 구체적**

미적분은 추상적:
- 극한: 무한히 가까워진다?
- 미분: 순간 변화율?
- 적분: 무한히 더하기?

**1:1에서**
- 그림으로 설명
- 다양한 예시
- 즉각 질문 해결

### 9.2 단계별 맞춤 학습

**선수 학습 점검**

미적분 전 필요:
- 함수 개념
- 그래프
- 극한 (무한)

**부족하면**
- 복습부터
- 천천히 진행
- 이해 확인

### 9.3 문제 풀이 전략

**유형별 접근**

- 극한: 치환? 유리화?
- 미분: 곱? 몫? 합성?

**실수 교정**
- 부호 실수

## 10. FAQ

### Q1. 미적분이 왜 중요한가요?

**모든 이공계의 기초입니다.**

대학에서:
- 공학: 필수
- 자연과학: 필수
- 경제·경영: 중요
- 의학: 필요

**실생활**:
- 최적화 문제
- 변화 예측
- 데이터 분석

### Q2. 극한을 이해하는 방법은?

**"접근"의 개념을 이해하세요.**

극한 ≠ 그 점에서의 값
극한 = 그 점에 가까워질 때의 경향

**그림으로 이해**:
- 그래프 그리기
- x→a일 때 y는?

### Q3. 미분 공식을 외워야 하나요?

**네, 반드시 외우세요.**

기본 공식:
- xⁿ → nxⁿ⁻¹
- eˣ → eˣ
- ln x → 1/x
- sin x → cos x
- cos x → -sin x

**법칙도 암기**:
- 곱의 미분
- 몫의 미분
- 연쇄법칙

### Q4. 적분은 미분의 역이라는데?

**맞습니다.**

∫f(x)dx = F(x) ⇔ F'(x) = f(x)

**확인 방법**:
- 적분한 것을 미분
- 원래 함수가 나오면 정답

**예시**:
- ∫x²dx = (1/3)x³ + C
- 검증: [(1/3)x³]' = x² ✓

### Q5. 치환적분은 언제 쓰나요?

**합성함수 형태일 때**

∫f(g(x))·g'(x)dx

**판단**:
- 안쪽 함수 g(x) 찾기
- g'(x)가 있는지 확인
- 있으면 치환!

### Q6. 정적분의 의미는?

**누적, 총량**

- 넓이: 곡선 아래 넓이
- 거리: 속도의 누적
- 부피: 단면의 누적

**계산**:
∫[a→b] f(x)dx = F(b) - F(a)

### Q7. 극값과 최댓값·최솟값의 차이는?

**극값**: 국소적 최대·최소
**최댓값·최솟값**: 전체에서 최대·최소

**예시**:
- f(x) = x³ - 3x², [0, 3]
- 극솟값: f(2) = -4
- 최솟값: f(2) = -4 (동일)
- 극댓값: f(0) = 0
- 최댓값: f(3) = 0

### Q8. 미적분 시험은 어떻게 준비하나요?

**3단계 전략**

**1단계**: 개념 이해

**2단계**: 계산 연습
- 미분·적분 공식
- 속도와 정확성

**3단계**: 응용 문제
- 접선, 극값, 넓이
- 종합 문제

### Q9. 수능 미적분은 얼마나 어렵나요?

**선택 과목 중 가장 어렵습니다.**

**난이도**:

**준비**:

### Q10. 1:1 과외가 효과적인가요?
