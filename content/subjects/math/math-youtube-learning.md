### 장점

**무료 접근**

```
장점:
- 완전 무료
- 다양한 강사
- 짧은 영상
- 언제든 시청

→ 접근성
```

```
단점:

→ 선택적 활용
```

## 추천 채널 유형

### 개념 설명

**기초**

```
특징:
쉽고 명확한 설명

활용:
새로운 개념 학습

→ 이해 중심
```

### 문제 풀이

**실전**

```
특징:
다양한 문제 풀이

활용:
유형 학습

→ 연습
```

### 단원 정리

**복습**

```
특징:
핵심 요약

활용:
시험 전 복습

→ 정리
```

## 효과적 시청법

### 시청 전

**준비**

```
준비:
- 노트, 필기구
- 관련 교과서
- 문제집

→ 능동적 준비
```

### 시청 중

**적극적**

```
방법:
× 수동 시청
○ 필기하며 시청

일시정지:
이해 안 되면 멈춤

→ 참여
```

### 시청 후

**실천**

```
필수:
- 노트 정리
- 문제 풀기
- 오답 확인

→ 복습
```

## 시간 활용

### 짧은 영상

**효율**

```
추천:
5-15분 영상

활용:
틈새 시간

→ 집중
```

### 재생 목록

**체계**

```
활용:
단원별 재생 목록

효과:
체계적 학습

→ 구조화
```

## 주의사항

### 함정 1: 관련 영상

**방해**

```
문제:
추천 영상 클릭

해결:
목표 영상만

→ 집중
```

### 함정 2: 수동 시청

**비효율**

```
문제:
그냥 보기만

해결:
필기하며 시청

→ 능동적
```

### 함정 3: 검증 X

**품질**

```
주의:
잘못된 설명 가능

확인:
구독자, 댓글 확인

→ 검증
```

## 보조 도구

### 재생 속도

**조절**

```
1배속: 처음

→ 효율
```

### 자막

**활용**

```
한글 자막:
이해 돕기

자막 필수

→ 보조
```

## 구독 관리

### 선택

**엄선**

```
기준:

- 업로드 규칙적

```

## 유튜브 vs 인강

### 비교

**차이**

```
유튜브:
장점: 무료, 짧음
단점: 체계 X

인강:
장점: 체계적
단점: 유료

→ 병행
```

## 자주 묻는 질문

**Q1. 유튜브로 수학 공부가 되나요?**

A. 보조 도구로는 매우 효과적입니다.
체계적 학습은 교과서나 인강으로 하세요.

**Q2. 추천 채널은요?**

A. 구독자 수와 댓글을 확인하세요.
여러 채널을 비교해보고 자신에게 맞는 것을 선택하세요.

**Q3. 몇 배속으로 봐야 하나요?**

A. 처음에는 1배속,
복습할 때는 1.25-1.5배속을 권장합니다.

**Q4. 짧은 영상만 봐도 되나요?**

A. 개념 확인이나 문제 풀이에는 좋지만,
전체적인 학습은 인강이나 교과서가 필요합니다.

**Q5. 영어 영상도 괜찮나요?**

A. 수학은 언어 장벽이 낮아 괜찮습니다.
자막을 활용하세요.

**Q6. 광고가 방해돼요.**

A. 광고 건너뛰기를 빠르게 하거나,
프리미엄을 고려할 수 있습니다.

**Q7. 어떻게 채널을 찾나요?**

A. "수학 개념 설명", "수학 문제 풀이" 등으로 검색하세요.
평가가 좋은 채널을 구독하세요.

**Q8. 댓글을 확인해야 하나요?**

A. 네, 댓글에 오류 정정이나 추가 설명이 있을 수 있습니다.

**Q9. 재생 목록을 활용하나요?**

A. 네, 단원별 재생 목록을 만들면 체계적입니다.

**Q10. 1:1 과외와 유튜브를 같이 써도 되나요?**

A. 네, 매우 효과적입니다.
유튜브로 개념을 미리 보고,
과외에서 심화 학습을 하면 효율적입니다.
