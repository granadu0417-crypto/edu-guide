## 1. 미디어 리터러시란?

### 1.1 정의와 중요성

**미디어 리터러시(Media Literacy)**
- 미디어 이해·분석·평가·창조 능력
- 비판적 수용
- 능동적 참여

**왜 중요한가?**
- 정보 홍수 시대
- 가짜뉴스 판별
- 광고·선전 구별
- 민주시민 역량

### 1.2 수능 국어와의 관계

**독서(비문학) 영역**
- 주장과 근거 파악
- 논리적 타당성 검증
- 정보의 신뢰성 평가
- 관점과 의도 분석

**매체 언어**
- 2020학년도부터 선택 과목
- 뉴스, 광고, SNS 분석
- 표현 방식 이해
- 수용자 영향 파악

## 2. 뉴스 읽기

### 2.1 뉴스의 구조

**역피라미드 구조**
- 리드(Lead): 핵심 정보
- 본문: 세부 내용
- 배경: 추가 정보

**5W1H**
- Who (누가)
- What (무엇을)
- When (언제)
- Where (어디서)
- Why (왜)
- How (어떻게)

### 2.2 비판적 뉴스 읽기

**질문하기**
- 정보 출처는?
- 근거는 충분한가?
- 다른 관점은?
- 생략된 정보는?

**사실 vs 의견**
- 사실: 검증 가능
- 의견: 주관적 판단
- 혼재 여부 확인

**객관성 평가**
- 편향된 표현?
- 균형 잡힌 보도?
- 반대 의견 제시?

### 2.3 가짜뉴스 판별

**특징**
- 선정적 제목
- 출처 불명
- 논리 비약
- 감정 자극

**확인 방법**
- 복수 출처 대조
- 팩트체크 사이트
- 원문 확인
- 날짜 확인

## 3. 광고 분석

### 3.1 광고의 전략

**AIDMA 모델**
- Attention (주의)
- Interest (흥미)

- Action (행동)

**설득 기법**
- 권위에의 호소
- 대중에의 호소
- 감정에의 호소
- 반복·과장

### 3.2 숨은 의도 파악

- 브랜드 이미지
- 소비 조장

**표현 분석**

- 긍정 편향

### 3.3 비판적 수용

**질문하기**
- 진짜 필요한가?

- 대안은?

**광고 윤리**

## 10. FAQ

### Q1. 미디어 리터러시가 수능에 나오나요?

**A1.** 네, 직간접적으로 나옵니다.
- 독서: 주장·근거, 논리 분석
- 매체: 뉴스·광고 분석
- 비판적 사고 전반

### Q2. 과외가 필요한가요?

**A2.** 선택사항입니다.
- 독학 가능: 기본 개념, 실습
- 과외 효과: 체계적 훈련, 피드백, 논리 분석

### Q3. 얼마나 공부해야 하나요?

**A3.** 꾸준히 하는 것이 핵심입니다.
- 매일 뉴스 읽기
- 주 1회 심층 분석
- 월 1회 프로젝트

### Q4. 어떤 뉴스를 읽어야 하나요?

**A4.** 다양하게 읽으세요.
- 주요 일간지 3곳
- 다양한 관점
- 팩트체크 기사

### Q5. SNS는 어떻게 활용하나요?

**A5.** 비판적으로 사용하세요.
- 정보 출처 확인
- 다양한 의견 접하기
- 윤리적 댓글

### Q6. 가짜뉴스를 어떻게 구별하나요?

**A6.** 체크리스트를 활용하세요.
- 출처 확인
- 복수 매체 대조
- 팩트체크 사이트
- 감정적 제목 의심

### Q7. 광고를 못 찾겠어요.

**A7.** 다양한 곳에 있습니다.
- TV·라디오
- 온라인 배너
- 유튜브 (협찬)
- PPL (간접 광고)

### Q8. 논리적 오류가 어려워요.

**A8.** 예시로 이해하세요.
- 개념 이해
- 실제 사례 찾기
- 반복 연습

### Q9. 디지털 에티켓이 중요한가요?

**A9.** 매우 중요합니다.
- 사이버 폭력 예방
- 법적 문제 방지
- 타인 존중

### Q10. 미디어 창작이 필요한가요?

**A10.** 선택이지만 도움됩니다.
- 깊은 이해
- 수용자 관점
- 창의성 개발
