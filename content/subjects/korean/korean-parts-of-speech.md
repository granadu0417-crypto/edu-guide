# 국어 품사 완벽 가이드

품사 구분이 어렵나요?

"명사와 대명사 차이가 뭐예요?"
"형용사와 동사 구분을 못하겠어요."
"조사가 너무 헷갈려요."

**품사 체계**를 제대로 이해하면 **모든 국어 문법이 쉬워집니다**.

이 가이드에서는 국어 9품사 완벽 정복법을 소개합니다.

## 1. 국어 품사란?

### 품사의 정의

품사는 **단어를 기능과 형태에 따라 분류**한 것입니다.

**분류 기준**:
- **형태**: 변하는가 (가변어/불변어)
- **기능**: 문장에서 역할
- **의미**: 나타내는 내용

### 9품사 체계

**체언** (3개):
- 명사
- 대명사
- 수사

**용언** (2개):
- 동사
- 형용사

**수식언** (2개):
- 관형사
- 부사

**관계언** (1개):
- 조사

**독립언** (1개):
- 감탄사

## 2. 체언 (명사, 대명사, 수사)

### 명사

**사람, 사물, 개념의 이름**을 나타냅니다.

**종류**:
1. **보통명사**: 학생, 책, 나무
2. **고유명사**: 서울, 한강, 김철수
3. **자립명사**: 하늘, 바다
4. **의존명사**: 것, 바, 수

**특징**:
- 조사와 결합
- 관형어의 수식 받음
- 주어, 목적어, 보어 역할

**예시**:
```
학생이 책을 읽는다.
(명사) (조사) (명사) (조사)
```

1:1 과외에서는 **실전 문장 분석**으로 명사 구분 능력을 키웁니다.

### 대명사

**사람이나 사물을 대신 가리키는** 말입니다.

**종류**:
1. **인칭 대명사**: 나, 너, 그, 우리
2. **지시 대명사**: 이, 그, 저
3. **의문 대명사**: 누구, 무엇, 어디

**특징**:
- 명사를 대신함
- 문맥에 따라 지칭 대상 변화
- 높임 표현 가능 (저, 당신)

### 수사

**수량이나 순서**를 나타냅니다.

**종류**:
1. **양수사**: 하나, 둘, 셋
2. **서수사**: 첫째, 둘째, 셋째

**특징**:
- 관형사로 쓰이기도 함
- 단위 명사와 결합

## 3. 용언 (동사, 형용사)

### 동사

**움직임이나 작용**을 나타냅니다.

**특징**:
- 어미 변화 (활용)
- 시제 표현 가능
- '-ㄴ다, -는다' 종결
- '-(으)ㄴ' 관형사형 불가

**예시**:
```
먹다 → 먹는다 (O)
      → 먹은 음식 (O)
      → 먹는 음식 (O)
```

**자동사 vs 타동사**:
- 자동사: 목적어 불필요 (가다, 오다)
- 타동사: 목적어 필요 (먹다, 읽다)

1:1 과외로 동사 활용 패턴을 완벽하게 익힐 수 있습니다.

### 형용사

**성질이나 상태**를 나타냅니다.

**특징**:
- 어미 변화 (활용)
- 시제 표현 불가
- '-다' 종결
- '-(으)ㄴ' 관형사형 가능

**예시**:
```
예쁘다 → 예쁘다 (O)
       → 예쁜 꽃 (O)
       → 예쁘는 꽃 (X)
```

**동사와 형용사 구분법**:
1. 현재 관형사형: -는 (동사), -(으)ㄴ (형용사)
2. 명령형 가능 여부: 가능(동사), 불가능(형용사)
3. 진행형: 가능(동사), 불가능(형용사)

## 4. 수식언 (관형사, 부사)

### 관형사

**체언을 꾸며주는** 말입니다.

**종류**:
1. **성상 관형사**: 새, 헌, 옛
2. **지시 관형사**: 이, 그, 저
3. **수 관형사**: 한, 두, 세

**특징**:
- 형태 불변
- 체언 앞에만 위치
- 조사 결합 불가

**관형사 vs 관형어**:
- 관형사: 품사 (불변어)
- 관형어: 문장성분 (관형사, 관형사형 어미 등)

### 부사

**용언, 다른 부사, 문장 전체를 꾸며주는** 말입니다.

**종류**:
1. **성분 부사**: 빨리, 매우, 자주
2. **문장 부사**: 과연, 설마, 제발

**특징**:
- 형태 불변
- '-게, -이, -히' 파생
- 위치 비교적 자유

**예시**:
```
매우 빨리 달린다.
(부사) (부사) (동사)
```

1:1 과외에서는 **부사 위치 변화**에 따른 의미 차이를 분석합니다.

## 5. 조사

**체언 뒤에 붙어 관계를 나타내거나 의미를 더하는** 말입니다.

### 격조사

문장 성분 관계를 표시합니다.

**종류**:
- **주격**: -이/가, -께서
- **목적격**: -을/를
- **관형격**: -의
- **부사격**: -에, -에서, -로
- **서술격**: -이다
- **호격**: -아/야, -여

### 보조사

특별한 의미를 더합니다.

**종류**:
- **는/은**: 대조, 주제
- **도**: 포함
- **만**: 한정
- **부터**: 기점
- **까지**: 종점
- **조차**: 극단적 포함

### 접속조사

**단어를 연결**합니다.
- -와/과, -하고, -(이)랑

## 6. 감탄사

**감정, 부름, 대답**을 나타냅니다.

**종류**:
1. **감동사**: 아!, 어머!, 이런!
2. **호격조사**: 여보세요, 얘들아
3. **응답사**: 네, 아니오, 예

**특징**:
- 독립적 사용
- 문장 성분 아님
- 형태 불변

## 7. 품사 구분 실전 전략

### 구분 순서

1. **형태 변화 여부 확인**
   - 변함: 용언 (동사/형용사)
   - 불변: 체언, 수식언, 조사, 감탄사

2. **체언 구분**
   - 조사 결합 여부
   - 의미 파악 (이름/대신/수량)

3. **용언 구분**
   - 현재 관형사형 (-는/-ㄴ)
   - 명령형 가능 여부

4. **수식언 구분**
   - 꾸며주는 대상 (체언/용언)

### 헷갈리기 쉬운 경우

**"한" 구분**:
```
한 개 (관형사)
한 사람 (관형사)
한국 (명사의 일부)
```

**"이" 구분**:
```
이 책 (관형사)
이가 좋다 (대명사)
학생이 (조사)
```

**"잘" 구분**:
```
잘 먹다 (부사)
잘을 자다 (명사 - 잠을 자다의 준말)
```

1:1 과외로 **헷갈리는 품사 구분**을 완벽하게 익힐 수 있습니다.

## 8. 품사 통용

하나의 단어가 **여러 품사로 쓰이는** 경우입니다.

### 관형사 ↔ 명사

```
새 옷 (관형사)
새가 날다 (명사)
```

### 관형사 ↔ 수사

```
한 개 (관형사)
하나 (수사)
```

### 명사 ↔ 부사

```
집 (명사)
집집마다 (부사)
```

## 10. 시험 대비 전략

### 기본 문제 유형

**유형1: 품사 고르기**
```
다음 중 부사는?
① 매우 ② 예쁜 ③ 학생 ④ 읽다
```

**유형2: 밑줄 친 단어 품사**
```
"그가 새 옷을 입었다"
→ "새": 관형사
```

**유형3: 같은 품사 찾기**
```
"빨리"와 품사가 같은 것?
① 높이 ② 높은 ③ 높다 ④ 높음
→ ①
```

### 고난도 문제 대비

**품사 통용**:
```
"한"의 품사가 다른 것?
① 한 개 (관형사)
② 한 명 (관형사)
③ 대한민국 (명사 일부)
④ 한 번 (관형사)
→ ③
```

**문맥적 판단**:
```
"이것이 중요하다"의 "이것"
→ 대명사 (지시대명사)

"이 물건이 중요하다"의 "이"
→ 관형사 (지시관형사)
```

### 실수 방지법

1. **조사 구분**: 격조사 vs 보조사
2. **관형사 vs 수사**: 단독/수식 여부
3. **동사 vs 형용사**: 관형사형 확인
4. **명사 vs 대명사**: 지칭 대상 확인

## 11. 품사별 학습 포인트

### 명사

- **의존명사 암기**: 것, 바, 수, 줄, 리, 듯
- **단위명사**: 개, 마리, 송이, 자루
- **고유명사 띄어쓰기**

### 동사/형용사

- **불규칙 활용**: ㅂ, ㄷ, ㅅ, 르, 우 불규칙
- **보조용언**: -아/어 있다, -아/어 하다
- **사동/피동**: -이-, -히-, -리-, -기-

### 조사

- **은/는 vs 이/가**: 주제 vs 주격
- **격조사 생략**: 구어체에서 자주 발생
- **복합조사**: -에게서, -로부터

### 부사

- **'-게' 부사**: 높게, 빠르게
- **'-이/-히' 부사**: 같이, 깨끗이
- **문장부사**: 과연, 설마, 제발

1:1 과외로 **품사별 핵심 포인트**를 효율적으로 학습할 수 있습니다.

## 12. 품사와 문장성분

### 품사 ≠ 문장성분

**품사**: 단어의 분류
**문장성분**: 문장에서의 역할

### 대응 관계

| 품사 | 주요 문장성분 |
|------|--------------|
| 체언 | 주어, 목적어, 보어 |
| 용언 | 서술어 |
| 관형사 | 관형어 |
| 부사 | 부사어 |
| 조사 | - |
| 감탄사 | 독립어 |

**주의사항**:
- 명사도 부사어 가능 (시간명사)
- 동사/형용사가 관형어, 부사어, 명사절 가능

## 자주 묻는 질문 (FAQ)

### Q1. 품사는 왜 배워야 하나요?

품사는 **국어 문법의 기초**입니다. 품사를 알아야 문장 구조를 이해하고, 올바른 문장을 쓸 수 있습니다. 또한 수능, 내신 시험에서 품사 관련 문제가 자주 출제됩니다.

### Q2. 동사와 형용사 구분이 어려워요.

**현재 관형사형**으로 구분하세요. 동사는 '-는'(먹는, 가는), 형용사는 '-(으)ㄴ'(예쁜, 좋은)입니다. 또한 명령형이 가능하면 동사, 불가능하면 형용사입니다.

### Q3. 관형사와 관형어의 차이는?

**관형사**는 품사(단어의 종류), **관형어**는 문장성분(문장에서의 역할)입니다. 관형사는 항상 관형어가 되지만, 관형어는 관형사 외에도 관형사형 어미, 명사 등이 될 수 있습니다.

### Q4. 조사는 어떻게 암기하나요?

**격조사**는 주격(이/가), 목적격(을/를), 관형격(의) 등으로 분류해서 암기하세요. **보조사**는 '는/은, 도, 만' 등 의미별로 묶어서 외우면 효과적입니다.

### Q5. 수사와 관형사는 어떻게 구분하나요?

**단독 사용 가능 여부**로 구분합니다. 수사는 "하나, 둘, 셋"처럼 단독 사용이 가능하고, 관형사 "한, 두, 세"는 반드시 체언을 꾸며야 합니다.

### Q6. 품사 통용이란?

하나의 단어가 **여러 품사로 쓰이는 것**입니다. 예: "새"는 "새 옷"(관형사), "새가 날다"(명사)처럼 문맥에 따라 품사가 달라집니다.

### Q7. 의존명사는 무엇인가요?

**조사 없이 홀로 쓰일 수 없는 명사**입니다. "것, 바, 수, 줄, 리, 듯" 등이 있으며, 반드시 관형어의 수식을 받아야 합니다. 예: "아는 것", "먹을 수", "갈 리".

### Q8. 품사 공부는 얼마나 걸리나요?

**기본 개념**은 1-2주, **완벽한 숙달**은 1-2개월 정도 걸립니다. 1:1 과외를 통해 학습하면 더욱 빠르고 효율적으로 마스터할 수 있습니다.

### Q9. 수능에서 품사가 중요한가요?

네, **매우 중요**합니다. 문법 문제의 기초가 되며, 품사를 정확히 알아야 문장 구조 분석, 어법 판단 등이 가능합니다. 매년 품사 관련 문제가 출제됩니다.

### Q10. 1:1 과외가 품사 학습에 도움이 되나요?

**매우 도움이 됩니다**. 품사는 헷갈리기 쉬운 개념이 많아 개인별 맞춤 설명이 필요합니다. 1:1 과외로 약점을 집중 보완하고, 실시간 질의응답으로 완벽히 이해할 수 있습니다.
