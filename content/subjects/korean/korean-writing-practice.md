**"글쓰기가 어려워요!"** 체계적 훈련으로 누구나 잘 쓸 수 있습니다.

글쓰기는 **생각을 표현하는 힘**이며, **꾸준한 연습**으로 향상됩니다.

## 1. 글쓰기란?

### 1.1 글쓰기의 의미

**정의**
- 생각을 글로 표현
- 독자에게 전달
- 의사소통의 도구

**중요성**
- 사고력 발달
- 의사 표현 능력
- 학습의 기초
- 평생 활용

### 1.2 좋은 글의 조건

**명확성**
- 주제가 분명
- 논리적 전개
- 이해하기 쉬움

**정확성**
- 맞춤법 정확
- 적절한 어휘
- 문법에 맞음

**풍부성**
- 다양한 표현
- 구체적 묘사
- 생동감

**독창성**
- 자신만의 생각
- 참신한 표현
- 개성

## 2. 글쓰기의 과정

### 2.1 주제 정하기

**방법**
- 관심사 탐색
- 경험 떠올리기
- 브레인스토밍
- 독서에서 발견

**좋은 주제**
- 구체적
- 자신만의 관점
- 독자 관심
- 충분한 자료

### 2.2 내용 구상하기

**마인드맵**
- 중심 주제
- 가지 뻗기
- 자유로운 연상

**개요 작성**
- 서론-본론-결론
- 문단별 주제
- 논리적 순서

**자료 수집**
- 책, 인터넷
- 관찰, 경험
- 메모 정리

### 2.3 초고 쓰기

**일단 쓰기**
- 완벽 추구하지 않기
- 생각나는 대로
- 일단 끝까지

**문단 나누기**
- 하나의 문단 = 하나의 생각
- 적절한 길이
- 연결 고려

**표현하기**
- 다양한 어휘
- 비유, 묘사
- 구체적으로

### 2.4 고쳐 쓰기

**내용 검토**
- 주제에 맞는가?
- 논리적인가?
- 근거 충분한가?

**표현 다듬기**
- 불필요한 말 삭제
- 어색한 표현 수정
- 더 좋은 어휘로

**맞춤법·문법**
- 철자 확인
- 문장 부호
- 문법 오류 수정

### 2.5 완성·발표

**최종 검토**
- 소리 내어 읽기
- 타인에게 보이기
- 피드백 받기

**정서**
- 깔끔하게 정리
- 적절한 형식
- 제출·발표

## 3. 문장 쓰기

### 3.1 문장의 기본

**주어와 서술어**
- 주어: 누가, 무엇이
- 서술어: 어떻게 하다
- 호응 관계

**문장 성분**
- 주성분: 주어, 서술어, 목적어, 보어
- 부속성분: 관형어, 부사어
- 독립성분: 독립어

**문장 종류**
- 평서문, 의문문
- 명령문, 청유문
- 감탄문

### 3.2 좋은 문장 쓰기

**간결하게**
- 불필요한 말 빼기
- 한 문장 하나의 생각
- 짧게

**명확하게**
- 주어-서술어 호응
- 지시어 명확히
- 중의적 표현 피하기

**자연스럽게**
- 구어체와 문어체
- 문장 길이 변화
- 리듬감

### 3.3 다양한 문장

**단문과 복문**
- 단문: 짧고 간결
- 복문: 풍부하고 세밀

**능동태와 피동태**
- 능동: 직접적, 명확
- 피동: 객관적, 부드러움

**긍정과 부정**
- 긍정: 적극적
- 부정: 강조

## 4. 문단 구성

### 4.1 문단이란?

**정의**
- 하나의 주제
- 여러 문장의 묶음
- 단락

**구조**
- 주제문
- 뒷받침 문장
- 마무리 문장

### 4.2 문단 나누기

**기준**
- 화제 바뀔 때
- 시간·장소 변할 때
- 적절한 길이

**연결**
- 접속어 활용
- 지시어로 연결
- 의미상 연결

### 4.3 문단 전개 방식

**시간 순서**
- 처음→나중
- 시계열

**공간 순서**
- 위→아래
- 가까운 곳→먼 곳

**인과 관계**
- 원인→결과
- 문제→해결

**일반→구체**
- 개괄→세부
- 주장→근거

**비교·대조**
- 공통점·차이점
- A와 B

## 5. 글의 구성

### 5.1 처음-중간-끝

**서론 (처음)**
- 주제 제시
- 독자 관심 유도
- 간결하게

**본론 (중간)**
- 주제 전개
- 근거 제시
- 가장 길게

**결론 (끝)**
- 요약·정리
- 주제 강조
- 여운 남기기

### 5.2 서론 쓰기

**방법**
- 질문으로 시작
- 인용으로 시작
- 일화로 시작
- 직접 제시

**주의**
- 너무 길지 않게
- 주제와 관련
- 흥미롭게

### 5.3 본론 쓰기

**전개 방법**
- 주장→근거
- 분류하여 설명
- 비교·대조
- 인과 관계

**풍부하게**
- 예시, 인용
- 통계, 자료
- 경험, 관찰

### 5.4 결론 쓰기

**방법**
- 요약·정리
- 주제 재강조
- 제안·촉구
- 전망 제시

**주의**
- 새로운 내용 넣지 않기
- 본론과 어긋나지 않게
- 인상적으로

## FAQ

### Q1. 글쓰기를 어떻게 시작하나요?

**A1.** 작은 것부터 시작하세요.
- 매일 일기 쓰기
- 한 문단부터
- 완벽 추구하지 않기
→ 꾸준함이 중요

### Q2. 글이 너무 짧아요.

**A2.** 구체적으로 쓰세요.
- 육하원칙 활용
- 예시 추가
- 감정, 생각 서술
- 묘사 풍부하게

### Q3. 어휘가 부족해요.

**A3.** 독서와 메모가 답입니다.
- 다양한 책 읽기
- 좋은 표현 메모
- 사전 활용
- 실제로 사용

### Q4. 맞춤법이 어려워요.

**A4.** 원리를 이해하고 연습하세요.
- 기본 규칙 학습
- 자주 틀리는 것 정리
- 맞춤법 검사기 활용
- 많이 읽고 쓰기

### Q5. 글의 주제를 어떻게 정하나요?

**A5.** 가까운 곳에서 찾으세요.
- 일상 경험
- 관심사
- 독서 내용
- 궁금한 것

### Q6. 서론을 어떻게 쓰나요?

**A6.** 독자의 관심을 끄세요.
- 질문으로 시작
- 재미있는 일화
- 인상적인 문장
- 주제 직접 제시

### Q7. 첨삭을 어떻게 활용하나요?

**A7.** 적극적으로 반영하세요.
1. 첨삭 내용 꼼꼼히 읽기
2. 왜 그런지 이해
3. 다시 쓰기
4. 비교하며 배우기

### Q8. 논설문과 설명문의 차이는?

**A8.** 목적이 다릅니다.
- 설명문: 정보 전달, 객관적
- 논설문: 주장 전달, 설득
- 설명문 → 이해
- 논설문 → 동의

### Q9. 글쓰기 과외가 필요한가요?

**A9.** 체계적 향상에 효과적입니다.
- 개별 첨삭
- 맞춤 지도
- 지속적 피드백
- 동기 부여

### Q10. 얼마나 써야 실력이 늘까요?

**A10.** 매일 조금씩이 중요합니다.
- 매일 쓰기 습관
- 최소 30분
- 다양한 주제
- 고쳐쓰기까지

## 결론

글쓰기는 **표현의 힘**이며, **생각의 도구**입니다.
