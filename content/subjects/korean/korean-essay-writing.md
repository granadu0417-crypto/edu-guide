# 국어 논술 완벽 마스터하기

논술이 막막하게 느껴지나요? 무엇을 써야 할지, 어떻게 논리적으로 전개해야 할지 모르겠나요? 논술은 체계적 훈련과 꾸준한 첨삭으로 반드시 향상됩니다. 1:1 맞춤 논술 과외로 논리 구성부터 표현력까지 완벽하게 키워보세요.

## 왜 논술 학습이 중요한가?

### 사고력의 집합체

논술은 단순히 글을 쓰는 것이 아니라, 주제를 분석하고, 논리를 구성하며, 설득력 있게 표현하는 종합적 사고력의 결과물입니다.

### 대입과 내신

- **논술 전형**: 일부 대학은 논술 전형으로 신입생 선발
- **수능 국어**: 서술형 문제 대비
- **내신 서술형**: 중·고등학교 국어 서술형 문제 비중 증가

### 실생활 응용

- **보고서 작성**: 학업, 직장에서 필수
- **자기소개서**: 입학, 취업 시 필요
- **비판적 사고**: 정보를 분석하고 판단하는 능력

### 타 과목 연계

논술 능력은 모든 과목의 서술형 문제, 보고서 작성에 도움이 됩니다.

## 1:1 논술 과외의 핵심 장점

### 개인 맞춤 첨삭

학생마다 약점이 다릅니다. 논리 구조가 약한지, 표현력이 부족한지, 맞춤법 오류가 많은지 정확히 진단하고 집중 보완합니다.

### 즉각적 피드백

- **구조적 문제**: 서론-본론-결론 흐름 개선
- **논리적 비약**: 근거와 주장의 연결 강화
- **표현의 모호함**: 명확하고 간결한 표현으로 수정
- **어휘 선택**: 더 적절하고 세련된 어휘 제안

### 체계적 훈련

- **1단계**: 짧은 글 (200~300자) 작성
- **2단계**: 중급 논술 (500~800자)
- **3단계**: 고급 논술 (1000자 이상)
- **4단계**: 복합 주제 논술

### 반복 연습과 확인

작성 → 첨삭 → 수정 → 재제출의 반복으로 실력이 빠르게 향상됩니다.

## 논술의 3요소

### 1. 내용 (Content)

**무엇을 쓸 것인가**

**주제 파악**:
- 제시문의 핵심 이해
- 논점 정확히 파악
- 요구하는 바 명확히 인식

**논거 마련**:
- 타당한 근거 수집
- 구체적 예시 준비
- 통계, 사례, 인용 활용

**깊이 있는 사고**:
- 표면적 이해를 넘어 본질 파악
- 다양한 관점 고려
- 창의적 시각 제시

### 2. 구조 (Structure)

**어떻게 전개할 것인가**

**기본 구조**:
```
서론 (10~15%)
  - 문제 제기 또는 배경 설명
  - 주제 제시
  - 글의 방향 예고

본론 (70~80%)
  - 주장 1 + 근거 + 예시
  - 주장 2 + 근거 + 예시
  - (필요시) 반론 제시 및 재반박

결론 (10~15%)
  - 요약 정리
  - 주장 재확인
  - 마무리 (제언, 전망)
```

**논리적 흐름**:
- 일관성: 서론의 주제가 본론과 결론까지 이어짐
- 연결성: 문단 간 자연스러운 전환
- 위계성: 중요도에 따른 배치

### 3. 표현 (Expression)

**어떻게 표현할 것인가**

**명확성**:
- 간결한 문장
- 모호함 없는 표현
- 불필요한 수식 제거

**설득력**:
- 구체적 어휘 사용
- 논리적 접속어 활용
- 비유와 인용의 적절한 활용

**품격**:
- 정확한 맞춤법과 띄어쓰기
- 적절한 어휘 수준
- 문장의 다양성

## 논술 유형별 전략

### 설명형 논술

**목표**: 개념, 현상, 원리를 명확히 설명

**작성법**:
1. **정의 제시**: 용어나 개념을 명확히 정의
2. **특징 설명**: 주요 특성과 요소 설명
3. **원인과 결과**: 왜 그런지, 어떤 결과를 낳는지
4. **구체적 사례**: 이해를 돕는 실례 제시
5. **종합 정리**: 핵심 내용 요약

**예시 주제**:
- "민주주의의 개념과 특징을 설명하시오"
- "지구 온난화의 원인과 영향을 서술하시오"

### 논증형 논술

**목표**: 주장을 논리적으로 입증

**구성**:
1. **명확한 주장**: 논제에 대한 자신의 입장 제시
2. **근거 1**: 주장을 뒷받침하는 첫 번째 이유
   - 논리적 설명
   - 구체적 예시 (사례, 통계, 인용)
3. **근거 2**: 두 번째 이유
4. **근거 3**: (필요시) 세 번째 이유
5. **반론 예상 및 재반박**: (선택) 반대 입장 제시 후 반박
6. **결론**: 주장 재확인

**예시 주제**:
- "사형 제도는 폐지되어야 하는가"
- "온라인 수업이 대면 수업보다 효과적인가"

### 비교·대조형 논술

**목표**: 두 대상의 공통점과 차이점 분석

**전략**:
1. **비교 기준 설정**: 어떤 관점에서 비교할 것인가
2. **체계적 비교**:
   - 기준 1에서 A와 B 비교
   - 기준 2에서 A와 B 비교
   - 기준 3에서 A와 B 비교
3. **균형 잡힌 분석**: 한쪽에 치우치지 않고 객관적으로
4. **종합적 평가**: 비교 결과를 종합하여 평가

**예시 주제**:
- "도시와 농촌의 생활 방식을 비교하시오"
- "전통 교육과 현대 교육의 차이를 논하시오"

### 문제 해결형 논술

**목표**: 문제 상황을 분석하고 해결책 제시

**단계**:
1. **문제 인식**: 현상과 문제점 파악
2. **원인 분석**: 왜 이런 문제가 발생했는가
3. **해결책 제시**:
   - 해결 방안 1 + 효과 및 실현 가능성
   - 해결 방안 2 + 효과 및 실현 가능성
4. **예상 효과**: 해결책 실행 시 기대되는 변화
5. **결론**: 종합적 제언

**예시 주제**:
- "청소년 스마트폰 중독 문제와 해결 방안"
- "저출산 문제 해결을 위한 정책 제안"

## 주제 분석 방법

### 제시문 정독

**1차 읽기**: 전체 내용 파악
- 어떤 주제인가?
- 어떤 정보가 제공되는가?

**2차 읽기**: 세부 분석
- 핵심 논점은?
- 제시된 근거와 예시는?
- 상반된 관점이 있는가?

### 논제 파악

**요구 사항 확인**:
- "설명하시오" → 설명형
- "논하시오" → 논증형
- "비교하시오" → 비교대조형
- "해결 방안을 제시하시오" → 문제해결형

**제한 사항 확인**:
- 글자 수 제한
- 시간 제한
- 특정 내용 포함 여부

### 브레인스토밍

**자유로운 발상**:
- 주제와 관련된 모든 생각 나열
- 떠오르는 예시, 사례, 통계 메모
- 찬성/반대 의견 모두 정리

**선택과 집중**:
- 가장 타당한 논거 선택
- 구체적이고 설득력 있는 예시 선별

## 개요 작성법

### 개요의 중요성

개요 없이 바로 쓰면 논리가 흐트러지고 방향을 잃기 쉽습니다. 개요는 논술의 설계도입니다.

### 개요 작성 단계

**1. 주장 명확화**:
- 나의 입장은?
- 핵심 주장 한 문장으로 정리

**2. 근거 나열**:
- 주장을 뒷받침하는 이유 3가지
- 우선순위 매기기

**3. 예시 배치**:
- 각 근거마다 구체적 예시 1~2개

**4. 구조 확정**:
```
서론: 문제 제기 + 주장
본론 1: 근거 1 + 예시
본론 2: 근거 2 + 예시
본론 3: 근거 3 + 예시
결론: 요약 + 재확인
```

## 서론-본론-결론 작성법

### 서론 작성법

**역할**:
- 독자의 관심 유발
- 주제 소개
- 글의 방향 제시

**작성 기법**:
- **문제 제기**: "최근 △△ 현상이 사회적 문제로 대두되고 있다"
- **질문**: "과연 ○○는 바람직한가?"
- **일화**: 구체적 사례로 시작
- **통계**: 충격적 수치 제시
- **인용**: 유명한 말이나 속담

**주의 사항**:
- 너무 길지 않게 (전체의 10~15%)
- 주제에서 벗어나지 않기

### 본론 작성법

**역할**:
- 주장의 타당성 입증
- 논리적 근거 제시
- 구체적 예시로 뒷받침

**작성 기법**:

**문단 구성 (PEEL 기법)**:
- **P**oint: 문단의 핵심 주장
- **E**vidence: 근거 (논리, 사실)
- **E**xample: 구체적 예시
- **L**ink: 다음 문단 또는 주제로 연결

**예시**:
```
(P) 첫째, 독서는 사고력을 향상시킨다.
(E) 책을 읽으며 저자의 논리를 따라가고 비판적으로 사고하는 과정에서 논리력이 길러진다.
(E) 실제로 하버드 대학 연구에 따르면 독서량이 많은 학생일수록 논리적 추론 능력이 높다고 한다.
(L) 이러한 사고력 향상은 학업 성취도로도 이어진다.
```

**접속어 활용**:
- 순서: 첫째, 둘째, 셋째 / 먼저, 다음으로, 마지막으로
- 인과: 따라서, 그러므로, 그 결과
- 부연: 즉, 다시 말해, 예를 들어
- 대조: 그러나, 반면에, 하지만
- 추가: 또한, 게다가, 뿐만 아니라

### 결론 작성법

**역할**:
- 논의 내용 요약
- 주장 재확인
- 마무리 인상 남기기

**작성 기법**:
- **요약 정리**: 본론의 핵심 내용 간략히 요약
- **주장 강조**: 주장을 다시 한 번 명확히
- **제언**: 앞으로 나아갈 방향 제시
- **전망**: 미래 예측
- **여운**: 독자에게 생각거리 제공

**주의 사항**:
- 새로운 내용 추가하지 않기
- 갑작스러운 방향 전환 금지
- 너무 길지 않게 (전체의 10~15%)

## 논리적 전개 방법

### 인과 관계 명확화

**원인 → 결과**의 연결이 논리적이어야 합니다.

**나쁜 예**:
- "게임을 하면 성적이 떨어진다. 따라서 게임은 나쁘다."
  (논리 비약, 다른 요인 무시)

**좋은 예**:
- "과도한 게임은 학습 시간을 감소시키고, 집중력을 저하시킨다. 연구에 따르면 하루 3시간 이상 게임을 하는 학생의 평균 성적이 그렇지 않은 학생보다 20% 낮다. 따라서 게임 시간 조절이 필요하다."

### 반론 제시와 재반박

**설득력 강화**:
- 예상되는 반론을 먼저 제시
- 이에 대한 재반박으로 주장 강화

**예시**:
```
(주장) 학교에서 스마트폰 사용을 금지해야 한다.
(근거) 스마트폰은 수업 집중을 방해하고 사이버 불링의 도구가 된다.
(반론) 일각에서는 스마트폰이 학습 도구로 활용될 수 있다고 주장한다.
(재반박) 그러나 실제 교실에서 스마트폰의 학습 활용 비율은 10% 미만이며, 대부분 SNS와 게임에 사용된다는 조사 결과가 있다. 따라서 학습용 태블릿 등 대체 수단을 제공하되, 개인 스마트폰 사용은 제한하는 것이 타당하다.
```

### 구체성 확보

**추상적 표현 → 구체적 표현**

**나쁜 예**:
- "환경 문제가 심각하다."

**좋은 예**:
- "2023년 서울의 미세먼지 나쁨 일수는 연간 80일을 넘었고, 플라스틱 쓰레기는 하루 평균 5,000톤이 발생한다."

## 표현력 향상 기법

### 어휘력 확장

**동의어 활용**:
- "중요하다" → 핵심적이다, 필수적이다, 본질적이다, 긴요하다
- "좋다" → 바람직하다, 긍정적이다, 유익하다, 효과적이다

**전문 용어 적절히 사용**:
- 남용하지 말되, 필요시 정확히 사용

### 문장 구조 다양화

**단문과 복문의 조화**:
- 단문: 강조하고 싶을 때
- 복문: 인과 관계나 조건 설명 시

**예시**:
- 단문: "독서는 중요하다."
- 복문: "독서는 지식을 쌓고 사고력을 키우며 정서적 안정을 주기 때문에 중요하다."

### 비유와 인용

**비유**:
- 추상적 개념을 구체적으로 설명
- "시간은 돈과 같다" (은유)
- "마치 ○○처럼" (직유)

**인용**:
- 권위 있는 인물이나 문헌 인용
- "소크라테스는 '무지의 자각'이 지혜의 시작이라 했다."

### 맞춤법과 띄어쓰기

**자주 틀리는 맞춤법**:
- 되/돼: "하게 되다" (O) / "하게 돼다" (X)
- -던/-든: 과거 회상 "-던" / 선택 "-든"
- 어떻게/어떡해: "어떻게 해야 하나" (O) / "어떡해" (구어)

**띄어쓰기 원칙**:
- 조사는 붙여 쓰기: "책을" "집에서"
- 의존 명사는 띄어 쓰기: "할 수 있다" "것이다"

## 첨삭의 중요성

### 자가 첨삭

**글 쓴 후 시간 두기**:
- 최소 30분~1시간 후 다시 읽기
- 객관적 시각 확보

**체크리스트**:
- [ ] 주제에서 벗어나지 않았는가?
- [ ] 논리적 비약은 없는가?
- [ ] 근거가 충분한가?
- [ ] 예시가 구체적인가?
- [ ] 맞춤법과 띄어쓰기는 정확한가?

### 전문가 첨삭

**1:1 과외 첨삭의 장점**:
- 학생이 보지 못한 문제점 지적
- 더 나은 표현 방법 제시
- 반복되는 오류 패턴 파악
- 즉각적 질의응답

**첨삭 후 수정**:
- 첨삭 내용을 반영하여 재작성
- 같은 실수 반복하지 않기
- 개선된 표현 내재화

## 자주 묻는 질문 (FAQ)

### 1. 논술은 언제부터 시작해야 하나요?

초등학교 고학년부터 간단한 글쓰기로 시작하고, 중학교 1학년부터 본격적인 논술 훈련을 시작하는 것이 좋습니다. 고등학교 가서 갑자기 시작하면 부담이 큽니다.

### 2. 논술 실력이 늘려면 얼마나 걸리나요?

주 1~2회 꾸준히 작성하고 첨삭 받으면 3~6개월 내 확실한 변화를 느낄 수 있습니다. 하지만 고급 논술까지는 1~2년의 꾸준한 훈련이 필요합니다.

### 3. 첨삭은 어떻게 받는 것이 좋나요?

1:1 과외가 가장 효과적입니다. 선생님이 직접 설명하며 첨삭해주므로 이해도가 높고, 즉각적으로 질문할 수 있어 빠른 향상이 가능합니다.

### 4. 논술 주제는 어디서 구하나요?

교과서 단원 마무리 문제, 논술 문제집, 신문 칼럼 주제, 시사 이슈 등을 활용할 수 있습니다. 과외 선생님이 수준에 맞는 주제를 제공하기도 합니다.

### 5. 논술과 독서의 관계는?

독서는 논술의 기초입니다. 배경 지식, 어휘력, 표현 기법을 독서로 쌓고, 논술로 표현하는 것이 이상적입니다. 논술 훈련과 함께 꾸준한 독서를 병행하세요.

### 6. 글자 수가 부족하거나 초과하면 어떡하나요?

개요 작성 단계에서 분량을 조절합니다. 부족하면 예시를 추가하고, 초과하면 덜 중요한 내용을 삭제합니다. 연습을 통해 분량 감각을 익히세요.

### 7. 논술에서 가장 중요한 것은 무엇인가요?

논리성입니다. 아무리 화려한 표현도 논리가 없으면 설득력이 없습니다. 주장-근거-예시의 연결이 명확하고 일관되어야 합니다.

### 8. 창의적인 내용을 쓰려면?

다양한 독서와 사고 훈련이 필요합니다. 한 주제를 여러 관점에서 생각해보고, 다른 사람의 의견도 들어보세요. 브레인스토밍도 도움이 됩니다.

### 9. 시험 시간 내에 논술을 완성하는 요령은?

시간 배분이 중요합니다. 전체 시간을 분석(20%) - 개요(20%) - 작성(50%) - 검토(10%)로 나눕니다. 평소 연습 시 타이머를 사용하여 시간 감각을 익히세요.

### 10. 과외에서는 주로 무엇을 배우나요?

주제 분석, 개요 작성, 논리적 전개, 표현력 향상, 첨삭 및 피드백 등을 학생의 수준과 목표에 맞춰 진행합니다. 정기적으로 논술을 작성하고 첨삭 받는 과정이 핵심입니다.
