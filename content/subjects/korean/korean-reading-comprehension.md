- 의미 해석
- 비판적 평가

**왜 중요한가?**
- 모든 과목 학습의 기본
- 시험 문제 이해
- 정보 습득
- 사고력 발달
- 의사소통

### 1.2 좋은 독해력

**정확성**

**속도**
- 적절한 속도
- 효율적 읽기

### 1.3 독해가 어려운 이유

**어휘 부족**
- 모르는 단어

- 생소한 주제

## 2. 문장 이해

### 2.1 문장 성분

**주성분**
- 주어: 누가, 무엇이
- 서술어: 어떻게 하다, 어떠하다
- 목적어: 무엇을
- 보어: 무엇이 되다

**부속 성분**
- 관형어: 어떤 (명사 꾸밈)
- 부사어: 어떻게 (동사·형용사 꾸밈)

**독립 성분**
- 독립어: 감탄사 등

### 2.2 문장 구조 파악

**단문**
- 주어 + 서술어
- 간단명료

**복문**
- 안은문장: 명사절, 관형절, 부사절, 서술절, 인용절
- 이어진문장: 대등, 종속

**긴 문장 읽기**
- 성분 구분
- 수식 관계
- 핵심 찾기

### 2.3 문맥으로 어휘 추론

**단서 찾기**
- 앞뒤 문장
- 예시
- 대조

**유추**
- 비슷한 말
- 반대말
- 상위어/하위어

## 4. 글 전체 이해

### 4.1 글의 구조

**서론**
- 화제 제시
- 문제 제기

**본론**
- 본격 전개
- 여러 문단

**결론**
- 요약
- 강조
- 제언

### 4.2 주제 파악

**주제란?**
- 글 전체의 중심 생각
- 필자가 말하고자 하는 것

**찾는 방법**
- 반복되는 내용
- 강조하는 부분
- 결론 부분
- 제목

**주제와 소재**
- 소재: 무엇에 대한 글인가 (대상)
- 주제: 그것에 대해 무엇을 말하는가 (생각)

### 4.3 글의 전개 방식

**두괄식**
- 주제 → 뒷받침
- 명확

**미괄식**
- 뒷받침 → 주제
- 설득적

**양괄식**
- 주제 → 뒷받침 → 주제
- 강조

**중괄식**
- 뒷받침 → 주제 → 뒷받침
- 균형
