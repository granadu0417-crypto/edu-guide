## 1. 생물 학습의 중요성

### 1.1 왜 생물인가

**생명을 이해하는 학문**

생물의 핵심:
- **세포**: 생명의 기본 단위
- **유전**: 형질의 전달
- **진화**: 생명의 다양성
- **생태계**: 생물과 환경의 관계

**실생활 응용**
- 의학: 질병 치료
- 농업: 품종 개량
- 환경: 생태계 보전
- 생명공학: 유전자 조작

### 1.2 대학 진학과의 연결

**필수 선수 과목**

| 전공 | 필요한 생물 | 난이도 |
|------|-------------|--------|
| 의학·치의학 | 고급 생물, 생리학 | ★★★ |
| 생명과학 | 분자생물학, 유전학 | ★★★ |
| 간호·보건 | 기초 생물, 인체 생리 | ★★☆ |
| 환경·농업 | 생태학, 식물학 | ★★☆ |

### 1.3 수능에서의 생물

**공통 과목 + 선택 과목**

| 과목 | 내용 | 비중 |
|------|------|------|
| 통합과학 | 생명과학 기초 | 25% |
| 생명과학I | 세포·유전·생태 | 100% (선택) |
| 생명과학II | 심화 생물 | 100% (선택) |

## 2. 세포 (Cell)

### 2.1 세포의 발견

**세포설 (Cell Theory)**

1. 모든 생물은 세포로 구성
2. 세포는 생명의 기본 단위
3. 모든 세포는 세포로부터

**발견 역사**
- 1665년 로버트 훅: 코르크 세포 관찰
- 1838년 슐라이덴: 식물세포설
- 1839년 슈반: 동물세포설

### 2.2 세포의 구조

**원핵세포 vs 진핵세포**

| 구분 | 원핵세포 | 진핵세포 |
|------|----------|----------|
| 핵 | 핵막 없음 | 핵막 있음 |
| 크기 | 작음 (1-10μm) | 큼 (10-100μm) |
| 세포소기관 | 없음 | 있음 |
| 예 | 세균 | 동물, 식물, 곰팡이 |

**진핵세포 구조**

**공통 구조**
- **세포막**: 물질 출입 조절
- **세포질**: 세포소기관이 있는 공간
- **핵**: 유전 정보 저장
- **미토콘드리아**: 세포 호흡
- **리보솜**: 단백질 합성

**식물세포 특유**
- **세포벽**: 세포 보호 (셀룰로오스)
- **엽록체**: 광합성
- **액포**: 물질 저장 (큼)

**동물세포 특유**
- **중심체**: 세포 분열

### 2.3 세포막

**구조**

**인지질 2중층**
- 친수성 머리: 바깥쪽
- 소수성 꼬리: 안쪽

**유동 모자이크 모델**
- 인지질은 유동적
- 단백질이 모자이크처럼 박혀 있음

**기능**
- 선택적 투과성
- 물질 출입 조절
- 세포 인식

### 2.4 세포 소기관

**핵 (Nucleus)**
- 유전 정보 저장 (DNA)
- 전사 (DNA → RNA)
- 핵막, 핵공, 핵소체

**미토콘드리아 (Mitochondria)**
- 세포 호흡
- ATP 생산 (에너지)
- 2중막, 자체 DNA

**엽록체 (Chloroplast)** - 식물만
- 광합성
- 포도당 합성
- 2중막, 틸라코이드, 자체 DNA

**소포체 (ER)**
- **조면 소포체**: 리보솜 있음, 단백질 합성
- **활면 소포체**: 리보솜 없음, 지질 합성

**골지체 (Golgi apparatus)**
- 단백질 가공·분비
- 소포 형성

**리보솜 (Ribosome)**
- 단백질 합성
- mRNA → 단백질

**리소좀 (Lysosome)**
- 소화 효소
- 세포 내 소화

### 2.5 물질 이동

**확산 (Diffusion)**
- 농도 높음 → 낮음
- 에너지 불필요
- 예: 향기

**삼투 (Osmosis)**
- 물의 확산
- 농도 낮음 → 높음
- 반투과성 막

**능동 수송 (Active Transport)**
- 농도 낮음 → 높음
- 에너지 필요 (ATP)
- 예: Na⁺-K⁺ 펌프

**세포 내 섭취/분비**
- 식세포 작용 (고체)
- 음세포 작용 (액체)
- 분비 (소포)

## 3. 유전 (Genetics)

### 3.1 DNA

**구조**

**이중 나선**
- 두 가닥이 나선 모양
- 왓슨과 크릭 (1953년)

**뉴클레오티드**
- 당 (디옥시리보스)
- 인산
- 염기 (A, T, G, C)

**염기 쌍**
- A-T: 2개 수소 결합
- G-C: 3개 수소 결합
- 상보적 결합

**예시**
- 한쪽: ATGCTA
- 반대쪽: TACGAT

### 3.2 DNA 복제

**반보존적 복제**

1. DNA 이중 나선 풀림
2. 각 가닥이 주형
3. 상보적 뉴클레오티드 결합
4. 2개의 DNA 생성

**결과**
- 원래 DNA 1개 → 2개
- 각 DNA: 1가닥 옛것 + 1가닥 새것

### 3.3 유전 정보의 흐름

**중심 원리**

DNA → RNA → 단백질

**전사 (Transcription)**

DNA → RNA
- 장소: 핵
- DNA 한쪽 가닥이 주형
- RNA 중합효소
- mRNA 생성

**번역 (Translation)**

RNA → 단백질
- 장소: 리보솜 (세포질)
- mRNA의 코돈
- tRNA의 안티코돈
- 아미노산 연결

### 3.4 유전 암호

**코돈 (Codon)**

- mRNA의 3개 염기 = 1개 아미노산
- 총 64가지 (4³)
- 20가지 아미노산

**특수 코돈**
- 시작 코돈: AUG (메티오닌)
- 종결 코돈: UAA, UAG, UGA

**예시**
- mRNA: AUG-GCU-UAA
- 아미노산: Met-Ala-Stop

### 3.5 멘델의 유전 법칙

**분리의 법칙**

대립 유전자는 분리되어 배우자로

**예시**
- 부모: Aa
- 배우자: A 또는 a
- 자손: AA, Aa, aa (1:2:1)

**독립의 법칙**

두 형질은 독립적으로 유전

**예시**
- 둥근(R) 노란(Y) 완두
- 주름진(r) 초록(y) 완두
- 자손 비율: 9:3:3:1

**우열의 법칙**
- 우성 유전자가 표현
- Aa = AA (우성)

## 4. 생명 활동

### 4.1 광합성 (Photosynthesis)

**정의**

빛 에너지 → 화학 에너지 (포도당)

**화학식**

6CO₂ + 6H₂O + 빛 → C₆H₁₂O₆ + 6O₂

**명반응 (명단계)**

- 장소: 틸라코이드
- 빛 필요
- 과정:
  1. 물 분해 (광분해)
  2. NADPH 생성
  3. ATP 생성

**암반응 (암단계, 캘빈 회로)**

- 장소: 스트로마
- 빛 불필요 (효소 반응)
- 과정:
  1. CO₂고정
  2. NADPH, ATP 사용
  3. 포도당 합성

### 4.2 세포 호흡 (Cellular Respiration)

**정의**

포도당 분해 → ATP 생성

**화학식**

C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + ATP

**3단계**

**1. 해당과정** (세포질)
- 포도당 → 피루브산
- ATP 2개 생성

**2. 크렙스 회로** (미토콘드리아 기질)
- 피루브산 → CO₂
- NADH, FADH₂ 생성

**3. 전자전달계** (미토콘드리아 내막)
- NADH, FADH₂ → ATP
- ATP 약 34개 생성
- O₂ → H₂O

**총 ATP**: 약 38개

### 4.3 소화

**목적**

큰 분자 → 작은 분자 (흡수 가능)

**소화관**
- 입 → 식도 → 위 → 소장 → 대장 → 항문

**영양소별 소화**

| 영양소 | 소화 장소 | 소화 효소 | 결과 |
|--------|-----------|-----------|------|
| 탄수화물 | 입, 소장 | 아밀레이스 | 포도당 |
| 단백질 | 위, 소장 | 펩신, 트립신 | 아미노산 |
| 지방 | 소장 | 리파아제 | 지방산, 글리세롤 |

**흡수**
- 주로 소장 융털
- 포도당, 아미노산 → 혈관
- 지방산, 글리세롤 → 림프관

### 4.4 순환

**혈액 순환**

**심장 (4개 방)**
- 좌심방, 좌심실
- 우심방, 우심실

**체순환**
- 좌심실 → 대동맥 → 온몸 → 대정맥 → 우심방

**폐순환**
- 우심실 → 폐동맥 → 폐 → 폐정맥 → 좌심방

**혈액 성분**
- 혈장 (55%): 물, 단백질
- 혈구 (45%):
  - 적혈구: 산소 운반
  - 백혈구: 면역
  - 혈소판: 지혈

### 4.5 호흡

**폐호흡**

**흡기**
- 횡격막 수축, 내려감
- 가슴 확장
- 공기 들어옴

**호기**
- 횡격막 이완, 올라감
- 가슴 축소
- 공기 나감

**가스 교환**

**폐포**
- O₂: 폐포 → 혈액
- CO₂: 혈액 → 폐포

**조직**
- O₂: 혈액 → 세포
- CO₂: 세포 → 혈액

## 5. 생태계 (Ecosystem)

### 5.1 생태계 구성

**생물적 요인**

**생산자 (Producer)**
- 식물, 조류
- 광합성
- 무기물 → 유기물

**소비자 (Consumer)**
- 1차: 초식 동물
- 2차: 1차 소비자 먹음
- 3차: 2차 소비자 먹음

**분해자 (Decomposer)**
- 세균, 곰팡이
- 유기물 → 무기물

**비생물적 요인**
- 빛, 온도, 물, 공기, 토양

### 5.2 먹이 사슬

**선형**

풀 → 메뚜기 → 개구리 → 뱀 → 독수리

**먹이 그물**

여러 먹이 사슬이 복잡하게 연결

**생태 피라미드**

- 생산자 (가장 많음)
- 1차 소비자
- 2차 소비자
- 3차 소비자 (가장 적음)

**10% 법칙**
- 에너지의 10%만 다음 단계로
- 90%는 호흡·열로 소실

### 5.3 물질 순환

**탄소 순환**

1. 광합성: CO₂ → 유기물
2. 호흡: 유기물 → CO₂
3. 분해: 유기물 → CO₂
4. 연소: 화석 연료 → CO₂

**질소 순환**

1. 질소 고정: N₂ → NH₃ (질소 고정 세균)
2. 질산화: NH₃ → NO₃⁻
3. 흡수: 식물이 NO₃⁻ 흡수
4. 탈질산화: NO₃⁻ → N₂

**물 순환**

증발 → 응결 → 강수 → 지표수/지하수

### 5.4 개체군

**개체군 크기**

- 출생률 - 사망률
- 이입 - 이출

**성장 곡선**

**J자형**
- 환경 저항 없음
- 기하급수적 증가

**S자형**
- 환경 저항 있음
- 처음 느림 → 급증 → 정체

**환경 수용력**
- 환경이 유지할 수 있는 최대 개체 수

### 5.5 생물 다양성

**3가지 수준**

**유전적 다양성**
- 같은 종 내 유전자 다양
- 예: 사람의 피부색, 키

**종 다양성**
- 여러 종 존재
- 종 수, 균등도

**생태계 다양성**
- 여러 생태계
- 산, 강, 바다, 사막

**중요성**
- 생태계 안정
- 자원 제공
- 유전자 보고

## 6. 진화 (Evolution)

### 6.1 진화의 증거

**화석**
- 과거 생물의 흔적
- 지층이 오래될수록 단순

**상동 기관**
- 기원 같음, 기능 다름
- 예: 사람 팔, 새 날개, 고래 지느러미

**흔적 기관**
- 기능 퇴화
- 예: 사람 맹장, 뱀 다리뼈

**발생 유사**
- 초기 배아 비슷

### 6.2 자연 선택설 (다윈)

**원리**

1. 변이: 개체마다 다름
2. 과잉 생산: 많이 태어남
3. 생존 경쟁: 자원 부족
4. 적자생존: 환경에 적합한 개체 생존
5. 유전: 유리한 형질 후대에 전달

**예시: 기린의 목**
- 변이: 목 길이 다양
- 경쟁: 높은 잎 먹기
- 선택: 목 긴 기린 유리
- 결과: 목 긴 기린 증가

## 7. 인체 생리

### 7.1 신경계

**구조**

**중추 신경계**
- 뇌: 대뇌, 소뇌, 간뇌, 중뇌, 연수
- 척수

**말초 신경계**
- 감각 신경: 자극 → 중추
- 운동 신경: 중추 → 반응

**반사**

자극 → 감각 신경 → 척수 → 운동 신경 → 반응
(뇌 거치지 않음, 빠름)

### 7.2 호르몬

**내분비샘**

| 샘 | 호르몬 | 기능 |
|-----|--------|------|
| 뇌하수체 | 성장 호르몬 | 성장 촉진 |
| 갑상샘 | 티록신 | 물질대사 촉진 |
| 이자 | 인슐린 | 혈당 감소 |
| 이자 | 글루카곤 | 혈당 증가 |
| 부신 | 아드레날린 | 비상 상황 대비 |

**혈당 조절**
- 혈당 높음 → 인슐린 분비 → 혈당 낮춤
- 혈당 낮음 → 글루카곤 분비 → 혈당 높임

### 7.3 항상성

## 10. FAQ

### Q1. 생물은 왜 중요한가요?

**생명의 원리를 이해합니다.**

대학:
- 의학: 필수
- 생명과학: 필수
- 간호·보건: 중요

실생활:
- 건강 관리
- 환경 보전
- 생명 윤리

### Q2. 생물은 외울 게 너무 많아요.

**이해하면 암기가 쉽습니다.**

전략:
- 원리 이해
- 그림으로 정리
- 연관 지어 암기
- 반복 복습

**체계적 정리**가 핵심입니다.

### Q3. 세포와 세포소기관은 어떻게 구분하나요?

**기능으로 구분하세요.**

- 핵: 유전 정보
- 미토콘드리아: 에너지
- 엽록체: 광합성
- 리보솜: 단백질 합성

**그림 그리기**가 효과적입니다.

### Q4. 광합성과 세포 호흡의 차이는?

**반대 과정입니다.**

| 구분 | 광합성 | 세포 호흡 |
|------|--------|-----------|
| 장소 | 엽록체 | 미토콘드리아 |
| 재료 | CO₂ + H₂O | 포도당 + O₂ |
| 산물 | 포도당 + O₂ | CO₂ + H₂O |
| 에너지 | 흡수 | 방출 |

### Q5. 유전 문제는 어떻게 풀나요?

**단계적으로 접근하세요.**

1. 부모 유전자형 파악
2. 배우자 유전자 나열
3. 펀넷 사각형
4. 자손 비율 계산

**연습**이 중요합니다.

### Q6. 생태계에서 에너지 흐름은?

**일방향 흐름입니다.**

생산자 → 1차 → 2차 → 3차

**10% 법칙**
- 다음 단계로 10%만
- 나머지 90% 소실

### Q7. 진화의 증거는?

**4가지 증거**

1. 화석
2. 상동 기관
3. 흔적 기관
4. 발생 유사

각각 예시와 함께 이해하세요.

### Q8. 항상성은 무엇인가요?

**내부 환경 일정 유지**

예시:
- 체온: 36.5°C
- 혈당: 100mg/dL
- 삼투압: 일정

**음성 피드백**으로 조절

### Q9. 수능 생명과학 준비법은?

**3단계 전략**

**1단계**: 개념 정리
- 용어 암기
- 원리 이해
- 그림 그리기

**2단계**: 문제 풀이
- 유형별 연습
- 실험 설계
- 그래프 해석

**3단계**: 기출 반복
- 10개년 기출
- 오답 분석
- 실전 연습

### Q10. 1:1 과외가 효과적인가요?

**생물은 1:1이 효과적입니다.**

- 개념 시각화
