# 과학 공부법

체계적인 과학 학습 방법과 탐구 능력 향상을 위한 가이드입니다.

## 학년별 과학 학습

### 초등 과학
- 호기심 기반 탐구 활동
- 실험 관찰 능력 기르기
- 과학적 사고력 발달

### 중학 과학
- 통합과학 학습 전략
- 실험 보고서 작성법
- 과학 개념 이해하기

### 고등 과학
- 물리/화학/생명과학/지구과학
- 탐구 활동 심화
- 과학 논술 대비

곧 다양한 과학 학습 콘텐츠가 업데이트됩니다.
