# 과학 공식 과외로 완벽 암기하기

과학 공식 암기가 어렵나요?

"공식이 너무 많아서 헷갈려요."
"외워도 금방 잊어버려요."
"언제 어떤 공식을 써야 할지 모르겠어요."
"단위 계산에서 자꾸 실수해요."

10년 넘게 과학을 가르치면서 느낀 점은,
공식 암기는 **이해와 체계적 반복**으로 반드시 가능하다는 것입니다.

1:1 과학 공식 과외에서는 각 공식의 의미를 이해하고,
체계적인 암기 전략으로 완벽하게 내 것으로 만듭니다.

## 과학 공식 암기가 어려운 이유

많은 학생들이 공식 암기에서 어려움을 겪습니다.

### 단순 암기의 한계

의미를 모르고 무작정 외우면:
- 금방 잊어버림
- 응용이 불가능함
- 비슷한 공식과 헷갈림
- 시험에서 실수 증가

### 체계적 학습 부족

공식을 무작위로 외우면:
- 연결고리가 없어 혼란스러움
- 유도 과정을 모르면 응용 불가
- 단원 간 관계 파악 실패
- 심화 문제 대응력 부족

### 실전 적용 부족

문제 풀이 연습이 부족하면:
- 어느 상황에 어떤 공식인지 모름
- 단위 환산에서 실수
- 복합 공식 적용 실패
- 문제 해결 속도 느림

## 과학 공식 암기의 3원칙

효과적인 공식 암기를 위한 핵심 원칙입니다.

### 1원칙: 이해가 먼저

단순 암기보다 이해가 100배 중요합니다.

**이해 방법:**
- 공식의 물리적/화학적 의미 파악
- 왜 이런 관계식이 나왔는지 이해
- 각 변수가 나타내는 의미 명확화
- 단위가 왜 그렇게 되는지 확인
- 실생활과의 연결고리 찾기

**이해의 힘:**
```
암기만: v = d/t → "속력은 거리 나누기 시간"
이해함: v = d/t → "같은 시간에 더 멀리 가면 속력이 크다"
```

### 2원칙: 체계적 반복

과학적으로 검증된 반복 주기를 활용합니다.

**에빙하우스 망각곡선 활용:**
```
1일차: 새 공식 학습 (100%)
2일차: 어제 공식 복습 (75% → 100%)
3일차: 최근 3일 공식 복습
7일차: 이번 주 전체 공식 복습
14일차: 2주차 통합 복습
30일차: 한 달 총정리
```

**일일 학습 루틴:**
- 아침: 어제 배운 공식 5분 복습
- 수업: 새 공식 학습 + 이해
- 저녁: 오늘 배운 공식 적용 문제 풀이
- 주말: 이번 주 전체 복습 + 약점 보완

### 3원칙: 즉각 적용

배운 공식을 바로 문제에 적용합니다.

**단계별 적용:**
1. **기본 문제 (1~2일차)**
   - 공식 직접 대입 문제
   - 단위 환산 연습
   - 변수 구하기

2. **유형별 문제 (3~5일차)**
   - 다양한 상황 적용
   - 조건 변화 이해
   - 복합 개념 연결

3. **심화 문제 (6~7일차)**
   - 여러 공식 종합 활용
   - 문제 해결 전략 수립
   - 실생활 문제 적용

4. **실전 테스트 (주말)**
   - 시간 제한 풀이
   - 오답 분석
   - 약점 보완

## 과목별 핵심 공식 완전 정복

각 과목별 필수 공식과 암기 전략입니다.

### 물리 공식 완벽 정리

#### 1. 운동과 힘

**핵심 공식:**
```
속력: v = d/t (m/s)
가속도: a = Δv/t (m/s²)
뉴턴 제2법칙: F = ma (N)
등가속도 운동: v = v₀ + at
이동거리: s = v₀t + (1/2)at²
```

**암기 포인트:**
- 속력은 '얼마나 빨리 이동하는가'
- 가속도는 '속력이 얼마나 빨리 변하는가'
- 힘은 '가속도를 만드는 원인'
- 등가속도는 '일정하게 빨라지거나 느려짐'

**흔한 실수:**
- 속력과 속도 혼동 (스칼라 vs 벡터)
- 가속도가 음수일 때 감속도로 착각
- 단위 환산 실수 (km/h ↔ m/s)

#### 2. 에너지와 일

**핵심 공식:**
```
일: W = F × d (J)
운동에너지: Ek = (1/2)mv² (J)
위치에너지: Ep = mgh (J)
역학적 에너지 보존: Ek + Ep = 일정
일률: P = W/t (W)
```

**암기 전략:**
- 일은 '힘이 물체를 이동시킨 정도'
- 운동에너지는 '운동하는 물체가 가진 에너지'
- 위치에너지는 '높이에 따른 에너지'
- 에너지 보존은 '형태만 바뀔 뿐 총량 불변'

**적용 연습:**
- 자유낙하 운동: Ep → Ek 전환
- 진자 운동: Ek ↔ Ep 변환
- 롤러코스터: 역학적 에너지 보존

#### 3. 전기

**핵심 공식:**
```
전류: I = Q/t (A)
전압: V = W/Q (V)
저항: R = V/I (Ω)
옴의 법칙: V = IR
전력: P = VI = I²R = V²/R (W)
전력량: E = Pt (Wh)
```

**이해 중심 암기:**
- 전류: 1초당 흐르는 전하량
- 전압: 전하를 이동시키는 능력
- 저항: 전류 흐름을 방해하는 정도
- 옴의 법칙: 전압↑ → 전류↑, 저항↑ → 전류↓

### 화학 공식 완벽 정리

#### 1. 몰(mole) 개념

**핵심 공식:**
```
몰수: n = m/M = V/Vm = N/NA
질량: m = n × M (g)
부피: V = n × Vm (L, 표준상태 22.4L)
입자수: N = n × NA (개, 6.02×10²³)
```

**암기 핵심:**
- 몰은 '물질의 양'을 나타내는 단위
- 1몰 = 6.02×10²³개 입자
- 표준상태(0℃, 1기압)에서 기체 1몰 = 22.4L
- M은 원자량 또는 분자량

**단위 환산 연습:**
```
예시: 물(H₂O) 18g은 몇 몰?
→ n = m/M = 18g / 18g/mol = 1몰
```

#### 2. 농도 계산

**핵심 공식:**
```
질량 백분율: (용질 질량/용액 질량) × 100 (%)
몰농도: C = n/V (M, mol/L)
ppm: (용질 질량/용액 질량) × 10⁶
```

**실전 적용:**
- 용질과 용매 구별하기
- 용액의 부피는 L 단위로 통일
- 묽힘: C₁V₁ = C₂V₂

#### 3. 화학 반응식과 양적 관계

**핵심 공식:**
```
화학 반응식의 계수비 = 몰수비 = 부피비(기체)
반응 전후 질량 보존
```

**계산 전략:**
1. 화학 반응식 완성 (계수 맞추기)
2. 주어진 물질의 몰수 계산
3. 계수비로 다른 물질 몰수 계산
4. 원하는 단위로 환산

#### 4. 기체 법칙

**핵심 공식:**
```
보일 법칙: PV = 일정 (온도 일정)
샤를 법칙: V/T = 일정 (압력 일정)
이상기체 방정식: PV = nRT
```

**암기 전략:**
- 보일: 압력↑ → 부피↓ (반비례)
- 샤를: 온도↑ → 부피↑ (정비례)
- 이상기체: R = 0.082 atm·L/mol·K
- 온도는 반드시 절대온도(K) 사용

### 생물 공식 완벽 정리

#### 1. 광합성과 호흡

**핵심 공식:**
```
광합성: 6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂
호흡: C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + 에너지
광합성 속도 = 생성된 산소량/시간
호흡 속도 = 소비된 산소량/시간
```

**이해 포인트:**
- 광합성과 호흡은 역반응 관계
- 낮: 광합성 > 호흡
- 밤: 호흡만 일어남
- 광보상점: 광합성 = 호흡

#### 2. 유전 확률

**핵심 공식:**
```
멘델 유전:
- F1 교배: Aa × Aa → AA(1) : Aa(2) : aa(1)
- 독립 유전: (3:1) × (3:1) = 9:3:3:1

확률 계산:
- 그리고(AND): 곱셈
- 또는(OR): 덧셈
```

**계산 전략:**
1. 부모 유전자형 파악
2. 생식세포 만들기
3. 자손 유전자형 조합
4. 표현형 분리비 계산

#### 3. 개체군 밀도

**핵심 공식:**
```
개체군 밀도 = 개체수/면적
개체군 증가율 = (출생률 - 사망률) + (이입률 - 이출률)
```

### 지구과학 공식 완벽 정리

#### 1. 해양 탐사

**핵심 공식:**
```
해양 깊이 = (소리 왕복 시간 × 소리 속력) / 2
소리 속력 ≈ 1500 m/s (해수)
```

**적용 예시:**
- 소리가 4초 만에 돌아옴
- 깊이 = (4초 × 1500m/s) / 2 = 3000m

#### 2. 천문 계산

**핵심 공식:**
```
별의 절대등급: M = m - 5log(d/10)
광도비: L₁/L₂ = 2.512^(m₂-m₁)
판의 이동 속도 = 이동 거리/시간
```

## 효과적인 암기 전략

실제로 효과가 검증된 암기 방법입니다.

### 공식 정리 노트 만들기

**노트 구성:**
```
[공식명] 속력 공식
[공식] v = d/t
[변수] v=속력(m/s), d=거리(m), t=시간(s)
[의미] 단위 시간당 이동 거리
[유도] 거리를 시간으로 나눔
[예제] 100m를 10초에 가면? v=100/10=10m/s
[주의] 속력(스칼라) ≠ 속도(벡터)
[연관] 가속도, 등속 운동
```

**작성 팁:**
- 손으로 직접 쓰기 (기억 강화)
- 색깔 펜으로 변수 구분
- 그림/도표 활용
- 예제는 간단하고 명확하게

### 공식 암기 카드 활용

**카드 제작:**
- 앞면: 공식명 + "무엇을 구하는가?"
- 뒷면: 공식 + 단위 + 핵심 의미

**복습 방법:**
1. 매일 전체 카드 섞기
2. 앞면 보고 공식 떠올리기
3. 틀린 카드만 따로 모으기
4. 틀린 카드 집중 복습
5. 3번 연속 맞으면 "완벽" 더미로 이동

### 공식 송(노래) 활용

**리듬에 맞춰 외우기:**
```
"브이는 디 나누기 티~ (v = d/t)
에프는 엠 곱하기 에이~ (F = ma)
몰수는 엠 나누기 엠~ (n = m/M)"
```

### 연상 기억법

**이미지로 연결:**
- F = ma: "힘(F)센 마(ma)동석"
- PV = nRT: "피부엔 알티"
- Ek = ½mv²: "이케 반 마부제곱"

- 테스트 출제 및 평가
- 학습 자료 제공
- 동기부여 및 격려

## 공식 암기 시간 관리

효율적인 시간 배분으로 완벽 암기합니다.

### 일일 학습 스케줄

**평일 (총 30분):**
```
06:30-06:35 (5분): 어제 공식 복습
17:00-17:20 (20분): 새 공식 학습 + 이해
21:00-21:05 (5분): 오늘 공식 복습
```

**주말 (총 1시간):**
```
토요일:
- 이번 주 전체 공식 복습 (30분)
- 약점 공식 집중 학습 (20분)
- 문제 풀이로 적용 (10분)

일요일:
- 암기 카드 전체 점검 (20분)
- 틀린 문제 재풀이 (30분)
- 다음 주 계획 (10분)
```

### 시험 기간 집중 전략

**시험 2주 전:**
- 전체 공식 1회독
- 약점 공식 리스트 작성
- 기출문제 풀이

**시험 1주 전:**
- 약점 공식 집중 암기
- 예상 문제 풀이
- 오답 분석

**시험 전날:**
- 공식 카드 전체 복습
- 자주 틀리는 공식만 정리
- 충분한 수면

## 흔한 실수와 해결법

많은 학생들이 하는 실수를 예방합니다.

### 실수 1: 단위 환산 오류

**문제:**
- km/h를 m/s로 바꿀 때 실수
- g과 kg 혼동
- mL와 L 착각

**해결법:**
- 단위 환산표 암기
- 계산 전 반드시 단위 통일
- 답 쓸 때 단위 재확인

**단위 환산 공식:**
```
1 km/h = 1000m / 3600s = 1/3.6 m/s
1 m/s = 3.6 km/h
1 kg = 1000 g
1 L = 1000 mL
```

### 실수 2: 변수 혼동

**문제:**
- v(속력)와 V(부피) 혼동
- m(질량)과 M(분자량) 착각
- n(몰수)과 N(입자수) 실수

**해결법:**
- 대문자와 소문자 정확히 구분
- 각 변수의 단위 함께 암기
- 문제에서 주어진 값 먼저 정리

### 실수 3: 공식 적용 오류

**문제:**
- 상황에 맞지 않는 공식 사용
- 조건 확인 실패 (온도, 압력 등)
- 여러 공식 혼합 적용 실패

**해결법:**
- 문제에서 조건 먼저 확인
- 어떤 공식이 필요한지 판단
- 단계별로 차근차근 계산

### 실수 4: 계산 실수

**문제:**
- 분수 계산 오류
- 제곱/제곱근 실수
- 소수점 위치 착각

**해결법:**
- 계산기 적극 활용
- 중간 계산 과정 기록
- 답의 타당성 확인 (너무 크거나 작지 않은지)

## 자주 묻는 질문

### Q1. 공식을 빨리 암기하는 가장 좋은 방법은?

A. **이해를 바탕으로 한 반복**이 가장 빠릅니다.

의미를 모르고 무작정 외우면 금방 잊어버리지만,
공식의 의미와 유도 과정을 이해하면 오래 기억됩니다.

추천 방법:
- 선생님께 공식의 의미 질문하기
- 유도 과정 한 번씩 따라해보기
- 예제 문제로 즉시 적용하기
- 매일 5분 복습으로 장기기억 전환

### Q2. 공식이 너무 많아서 다 외울 수가 없어요.

A. **한 번에 모두 외우려 하지 마세요.**

효과적인 전략:
- 단원별로 나눠서 학습
- 가장 자주 쓰이는 핵심 공식부터 암기
- 비슷한 공식끼리 묶어서 정리
- 한 주에 5~10개 정도만 목표 설정

예시:
1주차: 운동 관련 공식 5개
2주차: 힘과 에너지 공식 5개
3주차: 전기 공식 5개
→ 한 달에 20개씩 체계적으로 완성

### Q3. 공식은 외웠는데 문제에 적용이 안 돼요.

A. **암기와 적용은 별개의 능력**입니다.

적용력 향상 방법:
- 공식 암기 후 즉시 기본 문제 풀이
- 다양한 유형의 문제 접하기
- 문제에서 어떤 공식인지 판단하는 연습
- 선생님께 풀이 과정 설명 듣기

1:1 과외에서는:
- 문제 유형별 공식 적용 전략 제시
- 실시간으로 풀이 과정 피드백
- 약점 유형 집중 훈련

### Q4. 단위 계산에서 자주 틀려요.

A. **단위 환산이 공식만큼 중요**합니다.

단위 실수 예방법:
- 문제 풀기 전 주어진 값의 단위 먼저 확인
- 필요시 단위 통일 (m↔km, g↔kg, L↔mL)
- 계산 중간에도 단위 함께 쓰기
- 답 쓸 때 단위 다시 확인

자주 쓰이는 환산:
- 1 km/h = 1/3.6 m/s
- 1 atm = 1013 hPa = 760 mmHg
- 표준상태: 0℃ (273K), 1 atm

### Q5. 물리 공식이 특히 어려워요.

A. **물리는 가장 추상적**이라 어렵게 느껴집니다.

물리 공식 극복법:
- 실생활 예시와 연결 (자동차, 엘리베이터 등)
- 그림/도표로 시각화
- 실험 영상 시청
- 기본 문제부터 차근차근

1:1 물리 과외 효과:
- 추상적 개념을 구체적으로 설명
- 학생 수준에 맞는 예시 제시
- 단계별 난이도 조절
- 이해될 때까지 반복 설명

### Q6. 화학의 몰 계산이 너무 헷갈려요.

A. **몰 개념은 화학의 핵심**입니다.

몰 정복 전략:
- 몰 = "물질의 개수"라고 이해하기
- 몰 사각형 활용: n = m/M = V/Vm = N/NA
- 단위 확인: g, L, 개수 중 무엇인지
- 기본 문제 100개 이상 풀기

과외에서 집중 훈련:
- 몰 개념 완벽 이해시키기
- 유형별 문제 풀이 전략
- 실수 포인트 짚어주기
- 자신감 회복

### Q7. 생물은 계산 문제가 별로 없는데 공식이 필요한가요?

A. **생물도 계산 문제가 있습니다.**

생물 공식 활용:
- 광합성/호흡 속도 계산
- 유전 확률 계산 (멘델 유전)
- 개체군 밀도 계산
- 혈당량 변화 그래프 해석

생물 공식 특징:
- 개수는 적지만 정확히 알아야 함
- 그래프 해석 능력이 중요
- 실험 결과 분석에 활용

### Q8. 공식 노트를 만드는 게 도움이 되나요?

A. **매우 효과적**입니다.

공식 노트의 장점:
- 손으로 쓰면서 기억 강화
- 나만의 이해 방식으로 정리
- 시험 직전 빠른 복습 가능
- 약점 공식 한눈에 파악

효과적인 노트 작성법:
- 공식, 단위, 의미, 예제 포함
- 색깔 펜으로 중요 부분 강조
- 자주 틀리는 공식 별표 표시
- 주기적으로 업데이트

### Q9. 1:1 과외가 정말 도움이 되나요?

A. **네, 공식 학습에 매우 효과적**입니다.

과외의 장점:
- 학생의 이해도 실시간 확인
- 막히는 부분 즉시 해결
- 개인 맞춤 설명과 예시 제공
- 암기 전략 개인별 최적화
- 약점 공식 집중 보완
- 문제 풀이 전략 1:1 코칭

과외 vs 독학:
- 독학: 이해 안 되면 계속 막힘
- 과외: 즉시 해결하고 다음 단계로

특히 이런 학생에게 추천:
- 공식 암기가 안 되는 학생
- 공식은 아는데 적용이 안 되는 학생
- 특정 과목(물리/화학)이 약한 학생
- 시험 전 단기 집중 학습이 필요한 학생

### Q10. 단기간에 성적을 올릴 수 있나요?

A. **전략적 학습으로 가능**합니다.

1개월 단기 플랜:
- 1주차: 핵심 공식 20개 완벽 암기
- 2주차: 기본 문제 200개 풀이
- 3주차: 유형별 문제 풀이 + 약점 보완
- 4주차: 기출문제 + 실전 테스트

1:1 과외로 더 빠르게:
- 학습 우선순위 설정 (중요 공식부터)
- 약점 파악 후 집중 훈련
- 문제 풀이 스킬 전수
- 실전 시험 대비 전략

실제 사례:
- 중2 학생: 한 달 과외로 60점 → 85점
- 고1 학생: 물리 공식 집중 훈련으로 50점 → 78점
