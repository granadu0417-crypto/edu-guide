## 학습 부진 학생 지원 가이드 - 맞춤형 학원 과외 전략

학습 부진은 누구나 경험할 수 있으며, 적절한 지원으로 극복할 수 있습니다. 본 가이드에서는 학습 부진의 원인 파악, 1:1 과외를 활용한 맞춤 교육, 학습 동기 부여 방법, 그리고 단계별 학습 전략을 상세히 안내합니다.

## 1. 학습 부진의 원인

**인지적 원인:**
- 기초 학습 능력 부족
- 이해력 느림
- 기억력 약함
- 집중력 부족

**정서적 원인:**
- 자신감 부족
- 학습 불안
- 우울, 스트레스
- 학습 의욕 상실

**환경적 원인:**
- 가정 환경 불안정
- 학교 부적응
- 또래 관계 문제
- 학습 환경 열악

**신체적 원인:**
- 시력, 청력 문제
- 수면 부족
- 건강 문제
- ADHD 등

## 2. 원인 파악 방법

**1단계: 관찰**
- 어떤 과목이 부진한가?
- 언제부터 부진했는가?
- 어떤 상황에서 더 힘들어하는가?

**2단계: 상담**
- 학생과 대화
- 학교 선생님 상담
- 부모님 관찰 공유
- 1:1 과외 선생님 의견

**3단계: 평가**
- 학습 능력 평가
- 정서 상태 평가
- 필요시 전문가 상담 (심리, 의료)

**4단계: 계획**
- 맞춤 학습 계획 수립
- 1:1 과외 또는 학습 클리닉
- 정기적 점검

## 3. 1:1 과외 vs 학원

**1:1 과외 (강력 추천):**
- 장점: 개인 맞춤, 속도 조절, 집중 관리
- 단점: 비용 높음
- 추천: 기초 부족, 자신감 없음, 개별 관심 필요
- 비용: 월 50-80만 원

**소그룹 과외 (2-3명):**
- 장점: 부담 적음, 또래 학습
- 단점: 개인 맞춤 제한
- 추천: 중위권, 친구와 함께
- 비용: 월 30-50만 원/인

**학원 (비추천):**
- 장점: 비용 저렴, 체계적
- 단점: 개인 관심 부족, 따라가기 힘듦
- 추천: 일부 과목만 부진
- 비용: 월 20-50만 원

## 4. 1:1 과외 활용 전략

**선생님 선택:**
- 경험: 학습 부진 학생 지도 경험
- 성격: 인내심, 긍정적, 공감 능력
- 접근: 칭찬, 격려 중심
- 확인: 시범 수업 (1-2회)

**수업 방식:**
- 느린 속도로 설명
- 반복 학습
- 작은 성취감 제공
- 긍정적 피드백

**수업 빈도:**
- 주 2-3회 (집중 관리)
- 회당 1.5-2시간
- 장기적 관점 (6개월-1년)

**수업 후:**
- 복습 시간 확보 (30분-1시간)
- 작은 숙제 (부담 없이)
- 선생님과 부모님 소통

## 5. 단계별 학습 전략

**1단계: 기초 다지기 (2-4개월)**
- 목표: 기초 개념 완벽히
- 방법: 초등/중등 과정부터
- 속도: 천천히, 반복
- 평가: 작은 테스트 (칭찬 위주)

**2단계: 자신감 회복 (2-4개월)**
- 목표: "나도 할 수 있다"
- 방법: 쉬운 문제부터 성공 경험
- 칭찬: 과정 중심 (결과보다)
- 보상: 작은 보상 시스템

**3단계: 학교 진도 따라가기 (3-6개월)**
- 목표: 학교 수업 이해
- 방법: 예습 + 복습
- 1:1 과외로 보충
- 점진적 난이도 상승

**4단계: 독립적 학습 (6개월-1년)**
- 목표: 스스로 공부
- 방법: 학습 습관 형성
- 1:1 과외는 점차 줄이기
- 자기주도 학습 유도

## 6. 과목별 전략

**수학 (가장 많이 부진):**

**초등 수학 부족:**
- 연산부터 다시 (구구단, 사칙연산)
- 1:1 수학 과외로 기초 다지기
- 매일 20분 연산 연습
- 기간: 3-6개월

**중등 수학 부족:**
- 중1 수학부터 다시
- 개념 완벽 이해 (문제 풀이 X)
- 기본 문제만 반복
- 기간: 6개월-1년

**영어 (암기 부담):**

**초등 영어 부족:**
- 파닉스부터 (발음, 읽기)
- 기초 단어 (초등 800단어)
- 1:1 영어 과외로 재미있게
- 기간: 3-6개월

**중등 영어 부족:**
- 중1 단어, 문법
- 짧은 문장 읽기
- 듣기 매일 10분
- 기간: 6개월-1년

**국어 (독해력):**
- 짧은 글부터 (동화, 수필)
- 소리 내어 읽기
- 요약하기 연습
- 1:1 과외로 첨삭
- 기간: 6개월-1년

## 7. 동기 부여 전략

**성공 경험 제공:**
- 쉬운 문제부터 시작
- 작은 목표 달성
- 즉각적 칭찬

**칭찬 방법:**
- 과정 칭찬 (노력, 태도)
- 구체적 칭찬 ("집중 잘했어")
- 비교하지 않기 (다른 아이와)

**보상 시스템:**
- 작은 보상: 스티커, 칭찬 카드
- 중간 보상: 목표 달성 시 간식, 여가
- 큰 보상: 장기 목표 달성 시 선물

**실패 대응:**
- 비난 X, 위로
- "괜찮아, 다음엔 더 잘할 거야"
- 실패도 배움의 과정

## 8. 부모의 역할

**해야 할 것:**
- 인내심 있게 기다리기
- 작은 발전도 칭찬
- 정서적 지지
- 1:1 과외 선생님과 협력
- 규칙적인 생활 유도

**하지 말아야 할 것:**
- 다른 아이와 비교
- 과도한 압박
- 부정적 말 ("너는 왜 못해")
- 포기하는 모습
- 과외 선생님 탓

**소통:**
- 학생 의견 존중
- 1:1 과외 선생님과 주 1회 소통
- 학교 선생님과도 협력
- 전문가 상담 (필요시)

## 9. 실제 사례

**사례 1: 초등 수학 부진 → 중위권 회복**
- 상황: 초5, 구구단도 불안정
- 대응: 1:1 수학 과외 (6개월, 초등 과정 전체)
- 결과: 초6 중위권, 자신감 회복

**사례 2: 중등 전 과목 부진 → 기초 다지기**
- 상황: 중2, 전 과목 하위권
- 대응: 1:1 과외 (수학, 영어), 소그룹 (국어)
- 결과: 1년 후 중하위권, 학교 적응

**사례 3: 고등 수학 부진 → 중등부터 다시**
- 상황: 고1, 수학 9등급
- 대응: 1:1 수학 과외 (중등 과정부터, 1년)
- 결과: 고2 수학 5등급, 계속 상승 중

**공통점:**
- 기초부터 다시
- 1:1 과외 활용
- 장기적 관점 (6개월-1년)
- 끈기와 인내

## 10. 전문가 도움

**언제 필요한가:**
- 1년 이상 노력해도 효과 없음
- 심한 학습 불안, 우울
- ADHD, 난독증 의심
- 학교 부적응 심각

**전문가 종류:**
- 학습 클리닉: 학습 평가 및 치료
- 심리 상담: 정서적 문제
- 소아청소년 정신과: ADHD 등
- 특수교육: 학습 장애

**병행:**
- 전문가 치료 + 1:1 과외
- 학교 지원 + 가정 지원

## 11. 체크리스트

**원인 파악:**
- [ ] 어떤 과목이 부진한가?
- [ ] 기초 학습 능력은?
- [ ] 정서 상태는?
- [ ] 전문가 상담 필요한가?

**대응 계획:**
- [ ] 1:1 과외 또는 소그룹 과외

- [ ] 장기적 관점 (6개월-1년)
- [ ] 부모-선생님-학생 협력

**진행 중:**
- [ ] 작은 성취 칭찬
- [ ] 정기적 평가
- [ ] 동기 부여 유지
- [ ] 포기하지 않기

## FAQ

**Q1. 학습 부진은 치료 가능한가요?**
A1. 네, 대부분 적절한 지원으로 극복 가능합니다. 1:1 과외로 기초부터 천천히 다시 쌓으면 충분히 회복할 수 있습니다. 1대1 과외는 학습 부진 학생에게 가장 효과적입니다.

**Q2. 얼마나 걸리나요?**
A2. 원인과 정도에 따라 다르지만, 보통 6개월-1년 소요됩니다. 장기적 관점으로 인내심 있게 지원해야 합니다.

**Q3. 학원은 안 되나요?**
A3. 학습 부진 학생에게는 학원이 부적합합니다. 개인 관심이 부족하고 따라가기 힘듭니다. 1:1 과외를 강력히 권장합니다.

**Q4. 비용이 부담되는데요?**
A4. 장기 투자이지만 필요한 지출입니다. 1:1 과외로 기초를 다진 후 소그룹이나 학원으로 전환할 수 있습니다.

**Q5. 초등 과정부터 다시 해야 하나요?**
A5. 네, 기초가 부족하다면 초등 과정부터 다시 시작해야 합니다. 1:1 과외로 빠르게 따라잡을 수 있습니다.

**Q6. 자녀가 의욕이 없는데요?**
A6. 작은 성공 경험을 제공하고 칭찬하세요. 1:1 과외 선생님의 긍정적 피드백이 동기 부여에 큰 도움이 됩니다.

**Q7. 전 과목이 부진한데요?**
A7. 주요 과목(수학, 영어)부터 1:1 과외로 시작하세요. 모든 과목을 동시에 올리기는 어렵습니다.

**Q8. ADHD 의심되는데요?**
A8. 소아청소년 정신과 상담을 받으세요. 진단 후 적절한 치료와 1:1 과외를 병행하면 효과적입니다.

**Q9. 1:1 과외 선생님은 어떻게 선택하나요?**
A9. 학습 부진 학생 지도 경험이 있고, 인내심 있고 긍정적인 선생님을 선택하세요. 시범 수업으로 확인하세요.

**Q10. 언제 포기해야 하나요?**
A10. 포기하지 마세요! 느리더라도 계속 발전합니다. 1:1 과외와 전문가 도움으로 반드시 극복할 수 있습니다.
