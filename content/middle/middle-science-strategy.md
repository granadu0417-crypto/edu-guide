# 중학생 과학 학습 전략 세우기

과학이 어려운가요? 개념부터 탐구까지, 1:1 탐구 지도로 과학을 실험 중심법으로 정복하세요.

{{< cta-dual type="inline" >}}

## 중학 과학의 중요성

중학 과학은 물리, 화학, 생물, 지구과학의 기초입니다. 암기가 아닌 이해가 필요하며, 실험과 탐구를 통해 과학적 사고력을 키워야 합니다.

### 중학 과학이 중요한 이유

**1. 고등 과학의 토대**
- 고등학교 물리/화학/생물/지구과학의 기초 개념
- 중학 과학을 이해하지 못하면 고등 과학 포기
- 과학 선택과목 결정의 기준

**2. 과학적 사고력 형성**
- 관찰, 가설 설정, 실험, 결론 도출의 과학적 방법 습득
- 논리적 사고와 문제 해결 능력 배양
- 일상생활의 과학 현상 이해

**3. 이공계 진로의 출발점**
- 의대, 공대, 자연계열 진학의 필수 기초
- 과학 탐구 대회, 발명품 경진대회 등 활동
- 과학고, 영재학교 진학 대비

### 중학 과학의 어려움

**개념이 추상적**
- 눈에 보이지 않는 원자, 분자, 세포
- 복잡한 화학 반응식
- 물리 법칙과 수식

**암기와 이해의 균형**
- 단순 암기로는 문제 풀이 불가
- 개념 이해 없이 공식만 외우면 응용 실패
- 실험 원리를 이해해야 탐구 가능

**실험 결과 해석 능력**
- 실험 데이터 분석
- 그래프 해석
- 변인 통제 이해

### 과학 학습의 핵심 원리

**WHY(왜)를 중심으로**
- "이 현상이 왜 일어나는가?"
- "이 법칙은 왜 성립하는가?"
- "실험 결과가 왜 이렇게 나왔는가?"

**손으로 그리고 쓰면서**
- 세포 구조 직접 그리기
- 화학 반응식 손으로 쓰기
- 실험 과정 그림으로 표현

**일상과 연결하기**
- 교과서 속 과학을 실생활에서 찾기
- 뉴스의 과학 이슈 이해
- 주변 현상을 과학으로 설명

## 1:1 과학 학습의 장점

집단 수업에서는 실험을 보기만 하지만, 1:1 지도는 직접 실험하고 탐구하며 과학적 사고력을 완성합니다.

### 개념 시각화

복잡한 과학 개념을 눈으로 볼 수 있게 만들어야 이해가 빠릅니다.

**그림과 도표 활용**
- 세포 구조: 직접 그리면서 각 기관의 역할 이해
- 화학 반응: 원자 모형으로 반응 과정 시각화
- 물리 운동: 그래프로 속도와 가속도 표현
- 지구 구조: 단면도로 지각, 맨틀, 핵 이해

**실물 관찰**
- 현미경으로 세포 직접 관찰
- 실험 기구 직접 만지고 조작
- 화학 반응 직접 눈으로 확인
- 암석 표본 손으로 만지며 특징 파악

**애니메이션과 시뮬레이션**
- 분자 운동 시뮬레이션
- 천체 운동 애니메이션
- 화학 반응 3D 모델링
- 생태계 먹이사슬 시각화

### 실험 안전 교육

과학 실험은 위험할 수 있습니다. 1:1 지도로 안전하게 실험하는 방법을 배웁니다.

**실험 전 안전 수칙**
- 보안경, 실험복 착용
- 실험 기구 사용법 숙지
- 위험 물질 확인
- 비상 대처 방법 학습

**실험 중 주의사항**
- 화학 약품 취급 방법
- 가열 시 안전 거리
- 유리 기구 파손 대처
- 화재, 화상 응급 처치

**실험 후 정리**
- 약품 폐기 방법
- 실험 기구 세척
- 실험실 청소
- 사고 보고서 작성

### 탐구 설계 능력

과학은 정답을 외우는 것이 아니라 스스로 탐구하는 과정입니다.

**탐구 과정의 5단계**

**1. 문제 인식**
"왜 식물은 빛을 향해 자랄까?"
→ 궁금증을 과학적 질문으로 만들기

**2. 가설 설정**
"식물은 빛이 있는 쪽으로 성장 호르몬이 이동하기 때문이다"
→ 예상 답을 논리적으로 제시

**3. 탐구 설계**
- 독립 변인: 빛의 방향
- 종속 변인: 줄기의 굽음 정도
- 통제 변인: 식물 종류, 온도, 물
→ 변인 통제가 핵심

**4. 탐구 수행**
- 실험 진행
- 데이터 수집
- 관찰 기록
- 사진, 동영상 촬영

**5. 결과 분석 및 결론**
- 데이터 정리 (표, 그래프)
- 가설 검증
- 오차 분석
- 추가 탐구 계획

### 오개념 교정

과학에서는 잘못된 개념(오개념)을 가지고 있으면 아무리 공부해도 성적이 오르지 않습니다.

**흔한 오개념 예시**

**물리**
- (X) 무거운 물체가 더 빨리 떨어진다
- (O) 공기 저항 무시 시 모든 물체는 같은 속도로 낙하

**화학**
- (X) 얼음이 녹으면 물의 양이 늘어난다
- (O) 얼음이 녹아도 질량은 같고 부피만 줄어든다

**생물**
- (X) 식물은 낮에만 호흡한다
- (O) 식물은 24시간 호흡하고, 낮에 광합성도 함께 한다

**지구과학**
- (X) 여름이 더운 것은 태양이 가까이 있어서다
- (O) 지구 자전축이 기울어져 햇빛을 받는 각도가 달라서다

**1:1 오개념 교정 방법**
1. 현재 이해 상태 파악 (어떻게 생각하는지 질문)
2. 오개념 발견 (틀린 부분 찾기)
3. 인지적 갈등 유발 (반례 제시, 실험으로 증명)
4. 올바른 개념 정립 (정확한 설명)
5. 반복 확인 (유사 문제로 재점검)

## 과학 학습 4단계 전략

### 1단계: 개념 완벽 이해

과학은 암기가 아니라 이해입니다. 개념을 정확히 이해해야 문제를 풀 수 있습니다.

**교과서 정독법**

**1회독: 훑어보기 (스키밍)**
- 단원 제목과 소제목 파악
- 그림, 도표, 사진 먼저 보기
- 중요 용어 체크
- 학습 목표 확인

**2회독: 정독하기**
- 한 문장씩 천천히 읽기
- 모르는 용어는 즉시 찾기
- 그림과 본문 연결하며 읽기
- 중요 부분 밑줄 긋기

**3회독: 요약 정리**
- 핵심 개념 노트에 정리
- 그림 직접 그리며 이해
- 용어 정의 암기
- 개념 간 관계 파악

**개념 정리 노트 작성법**

```
[단원: 3. 광합성]

[핵심 개념]
광합성: 식물이 빛에너지를 이용해 이산화탄소와 물로
        포도당과 산소를 만드는 과정

[화학식]
6CO₂ + 6H₂O + 빛에너지 → C₆H₁₂O₆ + 6O₂
(이산화탄소) (물)        (포도당)  (산소)

[장소]
엽록체 (잎의 세포 속 녹색 소기관)

[조건]
1. 빛 (햇빛)
2. 이산화탄소 (공기 중)
3. 물 (뿌리에서 흡수)
4. 엽록소 (광합성 색소)

[중요 포인트]
- 낮에만 일어남 (빛 필요)
- 호흡은 24시간 (광합성과 별개)
- 산소는 부산물 (식물도 산소 필요)

[실생활]
- 나무가 공기를 맑게 하는 이유
- 실내 식물이 필요한 이유
```

**Why 질문으로 깊이 이해**

단순히 "무엇(What)"을 외우지 말고, "왜(Why)"를 물어야 합니다.

- Q: 왜 식물은 낮에만 광합성을 할까?
  A: 빛에너지가 필요하기 때문

- Q: 왜 광합성으로 산소가 나올까?
  A: 물(H₂O)을 분해하면서 O₂가 부산물로 생성

- Q: 왜 겨울에는 나뭇잎이 떨어질까?
  A: 빛이 약해 광합성 효율이 낮아 에너지 절약

### 2단계: 실험으로 원리 확인

과학은 실험 과목입니다. 직접 실험해야 원리를 이해합니다.

**교과서 핵심 실험**

**물리 실험**

**1. 속도 측정 실험**
- 목적: 평균 속도 = 이동 거리 / 걸린 시간
- 방법: 경사면에서 구슬 굴리며 시간 측정
- 변인: 경사각, 구슬 질량
- 결과: 그래프로 속도 변화 확인

**2. 빛의 굴절 실험**
- 목적: 빛이 다른 매질로 들어갈 때 꺾임
- 방법: 물에 빨대를 넣고 관찰
- 결과: 빨대가 꺾여 보임 → 굴절 원리

**화학 실험**

**1. 산과 염기 중화 실험**
- 목적: 산 + 염기 → 염 + 물
- 방법: 식초(산) + 베이킹소다(염기) → 거품 발생
- 지시약: 리트머스 종이 색 변화 관찰
- 결과: 중성에 가까워짐

**2. 화학 반응 전후 질량 보존**
- 목적: 반응 전후 질량은 변하지 않음
- 방법: 밀폐 용기에서 반응 → 저울로 측정
- 결과: 질량 보존 법칙 확인

**생물 실험**

**1. 세포 관찰**
- 목적: 현미경으로 세포 구조 확인
- 재료: 양파 표피, 구강 상피 세포
- 염색: 아세트산카민, 메틸렌블루
- 관찰: 핵, 세포막, 세포질 확인

**2. 광합성 실험 (녹말 검출)**
- 목적: 빛이 있을 때만 녹말 생성
- 방법: 알루미늄 포일로 일부 가림 → 요오드 용액
- 결과: 빛 받은 부분만 청람색 (녹말 있음)

**지구과학 실험**

**1. 암석 관찰**
- 화성암: 화강암, 현무암
- 퇴적암: 사암, 셰일, 석회암
- 변성암: 대리암, 편마암
- 특징: 색, 알갱이 크기, 경도 비교

**2. 날씨 관측**
- 온도계, 습도계 사용
- 기압 측정
- 바람 방향과 속도
- 구름 관찰 및 분류

**실험 보고서 작성법**

```
[실험 제목] 빛의 굴절 실험

[실험 목적]
빛이 서로 다른 매질을 통과할 때 굴절하는 현상을 관찰한다.

[실험 재료]
유리컵, 물, 빨대, 레이저 포인터

[실험 방법]
1. 빈 유리컵에 빨대를 똑바로 넣는다.
2. 빨대의 모양을 관찰한다. (똑바름)
3. 유리컵에 물을 절반 넣는다.
4. 빨대의 모양을 다시 관찰한다.
5. 레이저를 물에 비스듬히 쏘아 경로 관찰

[실험 결과]
- 물에 담긴 빨대가 꺾여 보임
- 레이저 빛이 수면에서 꺾임
- 입사각 > 굴절각 (공기→물)

[결론]
빛은 공기에서 물로 들어갈 때 굴절한다.
이는 매질의 밀도 차이 때문이다.

[오차 원인]
- 물의 흔들림
- 레이저 각도 측정 오차

[느낀 점]
일상에서 보는 현상(수영장이 얕아 보임)도
과학 원리로 설명할 수 있음을 알았다.
```

### 3단계: 탐구 활동으로 사고력 키우기

교과서 실험을 넘어서 스스로 탐구 주제를 정하고 실험합니다.

**탐구 주제 선정 방법**

**1. 일상 속 궁금증**
- "왜 비누는 기름때를 없앨까?"
- "왜 겨울에 입김이 보일까?"
- "왜 감자가 물에 뜨거나 가라앉을까?"

**2. 교과서 심화**
- "온도가 광합성 속도에 미치는 영향"
- "경사각과 물체의 가속도 관계"
- "식물 성장에 음악이 영향을 줄까?"

**3. 사회 이슈**
- "미세먼지 농도 측정과 원인 분석"
- "플라스틱 분해 시간 비교"
- "재생 에너지 효율성 비교"

**변인 통제의 중요성**

과학 탐구에서 가장 중요한 것은 **변인 통제**입니다.

**변인의 종류**
- **독립 변인**: 내가 바꾸는 것 (실험 조건)
- **종속 변인**: 결과로 나타나는 것 (측정 대상)
- **통제 변인**: 일정하게 유지하는 것

**예시: 비료 농도가 식물 성장에 미치는 영향**
```
독립 변인: 비료 농도 (0%, 1%, 2%, 3%)
종속 변인: 식물 키 (cm)
통제 변인:
  - 식물 종류 (동일 품종)
  - 화분 크기
  - 물 주는 양
  - 햇빛 양
  - 온도
  - 측정 기간
```

**변인 통제 실패 예시**
- 식물 A: 비료 1%, 햇빛 많이
- 식물 B: 비료 2%, 햇빛 적게
→ 식물 B가 작게 자라면? 비료 때문? 햇빛 때문?
→ 알 수 없음! (변인 통제 실패)

**탐구 보고서 작성**

```
[탐구 주제]
온도가 효소 활동에 미치는 영향

[탐구 동기]
겨울에 빨래가 잘 마르지 않는 이유가 궁금했다.
세제 속 효소가 온도에 따라 활동이 달라진다고 들었다.

[탐구 가설]
온도가 높을수록 효소 활동이 활발할 것이다.
단, 너무 높으면 효소가 파괴될 것이다.

[탐구 설계]
- 독립 변인: 온도 (10℃, 20℃, 30℃, 40℃, 50℃, 60℃)
- 종속 변인: 녹말 분해 속도 (요오드 반응 색 변화 시간)
- 통제 변인: 녹말 양, 침(효소) 양, 시험관 크기

[탐구 수행]
1. 녹말 용액 준비 (각 시험관에 동일량)
2. 침(아밀라아제 효소) 각 시험관에 동일량 넣기
3. 각각 다른 온도의 항온수조에 넣기
4. 1분마다 요오드 용액으로 녹말 확인
5. 청람색이 사라지는 시간 기록

[탐구 결과]
온도 | 녹말 분해 시간
10℃ | 20분
20℃ | 10분
30℃ | 5분
40℃ | 3분
50℃ | 8분
60℃ | 15분

[결과 분석]
- 40℃에서 가장 빠름 (최적 온도)
- 50℃ 이상에서 느려짐 (효소 변성)
- 그래프: 종 모양 곡선

[결론]
효소는 최적 온도(40℃ 근처)에서 가장 활발하다.
너무 높은 온도(50℃ 이상)에서는 효소가 변성되어
활동이 떨어진다.

[추가 탐구 계획]
- pH에 따른 효소 활동 변화
- 다른 효소(위의 펩신)의 최적 온도 비교
```

## 분야별 학습법

### 물리: 법칙과 공식 이해

물리는 자연의 법칙을 수식으로 표현한 학문입니다.

**핵심 개념**

**1. 운동**
```
속도 = 이동 거리 / 걸린 시간
v = s / t

가속도 = 속도 변화 / 걸린 시간
a = (v - v₀) / t
```

- 등속 운동: 속도가 일정 (가속도 0)
- 등가속도 운동: 가속도가 일정 (자유낙하)

**2. 힘**
```
뉴턴 제2법칙: F = ma
(힘 = 질량 × 가속도)
```

- 관성: 물체가 현재 운동 상태를 유지하려는 성질
- 작용 반작용: 힘은 쌍으로 작용

**3. 에너지**
```
운동 에너지 = 1/2 × 질량 × 속도²
위치 에너지 = 질량 × 중력가속도 × 높이

에너지 보존: 역학적 에너지는 일정
```

**4. 파동**
- 파동의 속력 = 진동수 × 파장
- 빛의 굴절: 스넬의 법칙
- 소리의 성질: 진동수(높낮이), 진폭(크기)

**물리 학습 팁**

**1. 공식 유도 과정 이해**
단순 암기 X → 왜 그런지 이해

**2. 그림 그리기**
- 힘의 방향 화살표로 표시
- 운동 경로 그리기
- 에너지 전환 도식화

**3. 일상 속 물리 찾기**
- 롤러코스터: 위치 에너지 ↔ 운동 에너지
- 자동차 안전벨트: 관성
- 스케이트: 작용 반작용

### 화학: 반응식과 개념 정리

화학은 물질의 변화를 다루는 학문입니다.

**핵심 개념**

**1. 원자와 분자**
```
원자: 물질의 기본 단위
분자: 원자 2개 이상 결합

예:
H₂O (물 분자) = H 원자 2개 + O 원자 1개
CO₂ (이산화탄소) = C 원자 1개 + O 원자 2개
```

**2. 화학 반응식**

**질량 보존 법칙**: 반응 전후 질량은 같다
→ 화학 반응식 양변의 원자 개수 일치

```
[연소 반응]
CH₄ + 2O₂ → CO₂ + 2H₂O
(메탄)(산소) (이산화탄소)(물)

검산:
왼쪽: C 1개, H 4개, O 4개
오른쪽: C 1개, H 4개, O 4개 ✓
```

**3. 산과 염기**
```
산: H⁺ 내놓음 (시큼, pH < 7)
  - 염산(HCl), 황산(H₂SO₄), 아세트산(CH₃COOH)

염기: OH⁻ 내놓음 (쓴맛, pH > 7)
  - 수산화나트륨(NaOH), 암모니아(NH₃)

중화 반응:
HCl + NaOH → NaCl + H₂O
(산) (염기)   (염)  (물)
```

**4. 이온**
```
양이온(+): 전자를 잃음 (Na⁺, Ca²⁺, Al³⁺)
음이온(-): 전자를 얻음 (Cl⁻, O²⁻, N³⁻)

이온 결합: Na⁺ + Cl⁻ → NaCl (염화나트륨)
```

**화학 학습 팁**

**1. 화학 반응식 직접 쓰기**
- 손으로 반복해서 쓰기
- 원자 개수 맞추기 연습
- 계수 맞추기 (최소 공배수)

**2. 주기율표 익히기**
- 원소 기호 암기 (최소 20개)
- H, He, Li, Be, B, C, N, O, F, Ne...
- 족(세로)과 주기(가로) 의미

**3. 실생활 화학 찾기**
- 요리: 베이킹소다(중화 반응)
- 청소: 식초(산), 비누(염기)
- 호흡: 산소 흡입, 이산화탄소 배출

### 생물: 구조와 기능 연결

생물은 생명체의 구조와 기능을 다루는 학문입니다.

**핵심 개념**

**1. 세포**
```
세포막: 물질 출입 조절
핵: 유전 정보 저장 (DNA)
세포질: 물질 대사 일어남
미토콘드리아: 에너지 생산 (호흡)
엽록체: 광합성 (식물 세포만)
```

**2. 광합성과 호흡**
```
광합성 (낮, 엽록체):
6CO₂ + 6H₂O + 빛 → C₆H₁₂O₆ + 6O₂

호흡 (24시간, 미토콘드리아):
C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + 에너지
```

**3. 소화와 흡수**
```
입: 침(아밀라아제) → 녹말 분해
위: 위액(펩신) → 단백질 분해
소장: 이자액, 쓸개즙 → 영양소 흡수
```

**4. 순환과 호흡**
```
심장: 펌프 역할
  - 좌심실 → 온몸 (동맥혈, 산소 ↑)
  - 우심실 → 폐 (정맥혈, 이산화탄소 ↑)

폐: 가스 교환
  - 산소 흡수, 이산화탄소 배출
```

**5. 유전**
```
DNA: 유전 정보 저장
염색체: DNA가 뭉쳐진 것
유전자: DNA의 특정 부분

멘델의 법칙:
  - 우성: 나타나는 형질 (A)
  - 열성: 숨겨지는 형질 (a)
  - Aa (우성 형질 나타남)
```

**생물 학습 팁**

**1. 그림 그리기**
- 세포 구조 직접 그리기
- 소화 기관 순서대로 그리기
- 심장 구조와 혈액 흐름 그리기

**2. 구조 → 기능 연결**
- "왜 이런 구조일까?" 생각
- 예: 소장이 긴 이유 → 흡수 표면적 넓히기

**3. 관찰과 실험**
- 현미경으로 세포 관찰
- 광합성 실험 (녹말 검출)
- 효소 실험 (소화)

### 지구과학: 지구와 우주 이해

지구과학은 지구와 우주를 다루는 학문입니다.

**핵심 개념**

**1. 지구 구조**
```
지각 (두께 평균 30km)
  - 대륙 지각: 두껍고 가벼움
  - 해양 지각: 얇고 무거움

맨틀 (깊이 2900km까지)
  - 대류 일어남 → 판 이동

외핵 (액체, 철과 니켈)
내핵 (고체, 철과 니켈)
```

**2. 판 구조론**
```
지각과 맨틀 윗부분 = 판(plate)
판의 이동:
  - 발산: 판이 멀어짐 (해령)
  - 수렴: 판이 가까워짐 (해구, 산맥)
  - 보존: 판이 어긋남 (단층)

지진, 화산: 판의 경계에서 발생
```

**3. 암석**
```
화성암: 마그마가 식어 굳음
  - 화강암 (천천히 식음, 알갱이 큼)
  - 현무암 (빨리 식음, 알갱이 작음)

퇴적암: 퇴적물이 쌓여 굳음
  - 사암, 이암, 석회암

변성암: 열과 압력으로 변함
  - 대리암 (석회암 변성)
  - 편마암 (화강암 변성)
```

**4. 날씨와 기후**
```
기압: 공기의 무게
  - 고기압: 하강 기류, 맑은 날씨
  - 저기압: 상승 기류, 흐린 날씨

전선:
  - 한랭 전선: 찬 공기가 밀고 들어옴 → 소나기
  - 온난 전선: 따뜻한 공기가 올라감 → 이슬비
```

**5. 태양계**
```
태양: 항성 (스스로 빛)
행성: 태양 주위 공전
  - 내행성: 수성, 금성
  - 외행성: 화성, 목성, 토성, 천왕성, 해왕성

달: 지구의 위성
  - 스스로 빛 X, 태양 빛 반사
  - 위상 변화: 초승달 → 상현달 → 보름달 → 하현달
```

**지구과학 학습 팁**

**1. 모형 만들기**
- 지구 내부 구조 단면 모형
- 판 경계 모형 (찰흙)
- 태양계 모형 (크기 비율)

**2. 관찰**
- 암석 표본 관찰
- 날씨 기록 (온도, 습도, 기압, 바람)
- 달 관찰 (위상 변화 기록)

**3. 시뮬레이션 활용**
- 판 이동 애니메이션
- 태양계 운동 시뮬레이션
- 날씨 예보 데이터 분석

**3. 문제 풀이 심화**

**4. 진로 탐색**

## 자주 묻는 질문 (FAQ)

### Q1. 과학은 무조건 암기해야 하나요?

**아닙니다! 이해가 먼저, 암기는 그 다음입니다.**

과학에서 암기할 것:
- 용어의 정의 (광합성, 중화 반응 등)
- 화학식 (H₂O, CO₂, NaCl 등)
- 기본 공식 (속도 = 거리/시간 등)

과학에서 이해할 것:
- 현상의 원리 (왜 그런 일이 일어나는가)
- 실험의 목적과 과정
- 개념 간의 관계

**올바른 학습 순서**
1. 원리 이해 (WHY)
2. 개념 정리 (WHAT)
3. 용어 암기 (TERM)
4. 문제 적용 (HOW)

예: 광합성
1. 이해: 왜 식물은 빛이 필요한가? → 에너지 필요
2. 개념: 빛 + 이산화탄소 + 물 → 포도당 + 산소
3. 암기: 6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂
4. 적용: 빛이 없으면? 이산화탄소가 없으면?

### Q2. 실험을 직접 못 하는데 어떻게 공부하나요?

**대안이 많습니다!**

**1. 학교 실험 시간 200% 활용**
- 적극적으로 참여
- 모든 과정 사진/동영상 촬영
- 자세한 실험 보고서 작성

**2. 가정에서 안전한 실험**
- 식초 + 베이킹소다 (중화 반응)
- 설탕물 농도별 계란 뜨기 (밀도)
- 양파 세포 관찰 (현미경)
- 빛의 굴절 (유리컵+물+빨대)

**3. 온라인 자료 활용**
- EBS 과학 실험 영상
- YouTube 과학 채널
- 가상 실험 시뮬레이션
- 3D 분자 모델링 프로그램

**4. 과학관 방문**
- 국립중앙과학관
- 서울시립과학관
- 대학 과학관
- 체험형 전시 적극 활용

### Q3. 과학 문제를 읽어도 무슨 말인지 모르겠어요.

**문제 독해력 부족입니다. 단계적으로 해결하세요.**

**1. 과학 용어 정리**
문제를 못 푸는 이유의 50%는 용어를 몰라서입니다.
- 교과서 핵심 용어 200개 암기
- 용어 사전 만들기
- 문제에 나온 모르는 용어는 즉시 정리

**2. 문제 분석 훈련**

**밑줄 그으며 읽기**
- 주어진 조건: 밑줄
- 구하는 것: 네모 박스
- 중요 숫자: 동그라미

예시 문제:
"질량이 **2kg**인 물체를 **10N**의 힘으로 밀었다.
이 물체의 □가속도□는 얼마인가?"

→ 주어진 것: 질량 2kg, 힘 10N
→ 구하는 것: 가속도
→ 공식: F = ma → a = F/m
→ 계산: a = 10/2 = 5 m/s²

**3. 그림 그리기**
- 문제 상황을 그림으로 표현
- 힘의 방향 화살표로 표시
- 물체의 움직임 그리기

**4. 쉬운 문제부터**
- 기본 문제 → 중급 문제 → 고난도 문제
- 한 번에 어려운 문제 풀려고 하지 말 것
- 단계적으로 난이도 상승

### Q4. 공식을 외워도 문제에 적용을 못해요.

**공식 암기만으로는 부족합니다. 활용 연습이 필요합니다.**

**1. 공식의 의미 이해**

나쁜 예: "속도는 v = s/t"
좋은 예: "속도는 '얼마나 빨리 이동했는가'를 나타낸다.
같은 거리를 짧은 시간에 가면 빠른 것이므로
속도 = 거리 / 시간이다."

**2. 공식 변형 연습**

기본: v = s / t

변형 1: s = v × t (거리 = 속도 × 시간)
변형 2: t = s / v (시간 = 거리 / 속도)

어느 것을 구하든 사용 가능하도록 연습

**3. 단위 확인**

속도: m/s (미터/초)
거리: m (미터)
시간: s (초)

잘못된 단위 사용 시 틀림
예: 시간을 분으로 주면 → 초로 변환

**4. 문제 유형별 연습**

같은 공식이라도 다양한 유형:
- 속도를 구하는 문제
- 거리를 구하는 문제
- 시간을 구하는 문제
- 그래프에서 속도 구하는 문제

### Q5. 시험 범위가 넓어서 뭘 공부해야 할지 모르겠어요.

**우선순위를 정해서 전략적으로 공부하세요.**

**시험 4주 전: 전체 훑기**
- 교과서 전체 1회독
- 핵심 개념 정리
- 중요 용어 암기
- 실험 정리

**시험 2주 전: 집중 학습**
- 중요 단원 집중 (보통 최근 배운 것)
- 문제집 풀이
- 오답 노트 정리
- 실험 원리 복습

**시험 1주 전: 최종 점검**
- 오답 노트 복습
- 헷갈리는 개념 정리
- 공식 암기 확인
- 예상 문제 풀이

**시험 전날: 가볍게**
- 정리 노트 훑어보기
- 공식 최종 확인
- 일찍 자기 (최소 7시간)

**과목별 공부 시간 배분**
- 약점 과목: 40%
- 중간 과목: 35%
- 강점 과목: 25%

예: 총 10시간 공부한다면
- 생물(약점): 4시간
- 화학(중간): 3.5시간
- 물리(강점): 2.5시간

### Q6. 과학은 잘하는데 수학을 못해서 문제가 안 풀려요.

**과학에 필요한 수학은 많지 않습니다.**

**중학 과학에 필요한 수학**

**1. 사칙연산**
- 속도 = 거리 / 시간
- 밀도 = 질량 / 부피

**2. 비례식**
- 압력과 부피 반비례 (보일 법칙)
- 온도와 부피 정비례 (샤를 법칙)

**3. 그래프 해석**
- x축, y축 의미 파악
- 기울기 의미 (속도, 가속도)
- 넓이 의미 (이동 거리)

**4. 백분율**
- 농도 계산 (질량 백분율)
- 효율 계산

**극복 방법**

**1. 공식에 숫자 바로 대입**
- 공식 쓰기
- 주어진 숫자 넣기
- 계산기로 계산

**2. 단위 확인**
- 단위를 함께 쓰면 실수 방지
- 예: v = 100m / 10s = 10 m/s

**3. 수학 기초만 보충**
- 분수 계산
- 소수 계산
- 비례식
- 그래프 읽기

→ 수학 전체가 아니라 필요한 부분만!

### Q7. 과학 선택과목을 어떻게 정하나요? (물리/화학/생물/지구과학)

**자신의 적성과 진로를 고려하세요.**

**물리 추천 대상**
- 수학을 좋아하고 잘함
- 논리적 사고 강함
- 공대, 기계, 전자 관련 진로
- 계산 문제 선호

**화학 추천 대상**
- 암기와 이해 균형
- 실험 좋아함
- 의대, 약대, 화공 관련 진로
- 반응식 쓰기 재미있음

**생물 추천 대상**
- 생명 현상에 관심
- 암기력 좋음
- 의대, 생명공학, 간호 관련 진로
- 그림 그리기 좋아함

**지구과학 추천 대상**
- 우주, 지구에 관심
- 폭넓은 내용 선호
- 기상, 해양, 천문 관련 진로
- 암기 + 이해 균형

**선택 팁**

**1. 중학교 성적 참고**
- 어느 분야가 재미있었나?
- 어느 분야 점수가 높았나?

**2. 고등학교 선생님 상담**
- 각 과목 특징 듣기
- 대학 입시 유불리
- 수업 난이도 파악

**3. 진로와 연결**
- 원하는 대학 학과의 필수 과목 확인
- 의대: 화학, 생물 필수
- 공대: 물리 필수
- 지구환경: 지구과학

**4. 조합도 중요**
- 물리 + 화학 (공대)
- 화학 + 생물 (의대)
- 생물 + 지구과학 (생명환경)

### Q8. 부모님이 과학을 도와주려면 어떻게 해야 하나요?

**직접 가르치는 것보다 환경과 동기를 제공하세요.**

**1. 과학 체험 기회**
- 과학관 방문 (월 1회)
- 자연사 박물관
- 동물원, 식물원
- 천문대 (별 관측)

**2. 과학 도서 및 영상**
- 과학 잡지 구독
- 과학 다큐멘터리 함께 시청
- 과학자 위인전
- 유튜브 과학 채널

**3. 일상 속 과학 대화**
- "왜 비누로 손을 씻으면 깨끗해질까?"
- "왜 겨울에 입김이 보일까?"
- "왜 냉장고는 시원할까?"

**4. 실험 지원**

- 실험 공간 제공
- 실험 보고서 격려
- 과학 대회 참가 지원

**5. 칭찬과 격려**
- 과학 성적보다 과정 칭찬
- "어떻게 그 답을 찾았니?" 질문
