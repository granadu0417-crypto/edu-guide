"과외를 언제까지 해야 하나요?"

**목표 달성하면 중단하세요.** 과외는 평생이 아니라 도구입니다.

## 과외 중단이 중요한 이유

**최종 목표:**
```
과외 의존 → 자기주도 학습

과외는 자전거 보조바퀴
언젠가는 떼야 함
```

**중단의 의미:**
- 졸업 = 실력이 올랐다
- 독립 = 혼자 할 수 있다
- 성장 = 다음 단계로

**장기 과외의 위험:**
```
❌ 자기주도력 저하
❌ 의존성 증가
❌ 비용 누적
❌ 스스로 공부 못 함

→ 적절한 시기에 중단 필수
```

## 중단해야 할 5가지 신호

### 신호 1: 목표를 달성했다

**명확한 목표 달성:**
```
목표: 3개월 내 수학 4등급 → 2등급
결과: 3개월 후 2등급 달성

→ 목표 달성! 중단 고려
```

**단계적 목표 달성:**
```
1단계 목표: 4등급 → 3등급 (달성)
2단계 목표: 3등급 → 2등급 (진행 중)

→ 1단계 달성 후 빈도 줄이기
→ 2단계 달성 후 완전 중단
```

**중단 판단:**
- 목표 등급 달성 ✓
- 3개월간 안정적 유지 ✓
- 다음 시험도 유지 ✓
→ 중단 고려

### 신호 2: 혼자 공부가 가능하다

**자기주도 학습 가능:**
```
✓ 스스로 계획 수립
✓ 혼자 문제 풀이
✓ 약점 스스로 파악
✓ 복습 습관 확립
✓ 질문 줄어듦 (이해했으니)
```

**테스트 방법:**
```
1주일 과외 없이 지내보기:
- 혼자 공부 가능한가?
- 진도 잘 나가나?
- 막히는 부분 스스로 해결?

→ 가능하면 중단 시기
```

**판단 기준:**
```
Before: "선생님 없으면 불안해요"
After: "혼자도 할 수 있어요"

→ 마음가짐의 변화
→ 중단 준비됨
```

### 신호 3: 6개월 이상 효과가 없다

**효과 없음의 신호:**
```
❌ 6개월 해도 성적 제자리
❌ 이해도 향상 없음
❌ 학습 태도 변화 없음
❌ 과외 시간 비효율
❌ 돈만 나감
```

**이유 파악:**
```
원인 1: 학생 태도 (숙제 안 함, 복습 안 함)
→ 해결: 태도 개선 or 중단

원인 2: 선생님 문제 (준비 부족, 불성실)
→ 해결: 선생님 교체 or 중단

원인 3: 과외 방식 안 맞음
→ 해결: 방식 변경 or 중단

원인 4: 학생에게 과외가 필요 없음
→ 해결: 즉시 중단
```

**중단 결정:**
- 6개월 효과 없음
- 원인 파악했지만 해결 안 됨
→ 과감히 중단

### 신호 4: 학생이 과외를 싫어한다

**싫어하는 신호:**
```
❌ 과외 시간 전 불평
❌ 과외 중 집중 안 됨
❌ 선생님과 대화 거부
❌ 숙제 계속 안 해옴
❌ "그만하고 싶어요" 표현
```

**원인 파악:**
```
일시적: 시험 스트레스, 컨디션 난조
→ 잠시 쉬기, 페이스 조정

지속적: 근본적으로 안 맞음
→ 중단 고려
```

**대화:**
```
부모: "과외 어때?"
학생: "싫어요, 그만하고 싶어요"
부모: "왜 싫어?"
학생: "선생님이 무서워요" / "지루해요" / "힘들어요"

→ 구체적 이유 파악
→ 해결 가능하면 조정
→ 불가능하면 중단
```

### 신호 5: 과부하 상태다

**과부하 신호:**
```
❌ 학원 + 과외 + 학교 = 과부하
❌ 휴식 시간 없음
❌ 항상 피곤
❌ 집중력 저하
❌ 성적 오히려 하락
❌ 건강 악화
```

**현실:**
```
많이 한다고 = 잘하는 것 X

오히려 과부하는
학습 효율 급감
번아웃 위험
```

**조정:**
```
1순위: 과외 빈도 줄이기 (주3 → 주1)
2순위: 학원 or 과외 중 하나 선택
3순위: 완전 휴식 기간 갖기

→ 건강이 최우선
```

## 계속해야 할 5가지 경우

### 경우 1: 목표 미달성, 효과 있음

**상황:**
```
목표: 4등급 → 2등급
현재: 3등급 (1단계 상승)
효과: 분명히 있음
기간: 3개월
```

**판단:**
→ 계속 필요
→ 목표까지 2-3개월 더

**조건:**
- 효과가 명확함
- 학생 동기 있음
- 방향 맞음

### 경우 2: 습관 형성 중

**상황:**
```
성적: 조금 오름 (4→3등급)
습관: 복습 습관 생김
태도: 적극적으로 변함
기간: 2개월
```

**판단:**
→ 계속 필요
→ 습관 완전 정착까지 2-3개월 더

**이유:**
- 습관 형성 중간에 중단하면 원점
- 최소 3-6개월은 지속해야 습관 고착

### 경우 3: 중요한 시기 (중3, 시험)

**상황:**
```
학년: 중3 (고등 준비)
시기: 중간고사 2달 전
상태: 관리 필요
```

**판단:**
→ 계속 필요
→ 중요한 시기 끝날 때까지

**중요 시기:**
- 중3 (특히 2학기)
- 시험 기간
- 방학 (약점 보완)

### 경우 4: 자기주도 아직 어려움

**상황:**
```
성적: 목표 달성 (2등급)
BUT 혼자 공부: 아직 어려움
계획: 스스로 못 세움
복습: 과외 없으면 안 함
```

**판단:**
→ 당장 중단은 이름
→ 점진적 중단 (빈도 줄이기)

**계획:**
```
1단계 (2개월): 주3 → 주2
2단계 (2개월): 주2 → 주1
3단계 (2개월): 시험 때만
4단계: 완전 중단
```

### 경우 5: 학생과 부모 모두 원함

**상황:**
```
학생: "더 하고 싶어요"
부모: "효과 있으니 계속해요"
선생님: "더 끌어올릴 수 있어요"
```

**판단:**
→ 계속 가능

**조건:**
- 명확한 다음 목표 설정
- 효과 지속적 점검
- 과부하 주의
- 점진적 중단 계획 수립

## 3개월 체크리스트

**매 3개월마다 점검:**

### A. 목표 달성도
```
□ 처음 목표 달성했나? (등급, 점수)
□ 중간 목표는? (이해도, 습관)
□ 예상대로 진행 중인가?
```

### B. 자기주도 학습 능력
```
□ 혼자 공부 시간 증가했나?
□ 스스로 계획 세우나?
□ 과외 없이 1주일 가능한가?
□ 질문이 줄었나? (이해했으니)
```

### C. 효과
```
□ 성적 올랐나?
□ 이해도 향상됐나?
□ 학습 태도 개선됐나?
□ 약점 보완됐나?
```

### D. 만족도
```
□ 학생 만족하나?
□ 부모 만족하나?
□ 선생님과 잘 맞나?
□ 과외 시간 알차나?
```

### E. 비용 대비 효과
```
□ 투자 대비 효과 있나?
□ 계속 감당 가능한가?
□ 다른 방법은 없나?
```

**판단:**
```
A, B 모두 Yes → 중단 고려
C, D 대부분 No → 중단 or 조정
E No → 중단 or 방법 변경
```

## 점진적 중단 3단계

### 1단계: 빈도 줄이기 (2개월)

**현재 → 조정:**
```
주 3회 → 주 2회
주 2회 → 주 1회
```

**목적:**
- 혼자 공부 시간 늘리기
- 자기주도 능력 키우기
- 의존도 낮추기

**관찰:**
```
✓ 혼자 공부 잘 하나?
✓ 성적 유지되나?
✓ 불편함 없나?

→ 괜찮으면 2단계로
```

### 2단계: 필요시만 (2-3개월)

**패턴:**
```
평소: 과외 없음
시험 2주 전: 과외 시작 (주2회)
시험 후: 다시 중단
```

**또는:**
```
평소: 혼자 공부
막히는 부분 생기면: 단기 과외 (1-2주)
해결되면: 다시 혼자
```

**효과:**
- 자기주도 연습
- 필요시만 도움
- 비용 절감
- 의존도 최소화

### 3단계: 완전 중단

**조건:**
```
✓ 목표 달성
✓ 혼자 공부 완전 가능
✓ 성적 안정적
✓ 자신감 있음
✓ 2단계 3개월 이상 성공
```

**중단 선언:**
```
학생: "이제 혼자 할 수 있어요"
부모: "충분히 성장했어"
선생님: "이제 독립할 때예요"

→ 과외 졸업!
```

**중단 후:**
```
1개월 후: 확인 (잘하고 있나?)
3개월 후: 점검 (성적 유지?)
6개월 후: 최종 평가

문제 생기면: 단기 재개 가능
```

## 잘못된 중단 vs 올바른 중단

### 잘못된 중단 (비추천)

**패턴 1: 조급한 중단**
```
❌ 1개월 만에 "효과 없다" 중단
→ 너무 이름, 최소 3개월

❌ 목표 반만 달성하고 중단
→ 중도 포기, 아까움
```

**패턴 2: 갑작스러운 중단**
```
❌ 점진적 과정 없이 갑자기 완전 중단
→ 학생 혼란, 성적 하락 위험
```

**패턴 3: 비용 때문에 급하게**
```
❌ "돈 아끼자" 급하게 중단
→ 목표 달성 직전일 수 있음
```

### 올바른 중단 (추천)

**패턴 1: 목표 달성 후**
```
✅ 목표 등급 달성
✅ 3개월 유지 확인
✅ 자신감 생김
→ 자연스러운 중단
```

**패턴 2: 점진적 중단**
```
✅ 3개월 단위 빈도 줄이기
✅ 학생 적응 확인
✅ 단계별 진행
→ 안정적 중단
```

**패턴 3: 명확한 기준**
```
✅ 체크리스트 기준 충족
✅ 학생/부모/선생님 합의
✅ 다음 계획 명확
→ 계획적 중단
```

## 중단 후 관리

### 중단 직후 (1개월)

**학생 할 일:**
```
✓ 과외 때 배운 것 계속 복습
✓ 과외 노트 주기적 보기
✓ 스스로 계획 세우기
✓ 혼자 문제 풀기
✓ 막히면 교과서/인강 활용
```

**부모 할 일:**
```
✓ 혼자 공부 잘 하는지 관찰
✓ 격려와 응원
✓ 성적 변화 모니터링
✓ 필요시 단기 과외 재개 준비
```

### 1-3개월 후

**점검 사항:**
```
□ 성적 유지되나?
□ 학습 습관 유지되나?
□ 혼자 공부 잘하나?
□ 자신감 있나?
```

**3개 이상 Yes:**
→ 성공적 중단! 계속 혼자

**2개 이하 Yes:**
→ 문제 발생, 원인 파악 필요

### 문제 발생 시

**성적 하락:**
```
원인 파악:
- 복습 안 했나?
- 혼자 공부 어려운가?
- 새로운 단원이 어려운가?

해결:
- 단기 과외 재개 (2-4주)
- 약점만 집중
- 다시 안정되면 중단
```

**학습 습관 무너짐:**
```
원인: 관리자(선생님) 없어지니 풀어짐

해결:
- 부모 관리 강화
- 스스로 계획표 작성
- 친구와 스터디
- 필요시 단기 재개
```

## 재개 vs 영구 중단

### 단기 재개가 필요한 경우

**상황:**
```
- 새 단원이 너무 어려움
- 시험 2주 앞인데 불안
- 특정 부분만 막힘
- 슬럼프
```

**방법:**
```
단기 과외 (2-4주)
→ 문제 해결
→ 다시 중단
→ 혼자 진행
```

**효과:**
- 문제 부분만 해결
- 비용 효율적
- 장기 의존 방지

### 영구 중단이 맞는 경우

**상황:**
```
✓ 목표 완전 달성
✓ 자기주도 완벽
✓ 성적 안정적 (6개월 이상)
✓ 학생 자신감 충만
✓ 혼자 충분히 가능
```

**선언:**
```
"과외 졸업!"

이제 진짜 혼자!
```

## 학년별 중단 타이밍

### 중1

**적정 과외 기간:**
```
6개월 - 1년

이유:
- 중학교 적응 기간
- 습관 형성 기간
- 기초 다지기 기간
```

**중단 시점:**
```
✓ 중학교 적응 완료
✓ 학습 습관 형성
✓ 기초 탄탄
✓ 2-3등급 안정적

→ 중1 말 or 중2 초 중단 고려
```

### 중2

**특징:**
```
- 자기주도 능력 형성 시기
- 과외 의존도 낮춰야 할 때
```

**중단 전략:**
```
✓ 목표 등급 달성하면
✓ 점진적 중단 시작
✓ 중2 말까지 완전 독립 목표

→ 중3은 혼자 가능하게
```

### 중3

**주의:**
```
❌ 중3은 함부로 중단 말 것
- 고등 준비 중요
- 내신 중요
- 관리 필요
```

**중단 가능:**
```
✓ 목표 고등 합격
✓ 1-2등급 안정적
✓ 혼자 확실히 가능

→ 고등 가기 전 중단
→ 고등은 새 출발
```

**계속 추천:**
```
- 목표 미달
- 불안정
- 고등 준비 필요

→ 중3 내내 or 고1까지
```

## 중단 결정 시 대화

### 학생과의 대화

**부모:**
```
"과외 어때? 계속 필요해?"
```

**학생 반응 A: "네, 더 필요해요"**
```
→ 이유 듣기
→ 구체적 목표 재설정
→ 기간 정하기 (예: 3개월 더)
```

**학생 반응 B: "이제 혼자 할 수 있을 것 같아요"**
```
→ 테스트 (1주일 혼자)
→ 잘하면 점진적 중단
```

**학생 반응 C: "잘 모르겠어요"**
```
→ 함께 체크리스트 작성
→ 객관적 판단
```

### 선생님과의 대화

**부모:**
```
"선생님, 저희 아이 이제 혼자 가능할까요?"
```

**선생님 의견:**
```
- 학생 현재 레벨
- 자기주도 능력
- 남은 약점
- 추천 계획
```

**함께 결정:**
- 계속 필요한가?
- 언제까지?
- 어떻게 중단?

## 자주 묻는 질문

### Q1. 목표 등급 달성했는데 혼자는 불안해요. 계속해야 하나요?

A. **점진적 중단을 추천합니다.**

**이유:**
```
목표 달성 = 실력 있음
불안 = 심리적 의존

→ 점진적 중단으로
   자신감 키우기
```

**계획:**
```
1단계 (2개월): 빈도 줄이기 (주3→주2)
→ "혼자도 괜찮네"

2단계 (2개월): 더 줄이기 (주2→주1)
→ "혼자 잘하네"

3단계: 완전 중단
→ "혼자 충분해"
```

**심리:**
```
갑자기 중단 = 불안 ↑
점진적 중단 = 자신감 ↑

천천히 적응하면
불안 사라짐
```

### Q2. 6개월 했는데 효과가 없어요. 바로 중단해야 하나요?

A. **원인을 먼저 파악하세요.**

**체크:**
```
□ 학생이 숙제를 하나요?
□ 복습을 하나요?
□ 과외 시간에 집중하나요?
□ 선생님이 성실한가요?
□ 과외 방식이 맞나요?
```

**판단:**
```
학생 문제 (숙제/복습 안 함):
→ 태도 개선 2개월 시도
→ 안 되면 중단

선생님 문제:
→ 선생님 교체
→ 2-3개월 재시도

근본적 문제 (과외 자체가 안 맞음):
→ 즉시 중단
→ 다른 방법 모색 (인강, 학원 등)
```

### Q3. 중3인데 과외 중단해도 될까요?

A. **신중해야 합니다. 조건을 확인하세요.**

**중단 가능:**
```
✓ 1-2등급 안정적 (6개월 이상)
✓ 혼자 공부 완벽
✓ 자신감 충만
✓ 목표 고등 합격 확실

→ 중단 가능
```

**중단 위험:**
```
❌ 3-4등급 불안정
❌ 혼자 공부 어려움
❌ 중요한 시험 앞둠
❌ 고등 준비 필요

→ 계속 추천
```

**타협안:**
```
빈도 줄이기:
주 3회 → 주 2회

또는 과목 줄이기:
수학+영어 → 수학만

→ 부담 줄이면서 관리
```

### Q4. 과외 중단 후 성적이 떨어지면 어떡하죠?

A. **자연스러운 일입니다. 단기 재개하세요.**

**현실:**
```
중단 직후 약간 하락 = 정상
(관리자 없어지니 일시적 하락)

1-2개월 후 회복 = 성공적 중단
지속적 하락 = 문제 있음
```

**대응:**
```
약간 하락 (1단계):
→ 지켜보기 (1-2개월)
→ 학생 스스로 회복 기회

큰 하락 (2단계 이상):
→ 단기 과외 재개 (2-4주)
→ 약점 보완
→ 다시 중단
```

**예방:**
```
✓ 점진적 중단 (갑자기 X)
✓ 중단 후에도 계속 복습
✓ 과외 노트 활용
✓ 부모 관심 유지
```

### Q5. 과외 완전히 중단한 후에도 가끔 상담 가능한가요?

A. **네, 가능합니다. 선생님과 합의하세요.**

**방식:**
```
Option 1: 월 1회 체크 상담
- 학습 상태 점검
- 질문 답변
- 방향 조언

Option 2: 필요시 단기 과외
- 막히는 부분 생기면
- 2-4회 단기 과외
- 해결 후 다시 혼자

Option 3: 시험 전만
- 평소: 혼자
- 시험 2주 전: 과외 재개
- 시험 후: 다시 중단
```

**비용:**
- 선생님과 협의
- 상담은 무료 or 소정
- 단기 과외는 정상

## 마무리

**과외 중단, 핵심 정리:**

**중단해야 할 때:**
```
✓ 목표 달성 (등급, 습관)
✓ 혼자 공부 가능
✓ 6개월 효과 없음
✓ 학생이 싫어함
✓ 과부하 상태
```

**계속해야 할 때:**
```
✓ 목표 미달, 효과 있음
✓ 습관 형성 중
✓ 중요한 시기 (중3, 시험)
✓ 자기주도 아직 어려움
✓ 모두 원함 + 효과 있음
```

**점진적 중단 3단계:**
```
1단계: 빈도 줄이기 (2개월)
2단계: 필요시만 (2-3개월)
3단계: 완전 중단

→ 천천히, 안정적으로
```

**최종 목표:**
```
과외 의존 → 자기주도

과외는 과정
자기주도가 목적

언젠가는 중단해야 함
그게 성공!
```

**판단 기준:**
```
3개월마다 체크리스트
→ 객관적 평가
→ 학생/부모/선생님 합의
→ 최선의 시기 결정

조급하지도, 늦지도 않게
적기에 중단!
```
