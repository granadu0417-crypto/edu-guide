"주중에 시간이 없어요. 주말만 가능한데요?"

**주말 과외도 충분히 효과적입니다.** 올바른 방법으로 활용하세요.

## 주말 과외란?

**정의:**
- 토요일 and/or 일요일
- 주 1-2회
- 긴 시간 (2-3시간)
- 집중 학습

**대상:**
```
✓ 주중 바쁜 학생
✓ 동아리/학원 많은 학생
✓ 집중력 있는 학생
✓ 자기주도 가능한 학생
```

## 주말 과외의 장점

### 1. 시간 여유
```
주중 과외: 1-1.5시간
주말 과외: 2-3시간
→ 깊이 있게 가능
```

### 2. 주중 부담 없음
- 학교 after 바로 집
- 동아리 활동 가능
- 친구와 시간
- 여유로운 주중

### 3. 집중 학습
```
긴 시간 → 한 주제 완전히
한 단원 집중
실전 문제 많이
```

### 4. 경제적
- 빈도 적음 (주 1-2회)
- 월 비용 절감
- 부담 적음

## 주말 과외의 단점

### 1. 빈도 부족
```
주 1-2회 = 연속성 떨어짐
복습 안 하면 까먹음
```

### 2. 컨디션 문제
```
주말 = 쉬고 싶음
피곤함
집중력 ↓ 가능성
```

### 3. 자기주도 필수
```
주중 혼자 공부 필수
복습 스스로
못 하면 효과 없음
```

### 4. 선생님 구하기 어려움
- 주말만 하는 선생님 적음
- 인기 선생님 주말 풀스케줄

## 주말 과외 추천 대상

**적합:**
- [ ] 주중 정말 바쁨 (학원 3-4개)
- [ ] 자기주도 가능
- [ ] 복습 잘 함
- [ ] 2-3등급

**3개 이상: 주말 과외 적합**

**부적합:**
- 4-5등급 (빈도 필요)
- 자기주도 어려움
- 복습 안 함
- 학습 습관 없음

## 효과적인 주말 과외 운영

### 토요일 과외 (2-3시간)

**1부: 복습 (30-40분)**
```
✓ 한 주 배운 것 점검
✓ 숙제 확인
✓ 틀린 문제 분석
✓ 질문 답변
```

**2부: 새 내용 (60-90분)**
```
✓ 새 개념 설명
✓ 예제 풀이
✓ 기본 문제
✓ 이해 확인
```

**3부: 문제 풀이 (40-60분)**
```
✓ 응용 문제
✓ 실전 연습
✓ 오답 분석
```

**마무리 (10분)**
- 오늘 정리
- 주중 과제
- 다음 주 계획

### 일요일 과외 (2시간)

**Option 1: 다른 과목**
```
토: 수학 (3시간)
일: 영어 (2시간)
```

**Option 2: 같은 과목 심화**
```
토: 수학 개념
일: 수학 문제 풀이
```

**Option 3: 복습 + 예습**
```
토: 이번 주 복습
일: 다음 주 예습
```

### 주중 과제 (필수)

**매일 30분-1시간:**
```
월: 토요일 내용 복습
화: 문제 풀이
수: 복습
목: 문제 풀이
금: 주말 과외 준비
```

**주중 과제 중요:**
- 주말만으로는 부족
- 복습 필수
- 연속성 유지

## 주말 과외 vs 주중 과외

### 빈도
```
주중: 주 2-3회
주말: 주 1-2회
```

### 시간
```
주중: 1-1.5시간/회
주말: 2-3시간/회

총 시간 비슷
```

### 효과
```
주중: 연속성 ↑, 습관 형성
주말: 집중도 ↑, 깊이 학습
```

### 비용
```
주중: 월 비용 높음
주말: 월 비용 낮음
```

### 추천
```
4-5등급: 주중 (빈도 필요)
2-3등급: 주말 OK
1등급: 주말 충분
```

## 과목별 주말 과외 전략

### 수학 주말 과외

**토요일 (3시간):**
```
30분: 지난 주 복습
90분: 새 단원 (개념 + 예제)
60분: 문제 풀이 (기본 + 응용)
```

**주중 과제:**
- 매일 계산 10문제
- 개념 복습
- 문제집 풀이

**효과:**
```
주 1회인데도
복습 잘 하면
주 2회 만큼 효과
```

### 영어 주말 과외

**토요일 (2-3시간):**
```
30분: 단어 테스트
60분: 문법 + 독해
60분: 문제 풀이
30분: 서술형 연습
```

**주중 과제:**
- 단어 매일 30개
- 문법 예문 암기
- 지문 복습

### 국어 주말 과외

**집중도:**
- 작품 분석 심화
- 비문학 독해 훈련
- 서술형 연습

**주중:**
- 교과서 복습
- 작품 읽기
- 어휘 정리

## 주말 과외 성공 조건

### 1. 철저한 주중 과제

**매일:**
```
✓ 30분-1시간 복습
✓ 과제 완수
✓ 틀린 문제 다시
```

**안 하면:**
- 주말에 다 까먹음
- 효과 없음
- 시간 낭비

### 2. 긴 시간 집중력

**2-3시간 집중:**
- 필수 능력
- 집중 못 하면 비효율
- 휴식 포함 (10분씩)

**훈련:**
- 처음엔 2시간부터
- 점진적 증가
- 휴식 적절히

### 3. 자기주도 학습

**주중 혼자:**
- 복습
- 문제 풀이
- 질문 정리

**불가능하면:**
- 주말 과외 부적합
- 주중 과외 추천

### 4. 계획성

**주간 계획:**
```
토: 과외 3시간
월-금: 매일 30분씩 복습
일: 다음 주 예습 or 추가 과외
```

## 주말 과외 시간 선택

### 토요일 오전 (9-12시)
**장점:**
- 컨디션 좋음
- 집중력 최고
- 오후 자유

**단점:**
- 늦잠 자고 싶음
- 일찍 일어나기 힘듦

**추천:**
- 아침형 인간
- 오후 활동 많은 학생

### 토요일 오후 (2-5시)
**장점:**
- 충분한 수면
- 점심 후 시작
- 편안함

**단점:**
- 점심 후 졸림
- 저녁 시간 촉박

**추천:**
- 대부분의 학생
- 가장 일반적

### 토요일 저녁 (6-9시)
**장점:**
- 하루 여유
- 저녁 활용

**단점:**
- 피곤함
- 집중력 ↓

**비추천:**
- 효율 낮음

### 일요일 활용

**오전/오후:**
- 토요일과 유사
- 다른 과목 or 보충

**주의:**
- 일요일 저녁 피하기
- 월요일 준비 시간 필요

## 주말만 vs 주말+주중 1회

### 주말만 (주 1-2회)

**총 시간:**
```
토 3시간 + 일 2시간 = 5시간/주
```

**장점:**
- 주중 여유
- 비용 저렴

**단점:**
- 연속성 부족
- 복습 필수

**추천:**
- 2-3등급
- 자기주도 가능

### 주말 + 주중 1회

**총 시간:**
```
주중 1회 (1.5h) + 토 (3h) = 4.5시간/주
```

**장점:**
- 연속성 ↑
- 효과 더 좋음
- 복습 체크

**단점:**
- 주중 부담
- 비용 증가

**추천:**
- 3-4등급
- 안정적 관리 필요

## 비용

**주말 과외 비용:**
- 시간당: 주중과 비슷
- 월 총액: 더 저렴 (빈도 적음)
- 가성비: 좋음 (자기주도 가능시)

**예시:**
```
주중 주 2회 (월 8회): 월 XX원
주말 주 1회 (월 4회): 월 XX원의 50%
```

## 성공 사례

### 사례 1: 주중 바쁜 학생

**학생:**
- 중2, 수학 2등급
- 학원 3개 + 동아리
- 주중 시간 없음

**과외:**
```
토요일 3시간
주중 매일 30분 복습
```

**결과:**
```
꾸준히 1등급 유지
주중 부담 없이
```

### 사례 2: 두 과목 주말 집중

**학생:**
- 중3, 수학 3등급, 영어 3등급
- 주말 집중 관리 원함

**과외:**
```
토: 수학 3시간
일: 영어 2시간
주중: 각 과목 30분씩 복습
```

**결과:**
```
3개월 후
수학 2등급, 영어 2등급
모두 1등급 상승
```

## 자주 묻는 질문

### Q1. 주 1회만으로 효과 있나요?

A. **자기주도가 되면 충분히 효과적입니다.**

**조건:**
```
✓ 주중 매일 30분 복습
✓ 과제 완수
✓ 자기주도 가능
✓ 2-3등급 이상
```

**효과:**
- 주중 과외 주 2회와 비슷
- 단, 복습 필수

**불가능:**
- 4-5등급 (빈도 필요)
- 복습 안 함
- 자기주도 어려움

### Q2. 토요일과 일요일 중 언제가 좋나요?

A. **토요일이 더 효과적입니다.**

**이유:**
```
토요일:
✓ 컨디션 좋음
✓ 일주일 피로 풀림
✓ 일요일 복습 가능

일요일:
△ 월요일 걱정
△ 주말 끝나는 아쉬움
△ 컨디션 애매
```

**추천:**
```
Best: 토요일 오전/오후
Good: 일요일 추가 (2과목)
Avoid: 일요일만
```

### Q3. 2-3시간 집중 가능할까요?

A. **연습하면 가능합니다.**

**단계적 접근:**
```
1개월: 2시간 (휴식 10분 1회)
2개월: 2.5시간 (휴식 10분 2회)
3개월: 3시간 (휴식 10분 2회)
```

**휴식 활용:**
- 50분 학습 → 10분 휴식
- 스트레칭, 물 마시기
- 재충전

**힘들면:**
- 2시간으로 시작
- 점진적 증가
- 무리하지 말기

### Q4. 주중 복습 꼭 해야 하나요?

A. **필수입니다. 안 하면 효과 없습니다.**

**이유:**
```
주말만: 일주일 간격
→ 다 까먹음
→ 다시 배우기
→ 비효율

주중 복습: 연속성 유지
→ 기억 유지
→ 쌓임
→ 효과적
```

**최소 복습:**
```
매일 20-30분
토요일 배운 것
문제 다시 풀기
개념 읽기
```

### Q5. 주말 과외 후 자기주도로 전환 가능한가요?

A. **네, 그것이 목표입니다.**

**과정:**
```
6개월: 주말 과외 + 주중 복습
      → 습관 형성
      → 자기주도 능력 ↑

1년 후: 과외 빈도 ↓
       → 월 1-2회 체크
       → 거의 자기주도

완전 독립!
```

**목표:**
- 과외 = 학습 방법 배우기
- 최종 = 혼자서도 가능
- 점진적 독립

## 마무리

**주말 과외, 이것만 기억하세요:**

**적합한 학생:**
```
✓ 주중 바쁨
✓ 자기주도 가능
✓ 2-3등급
✓ 복습 잘 함
```

**성공 조건:**
```
1. 주중 매일 복습 (필수!)
2. 긴 시간 집중력
3. 자기주도 학습
4. 계획적 관리
```

**효과:**
```
주 1-2회인데도
복습 잘 하면
주중 과외 만큼 효과
```

**주의:**
```
❌ 복습 안 하면 효과 없음
❌ 4-5등급엔 부적합
❌ 주말만 의존 (X)

✅ 주중 복습 필수
✅ 자기주도 훈련
✅ 점진적 독립 목표
```
