과외를 시작하면서 **흔히 저지르는 실수** 때문에 시간과 돈을 낭비하는 경우가 많습니다.

같은 실수를 반복하지 않도록, **23가지 흔한 실수**와 **올바른 해결 방법**을 정리했습니다.

이 글 하나로 **시행착오를 줄이고** 과외 효과를 극대화하세요.

## 과외 시작 전 실수

### 실수 1: 과외가 만능이라고 생각하기

**잘못된 생각:**
"과외만 하면 성적이 오를 거야"

**현실:**
- 과외는 도구일 뿐입니다
- 학생의 의지가 80%
- 과외는 20% 보조 역할

**해결:**
- 과외 전 학생 의지 확인
- "정말 성적 올리고 싶어?"
- 의지 없으면 과외 효과 없음

**올바른 기대:**
- 과외 = 방향 제시 + 약점 보완
- 실제 공부는 학생이 해야 함
- 숙제, 복습 철저히 해야 효과

### 실수 2: 너무 늦게 시작하기

**흔한 경우:**
- 시험 2주 전에 시작
- 성적표 받고 나서 시작
- 이미 뒤처진 후 시작

**문제점:**
- 기초부터 다시 쌓기 어려움
- 단기간 성과 기대 어려움
- 학생 스트레스 증가

**올바른 시작 시기:**
- 학기 시작 1개월 전 (방학 때)
- 또는 학기 시작과 동시에
- 최소 3개월 여유 두고 시작

**이유:**
- 기초 다지기: 1개월
- 실력 향상: 2개월
- 성적 반영: 3개월

### 실수 3: 학생 수준 무시하고 선행만 강요

**잘못된 접근:**
- 현재 70점인데 선행학습
- 기초 부족한데 진도 나가기
- 주변 친구 따라하기

**결과:**
- 기초 계속 부족
- 선행도 제대로 안 됨
- 시험 성적 그대로

**올바른 접근:**

**70점 미만:**
- 복습: 80%
- 선행: 20%
- 기초 완성 최우선

**70-85점:**
- 복습: 50%
- 선행: 50%
- 균형 있게

**85점 이상:**
- 복습: 30%
- 선행: 70%
- 선행으로 경쟁력

## 과외 선생님 선택 실수

### 실수 4: 학벌만 보고 선택하기

**흔한 착각:**
"서울대생이면 잘 가르치겠지"

**현실:**
- 학벌 ≠ 티칭 능력
- 공부 잘한 사람 ≠ 가르치기 잘하는 사람
- 의사소통 능력이 더 중요

**확인해야 할 것:**
1. 티칭 경험 (최소 1년)
2. 학생과의 케미
3. 설명 능력 (시범 수업)
4. 인내심과 성격

**시범 수업 때 체크:**
- 학생 눈높이에 맞춰 설명하는가
- 이해 안 되면 다시 설명하는가
- 학생이 편하게 질문하는가
- 숙제 확인을 꼼꼼히 하는가

### 실수 5: 첫 선생님과 바로 계약하기

**위험:**
- 다른 선생님 비교 없이 선택
- 나중에 후회해도 바꾸기 어려움
- 케미 안 맞아도 참고 계속

**올바른 방법:**

**1. 최소 3명 시범 수업:**
- 같은 주제로 설명 들어보기
- 학생 반응 확인
- 학부모도 참관

**2. 비교 포인트:**
- 설명의 명확성
- 학생과의 상호작용
- 숙제 확인 방식
- 전반적인 인상

**3. 학생 의견 최우선:**
"어느 선생님이 가장 이해하기 쉬웠어?"

### 실수 6: 너무 자주 선생님 바꾸기

**반대 극단:**
- 한 달마다 선생님 교체
- 조금만 불만 있으면 바로 교체
- 안정성 없음

**문제점:**
- 학생-선생님 신뢰 형성 안 됨
- 매번 새로 적응해야 함
- 학습 연속성 끊김

**올바른 기준:**

**바꿔야 할 때:**
- 3개월 지나도 효과 전혀 없음
- 학생이 선생님 만나기 싫어함
- 숙제 확인 안 함, 불성실

**참고 계속할 때:**
- 조금씩이라도 성적 상승
- 학생이 선생님 좋아함
- 성실하게 가르침

**최소 기간:**
3개월은 지켜봐야 효과 판단 가능

## 과외 방식 실수

### 실수 7: 과외 시간에 숙제만 하기

**흔한 장면:**
- 학생: 숙제 풀이
- 선생님: 옆에서 지켜보다가 틀리면 설명

**문제점:**
- 과외 시간 낭비
- 숙제는 혼자서 해야 함
- 자기주도 학습 능력 안 생김

**올바른 과외 시간 활용:**

**과외 전 (숙제는 미리):**
- 학생이 숙제 먼저 풀어옴
- 모르는 문제만 표시

**과외 시간:**
- 모르는 문제 설명 (30%)
- 새로운 개념 학습 (40%)
- 문제 풀이 연습 (20%)
- 다음 주 계획 (10%)

### 실수 8: 선생님이 다 풀어주기

**잘못된 방식:**
- 선생님: 문제 읽고 바로 풀이
- 학생: 받아적기만
- 이해했는지 확인 안 함

**결과:**
- 그 자리에서는 이해한 것 같음
- 혼자 풀면 못 풀어
- 시험에서 못 풀어

**올바른 방식:**

**1. 힌트만 주기:**
"이 문제는 어떤 공식을 써야 할까?"

**2. 학생이 먼저 풀기:**
"너가 먼저 풀어봐. 막히면 물어봐"

**3. 틀려도 혼자 끝까지:**
"틀렸지만 과정은 좋았어. 여기서 실수했네"

**4. 다시 풀어보기:**
"이제 비슷한 문제 혼자 풀어봐"

### 실수 9: 진도만 나가고 복습 안 하기

**흔한 패턴:**
- 매주 새로운 단원
- 지난주 내용 확인 안 함
- 학기 말에 다 까먹음

**문제점:**
- 단기 기억에만 저장
- 시험 때 기억 안 남
- 기초 계속 부족

**황금 비율:**

**매 과외 시간 (90분 기준):**
```
- 지난 시간 복습 테스트: 15분
- 새로운 개념: 40분
- 문제 풀이: 25분
- 오늘 배운 것 정리: 10분
```

**주기적 복습:**
- 1주 후: 1회 복습
- 2주 후: 1회 복습
- 시험 전: 전체 복습

## 학습 관리 실수

### 실수 10: 숙제 확인 안 하기

**부모님:**
"선생님한테 맡겼으니 알아서 하겠지"

**학생:**
"안 해도 아무 말 안 하시네"

**결과:**
- 숙제 안 함
- 과외 효과 0%

**올바른 관리:**

**부모님:**
- 매주 과외 후 선생님과 통화
- "이번 주 숙제 다 했나요?"
- "다음 주 숙제는 뭔가요?"

**선생님:**
- 숙제 확인 철저히
- 안 하면 과외 시간에 지금 하기
- 부모님께 보고

**학생:**
- 숙제 노트 따로 만들기
- 매일 조금씩 하기
- 모르면 바로 표시

### 실수 11: 과외만 믿고 학교 수업 소홀

**학생 생각:**
"어차피 과외에서 다시 배우니까 학교 수업은 대충"

**문제점:**
- 학교 수업이 기본
- 과외는 보충일 뿐
- 내신은 학교 시험

**올바른 태도:**

**학교 수업 (80%):**
- 집중해서 듣기
- 필기 철저히
- 모르는 것 바로 질문

**과외 (20%):**
- 학교에서 이해 안 된 부분
- 심화 문제
- 오답 정리

**선순환:**
학교 → 이해 안 됨 → 과외에서 해결 → 학교 수업 더 잘 이해 → 성적 상승

### 실수 12: 과외 과목만 공부하기

**흔한 경우:**
- 수학 과외: 수학만 공부
- 영어, 국어, 과학, 사회 방치
- 전체 평균 하락

**문제점:**
- 내신은 전 과목 평균
- 한 과목만 높아도 의미 없음

**올바른 시간 배분:**

**수학 과외 받을 때:**
- 수학: 40%
- 영어: 25%
- 국어: 15%
- 과학: 10%
- 사회: 10%

**시험 기간:**
- 모든 과목 균등하게
- 과외 받는 과목 조금 더

### 실수 13: 과외 횟수만 늘리기

**잘못된 생각:**
"주 2회보다 주 5회가 더 좋겠지"

**문제점:**
- 스스로 공부할 시간 없음
- 과외 의존도 증가
- 자기주도 학습 능력 저하

**적정 횟수:**

**일반 학생:**
- 주 2-3회 (과목당 주 1-2회)
- 자율 학습 시간 확보 필수

**시험 2주 전:**
- 주 3-4회로 증가 가능
- 단기간만

**과외보다 중요한 것:**
- 혼자 공부하는 시간
- 스스로 생각하는 시간
- 문제 풀이 연습 시간

## 목표 설정 실수

### 실수 14: 비현실적 목표 세우기

**흔한 목표:**
- "이번 시험 100점!"
- "3개월 안에 1등급!"
- "학교 1등!"

**문제점:**
- 달성 불가능
- 실패 시 좌절
- 동기 부여 하락

**현실적 목표:**

**현재 50점대:**
- 단기 (3개월): 65-70점
- 중기 (6개월): 75-80점
- 장기 (1년): 80-85점

**현재 70점대:**
- 단기: 80점
- 중기: 85점
- 장기: 90점 (1등급)

**핵심:**
- 한 번에 10-15점 상승 목표
- 단계적으로 접근
- 작은 성공 경험 쌓기

### 실수 15: 결과만 목표로 삼기

**잘못된 목표:**
- "90점 받기"
- "1등급 받기"

**문제:**
- 과정은 없고 결과만
- 구체적 실행 방법 없음

**올바른 목표:**

**결과 목표 + 과정 목표:**
- 결과: 90점
- 과정:
  - 매일 2시간 공부
  - 학교 프린트 3회 반복
  - 오답노트 완성
  - 과외 숙제 100% 완료

**측정 가능한 목표:**
- "이번 주 수학 문제집 20문제"
- "영어 단어 100개 암기"
- "과학 실험 5개 복습"

## 시험 기간 실수

### 실수 16: 시험 2주 전에도 선행만 하기

**잘못된 선택:**
- 시험 범위 복습보다 선행 진도
- "선행이 더 중요해"

**결과:**
- 시험 망침
- 선행도 제대로 안 됨

**올바른 접근:**

**시험 3주 전부터:**
- 선행 중단
- 시험 범위 집중
- 과외도 시험 대비로 전환

**시험 후:**
- 선행 재개
- 다음 학기 준비

### 실수 17: 벼락치기로 커버하려고 하기

**학생:**
"평소엔 대충, 시험 1주 전에 몰아서!"

**문제점:**
- 전체 범위 소화 불가능
- 단기 기억, 시험 후 망각
- 극심한 스트레스
- 건강 악화

**올바른 준비:**

**시험 3주 전:**
- 전체 범위 파악
- 계획 수립

**시험 2주 전:**
- 1회독 완료
- 약점 파악

**시험 1주 전:**
- 오답 정리
- 반복 학습

**시험 전날:**
- 가벼운 복습
- 일찍 취침

### 실수 18: 과외 없는 날은 공부 안 하기

**학생:**
"과외 있는 날만 공부"

**결과:**
- 주 2회 과외 = 주 2일만 공부
- 나머지 5일 놀음
- 성적 안 오름

**올바른 학습:**

**과외 있는 날:**
- 과외 90분
- 복습 30분
- 숙제 60분

**과외 없는 날:**
- 스스로 학습 120분
- 숙제 60분
- 복습 30분

**핵심:**
- 매일 공부하는 습관
- 과외는 보조 수단
- 스스로 공부가 더 중요

## 성적 관리 실수

### 실수 19: 성적표 받고 방치하기

**흔한 패턴:**
- 성적표 받음
- 잠깐 실망 or 기쁨
- 그대로 방치

**문제:**
- 왜 잘했는지/못했는지 분석 안 함
- 다음 시험도 같은 실수 반복

**올바른 성적 관리:**

**시험 직후:**
1. 오답 정리 (1주 이내)
2. 왜 틀렸는지 분석
3. 개선 방안 도출

**성적표 받은 후:**
1. 과목별 분석
2. 강점/약점 파악
3. 다음 시험 전략 수립

**과외 선생님과:**
1. 성적 분석 회의
2. 다음 목표 설정
3. 커리큘럼 조정

### 실수 20: 1등급 받고 방심하기

**흔한 경우:**
- 한 번 1등급 받음
- "이제 됐다" 방심
- 다음 시험 2-3등급

**이유:**
- 공부량 줄임
- 과외 빠짐
- 숙제 소홀

**1등급 유지 전략:**
- 매 시험 같은 루틴 유지
- 방심하지 않기
- 과외 계속 참여
- 심화 학습 지속

### 실수 21: 성적 안 오르면 바로 과외 그만두기

**조급한 판단:**
- 1개월 과외했는데 성적 안 오름
- "효과 없네" 바로 중단

**문제:**
- 3개월은 지켜봐야 효과 나타남
- 너무 빠른 포기

**올바른 판단 시기:**

**1개월 후:**
- 효과 판단 불가
- 계속 진행

**3개월 후:**
- 조금이라도 상승: 계속
- 전혀 변화 없음: 원인 분석

**원인 분석:**
- 학생이 숙제 안 함?
- 선생님과 케미 안 맞음?
- 목표가 비현실적?

**대응:**
- 숙제 관리 강화
- 또는 선생님 교체
- 또는 목표 재조정

## 학부모 실수

### 실수 22: 과외 선생님한테 모든 걸 맡기기

**잘못된 태도:**
"과외 시켰으니 알아서 하겠지"

**문제:**
- 학생 관리 소홀
- 숙제 확인 안 함
- 진도 파악 안 함

**올바른 학부모 역할:**

**주 1회 선생님과 통화:**
- 이번 주 진도
- 학생 이해도
- 다음 주 계획
- 숙제 완료 여부

**매일 학생 확인:**
- "오늘 뭐 배웠어?"
- "숙제 다 했어?"
- "모르는 거 있어?"

**월 1회 삼자 면담:**
- 학생 + 선생님 + 부모님
- 한 달 평가
- 다음 달 목표

### 실수 23: 학생 의견 무시하고 부모 뜻대로만

**흔한 경우:**
- 학생: "이 선생님 싫어요"
- 부모: "그래도 계속 해"

**결과:**
- 학생 동기 부여 0
- 과외 효과 없음
- 돈 낭비

**올바른 접근:**

**선생님 선택:**
- 학생 의견 최우선
- "어느 선생님이 좋아?"

**과목 선택:**
- 학생과 상의
- "어느 과목이 가장 힘들어?"

**횟수 결정:**
- 학생 의견 반영
- "주 2회가 좋아, 3회가 좋아?"

**핵심:**
- 학생이 주체
- 부모는 조력자
- 강요하면 역효과

## 자주 묻는 질문 (FAQ)

### Q1. 과외 효과가 없는데 계속해야 하나요?

A. 기간과 원인에 따라 다릅니다.

**1개월 후:**
- 아직 효과 판단 시기 아님
- 계속 진행

**3개월 후에도 효과 없으면:**
1. 학생이 숙제를 안 한다 → 숙제 관리 강화
2. 선생님과 케미 안 맞는다 → 선생님 교체
3. 목표가 비현실적이다 → 목표 재조정
4. 과외 방식이 잘못됐다 → 방식 변경

**6개월 후에도 전혀 변화 없으면:**
- 과외 중단 고려
- 다른 방법 모색 (학원, 인강 등)

### Q2. 선생님을 바꾸고 싶은데 언제 바꿔야 하나요?

A. 명확한 이유가 있을 때만 바꾸세요.

**바꿔야 할 때:**
- 학생이 선생님 만나기 싫어함
- 숙제 확인 안 함, 불성실
- 3개월 지나도 효과 전혀 없음
- 약속 시간 자주 어김

**바꾸지 말아야 할 때:**
- 조금이라도 성적 상승
- 학생이 선생님 좋아함
- 아직 1-2개월밖에 안 됨

**바꿀 때 주의:**
- 새 선생님도 시범 수업
- 왜 바꾸는지 명확히
- 같은 실수 반복하지 않기

### Q3. 과외와 학원 중 뭐가 나을까요?

A. 학생 성향에 따라 다릅니다.

**과외가 나은 경우:**
- 개인별 약점이 명확
- 1:1 설명 필요
- 자기주도 학습 가능
- 시간 유연성 필요

**학원이 나은 경우:**
- 또래 친구들과 경쟁 필요
- 체계적인 커리큘럼 선호
- 자기 관리 어려움

**병행:**
- 학원: 진도
- 과외: 개인별 보충
- 단, 시간 부담 주의

### Q4. 과외 횟수는 몇 회가 적당한가요?

A. 학생 수준과 과목에 따라 다릅니다.

**일반적 권장:**
- 주 2-3회 (과목당 1-2회)
- 90-120분씩

**성적대별:**
- 70점 미만: 주 3-4회
- 70-85점: 주 2-3회
- 85점 이상: 주 1-2회

**시험 기간:**
- 평소보다 1-2회 증가 가능
- 단기간만

**주의:**
- 너무 많으면 자율 학습 시간 부족
- 스스로 공부하는 시간이 더 중요

### Q5. 숙제를 안 하는데 어떻게 하나요?

A. 단계적으로 접근하세요.

**1단계: 이유 파악**
- 너무 많아서? → 양 줄이기
- 너무 어려워서? → 난이도 조정
- 귀찮아서? → 동기 부여

**2단계: 관리 강화**
- 매일 부모님 확인
- 선생님 철저히 체크
- 안 하면 과외 시간에 하기

**3단계: 보상/제재**
- 다 하면 칭찬 + 보상
- 안 하면 과외 시간 손해
- 지속 시 과외 중단 경고

### Q6. 성적이 떨어졌는데 과외 문제인가요?

A. 여러 원인을 종합적으로 봐야 합니다.

**확인할 것:**
1. 학생이 숙제를 했는가?
2. 학교 수업을 집중했는가?
3. 시험 범위를 다 공부했는가?
4. 컨디션은 괜찮았는가?
5. 시험 난이도는 어땠는가?

**과외 문제일 수도:**
- 시험 대비를 안 함
- 선행만 하고 복습 안 함
- 선생님이 불성실

**학생 문제일 수도:**
- 숙제 안 함
- 학교 수업 소홀
- 복습 안 함

### Q7. 1등급인데 과외가 필요한가요?

A. 필요합니다. 하지만 방식이 달라야 합니다.

**1등급 유지 과외:**
- 개념 설명 X, 심화 O
- 기본 문제 X, 고난도 O
- 양 X, 질 O

**횟수:**
- 평소: 주 1-2회
- 시험 기간: 주 2-3회

**내용:**
- 심화 개념 탐구
- 고난도 문제 풀이
- 실수 방지 전략
- 변별력 문제 대비

### Q8. 과외를 언제 그만둬야 하나요?

A. 목적을 달성했거나 효과가 없을 때입니다.

**그만둬도 되는 경우:**
- 목표 성적 달성 + 안정적 유지
- 스스로 공부 가능
- 학습 습관 완전히 정착

**그만둬야 하는 경우:**
- 6개월 이상 효과 전혀 없음
- 학생이 극도로 싫어함
- 경제적 부담이 너무 큼

**계속해야 하는 경우:**
- 조금씩 성적 상승 중
- 학생이 도움 받고 있다고 느낌
- 목표 아직 미달성

### Q9. 과외 선생님과 갈등이 생겼는데요?

A. 즉시 소통하세요.

**흔한 갈등:**
- 시간 약속 어김
- 숙제 확인 소홀
- 진도가 너무 빠름/느림
- 학생과 케미 안 맞음

**해결 방법:**
1. 선생님과 직접 대화
2. 문제점 명확히 지적
3. 개선 요청
4. 1개월 지켜보기

**개선 안 되면:**
- 선생님 교체
- 같은 실수 반복하지 않기

### Q10. 과외비가 부담스러운데 효과를 높이려면?

A. 효율을 극대화하세요.

**횟수 줄이기:**
- 주 3회 → 주 2회로
- 대신 자율 학습 시간 증가

**시간 효율:**
- 숙제 미리 하고 가기
- 모르는 것만 질문
- 과외 시간 낭비 금지

**학원 병행 고려:**
- 학원: 진도 (저렴)
- 과외: 보충 (주 1회만)

**인강 활용:**
- 개념 학습은 인강
- 오답 정리만 과외

## 실패 사례 vs 성공 사례

### 실패 사례 1: 과외만 믿고 숙제 안 함

**배경:**
- 중1 남학생
- 수학 과외 주 3회
- 성적: 60점

**문제:**
- 과외 때만 공부
- 숙제 전혀 안 함
- 복습 안 함

**6개월 후:**
- 성적: 62점 (2점 상승)
- 과외비만 낭비
- 과외 중단

**교훈:**
- 과외는 도구일 뿐
- 숙제와 복습이 핵심
- 학생의 노력 없으면 효과 0

### 성공 사례 1: 숙제 철저 + 복습으로 1등급

**배경:**
- 중2 여학생
- 수학 과외 주 2회
- 성적: 75점

**방법:**
- 숙제 100% 완료
- 매일 30분 복습
- 과외 시간 집중
- 학교 수업 충실

**6개월 후:**
- 성적: 92점 (1등급)
- 17점 상승
- 과외 계속 유지

**성공 요인:**
- 성실한 숙제 완료
- 꾸준한 복습
- 과외 + 자율 학습 병행

### 실패 사례 2: 너무 자주 선생님 교체

**배경:**
- 중3 남학생
- 6개월 동안 선생님 4명 교체
- 성적: 70점 → 68점

**문제:**
- 조금만 불만 있으면 교체
- 매번 새로 적응
- 학습 연속성 없음

**결과:**
- 성적 오히려 하락
- 과외 효과 0
- 신뢰 관계 형성 실패

**교훈:**
- 최소 3개월은 지켜봐야 함
- 너무 자주 바꾸면 역효과
- 인내심 필요

### 성공 사례 2: 한 선생님과 2년 유지

**배경:**
- 중1 여학생
- 한 선생님과 2년 지속
- 성적: 65점 → 85점

**방법:**
- 선생님과 신뢰 구축
- 약점 정확히 파악
- 맞춤형 커리큘럼
- 꾸준한 발전

**2년 후:**
- 20점 상승
- 1등급 근접
- 학습 습관 완성

**성공 요인:**
- 선생님과의 신뢰
- 장기적 관점
- 꾸준함

## 마무리: 실수는 피할 수 있습니다

과외에서 **흔히 저지르는 23가지 실수**를 정리했습니다.

**핵심 포인트:**

1. **과외 = 만능 아님**: 학생 의지가 80%
2. **충분한 시간**: 최소 3개월
3. **현실적 목표**: 한 번에 10-15점
4. **선생님 신중히**: 3명 이상 비교
5. **숙제 철저히**: 효과의 핵심
6. **복습 필수**: 단기가 아닌 장기 기억
7. **학교 수업 집중**: 기본이 중요
8. **모든 과목 균형**: 전체 평균 관리
9. **학부모 역할**: 적절한 관심과 지원
10. **학생 주체성**: 의견 존중

**기억하세요:**
- 실수는 누구나 합니다
- 중요한 건 **같은 실수 반복하지 않기**
- 이 글을 체크리스트로 활용하세요
- **시행착오를 줄이고 효과를 극대화**하세요

**과외는 도구일 뿐, 주인공은 학생입니다!**
