"시험 2주 남았는데 과외 할 수 있나요?"

**가능합니다!** 단기 집중으로 효과를 보세요.

**2-4주 안에 최대 효과를 내는 전략을 알려드립니다.**

## 단기 집중 과외란?

### 정의

**짧은 기간, 높은 밀도:**
- 기간: 2-4주
- 빈도: 주 4-6회
- 시간: 1회 1-2시간
- 목표: 명확하고 구체적

**장기 과외와의 차이:**
```
장기 과외: 6개월-1년, 근본 실력 향상
단기 과외: 2-4주, 즉각적 목표 달성
```

### 특징

**집중도 높음:**
- 매일 or 격일
- 한 가지 목표
- 빠른 피드백
- 밀착 관리

**목표 명확:**
```
❌ "성적 올리기" (막연)
✅ "이번 시험 80점 받기" (구체적)
```

## 단기 집중 과외 유형

### 1. 시험 대비 (2-3주)

**시기:**
- 중간/기말고사 2-3주 전
- 학교 진도 끝난 후
- 시험 직전

**목표:**
```
✓ 시험 범위 정리
✓ 예상 문제 풀이
✓ 약점 집중 보완
✓ 점수 10-20점 향상
```

**진행:**
- 주 4-5회
- 1회 1.5-2시간
- 시험 범위 전체 커버

**효과:**
```
Before: 70점 (3등급)
After: 80-85점 (2등급)
= 10-15점 향상
```

### 2. 방학 집중 (4주)

**시기:**
- 여름/겨울 방학
- 한 달 기간
- 다음 학기 준비

**목표:**
```
✓ 한 과목 완성
✓ 특정 단원 마스터
✓ 선행 학습
✓ 약점 보완
```

**진행:**
- 주 5-6회
- 1회 2시간
- 체계적 진도

**효과:**
```
✓ 한 학기 선행 가능
✓ 약점 단원 완전 정복
✓ 자신감 향상
```

### 3. 특정 단원 집중 (1-2주)

**시기:**
- 특정 단원 막힘
- 그 단원만 집중
- 짧고 강하게

**목표:**
```
예: "일차함수만 완전히"
    "영어 시제만 정복"
    "서술형만 집중"
```

**진행:**
- 주 5회
- 1회 1시간
- 해당 부분만

**효과:**
```
막혔던 부분 뚫림
→ 전체 이해도 ↑
```

### 4. 긴급 상황 (1주)

**시기:**
- 시험 1주 전
- 정말 급한 경우
- 최소한의 준비

**목표:**
```
✓ 핵심만 빠르게
✓ 출제 예상 문제
✓ 최소한 점수 확보
```

**진행:**
- 매일
- 1회 2시간
- 실전 대비

**주의:**
```
❌ 완벽한 준비 기대 (X)
✅ 최선의 응급 처치 (O)
```

## 단기 과외의 장단점

### 장점

**1. 즉각적 효과**
```
2-3주 만에 결과
시험 점수로 확인
빠른 성취감
```

**2. 목표 명확**
```
"이번 시험"
"이 단원"
→ 집중 가능
```

**3. 경제적**
```
2-4주만 비용
총액 부담 ↓
필요할 때만
```

**4. 동기 부여**
```
짧은 기간
→ 집중력 유지
→ 끝까지 완수
```

### 단점

**1. 근본 해결 아님**
```
단기 점수 ↑
but 기초 여전히 약함
→ 다음에 또 어려움
```

**2. 고강도**
```
매일 과외
→ 힘들 수 있음
→ 번아웃 위험
```

**3. 제한적 범위**
```
많은 것 불가능
한두 가지만
욕심 부리면 실패
```

**4. 선생님 구하기 어려움**
```
단기만 하는 선생님
→ 적음
→ 비용 높을 수 있음
```

```
문법 문제 만점
```

### 사례 3: 긴급 대비 (1주)

**상황:**
- 중1, 수학 시험 1주 전
- 준비 전혀 안 함
- 50점대 예상

**과외 내용:**
```
매일 2시간, 7일
핵심만 빠르게
기출 문제 집중
```

**결과:**
```
시험: 72점
최소한 3등급 확보
완벽하진 않지만 성공
```

## 자주 묻는 질문

### Q1. 2주만으로 효과 있나요?

A. **네, 조건이 맞으면 10-20점 향상 가능합니다.**

**조건:**
```
✓ 기본 실력 있음 (4등급 이상)
✓ 목표 명확
✓ 집중적 학습 (주 4-5회)
✓ 복습 철저
```

**예상 효과:**
```
Before: 70-75점
After: 80-90점
= 10-15점 ↑
```

**불가능한 경우:**
```
5등급 → 1등급 (불가능)
기초 없음 → 완성 (불가능)
```

### Q2. 시험 며칠 전부터 시작해야 하나요?

A. **최소 2주, 권장 3주 전입니다.**

**이상적:**
```
3주 전: 충분한 준비
→ 개념 + 문제 + 실전
→ 여유 있게
```

**최소:**
```
2주 전: 최소 기간
→ 빡빡하지만 가능
→ 집중 필수
```

**1주 전:**
```
응급 처치만 가능
완벽한 준비 어려움
최소한 확보
```

### Q3. 매일 해야 하나요?

A. **최소 주 3-4회, 권장 주 5-6회입니다.**

**빈도별 효과:**
```
주 2회: 효과 적음 (단기엔 부족)
주 3-4회: 최소 효과
주 5-6회: 이상적
매일: 최고 (번아웃 주의)
```

**추천:**
```
시험 대비: 주 5회
방학 집중: 주 5-6회
특정 단원: 주 4-5회
```

### Q4. 단기 과외 후에는 어떻게 하나요?

A. **목표에 따라 다릅니다.**

**시험 대비 후:**
```
Option 1: 중단 (목표 달성)
Option 2: 정기 과외로 전환
Option 3: 다음 시험 전 again
```

**방학 집중 후:**
```
Option 1: 중단 (자기주도)
Option 2: 정기 과외 전환 (지속)
Option 3: 다른 과목 시작
```

**추천:**
```
단기 = 응급 처치

근본 해결 필요하면:
→ 정기 과외로 전환

일시적 도움이면:
→ 중단 OK
```

### Q5. 단기 과외로 습관도 잡히나요?

A. **습관은 어렵습니다. 장기 과외가 필요합니다.**

**단기 (2-4주):**
```
✓ 점수 향상
✓ 약점 보완
✓ 동기 부여
✗ 습관 형성 (어려움)
```

**습관 형성:**
```
최소 3개월 필요
매일 공부 루틴
꾸준한 복습
→ 장기 과외 추천
```

**결론:**
```
단기 = 점수
장기 = 습관
```

## 마무리하며

**단기 집중 과외, 이것만 기억하세요:**

**핵심 원칙:**
```
1. 목표 명확히
2. 핵심만 집중
3. 높은 빈도 (주 4-5회)
4. 철저한 복습
5. 끝까지 집중
```

**가능한 것:**
```
✓ 10-20점 향상
✓ 1등급 상승
✓ 약점 보완
✓ 시험 범위 정리
✓ 특정 단원 마스터
```

**불가능한 것:**
```
✗ 완전한 변화 (5→1등급)
✗ 기초 완성
✗ 습관 형성
✗ 모든 과목
```

**최적 시기:**
```
✓ 시험 2-3주 전
✓ 방학 4주
✓ 특정 단원 막힐 때
```

**성공 조건:**
```
✓ 기본 실력 있음 (4등급 이상)
✓ 명확한 목표
✓ 집중력
✓ 철저한 복습
✓ 학생 각오
```

**조언:**
```
단기 과외 = 특효약
응급 상황에 효과적

하지만:
근본 치료는 장기 과외
상황에 맞게 선택하세요
```
