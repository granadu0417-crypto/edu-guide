"친구랑 같이 과외 받으면 안 될까요?"

**그룹과외도 좋은 선택**입니다. 단, 조건이 맞아야 합니다.

**친구와 함께하는 과외의 모든 것을 알려드립니다.**

## 그룹과외란?

### 정의

**2명 이상의 학생이 함께:**
- 한 선생님에게
- 동시에 수업
- 같은 내용 학습

### 유형별 분류

**2:1 과외:**
- 학생 2명 : 선생님 1명
- 가장 효율적
- 친구, 형제자매

**3:1 소그룹:**
- 학생 3명 : 선생님 1명
- 작은 학습 그룹
- 비슷한 수준

**4:1 그룹:**
- 학생 4명 : 선생님 1명
- 소형 학원과 비슷
- 경제적

## 그룹과외 vs 1:1 비교

### 공통점

**둘 다:**
- 선생님의 직접 지도
- 질문 가능
- 맞춤 학습 (정도 차이)

### 차이점

**1:1 과외:**
```
장점: 완전 맞춤, 집중 관리
단점: 비용 높음, 경쟁 없음
```

**그룹과외:**
```
장점: 경제적, 경쟁 자극
단점: 개인 맞춤 ↓, 시간 분산
```

## 그룹과외의 장점

### 1. 경제적

**비용 절감:**
```
1:1 과외 100%라면
2:1 과외: 50-70%
3:1 과외: 40-50%
4:1 과외: 30-40%
```

**예시:**
- 1:1 시간당 6만원
- 2:1 시간당 4만원 (각자)
- 월 8회 기준 16만원 절약

**부담 감소:**
- 가정 경제적 부담 ↓
- 장기간 지속 가능
- 여러 과목 가능

### 2. 경쟁과 자극

**긍정적 경쟁:**
- "친구는 푸는데 나만 못 풀어?"
- 서로 자극
- 동기 부여 ↑

**협력 학습:**
- 서로 가르쳐주기
- 함께 토론
- 설명하며 이해 ↑

**예시:**
"철수가 이번에 80점 받았대. 나도 열심히 해야지!"
→ 건강한 경쟁심

### 3. 사회성 발달

**함께 배우기:**
- 외롭지 않음
- 친구와 함께
- 재미있음

**소통 능력:**
- 질문하며 배우기
- 설명하며 익히기
- 협력하는 법

**스트레스 감소:**
- 혼자가 아님
- 부담 덜함
- 편안한 분위기

### 4. 지속 가능성

**꾸준함:**
- 친구와 약속 → 빠지기 어려움
- 서로 체크
- 장기 지속

**습관 형성:**
- 규칙적 만남
- 학습 루틴
- 책임감

## 그룹과외의 단점

### 1. 개인 맞춤 부족

**1:1과 비교:**
```
1:1: 100% 나에게 집중
2:1: 50% 나에게
3:1: 33% 나에게
```

**문제:**
- 내 약점만 집중 어려움
- 진도 조율 필요
- 개인 상담 시간 ↓

### 2. 실력 차이 문제

**문제 상황:**
- 한 명은 잘하고
- 한 명은 못 하면
- 수업 진행 곤란

**결과:**
- 잘하는 학생: 지루함
- 못하는 학생: 못 따라감
- 둘 다 불만

**중요:**
- 비슷한 실력 필수
- 등급 차이 1등급 이내

### 3. 시간 분배

**질문 시간:**
```
1:1: 질문 즉시 답변
그룹: 차례대로, 기다림
```

**설명 시간:**
- 각자에게 설명
- 시간 분산
- 집중도 ↓

### 4. 갈등 가능성

**친구 간 문제:**
- 실력 차이로 불편
- 숙제 안 하는 친구
- 태도 문제

**예시:**
"친구는 맨날 숙제 안 해와서 선생님이 그 친구한테만 시간 쓰셔요."

## 그룹과외 추천 대상

### 추천하는 경우

**다음 중 3개 이상이면 추천:**

- [ ] 비슷한 실력의 친구 있음 (등급 차이 1등급 이내)
- [ ] 경제적 부담이 큼
- [ ] 경쟁 동기가 필요함
- [ ] 혼자 하면 외로움
- [ ] 내신 2-3등급
- [ ] 친구와 관계 좋음

**특히 좋은 경우:**
- 쌍둥이, 형제자매
- 절친한 친구
- 비슷한 학습 스타일

### 비추천하는 경우

**다음 중 2개 이상이면 1:1 추천:**

- [ ] 실력 차이 큼 (2등급 이상 차이)
- [ ] 기초가 매우 약함 (4-5등급)
- [ ] 개인 맞춤 관리 필요
- [ ] 친구와 자주 싸움
- [ ] 집중력 매우 낮음
- [ ] 경쟁이 스트레스

## 성공하는 그룹과외 조건

### 1. 비슷한 실력 (최중요)

**기준:**
- 내신 등급 차이 1등급 이내
- 진도 비슷
- 이해 속도 비슷

**체크 방법:**
- 최근 시험 성적 비교
- 학교 진도 확인
- 문제 풀이 속도

**예시:**
```
OK: 둘 다 70-80점대
OK: 둘 다 3등급
NG: 한 명 90점, 한 명 60점
```

### 2. 좋은 관계

**필수 요소:**
- 서로 존중
- 신뢰 관계
- 협력 가능

**피해야 할 관계:**
- 경쟁이 과열
- 자주 싸움
- 한 명이 무시

**테스트:**
"같이 공부하면 스트레스 받니?"
→ YES면 그룹 비추천

### 3. 명확한 규칙

**수업 규칙:**
```
1. 지각 금지
2. 숙제 필수
3. 집중 약속
4. 질문 차례대로
5. 서로 존중
```

**위반 시:**
- 경고 1회
- 반복 시 개별 과외로

### 4. 좋은 선생님

**그룹 지도 능력:**
- 시간 배분 능력
- 공정한 관리
- 갈등 조율
- 수준 조절

**확인 질문:**
"그룹과외 경험 많으신가요?"
"실력 차이 나면 어떻게 하시나요?"

## 그룹 구성 방법

### 친구 찾기

**어디서 찾을까:**
- 같은 학교 친구
- 같은 학원 친구
- 이웃 친구

**조건 맞추기:**
- 실력 비슷한지
- 시간 맞는지
- 부모님들 동의

### 첫 상담

**선생님과:**
1. 두 학생 모두 참석
2. 각자 실력 체크
3. 가능 여부 판단
4. 규칙 약속

**체험 수업:**
- 1-2회 필수
- 분위기 확인
- 효과 판단

## 그룹과외 진행 방법

### 일반 수업 (60분 기준)

**10분: 숙제 확인**
- 두 학생 번갈아
- 틀린 문제 체크

**20분: 새 개념 (함께)**
- 두 학생 함께 설명
- 질문 받기
- 이해 확인

**20분: 문제 풀이 (개별)**
- 각자 문제 풀기
- 선생님 순회
- 개별 지도

**10분: 정리 및 숙제**
- 오늘 배운 것 정리
- 숙제 내주기
- 다음 시간 약속

### 실력 차이 대처

**조금 차이 날 때:**
- 쉬운 문제: 함께
- 어려운 문제: 잘하는 학생만
- 못하는 학생: 기초 보충

**많이 차이 날 때:**
- 그룹 해체 고려
- 또는 수준 다른 문제
- 번갈아 개별 지도

## 그룹과외 효과

### 기대 효과

**3개월 후:**
- 성적 상승 (1등급 정도)
- 학습 습관 형성
- 서로 자극 효과

**6개월 후:**
- 안정적 내신 관리
- 건강한 경쟁 관계
- 자기주도 학습 시작

**조건:**
- 두 학생 모두 노력
- 규칙 준수
- 선생님 역량

### 효과 판단 (3개월 후)

**체크리스트:**
- [ ] 두 학생 모두 성적 올랐나?
- [ ] 서로 자극 받나?
- [ ] 관계 좋나?
- [ ] 갈등 없나?
- [ ] 수업 만족하나?

**3개 이상 YES: 효과 있음 ✅**
**2개 이하: 문제 있음 → 조정 필요**

## 그룹과외 주의사항

### 갈등 관리

**미리 예방:**
- 규칙 명확히
- 공정한 시간 배분
- 서로 존중 분위기

**갈등 발생 시:**
- 즉시 대화
- 선생님 중재
- 해결 안 되면 해체

### 무임승차 방지

**문제:**
"한 친구만 열심히 하고, 다른 친구는 놀아요."

**해결:**
- 개별 숙제 체크
- 개별 테스트
- 노력 안 하면 개별로

### 비교하지 않기

**부모가 하지 말아야 할 것:**
- "너는 왜 친구만 못하니?"
- "친구는 올랐는데 너는..."

**대신:**
- "너도 많이 늘었어"
- 과거 자신과 비교
- 과정 칭찬

## 2:1 vs 3:1 vs 4:1

### 2:1 과외 ⭐⭐⭐ (가장 추천)

**장점:**
- 1:1과 비슷한 효과
- 경제적
- 관리 가능

**추천:**
- 대부분의 경우
- 친구, 형제자매

### 3:1 소그룹

**장점:**
- 더 경제적
- 다양한 자극

**단점:**
- 관리 어려움
- 시간 부족

**추천:**
- 2-3등급 학생
- 자기주도 가능

### 4:1 그룹

**장점:**
- 가장 경제적

**단점:**
- 소형 학원과 비슷
- 개인 맞춤 어려움

**추천:**
- 경제적 이유
- 기본 실력 있음

## 자주 묻는 질문

### Q1. 2:1이 1:1의 절반 효과인가요?

A. **아닙니다. 70-80% 효과입니다.**

이유:
- 경쟁 자극 효과
- 협력 학습 효과
- 설명하며 이해 ↑

조건 맞으면 1:1만큼 좋을 수 있습니다.

### Q2. 실력 차이 얼마까지 가능한가요?

A. **등급 차이 1등급 이내 권장합니다.**

예시:
- OK: 둘 다 3등급
- OK: 2등급 + 3등급
- NG: 2등급 + 4등급

2등급 이상 차이 나면 어렵습니다.

### Q3. 친구가 아니어도 되나요?

A. **비슷한 수준이면 괜찮습니다.**

중요한 것:
- 친분보다 실력
- 서로 존중
- 협력 가능

모르는 사람끼리도 가능합니다.

### Q4. 중간에 한 명 빠지면 어쩌죠?

A. **계약 내용에 따라 다릅니다.**

일반적으로:
- 1:1로 전환 (비용 ↑)
- 다른 학생 충원
- 종료

미리 약속하세요.

### Q5. 형제자매도 가능한가요?

A. **학년 차이 없으면 최고입니다.**

장점:
- 같은 집에서
- 이동 편리
- 경제적

주의:
- 학년 차이 크면 어려움
- 같은 학년이면 최고

## 마무리하며

**그룹과외는 조건만 맞으면 훌륭한 선택입니다.**

**핵심 조건:**
1. 비슷한 실력 (필수)
2. 좋은 관계
3. 명확한 규칙
4. 경제적 효율

**잘 활용하면 1:1 못지않은 효과를 볼 수 있습니다.**
