# 중학생 어휘력 확장하기

어려운 단어가 많나요? 한자어부터 문맥까지, 1:1 집중 확장으로 어휘력을 폭발적으로 성장시키세요.

## 어휘력의 중요성

풍부한 어휘력은 독해와 작문의 핵심입니다. 중학생 시기는 어휘력이 급격히 확장되는 결정적 시기이며, 이때 체계적으로 어휘를 쌓아야 고등학교 진학 후 학습이 수월합니다.

### 어휘력이 모든 과목의 기초인 이유

**1. 독해력의 핵심**
- 글의 내용 이해는 어휘력에서 시작
- 모르는 단어가 많으면 문맥 파악 불가능
- 한 문장에 3개 이상 모르는 단어 → 독해 실패

**2. 작문 능력 향상**
- 풍부한 어휘로 정확한 표현 가능
- 같은 의미도 다양하게 표현
- 설득력 있는 글쓰기

**3. 전 과목 성적 향상**
- 수학: 문제 해석 능력
- 과학/사회: 전문 용어 이해
- 영어: 한국어 어휘력이 영어 어휘력의 기초

### 중학생 어휘력의 현실

**초등 vs 중등 어휘 수준**
- 초등 6학년: 약 3,000~5,000개 어휘
- 중학 3학년: 약 10,000~15,000개 어휘 필요
- 3년간 2~3배 증가해야 함

**어휘력 부족 시 문제점**
- 교과서 내용 이해 어려움
- 시험 문제 독해 실패
- 자신감 저하 → 학습 의욕 감소
- 고등학교 진학 후 더 큰 어려움

### 어휘력 확장의 황금기

중학교 시기는 뇌의 언어 발달이 활발한 시기입니다.
- **추상적 사고 발달**: 복잡한 개념어 이해 가능
- **장기 기억 형성**: 체계적 학습으로 평생 어휘 습득
- **다양한 지문 노출**: 교과서, 독서로 자연스런 어휘 확장

## 1:1 어휘력 확장의 장점

집단 수업에서는 평균적인 어휘 수준에 맞춰 진행하지만, 1:1 지도는 개인의 어휘 수준을 정확히 진단하고 맞춤형 확장 전략을 제공합니다.

### 수준별 어휘 학습

**정확한 어휘 수준 진단**
- 현재 보유 어휘 개수 측정
- 한자어, 고유어, 외래어 비율 분석
- 교과별 전문 용어 숙지도 확인
- 독서 수준에 따른 어휘 수준 파악

**맞춤형 어휘 목록 제공**
- 학년별 필수 어휘 우선 학습
- 취약한 분야 집중 보완
- 독서 수준에 맞는 어휘 확장
- 과목별 전문 용어 정리

**단계별 학습 설계**
- 기초: 일상 어휘 → 학습 어휘
- 중급: 교과서 어휘 → 심화 어휘
- 고급: 추상 어휘 → 학술 어휘

### 한자어 집중 학습

한국어 어휘의 60% 이상이 한자어입니다. 한자 구조를 이해하면 어휘력이 폭발적으로 증가합니다.

**한자 부수 학습**
- 214개 부수의 의미 이해
- 부수로 한자의 뜻 유추
- 예: 氵(물 수) → 江(강), 河(하), 海(바다)

**자주 쓰이는 한자 300개 집중**
- 중학생 필수 한자 300개
- 이 한자로 수천 개 어휘 이해 가능
- 예: 敎(가르칠 교) → 교육, 교사, 교과서, 교양

**한자어 조합 원리**
- 2자 한자어: 敎育(가르칠 교 + 기를 육)
- 3자 한자어: 敎育法(교육 + 법)
- 4자 성어: 敎學相長(가르치고 배우며 함께 성장)

### 문맥 속 어휘 학습

단어만 외우면 금방 잊어버립니다. 문맥 속에서 어휘를 익혀야 오래 기억됩니다.

**실제 지문 활용**
- 교과서 지문에서 어휘 추출
- 독서 작품에서 문맥 파악
- 신문 기사로 시사 어휘 확장

**문맥으로 의미 추론**
- 앞뒤 문장으로 뜻 유추
- 접속어로 관계 파악
- 전체 주제로 의미 확정

**어휘 노트 작성**
```
[어휘: 편견(偏見)]
- 뜻: 한쪽으로 치우친 생각
- 한자: 偏(치우칠 편) + 見(볼 견)
- 문맥: "우리는 편견 없이 사람을 대해야 한다."
- 유사어: 선입견, 고정관념
- 반의어: 공정, 객관
- 예문: 성별에 대한 편견을 버려야 평등한 사회가 된다.
```

### 개인별 암기 전략

암기 방법은 사람마다 다릅니다. 자신에게 맞는 방법을 찾아야 합니다.

**시각형 학습자**
- 어휘 카드 만들기
- 마인드맵으로 어휘 연결
- 그림으로 의미 표현

**청각형 학습자**
- 어휘 소리내어 읽기
- 예문 녹음 후 반복 듣기
- 어휘 노래로 만들기

**운동감각형 학습자**
- 손으로 쓰면서 암기
- 몸으로 단어 표현
- 문장 만들기

### 2단계: 어원으로 의미 추론

단어의 어원을 알면 처음 보는 단어도 의미를 유추할 수 있습니다.

**한자어 어원 분석**

**예시 1: 문화(文化)**
- 文(무늬 문): 글자, 문양, 아름다움
- 化(될 화): 변하다, 되다
- 합쳐진 뜻: "문명으로 변화시키다" → 인간의 정신적 성취

**예시 2: 민주주의(民主主義)**
- 民(백성 민): 사람들
- 主(주인 주): 주인
- 主(주인 주): 중심
- 義(옳을 의): 올바른 도리
- 합쳐진 뜻: "백성이 주인이 되는 올바른 도리"

**고유어 어원**

한국어 고유어는 대부분 소리와 모양을 본뜬 의성어, 의태어에서 출발했습니다.

- 꼬불꼬불: 구부러진 모양 (ㄱ, ㅂ은 꺾이는 느낌)
- 반짝반짝: 빛나는 모습 (ㅂ, ㅈ은 짧고 강한 느낌)
- 엉금엉금: 느리게 기는 모습 (ㅇ은 둥글고 느린 느낌)

**외래어 어원**

영어 단어의 어근을 알면 수많은 단어를 유추할 수 있습니다.

**자주 쓰이는 어근**
```
vis, vid = 보다
  - vision(시력), video(비디오), visit(방문)

port = 나르다
  - transport(운송), export(수출), import(수입)

dict = 말하다
  - dictionary(사전), predict(예언), contradict(반박)

scrib, script = 쓰다
  - describe(설명), manuscript(원고), subscribe(구독)
```

**접두사 활용**
```
pre- (미리)
  - preview(미리보기), predict(예측), prepare(준비)

re- (다시)
  - return(돌아오다), review(복습), rewrite(다시쓰다)

un-, in-, dis- (반대)
  - unable(불가능), incorrect(부정확), disagree(반대)

sub- (아래)
  - subway(지하철), submarine(잠수함), subtitle(자막)
```

### 3단계: 문맥 속에서 어휘 학습

단어는 문맥 속에서 익혀야 오래 기억되고 실제로 사용할 수 있습니다.

**문맥으로 의미 추론하기**

**예시 지문**
```
"그는 친구의 거짓말에 분개했다.
평소 믿었던 친구가 자신을 속였다는 사실에
견딜 수 없는 배신감을 느꼈다."
```

**분개(憤慨) 의미 추론 과정**
1. 상황: 친구가 거짓말을 함
2. 감정: "견딜 수 없는 배신감"
3. 추론: 분개 = 몹시 화내고 분한 마음
4. 한자: 憤(성낼 분) + 慨(슬퍼할 개) = 분노와 서러움

**문맥 단서 찾기**

**1. 동의어/유사어**
"그는 매우 당황했다. 즉, 황당하여 어찌할 바를 몰랐다."
→ 당황 = 황당하여 어쩔 줄 모름

**2. 반의어/대조**
"그는 관대한 사람이 아니라 인색한 성격이었다."
→ 인색 ↔ 관대 (반대)

**3. 예시**
"그는 다양한 취미를 가지고 있다. 예를 들어 독서, 등산, 사진 등이다."
→ 취미 = 여가에 즐기는 활동

**4. 인과관계**
"가뭄이 계속되었다. 그 결과 농작물이 말라 죽었다."
→ 가뭄 = 비가 오지 않아 농사가 어려움

**교과서 지문 활용**

**국어 교과서**
- 문학 작품: 감정, 심리 어휘 풍부
- 설명문: 학술, 전문 어휘 습득
- 논설문: 추상 개념 어휘 확장

**과학 교과서**
- 실험: 관찰, 측정 관련 어휘
- 개념: 과학 전문 용어
- 탐구: 논리, 추론 어휘

**사회 교과서**
- 역사: 시대, 제도 어휘
- 지리: 자연, 지형 어휘
- 일반사회: 정치, 경제, 사회 용어

### 4단계: 실전 적용 - 독해와 작문

어휘를 알기만 해서는 부족합니다. 독해와 작문에 실제로 사용해야 내 것이 됩니다.

**독해 실전**

**전략 1: 모르는 어휘 표시**
지문을 읽으면서 모르는 단어에 밑줄 긋기
→ 문맥으로 추론 → 사전 확인 → 어휘 노트 정리

**전략 2: 핵심 어휘 파악**
- 제목과 첫 문장의 주요 어휘
- 반복되는 핵심 단어
- 지문의 주제를 담은 추상 어휘

**전략 3: 문제 해결**
시험 문제의 선택지도 어휘력 확장 기회
→ 틀린 선택지의 어휘도 모두 정리

**작문 실전**

**전략 1: 어휘 대체하기**
```
[기본] "좋다" → [확장]
  - 훌륭하다, 우수하다, 뛰어나다, 탁월하다
  - 바람직하다, 적절하다, 적합하다

[기본] "나쁘다" → [확장]
  - 부정적이다, 부적절하다, 불합리하다
  - 문제가 있다, 바람직하지 않다
```

**전략 2: 정확한 어휘 선택**
- "매우 좋다" (X) → "탁월하다" (O)
- "많이 화났다" (X) → "격분했다" (O)
- "조금 슬프다" (X) → "서글프다" (O)

**전략 3: 주제별 어휘 활용**
```
[환경 주제]
오염, 보존, 지속가능, 생태계, 순환,
재생에너지, 탄소배출, 온실가스

[교육 주제]
함양, 육성, 계발, 배양, 습득,
자질, 역량, 소양, 인성
```

## 영역별 어휘 학습법

### 국어 어휘 확장

**1. 고유어**

한국어의 뿌리인 고유어는 감정과 일상을 섬세하게 표현합니다.

**감정 표현 고유어**
```
기쁨: 반갑다, 즐겁다, 신나다, 흐뭇하다
슬픔: 서럽다, 서글프다, 애달프다, 처연하다
화남: 분하다, 억울하다, 약오르다, 치밀다
놀람: 깜짝, 덜컥, 화들짝, 기겁하다
```

**모양 표현 고유어**

## 어휘 암기 효율 극대화 방법

### 기억 곡선 활용

**에빙하우스 망각 곡선**
- 학습 후 1시간: 50% 망각
- 학습 후 1일: 70% 망각
- 학습 후 1주: 80% 망각

**효율적 복습 주기**
```
Day 1: 새 어휘 20개 학습
Day 2: 1차 복습 (20개)
Day 4: 2차 복습 (20개)
Day 8: 3차 복습 (20개)
Day 16: 4차 복습 (20개)
Day 32: 5차 복습 (20개)
→ 장기 기억 완성
```

### 연상 기법

**1. 이미지 연상**
- 단어를 그림으로 표현
- 예: "분개" → 화난 얼굴 그림

**2. 이야기 만들기**
- 여러 단어를 스토리로 연결
- 예: "편견 없는 관대한 사람이 부당한 차별에 분개했다"

**3. 어원 연상**
- 한자 구조로 의미 기억
- 예: 편견(偏見) = 한쪽으로 치우쳐(偏) 봄(見)

### 다중 감각 활용

**시각 + 청각 + 운동**
1. 어휘 카드를 **보면서** (시각)
2. 소리내어 **읽고** (청각)
3. 손으로 **쓴다** (운동)

→ 뇌의 여러 영역 활성화로 기억 강화

### 게임으로 즐기기

**혼자 하는 게임**
- 어휘 십자말풀이
- 한자 조합 만들기
- 동음이의어 찾기

**함께 하는 게임**
- 끝말잇기 (한자어만, 4글자만 등)
- 어휘 스피드 퀴즈
- 문장 만들기 대결

## 자주 묻는 질문 (FAQ)

### Q1. 어휘를 외워도 금방 잊어버려요. 어떻게 하나요?

**원인: 단순 암기만 하고 활용하지 않기 때문**

**해결 방법**

**1. 문맥 속에서 암기**
단어만 외우지 말고 문장 전체를 외우세요.
```
[나쁜 예] "분개 = 분하고 개탄함"
[좋은 예] "그는 부당한 처사에 분개했다."
```

**2. 즉시 활용**
- 그날 배운 어휘로 일기 쓰기
- 대화에서 사용해보기
- 작문 시험에 적용

**3. 복습 주기 준수**
- 당일 복습 (100%)
- 1일 후 복습 (70%)
- 3일 후 복습 (50%)
- 1주일 후 복습 (30%)
→ 장기 기억으로 전환

**4. 관련 어휘 함께 암기**
- 유의어: 분개 ↔ 격분, 분노
- 반의어: 분개 ↔ 관대, 너그러움
- 관련어: 분개 → 억울, 부당, 항의

**시험 직전 (2주~당일)**
- 핵심 어휘만 최종 점검
- 헷갈리는 어휘 정리
- 자주 틀리는 한자어 확인

### Q6. 어휘력과 독해력, 무엇을 먼저 키워야 하나요?

**둘 다 동시에!**

어휘력과 독해력은 상호보완 관계입니다.

**선순환 구조**
```
어휘력 증가
    ↓
독해 가능한 지문 늘어남
    ↓
더 많은 독서
    ↓
더 많은 어휘 습득
    ↓
어휘력 증가 (반복)
```

**구체적 전략**

**어휘력 80% + 독해력 20% 수준일 때**
→ 독해 연습 집중 (어휘는 충분, 활용 부족)

**어휘력 50% + 독해력 50% 수준일 때**
→ 균형있게 둘 다 (가장 이상적)

**어휘력 30% + 독해력 70% 수준일 때**
→ 어휘 암기 집중 (어휘 부족으로 독해 한계)

**자가 진단**
□ 지문에서 모르는 단어가 한 문장에 3개 이상인가?
  → YES: 어휘력 집중
  → NO: 독해력 집중

### Q7. 영어 단어도 많이 외워야 하는데, 동시에 가능한가요?

**가능합니다!**

오히려 한국어 어휘력이 영어 어휘력을 돕습니다.

**한국어 → 영어 연결**

**1. 한자어 = 영어 어근**
```
敎育(교육) = education
  - 敎(가르치다) = educate
  - 育(기르다) = cultivate

進步(진보) = progress
  - 進(나아가다) = proceed, advance
  - 步(걸음) = step
```

**2. 개념 이해**
한국어로 개념을 정확히 이해한 후
→ 영어 단어는 쉽게 매칭

**3. 추상 어휘**
한국어로 추상 개념을 먼저 잡고
→ 영어 단어 암기

**효율적 방법**
- 아침: 한국어 어휘 10개 (20분)
- 저녁: 영어 단어 20개 (20분)
- 주말: 둘 다 복습 (1시간)

### Q8. 부모님이 도와주려면 어떻게 해야 하나요?

**가정에서 할 수 있는 어휘력 향상 활동**

**1. 함께 뉴스 보기**
- 저녁 9시 뉴스 시청
- 어려운 용어 설명해주기
- 뉴스 내용 대화로 정리

**2. 대화 시 정확한 어휘 사용**
- "그거" 대신 정확한 명칭
- "좋다" 대신 구체적 표현
- "나쁘다" 대신 적절한 어휘

**3. 책 선물**
- 생일, 명절에 책 선물
- 서점 함께 가기
- 독서 후 대화 나누기

**4. 한자 학습 지원**
- 한자 카드 게임
- 한자능력검정 응시 지원
- 생활 속 한자 찾기 놀이

**5. 칭찬과 격려**
- 새로운 어휘 사용 시 칭찬
- 어휘 노트 작성 격려
- 작은 성취 인정

**하지 말아야 할 것**
- 무조건 외우라고 강요 (X)
- 못 외우면 꾸짖기 (X)
- 본인은 어휘를 틀리면서 아이만 비판 (X)

**부모님도 함께**
- 부모님도 어휘 노트 작성
- 가족 어휘 게임
- 한 달에 한 번 "이달의 어휘" 선정
