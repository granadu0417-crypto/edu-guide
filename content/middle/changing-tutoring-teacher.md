"선생님을 바꿔야 할 것 같은데 어떻게 해야 하나요?"

**즉시 교체 vs 기다리기**, 상황을 정확히 판단하세요.

## 교체 판단이 어려운 이유

**감정 vs 효과:**
```
학생: "선생님 싫어요"
BUT 성적: 오르고 있음

OR

학생: "선생님 좋아요"
BUT 성적: 6개월째 제자리

→ 어떻게 판단?
```

**타이밍의 딜레마:**
```
너무 빨리 교체: 적응 기회 없음
너무 늦게 교체: 시간/돈 낭비

→ 적절한 시점은?
```

**관계의 부담:**
- 선생님께 어떻게 말씀?
- 학생 상처 안 줄까?
- 새 선생님은 나을까?

## 즉시 교체해야 할 5가지 경우

### 1. 선생님의 명백한 문제

**불성실:**
```
❌ 지각 잦음 (월 2회 이상)
❌ 결석 잦음 (사전 공지 없이)
❌ 준비 없이 오심
❌ 수업 중 휴대폰 봄
❌ 시간 채우기식 수업
```

**부적절한 언행:**
```
❌ 학생 무시/비하
❌ 화 자주 냄
❌ 욕설 or 폭언
❌ 부적절한 신체 접촉
❌ 사생활 과도한 질문
```

**실력 부족:**
```
❌ 문제 못 풀음
❌ 설명이 틀림
❌ "다음에 알아올게요" 반복
❌ 교재 의존만
```

**판단:**
→ 위 항목 1개라도 해당하면
→ **즉시 교체**
→ 개선 기회 줄 필요 없음

### 2. 학생이 극도로 싫어함

**신호:**
```
❌ 과외 시간 전 울음/불안
❌ "절대 안 하고 싶어요" 반복
❌ 과외 시간에 공황
❌ 선생님 오면 방 안 나옴
❌ 심한 스트레스 (두통, 복통)
```

**원인:**
- 선생님이 무서움
- 성격이 완전히 안 맞음
- 트라우마 수준

**판단:**
→ 학생 정신 건강 최우선
→ **즉시 교체**
→ 억지로 계속하면 공부 자체 싫어짐

### 3. 6개월 이상 효과 전혀 없음

**명확한 무효과:**
```
6개월 경과:
❌ 성적 변화 0
❌ 이해도 향상 0
❌ 학습 태도 개선 0
❌ 질문 사항 해결 0
```

**원인 파악 시도:**
```
시도 1: 학생 태도 개선 (2개월)
결과: 변화 없음

시도 2: 수업 방식 조정 (2개월)
결과: 변화 없음

→ 선생님과 근본적 안 맞음
```

**판단:**
→ 6개월은 충분한 기간
→ **교체 결정**

### 4. 수업 방식이 근본적으로 안 맞음

**예시:**
```
학생: 차근차근 설명 필요
선생님: 빠르게 진도 나감
→ 학생 못 따라감

학생: 많은 문제 풀이 원함
선생님: 개념 설명만 집중
→ 실전 연습 부족

학생: 엄격한 관리 필요
선생님: 자유방임
→ 학생 늘어짐
```

**조정 시도:**
```
부모: "좀 더 천천히 설명해주세요"
선생님: 시도했으나 스타일상 어려움

OR

선생님: "저는 이 방식이 효과적입니다"
→ 변화 거부
```

**판단:**
→ 스타일은 바꾸기 어려움
→ **교체하는 게 맞음**

### 5. 신뢰가 완전히 무너짐

**신뢰 파괴:**
```
❌ 거짓말 발각 (학력, 경력 등)
❌ 돈 문제 (과다 청구 등)
❌ 학생 비밀 누설
❌ 부모님께 거짓 보고
❌ 약속 반복 위반
```

**관계:**
```
신뢰 없는 과외 = 효과 0

신뢰 회복은 거의 불가능
```

**판단:**
→ **즉시 교체**
→ 계속해봐야 의미 없음

## 기다려야 할 4가지 경우

### 1. 시작한 지 얼마 안 됨 (1-2개월)

**현실:**
```
1개월: 서로 파악 중
2개월: 막 적응 시작
3개월: 효과 나타나기 시작

→ 1-2개월은 너무 이름!
```

**흔한 착각:**
```
"1달 했는데 성적 안 올라요"
→ 당연함, 아직 시간 필요

"선생님 설명이 좀 어려워요"
→ 적응 기간 필요
```

**판단:**
→ 명백한 문제 없으면
→ **최소 3-4개월은 지켜보기**

### 2. 효과가 서서히 나타나는 중

**긍정 신호:**
```
✓ 이해도 조금씩 향상
✓ 질문이 구체적으로 변함
✓ 학습 태도 개선 중
✓ 작은 발전 있음 (성적은 아직)
```

**현실:**
```
성적 반영은 느림:
1-2개월: 이해도 향상
3-4개월: 작은 변화
5-6개월: 성적 반영

→ 과정 중에 교체하면 아까움
```

**판단:**
→ 방향이 맞다면
→ **계속 진행**

### 3. 작은 불만은 있지만 조정 가능

**사소한 불만:**
```
- 숙제가 조금 많음/적음
- 설명 속도가 약간 빠름/느림
- 수업 시간 5-10분 차이
- 스타일 약간 다름
```

**해결책:**
```
선생님과 대화:
"숙제 양을 조금 줄여주시면..."
"설명을 조금 천천히..."

→ 대부분 조정 가능
```

**판단:**
→ 조정 가능한 수준이면
→ **교체 말고 소통**

### 4. 학생의 일시적 반항/피로

**구분:**
```
지속적 거부: 처음부터 싫어함
일시적 반항: 갑자기 싫다고 함

일시적 원인:
- 시험 스트레스
- 학교 문제
- 친구 문제
- 사춘기
```

**테스트:**
```
1-2주 지켜보기:
→ 일시적이면 다시 괜찮아짐
→ 지속되면 진짜 문제
```

**판단:**
→ 일시적 감정이면
→ **기다려 보기**

## 교체 판단 체크리스트

### Step 1: 객관적 평가 (3개월 기준)

```
□ 선생님이 성실한가? (지각/결석 없음)
□ 준비를 잘 해오시나? (교재, 자료)
□ 수업이 체계적인가?
□ 학생과 소통하시나?
□ 숙제/피드백 꼼꼼한가?
□ 학생 이해도를 확인하시나?
□ 약속을 잘 지키시나?
```

**5개 이상 No: 교체 고려 강력**
**3-4개 No: 선생님과 대화 필요**
**2개 이하 No: 계속 가능**

### Step 2: 효과 평가

```
□ 이해도가 향상되었나?
□ 성적이 조금이라도 올랐나?
□ 학습 태도가 개선되었나?
□ 약점이 보완되고 있나?
□ 학생이 만족하나?
□ 부모가 만족하나?
```

**4개 이상 Yes: 계속**
**2-3개 Yes: 3개월 더 지켜보기**
**1개 이하 Yes: 교체 고려**

### Step 3: 학생 의견

```
학생에게 물어보기:
"선생님 어때?"
"계속하고 싶어?"
"뭐가 좋고 뭐가 별로야?"
```

**구체적으로:**
- 싫으면 왜 싫은지
- 좋으면 뭐가 좋은지
- 바꾸고 싶은지

### Step 4: 최종 판단

```
객관적 평가 + 효과 + 학생 의견
→ 종합 판단

즉시 교체: 명백한 문제
3개월 더: 효과 애매
계속: 모든 면 OK
```

## 교체 과정 5단계

### 1단계: 문제 명확히 하기

**정확한 진단:**
```
Q. 정확히 뭐가 문제인가?
- 선생님 문제인가?
- 학생 문제인가?
- 방식 문제인가?
- 관계 문제인가?
```

**가족 회의:**
- 부모 의견
- 학생 의견
- 객관적 사실
→ 종합 판단

### 2단계: 선생님과 대화 (선택)

**대화할 경우:**
```
문제가 조정 가능하면:
"선생님, 말씀드릴 게 있는데요..."
"수업 방식을 조금 조정해주실 수 있을까요?"

→ 개선 기회 2-4주
→ 안 되면 교체
```

**대화 생략:**
```
즉시 교체 사유:
- 불성실
- 부적절 언행
- 신뢰 파괴

→ 대화 없이 바로 교체 통보
```

### 3단계: 정중한 교체 통보

**시기:**
- 수업 끝난 직후 (학생 없는 자리)
- 또는 전화/메시지

**말씀 드리기:**
```
"선생님, 죄송한데요,
 여러 가지 사정으로
 과외를 중단하려고 합니다.

 그동안 감사했습니다."
```

**이유 설명:**
```
구체적 X: "사정이 생겨서..."
구체적 O (선택): "아이가 적응을 못 해서..."

→ 너무 구체적이면 불편
→ 간단히 하는 게 좋음
```

**비용 정산:**
- 깔끔하게
- 수업한 만큼만
- 영수증 받기

### 4단계: 새 선생님 찾기

**이전 문제 반성:**
```
이번엔 뭘 중점적으로 볼까?
- 성실성?
- 소통?
- 스타일?
- 성별/나이?
```

**체험 수업 필수:**
```
1-2회 체험:
✓ 학생 반응 확인
✓ 수업 스타일 확인
✓ 소통 가능성 확인

→ 괜찮으면 시작
→ 아니면 다른 분
```

**학생과 상의:**
- "이번 선생님은 어때?"
- "계속해볼까?"
- 학생 의견 존중

### 5단계: 첫 1개월 집중 관찰

**체크 포인트:**
```
✓ 이전 문제 반복 안 되나?
✓ 학생 적응 잘 하나?
✓ 수업 만족스러운가?
✓ 방향이 맞는가?
```

**빠른 피드백:**
- 문제 보이면 즉시 말씀
- 조정 가능한 시기
- 1개월 지나면 조정 어려움

## 교체 시 실수 5가지

### 실수 1: 너무 자주 교체

**패턴:**
```
A선생님 2개월 → 교체
B선생님 2개월 → 교체
C선생님 2개월 → 교체

→ 문제는 학생일 수 있음!
```

**문제:**
- 적응 기회 없음
- 돈만 낭비
- 학생만 혼란
- 선생님들도 기피

**해결:**
- 최소 3-4개월은 해보기
- 학생 태도 점검
- 부모 개입 줄이기

### 실수 2: 감정적 판단

**잘못된 이유:**
```
❌ "학생이 싫대요" (근거 없음)
❌ "마음에 안 들어요" (막연함)
❌ "다른 선생님이 더 나을 것 같아요" (추측)
```

**올바른 판단:**
```
✅ 객관적 사실 기반
✅ 효과 측정
✅ 구체적 문제 파악
✅ 대안 명확
```

### 실수 3: 학생 의견만 따름

**위험:**
```
학생: "싫어요, 바꿔주세요"
부모: "그래, 바꾸자"

→ 학생 입맛대로
→ 조금만 힘들면 또 바꾸기
→ 끈기 없어짐
```

**균형:**
```
학생 의견 + 객관적 평가
→ 종합 판단

진짜 문제 vs 단순 회피
→ 구분 필요
```

### 실수 4: 통보 방식 잘못

**나쁜 예:**
```
❌ 문자로 갑자기 "오늘부터 안 해요"
❌ 학생 앞에서 통보
❌ 비난하며 통보
❌ 비용 미지급하고 연락 끊기
```

**좋은 예:**
```
✅ 정중하게 사전 통보 (1주일 전)
✅ 학생 없는 자리에서
✅ 감사 인사
✅ 비용 깔끔히 정산
```

### 실수 5: 새 선생님 준비 없이 교체

**문제:**
```
A선생님 중단
→ 새 선생님 못 구함
→ 1-2개월 공백
→ 학생 공부 안 함
→ 성적 하락
```

**올바른 순서:**
```
1. 새 선생님 알아보기
2. 체험 수업
3. 확정
4. 현 선생님께 통보
5. 겹치지 않게 전환
```

## 선생님께 정중히 말씀드리는 법

### 상황별 멘트

**일반적 중단:**
```
"선생님, 그동안 정말 감사했는데요,
 저희 사정상 과외를 중단하게 되었습니다.
 죄송하고 감사합니다."
```

**학생 적응 문제:**
```
"선생님은 좋으신데,
 저희 아이가 적응을 못 하네요.
 죄송합니다."
```

**효과 없음:**
```
"선생님, 죄송한데요,
 여러 가지로 효과가 기대에 못 미쳐서
 다른 방법을 시도해보려고 합니다."
```

**명백한 문제 (불성실):**
```
"선생님, 수업 태도에 문제가 있어
 과외를 중단하겠습니다.
 비용은 수업한 만큼만 드리겠습니다."
```

### 피해야 할 말

**공격적:**
```
❌ "선생님 실력이 없으신 것 같아요"
❌ "우리 아이 성적 떨어뜨렸어요"
❌ "돈만 받고 일 안 하시네요"
```

**거짓말:**
```
❌ "이사 가요" (나중에 마주칠 수 있음)
❌ "경제적 문제" (나중에 다른 과외 하면 어색)

→ 간단히 "사정상" 이 최선
```

## 새 선생님 선택 시 체크

### 이전 문제 분석

```
이전 선생님 문제:
1. _____________
2. _____________
3. _____________

→ 이번엔 이 부분 중점 확인
```

### 체험 수업 시 확인 사항

```
✓ 준비성: 자료 준비해오셨나?
✓ 소통: 학생과 대화 잘 하시나?
✓ 설명: 이해하기 쉬운가?
✓ 성실성: 시간 지키시나?
✓ 관심: 학생에게 관심 있으신가?
✓ 계획: 앞으로 계획 있으신가?
```

### 학생 반응 체크

```
체험 후 학생에게:
"어땠어?"
"이해 잘 됐어?"
"선생님 어떠셨어?"
"계속하고 싶어?"

→ 긍정적이면 시작
→ 부정적이면 다른 분
```

## 자주 묻는 질문

### Q1. 몇 회 정도 해보고 교체 판단해야 하나요?

A. **최소 10-12회 (3개월)는 해보세요.**

**이유:**
```
1-4회: 서로 파악, 적응 기간
5-8회: 본격 수업, 효과 시작
9-12회: 효과 판단 가능

→ 3개월 미만은 너무 이름
```

**예외 (즉시 교체):**
```
- 불성실 (지각, 결석 잦음)
- 부적절 언행
- 학생 극도 거부
```

**판단 시점:**
```
1개월 (4회): 즉시 교체 사유만
3개월 (12회): 본격 효과 판단
6개월 (24회): 최종 결정
```

### Q2. 선생님께 솔직히 이유를 말씀드려야 하나요?

A. **상황에 따라 다릅니다.**

**간단히 (추천):**
```
"여러 사정으로 중단하게 되었습니다"

장점:
- 불편 최소화
- 관계 유지
- 간단명료
```

**구체적으로:**
```
"수업 방식이 우리 아이와 안 맞는 것 같아요"

장점:
- 선생님께 피드백
- 명확함

단점:
- 논쟁 가능성
- 불편할 수 있음
```

**명백한 문제:**
```
"지각이 너무 잦아서 중단합니다"

→ 사실 그대로 말씀
```

### Q3. 교체 후 다시 안 맞으면 어떡하죠?

A. **체험 수업을 더 꼼꼼히 하세요.**

**1차 교체 후:**
```
체험 1-2회 → 시작
```

**2차 교체 후:**
```
체험 3-4회 → 신중히 시작
+ 체크리스트 작성
+ 학생 의견 더 반영
```

**반복 교체 시:**
```
문제가 선생님인가, 학생인가?

학생 문제일 수 있음:
- 학습 태도
- 과도한 기대
- 끈기 부족

→ 학생 상담/교육 먼저
```

### Q4. 선생님과 친해졌는데 어떻게 바꾸나요?

A. **어렵지만, 학생 성장이 우선입니다.**

**마음:**
```
부모: "선생님이 좋은 분이신데..."
BUT: 효과가 없거나 문제가 있다면

→ 친분보다 학생 성장이 우선
```

**정중하게:**
```
"선생님, 정말 좋으신 분이고
 감사한데, 여러 사정으로
 중단하게 되었습니다.

 혹시 나중에 다른 도움 필요하면
 연락드려도 될까요?"

→ 관계 유지하면서 중단
```

**현실:**
```
대부분 선생님들도 이해하심
- 프로답게 받아들임
- 감사 인사
- 좋은 관계 유지 가능
```

### Q5. 교체하면 학생에게 안 좋은 영향 있나요?

A. **올바르게 교체하면 오히려 좋습니다.**

**부정적 영향:**
```
자주 교체 (3개월마다):
→ 학생 혼란
→ 적응 피로
→ 불안정

OR

억지로 계속:
→ 학생 스트레스
→ 효과 0
→ 공부 싫어짐
```

**긍정적 영향:**
```
적절한 교체:
→ "나에게 맞는 선생님 있구나"
→ 학습 효과 ↑
→ 만족도 ↑
→ 성장

+ 배움:
"안 맞으면 변화 필요"
"포기 vs 현명한 선택 구분"
```

**학생에게 설명:**
```
"선생님과 우리가 조금 안 맞는 것 같아.
 더 잘 맞는 선생님 찾아보자.
 괜찮지?"

→ 학생도 이해시키기
```

## 마무리

**선생님 교체, 핵심 정리:**

**즉시 교체:**
```
✓ 불성실 (지각, 결석, 무준비)
✓ 부적절 언행
✓ 학생 극도 거부
✓ 6개월 효과 0
✓ 신뢰 완전 파괴
```

**3개월 기다리기:**
```
✓ 시작한 지 얼마 안 됨
✓ 효과 서서히 나타남
✓ 작은 불만 (조정 가능)
✓ 일시적 문제
```

**교체 과정:**
```
1. 문제 명확히
2. 선생님과 대화 (선택)
3. 정중한 통보
4. 새 선생님 체험
5. 신중히 시작
```

**주의사항:**
```
❌ 너무 자주 교체
❌ 감정적 판단
❌ 학생 의견만
❌ 무례한 통보
❌ 준비 없이 교체
```

**판단 기준:**
```
객관적 평가 + 효과 + 학생 의견
→ 종합 판단

3개월 = 기본 판단 시점
6개월 = 최종 결정 시점

명백한 문제 = 즉시
```
