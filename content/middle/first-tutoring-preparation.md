"처음 과외를 시작하는데 무엇부터 해야 할까요?"

**첫 시작이 중요합니다.** 제대로 준비하면 효과가 2배입니다.

## 첫 과외 준비가 중요한 이유

**시작이 반:**
```
준비 잘 함 → 첫 수업부터 효과적
준비 부족 → 시행착오, 시간 낭비
```

**첫인상:**
- 선생님도 학생 파악
- 학생도 과외 적응
- 서로 신뢰 형성
- 방향 설정

**효율:**
```
1회차: 제대로 준비하면 바로 시작
1회차: 준비 없으면 상담만 하고 끝
→ 1회차부터 차이
```

## 과외 시작 전 준비 (D-7일)

### 1단계: 목표 명확히 하기 (D-7일)

**왜 과외를 시작하나요?**
```
□ 성적이 떨어져서 (몇 등급 → 목표 등급)
□ 특정 과목이 약해서 (어떤 부분)
□ 학원만으론 부족해서 (무엇이 부족)
□ 시험 대비 (언제 시험)
□ 기초가 부족해서 (어디서부터)
```

**구체적 목표 설정:**
```
나쁜 예: "수학 잘하고 싶어요"
→ 막연함

좋은 예: "수학 4등급을 3개월 내 2등급으로"
→ 구체적, 측정 가능, 기한 있음
```

**목표 작성 템플릿:**
```
과목: __________
현재: __등급 (약점: _______)
목표: __등급
기간: __개월
특별히 집중할 부분: __________
```

**예시:**
```
과목: 수학
현재: 3등급 (도형 약함)
목표: 2등급
기간: 3개월
특별히 집중할 부분: 도형, 함수
```

### 2단계: 현재 상태 정확히 파악 (D-6일)

**성적 분석:**
```
✓ 최근 3회 시험 성적 정리
✓ 단원별 점수 확인
✓ 자주 틀리는 유형 체크
✓ 시간 부족 vs 이해 부족 구분
```

**학습 습관 체크:**
```
□ 혼자 공부 시간: 하루 ___분
□ 복습 습관: 있음 / 없음
□ 숙제 완성도: 항상 / 보통 / 자주 못함
□ 질문 습관: 많음 / 보통 / 거의 없음
□ 집중 시간: ___분 정도 가능
```

**약점 정리:**
```
1순위 약점: __________
2순위 약점: __________
3순위 약점: __________
```

**예시 (수학):**
```
1순위: 도형 (증명 문제 거의 못 풀음)
2순위: 함수 (응용 문제 약함)
3순위: 계산 실수 많음
```

### 3단계: 과목 및 선생님 선택 (D-5일)

**과목 우선순위:**
```
많이 약한 과목부터
OR
빨리 올릴 수 있는 과목부터
```

**판단 기준:**
```
중1-2: 수학, 영어 중 1-2과목
중3: 수학, 영어 + 선택 1과목

처음엔 1-2과목만!
욕심 내면 과부하
```

**선생님 선택 기준:**
```
✓ 경력 (비슷한 레벨 학생 지도 경험)
✓ 소통 (설명 이해하기 쉬운지)
✓ 성향 (엄격 vs 친근, 학생과 맞는지)
✓ 거리/시간 (무리 없는지)
```

### 4단계: 스케줄 계획 (D-4일)

**주간 과외 시간 정하기:**
```
초보: 주 1-2회, 회당 1-1.5시간
표준: 주 2-3회, 회당 1.5-2시간
집중: 주 3-4회, 회당 2시간
```

**시간대 선택:**
```
평일 방과후: 5-6시 시작 (추천)
평일 저녁: 7-8시 시작 (괜찮음)
주말 오전: 10-11시 시작 (추천)
주말 오후: 2-3시 시작 (괜찮음)
```

**가장 집중 잘 되는 시간:**
- 학생마다 다름
- 식사 직후 피하기
- 너무 늦은 시간 피하기

**주간 계획표 작성:**
```
월: 학교 → 저녁 → [과외 7-8:30pm]
화: 학교 → 복습
수: 학교 → 저녁 → [과외 7-8:30pm]
목: 학교 → 복습
금: 학교 → 과제
토: [과외 2-3:30pm] → 자유시간
일: 복습 + 휴식
```

### 5단계: 공간 준비 (D-3일)

**과외 장소 결정:**
```
Option 1: 학생 집 (가장 일반적)
Option 2: 선생님 집 / 스터디카페
Option 3: 온라인
```

**집에서 할 경우 공간 세팅:**
```
✓ 조용한 방 (거실 X, 방문 닫기)
✓ 책상 정리 (깔끔하게)
✓ 의자 2개 (학생용, 선생님용)
✓ 조명 밝게 (눈 안 피곤하게)
✓ 화이트보드 or 칠판 (있으면 좋음)
✓ 온도 적당히 (너무 덥거나 춥지 않게)
```

**주의사항:**
```
❌ TV 소리 들림
❌ 동생 방해
❌ 부모님 자주 드나듦
❌ 핸드폰, 게임기 눈에 보임

→ 집중 방해 요소 제거
```

### 6단계: 학습 자료 준비 (D-2일)

**필수 자료:**
```
✓ 교과서 (현재 학기)
✓ 학교 프린트/학습지 (최근 것)
✓ 최근 시험 문제 + 답안지 (최소 2-3회)
✓ 문제집 (있으면, 없어도 선생님이 준비)
✓ 노트 (과외 전용)
✓ 필기구 (펜, 연필, 지우개, 형광펜)
```

**중요 자료 (반드시 준비):**
```
★ 최근 시험지 2-3회
→ 약점 파악에 필수!
→ 어떤 문제 틀렸는지
→ 왜 틀렸는지 분석
```

**있으면 좋은 것:**
```
- 오답노트 (만들었다면)
- 학원 교재 (다니면)
- 이전 학년 교과서 (기초 부족 시)
```

**정리 방법:**
```
1. 파일 or 폴더에 과목별 정리
2. 최근 것부터 순서대로
3. 첫 수업 때 선생님께 모두 보여주기
```

### 7단계: 마음 준비 (D-1일)

**학생 마음가짐:**
```
✓ 솔직하게: 모르면 모른다고 말하기
✓ 적극적으로: 질문 많이 하기
✓ 열린 마음: 새로운 방법 받아들이기
✓ 인내심: 바로 안 올라도 괜찮음
```

**부모님 마음가짐:**
```
✓ 신뢰: 선생님 믿고 맡기기
✓ 인내: 최소 2-3개월은 지켜보기
✓ 거리: 과외 시간엔 간섭 X
✓ 소통: 정기적 피드백 받기
```

**첫 수업 기대치 조정:**
```
❌ "첫 수업부터 바로 성적 오르겠지?"
→ 비현실적

✅ "첫 수업은 서로 알아가는 시간"
→ 현실적

첫 3-4회: 파악 + 적응
1-2개월: 변화 시작
3개월: 확실한 효과
```

## D-Day: 첫 수업 당일

### 수업 1시간 전

**최종 체크:**
```
□ 준비물 챙김 (교과서, 시험지, 노트 등)
□ 방 정리 및 환기
□ 간식/물 준비 (휴식 시간용)
□ 집중 방해 요소 제거
□ 가족에게 공지 (조용히 해달라고)
```

**학생 컨디션:**
- 충분히 쉬었는지
- 너무 긴장 안 했는지
- 화장실 다녀오기
- 물 마시기

### 첫 수업 시작 (0-15분)

**인사 및 소개:**
```
학생: "안녕하세요, ○○○입니다"
부모: 선생님께 인사, 간단히 상황 설명
선생님: 자기소개, 수업 방식 설명
```

**부모님은 이 시점에 자리 비우기:**
- 학생과 선생님끼리
- 편한 분위기
- 솔직한 대화

### 첫 수업 중반 (15-45분)

**선생님이 할 것:**
```
✓ 학생 레벨 테스트 (간단히)
✓ 약점 파악
✓ 학습 스타일 파악
✓ 목표 확인
```

**학생이 할 것:**
```
✓ 솔직하게 대답
✓ 모르는 거 숨기지 않기
✓ 편한 스타일 말하기
✓ 궁금한 거 질문
```

**이 시간에 나눌 이야기:**
```
- 현재 성적, 어려운 점
- 학교 진도, 다음 시험 일정
- 학원 다니는지, 어떤지
- 공부 습관, 집중 시간
- 과외 목표, 기대
```

### 첫 수업 후반 (45-90분)

**실제 수업 맛보기:**
```
✓ 간단한 개념 설명
✓ 문제 1-2개 풀어보기
✓ 학생 이해도 확인
✓ 다음 시간 계획
```

**마무리 (90-100분):**
```
✓ 오늘 배운 것 정리
✓ 숙제 (부담 없게)
✓ 다음 수업 계획
✓ 질문 있는지 확인
```

## 첫 수업 후 (D+1일)

### 학생 자가 평가

**체크리스트:**
```
□ 선생님 설명 이해하기 쉬웠나?
□ 질문하기 편했나?
□ 시간이 빨리 지나갔나? (집중했다는 뜻)
□ 다음에도 하고 싶나?
□ 뭔가 배운 느낌이 드나?
```

**3개 이상 Yes: 좋은 시작**

### 부모 관찰 포인트

**수업 중 (멀리서):**
```
□ 학생이 집중하는가?
□ 적극적으로 참여하는가?
□ 선생님이 꼼꼼한가?
□ 분위기가 좋은가?
```

**수업 후 대화:**
```
부모: "어땠어?"
학생: 솔직한 느낌 듣기

부모: "선생님 어떠셨어요?"
선생님: 학생 상태, 앞으로 계획
```

### 피드백 및 조정

**좋았으면:**
```
→ 그대로 진행
→ 2-3회 더 해보기
→ 루틴 확립
```

**아쉬웠으면:**
```
→ 구체적으로 뭐가 불편했나?
→ 선생님께 말씀드리기
→ 조정 가능한지 확인
→ 2-3회는 더 해보기 (첫날 판단 이름)
```

**정말 안 맞으면:**
```
→ 4회 정도 해보고 판단
→ 여전히 안 맞으면 선생님 교체 고려
```

## 첫 1개월 계획

### 1주차: 적응 기간

**목표:**
```
✓ 서로 알아가기
✓ 학생 약점 정확히 파악
✓ 루틴 확립
```

**수업 내용:**
```
- 진단 테스트
- 기초 점검
- 학습 스타일 파악
- 수업 방식 조율
```

**숙제:**
- 부담 없는 양
- 복습 위주
- 습관 형성

### 2-3주차: 본격 시작

**목표:**
```
✓ 본격 학습 시작
✓ 약점 보완 시작
✓ 학습 습관 형성
```

**수업 내용:**
```
- 약한 부분 집중
- 개념 다시 잡기
- 기본 문제 풀이
- 오답 분석
```

**숙제:**
- 점진적 증가
- 복습 + 문제 풀이
- 꼭 해오기

### 4주차: 첫 점검

**평가:**
```
✓ 1개월 간 변화
✓ 이해도 향상
✓ 학습 태도 개선
✓ 작은 성공 경험
```

**조정:**
```
- 빈도 조절 필요한지
- 난이도 조절
- 수업 방식 개선
- 2-3개월 계획 수립
```

## 준비물 체크리스트

### 필수 준비물 (반드시!)

**학습 자료:**
```
□ 교과서 (현재 학기)
□ 최근 시험지 2-3회분 + 답안지
□ 학교 프린트/학습지
□ 노트 (과외 전용, 새 거)
□ 필기구 (펜, 연필, 지우개, 형광펜)
```

**공간:**
```
□ 조용한 방
□ 정리된 책상
□ 의자 2개
□ 밝은 조명
```

**마음:**
```
□ 목표 설정 완료
□ 현재 상태 파악 완료
□ 열린 마음
□ 인내심
```

### 있으면 좋은 것

```
□ 문제집 (있으면)
□ 오답노트 (만들었으면)
□ 학원 교재 (다니면)
□ 화이트보드/칠판
□ 스탠드 조명
□ 타이머
```

### 준비하지 않아도 되는 것

```
✗ 새 문제집 (선생님과 상의 후)
✗ 비싼 교재 (필요하면 선생님이 말함)
✗ 특별한 장비
```

## 첫 수업에서 꼭 물어볼 것

### 학습 방향

```
Q1. "저의 가장 큰 약점이 뭔가요?"
→ 선생님 진단 듣기

Q2. "목표 등급까지 얼마나 걸릴까요?"
→ 현실적 기대치 파악

Q3. "어떤 순서로 공부하나요?"
→ 학습 계획 이해
```

### 수업 방식

```
Q4. "숙제는 얼마나 나가나요?"
→ 부담 정도 파악

Q5. "질문은 언제 해도 되나요?"
→ 수업 외 질문 가능 여부

Q6. "진도는 어떻게 나가나요?"
→ 선행 vs 복습 vs 병행
```

### 피드백

```
Q7. "부모님께 피드백은 어떻게 하시나요?"
→ 소통 방식 확인

Q8. "효과는 언제쯤 나타날까요?"
→ 기대 시점 확인
```

## 흔한 첫 과외 실수

### 실수 1: 과도한 준비

**잘못된 예:**
```
❌ 새 문제집 10권 구입
❌ 비싼 교재 세트 구입
❌ 학습 기구 여러 개 구입
```

**문제:**
- 돈 낭비
- 선생님 스타일과 안 맞을 수 있음
- 부담감만 증가

**올바른 방법:**
```
✅ 기본만 준비 (교과서, 시험지, 노트)
✅ 추가 자료는 선생님과 상의 후
✅ 필요한 것만 점진적 구입
```

### 실수 2: 첫날 판단

**잘못된 생각:**
```
❌ "첫 수업 별로네, 바꿔야겠다"
❌ "애가 별로 안 집중하네, 과외 소용없나?"
```

**문제:**
- 첫날은 서로 적응 기간
- 너무 빠른 판단

**올바른 방법:**
```
✅ 최소 3-4회는 해보기
✅ 적응 기간 감안
✅ 변화 천천히 지켜보기
```

### 실수 3: 목표 불명확

**잘못된 목표:**
```
❌ "수학 잘하고 싶어요"
❌ "성적 올리고 싶어요"
→ 너무 막연함
```

**문제:**
- 진도 계획 세우기 어려움
- 효과 측정 불가
- 동기부여 부족

**올바른 목표:**
```
✅ "3개월 내 수학 4등급 → 2등급"
✅ "중간고사 도형 문제 80% 이상"
✅ "방학 동안 1학기 수학 완벽 복습"
```

### 실수 4: 부모 과잉 간섭

**잘못된 행동:**
```
❌ 수업 중 들어와서 확인
❌ 선생님 수업 방식에 계속 간섭
❌ "이것도 가르쳐주세요, 저것도..."
❌ 아이 앞에서 선생님께 불만 표현
```

**문제:**
- 수업 분위기 망침
- 선생님 위축
- 학생 집중 방해
- 신뢰 관계 깨짐

**올바른 방법:**
```
✅ 수업 중엔 간섭 X
✅ 피드백은 따로 시간 내서
✅ 선생님 믿고 맡기기
✅ 정기 상담으로 소통
```

### 실수 5: 비현실적 기대

**잘못된 기대:**
```
❌ "1개월이면 1등급 되겠지?"
❌ "과외만 하면 알아서 오르겠지?"
❌ "숙제 안 해도 괜찮겠지?"
```

**현실:**
```
✅ 3-6개월은 지켜봐야 함
✅ 과외 + 학생 노력 = 성과
✅ 숙제는 필수!
```

## 학년별 첫 과외 포인트

### 중1 첫 과외

**특징:**
- 과외 자체가 처음
- 1:1 수업 낯섦
- 긴장 많이 함

**준비 포인트:**
```
✓ 편안한 분위기 중요
✓ 친근한 선생님 추천
✓ 짧은 시간부터 (1-1.5h)
✓ 칭찬과 격려 많이
```

**첫 수업 초점:**
- 긴장 풀기
- 라포 형성
- 학습 스타일 파악
- 재미있는 경험

### 중2 첫 과외

**특징:**
- 학원 경험 있을 수 있음
- 학습 습관 어느 정도 형성
- 목표 의식 있음

**준비 포인트:**
```
✓ 명확한 목표 설정
✓ 체계적 계획
✓ 약점 정확히 파악
✓ 효율적 수업
```

**첫 수업 초점:**
- 빠른 레벨 파악
- 약점 진단
- 구체적 계획 수립
- 단기 목표 설정

### 중3 첫 과외

**특징:**
- 고등 준비
- 시간 촉박
- 목표 명확
- 압박감 있음

**준비 포인트:**
```
✓ 즉시 시작 가능하게
✓ 자료 완벽 준비
✓ 시험 일정 공유
✓ 효율 최우선
```

**첫 수업 초점:**
- 빠른 진단
- 즉시 약점 보완 시작
- 타이트한 계획
- 시간 관리 중요

## 성공적인 첫 과외 사례

### 사례 1: 완벽 준비로 빠른 시작

**학생:**
- 중2, 수학 4등급
- 첫 과외

**준비:**
```
✓ 목표 명확: "3개월 내 2등급"
✓ 최근 시험지 3회분 준비
✓ 약점 정리 (도형, 함수)
✓ 공부방 완벽 세팅
✓ 주간 계획표 미리 작성
```

**첫 수업:**
```
- 자료 덕분에 즉시 진단 가능
- 약점 파악 빠름
- 바로 학습 시작
- 2-3개월 계획 수립
```

**결과:**
```
1회차부터 효율적
시간 낭비 없음
3개월 후 2등급 달성!
```

### 사례 2: 솔직함이 핵심

**학생:**
- 중1, 영어 5등급
- 기초 많이 부족
- 부끄러워서 숨기려 함

**첫 수업:**
```
학생: (처음엔 아는 척)
선생님: "괜찮아, 모르는 거 솔직히 말해"
학생: "사실 알파벳도 헷갈려요..."
선생님: "그럼 거기서부터 시작하자!"
```

**계획 조정:**
- 당초: 중1 교과서 진도
- 조정: 기초 (알파벳, 파닉스)부터
- 천천히 올라가기

**결과:**
```
3개월: 기초 완성
6개월: 중1 교과서 따라감
1년: 4등급 달성
→ 솔직해서 제대로 된 시작
```

### 사례 3: 부모님의 현명한 거리두기

**상황:**
- 중3, 수학 과외 시작
- 부모님 교육열 높음
- 간섭 우려

**부모님의 선택:**
```
✓ 첫 수업: 인사만 하고 나감
✓ 수업 중: 절대 안 들어감
✓ 피드백: 수업 끝난 후 5분만
✓ 상담: 2주 1회 별도 시간
✓ 신뢰: 선생님께 전적으로 맡김
```

**효과:**
```
- 학생 편안함
- 선생님과 신뢰 관계 빠르게 형성
- 솔직한 소통
- 집중도 ↑
- 6개월 만에 2등급 달성
```

## 자주 묻는 질문

### Q1. 첫 수업 때 꼭 부모가 같이 있어야 하나요?

A. **처음 10-15분만 함께 하세요.**

**추천 방식:**
```
0-10분: 부모님 + 학생 + 선생님
→ 인사, 간단한 상황 설명
→ 수업 방식 확인

10분 이후: 학생 + 선생님만
→ 편한 분위기
→ 솔직한 대화 가능
→ 집중 수업
```

**이유:**
- 부모님 계시면 학생 위축
- 모르는 것 솔직히 못 말함
- 선생님도 불편
- 1:1 효과 감소

**예외:**
- 중1 첫 과외 + 극도로 수줍음
→ 첫 수업만 옆에서 지켜보기
→ 2회차부터는 나가기

### Q2. 첫 수업에서 진도를 나가나요?

A. **반반입니다. 선생님마다 다릅니다.**

**패턴 1: 진단 위주 (50%)**
```
1회차: 레벨 테스트 + 상담 + 계획 수립
2회차: 본격 수업 시작
```

**패턴 2: 바로 수업 (50%)**
```
1회차: 간단 진단 + 바로 약점 보완 시작
→ 더 효율적
```

**어느 쪽이든 괜찮음:**
- 선생님 스타일
- 자료 준비 여부에 따라
- 학생 상태에 따라

**준비 잘 하면:**
→ 1회차부터 바로 수업 가능
→ 더 효율적!

### Q3. 첫 과외 비용은 언제 드리나요?

A. **보통 첫 수업 끝나고 드립니다.**

**일반적 방식:**
```
Option 1: 첫 수업 후 (가장 일반적)
Option 2: 한 달 선불 (월초)
Option 3: 한 달 후불 (월말)
```

**확인 사항:**
```
- 결제 방식 (현금 / 계좌이체)
- 결제 시점 (언제)
- 계산 방법 (회당 / 월)
```

**첫 수업 전 미리 물어보기:**
- "비용은 언제 어떻게 드리면 될까요?"
- 명확히 해두면 나중에 불편 없음

### Q4. 몇 회 해보고 선생님을 바꿀 수 있나요?

A. **4-5회는 해보고 판단하는 게 좋습니다.**

**판단 시점:**
```
1-2회: 너무 이름 (적응 기간)
3-4회: 어느 정도 파악 가능
5회 이상: 확실히 판단 가능
```

**바로 바꿔야 하는 경우:**
```
❌ 선생님이 지각/결석 잦음
❌ 준비 없이 오심
❌ 학생이 극도로 거부
❌ 수업 태도 불성실
```

**조금 더 지켜봐야 하는 경우:**
```
△ "설명이 좀 어려워요"
  → 선생님께 말씀드리기
  → 조정 가능

△ "생각보다 안 재미있어요"
  → 과외는 공부, 재미만 추구 X
  → 효과 중심으로 판단

△ "아직 잘 모르겠어요"
  → 3-4회는 더 해보기
```

**교체 방법:**
```
1. 선생님께 정중히 말씀
2. "죄송하지만 스타일이 안 맞는 것 같아요"
3. 비용 정산
4. 다른 선생님 찾기
```

### Q5. 첫 수업부터 숙제가 나오나요?

A. **네, 보통 나옵니다. 하지만 부담 없는 양입니다.**

**첫 숙제 특징:**
```
✓ 적은 양 (30분-1시간 분량)
✓ 복습 위주
✓ 쉬운 문제
✓ 습관 형성 목적
```

**예시:**
```
수학: 오늘 푼 문제 다시 풀기 + 비슷한 문제 5개
영어: 오늘 배운 단어 복습 + 문장 5개 암기
국어: 오늘 분석한 작품 다시 읽기
```

**숙제 안 하면:**
- 다음 수업 비효율
- 복습 안 되면 까먹음
- 선생님도 판단 어려움

**꼭 해가기:**
```
✅ 모르는 거 있어도 시도
✅ 못 풀었으면 표시해두기
✅ 질문 정리해가기
✅ 솔직하게 얘기하기
```

**D-7일 체크리스트:**
```
□ 목표 명확히 (구체적 + 기한)
□ 현재 상태 파악 (성적, 약점, 습관)
□ 최근 시험지 2-3회 준비
□ 공부 공간 세팅
□ 스케줄 계획
□ 마음 준비 (학생 + 부모)
```

**첫 수업 성공 조건:**
```
✓ 학생: 솔직 + 적극 + 열린 마음
✓ 부모: 신뢰 + 거리두기 + 인내
✓ 준비: 자료 + 공간 + 목표
```

**첫 1개월 목표:**
```
1주: 적응 + 파악
2-3주: 본격 시작
4주: 첫 점검

→ 습관 형성이 우선
→ 성적은 2-3개월 후
```

**피하세요:**
```
❌ 과도한 준비 (자료 과다 구입)

❌ 목표 불명확
❌ 부모 과잉 간섭
❌ 비현실적 기대
```

**핵심:**
```
첫 시작이 반!
제대로 준비하면
효과 2배

시작이 좋으면
끝도 좋다!
```
