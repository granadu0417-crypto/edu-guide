"중등 과외, 얼마가 적정할까요?"

**비용보다 중요한 것은 가성비**입니다. 합리적으로 선택하세요.

**과외 비용을 결정하는 모든 요소를 알려드립니다.**

## 과외 비용의 이해

### 비용이 중요한 이유

**장기적 지출:**
- 월 단위 지속
- 3-6개월 기본
- 1-2년도 흔함

**가계 부담:**
- 교육비의 큰 부분
- 다른 지출 고려
- 합리적 판단 필요

**효과와의 관계:**
- 비싸다고 좋은 것 아님
- 싸다고 나쁜 것 아님
- 적합성이 중요

### 비용 결정 요소

**5가지 주요 요소:**
1. 선생님 유형 (대학생/경력/전문)
2. 과목 (수학/영어/국어 등)
3. 수업 형태 (1:1/그룹/온라인)
4. 지역 (서울/수도권/지방)
5. 학생 수준 (기초/심화)

## 선생님 유형별 비용

### 대학생 선생님

**특징:**
- 최근 입시 경험
- 학생과 나이 차 적음
- 친근한 관계
- 열정적

**비용 수준:**
- 상대적으로 저렴
- 경력 선생님의 60-70%
- 초임 시작

**장점:**
- 경제적
- 최신 입시 정보
- 공감대 형성
- 에너지 넘침

**단점:**
- 경험 부족
- 체계 미흡할 수 있음
- 안정성 낮음 (군대, 취업 등)

**추천 대상:**
- 2-3등급 학생
- 친근한 분위기 선호
- 경제적 부담 있는 가정
- 동기 부여 필요한 학생

**체크 포인트:**
```
✓ 명문대 재학 중인가?
✓ 과외 경험 있나?
✓ 성실한가?
✓ 장기 계획 가능한가?
```

### 경력 선생님 (학원 강사, 전직 교사)

**특징:**
- 5-10년 이상 경력
- 체계적 커리큘럼
- 학생 유형별 대처
- 안정적 관리

**비용 수준:**
- 중간-높음
- 전문 과외 선생님의 70-80%
- 경력에 따라 차이

**장점:**
- 풍부한 경험
- 체계적 수업
- 문제 대처 능력
- 학습 관리 능력

**단점:**
- 비교적 높은 비용
- 스케줄 제한적
- 형식적일 수 있음

**추천 대상:**
- 3-4등급 학생 (기초 다지기)
- 체계적 관리 필요
- 장기 플랜 원하는 경우
- 학원 대안 찾는 경우

**체크 포인트:**
```
✓ 어디서 강의했나?
✓ 실제 합격 실적?
✓ 학생 관리 방식은?
✓ 커리큘럼 있나?
```

### 전문 과외 선생님

**특징:**
- 과외만 전문
- 10년 이상 경력
- 입소문으로 유명
- 관리 철저

**비용 수준:**
- 가장 높음
- 프리미엄 수준
- 대기 있을 수 있음

**장점:**
- 최고 수준 실력
- 완벽한 관리
- 입시 정보 풍부
- 1:1 맞춤 최고

**단점:**
- 높은 비용
- 구하기 어려움
- 스케줄 맞추기 어려움

**추천 대상:**
- 4-5등급 (집중 관리 필요)
- 1-2등급 (최상위 목표)
- 특목고 준비
- 경제적 여유 있음

**체크 포인트:**
```
✓ 전문 분야는?
✓ 학생 후기는?
✓ 합격 실적은?
✓ 관리 방식은?
```

## 과목별 비용 차이

### 주요 과목: 수학, 영어

**비용이 높은 이유:**
- 수요가 가장 많음
- 중요도 높음
- 좋은 선생님 많이 찾음
- 내신+수능 모두 중요

**특징:**
- 선생님 구하기 쉬움
- 선택 폭 넓음
- 경쟁 치열

**전략:**
```
우선순위 1순위
가장 많이 투자
효과도 큰 과목
```

### 국어

**비용:**
- 수학/영어보다 약간 낮음
- 수요 증가 추세
- 선생님 적음

**특징:**
- 최근 중요도 상승
- 내신 비문학 어려움
- 독해력 필수

**전략:**
```
3-4등급이면 고려
독해 약하면 필수
장기적으로 중요
```

### 과학, 사회

**비용:**
- 상대적으로 저렴
- 수요 적음
- 선택 과목

**특징:**
- 시험 전 집중
- 단기 과외 가능
- 특목고 준비 시 중요

**전략:**
```
필요할 때만
시험 2주 전
단기 집중 추천
```

## 수업 형태별 비용

### 1:1 개인과외

**비용:**
- 가장 높음
- 100% 기준

**효과:**
- 최고
- 완전 맞춤
- 집중 관리

**추천:**
- 기초가 약함 (4-5등급)
- 빠른 성적 향상 필요
- 개인 맞춤 필요
- 경제적 여유

**예산 절약법:**
- 온라인으로 전환
- 빈도 줄이기 (주 3→2회)
- 시험 기간만 집중

### 2:1 그룹과외

**비용:**
- 1:1의 60-70% 수준
- 각자 부담

**효과:**
- 1:1의 70-80%
- 경쟁 자극 효과

**추천:**
- 비슷한 실력 친구 있음
- 경제적 부담 큼
- 2-3등급 학생

**주의:**
```
실력 차이 1등급 이내
서로 좋은 관계
둘 다 성실해야
```

### 3:1 이상 소그룹

**비용:**
- 1:1의 40-50% 수준
- 가장 경제적

**효과:**
- 1:1의 50-60%
- 소형 학원과 비슷

**추천:**
- 경제적 이유
- 기본 실력 있음
- 자기주도 가능

**단점:**
```
개인 맞춤 어려움
시간 분산
관리 한계
```

### 온라인 과외

**비용:**
- 방문 과외의 80-90%
- 이동 비용 절약

**효과:**
- 방문과 비슷
- 기술 적응 필요

**추천:**
- 집중력 좋음
- 이동 시간 아까움
- 다양한 선생님 만나고 싶음

**장점:**
```
✓ 이동 시간 없음
✓ 전국 선생님 가능
✓ 녹화 복습 가능
✓ 약간 저렴
```

## 지역별 비용 차이

### 서울 (특히 강남/목동/중계)

**비용 수준:**
- 가장 높음
- 선생님 공급 많음
- 수요도 많음

**특징:**
- 선택 폭 넓음
- 경쟁 치열
- 수준 높은 선생님 많음

### 수도권 (경기/인천)

**비용 수준:**
- 서울의 80-90%
- 지역별 차이 있음

**특징:**
- 서울 선생님도 가능
- 적정 수준

### 지방

**비용 수준:**
- 서울의 60-80%
- 공급 적을 수 있음

**특징:**
- 선생님 구하기 어려울 수 있음
- 온라인 과외 고려

**팁:**
```
지방이면 온라인 추천
서울 좋은 선생님
이동 시간 없음
비용도 절약
```

## 합리적 선택 기준

### 가성비 판단법

**비용 대비 효과:**
```
효과 없으면 비싼 것 = 손해
효과 있으면 비싼 것 = 가치 있음
```

**판단 기준 (3개월 후):**
- [ ] 성적 올랐나?
- [ ] 학습 습관 생겼나?
- [ ] 아이가 만족하나?
- [ ] 선생님 성실한가?
- [ ] 관리 잘 되나?

**3개 이상 YES: 계속 ✅**
**2개 이하: 변경 고려**

### 예산 수립 방법

**1단계: 목표 설정**
```
현재: 4등급
목표: 2등급
기간: 6개월
```

**2단계: 우선순위**
```
1순위: 수학 (가장 약함)
2순위: 영어 (보통)
3순위: 국어 (나중에)
```

**3단계: 형태 선택**
```
수학: 1:1 (집중 필요)
영어: 온라인 (경제적)
국어: 시험 전만
```

**4단계: 월 예산**
```
수학 1:1: XX원 (주 2회)
영어 온라인: XX원 (주 2회)
총: XX원/월
```

**5단계: 지속 가능성**
```
6개월 지속 가능?
효과 없으면?
중간 평가 시기?
```

### 투자 우선순위

**1순위: 가장 약한 과목**
```
4-5등급 과목
기초부터 다시
1:1 집중 투자
```

**2순위: 중요 과목**
```
수학, 영어
주요 과목
장기 투자
```

**3순위: 나머지 과목**
```
국어, 과학, 사회
필요시만
단기 집중
```

## 비용 절약 전략

### 1. 그룹과외 활용

**절약 효과:**
- 1:1 대비 30-40% 절약
- 친구와 함께
- 경쟁 효과도

**조건:**
- 비슷한 실력 친구
- 좋은 관계
- 둘 다 성실

**예시:**
```
Before: 수학 1:1 주 2회
After: 수학 2:1 주 2회
절약: 월 30-40% 줄어듦
효과: 70-80% 유지
```

### 2. 온라인 전환

**절약 효과:**
- 방문 대비 10-20% 절약
- 이동 시간 절약
- 선택 폭 넓음

**적합한 과목:**
- 수학 (화면 공유)
- 영어 (회화 제외)
- 국어, 과학, 사회

**부적합:**
- 초등 저학년
- 집중력 낮음
- 기기 다루기 어려움

### 3. 빈도 조절

**효과적 감축:**
```
Before: 주 3회
After: 주 2회
절약: 33%
효과: 80% 유지 가능
```

**조건:**
- 기본 실력 있음
- 자기주도 가능
- 복습 잘 함

**주의:**
```
너무 줄이면 효과 ↓
최소 주 1-2회 권장
```

### 4. 시기 집중

**방학 집중 전략:**
```
학기 중: 없음 or 최소
방학: 집중 투자
효과: 선행, 복습
```

**시험 기간 전략:**
```
평소: 최소
시험 2주 전: 집중
효과: 내신 대비
```

**비용:**
- 연간 총액 감소
- 필요한 시기만
- 효율적

### 5. 과목 선택

**전략적 선택:**
```
모든 과목 (X)
1-2과목 집중 (O)
```

**선택 기준:**
1. 가장 약한 과목
2. 가장 중요한 과목
3. 효과 빠른 과목

**예시:**
```
Bad: 수학+영어+국어+과학 = 과부하
Good: 수학 집중 → 성적 오르면 → 영어 추가
```

### 6. 학원 병행

**전략:**
```
학원: 개념, 기본 문제
과외: 어려운 문제, 1:1 관리
```

**효과:**
- 비용 절감
- 보완적 학습
- 효율적

**주의:**
```
너무 많으면 과부하
시간 관리 중요
우선순위 명확히
```

## 흔한 비용 실수

### 실수 1: 무조건 비싼 선생님

**문제:**
"비싸니까 좋겠지"

**현실:**
- 아이와 안 맞을 수 있음
- 과한 투자
- 부담만 커짐

**올바른 방법:**
```
✓ 체험 수업 필수
✓ 아이 의견 듣기
✓ 적합성 판단
✓ 예산 고려
```

### 실수 2: 무조건 싼 선생님

**문제:**
"싸면 좋지"

**현실:**
- 효과 없으면 손해
- 시간 낭비
- 성적 안 오름

**올바른 방법:**
```
✓ 실력 먼저 확인
✓ 후기 체크
✓ 체험 수업
✓ 적정 비용 지불
```

### 실수 3: 너무 많은 과목

**문제:**
"다 하면 좋겠지"

**현실:**
- 과부하
- 소화 안 됨
- 비용만 많이

**올바른 방법:**
```
1-2과목 집중
효과 보면 추가
선택과 집중
```

### 실수 4: 단기 판단

**문제:**
"1개월인데 효과 없어요"

**현실:**
- 효과는 3-6개월 후
- 너무 빨리 바꿈
- 적응 못 함

**올바른 방법:**
```
최소 3개월 기다리기
명백한 문제 아니면
인내심 갖기
```

### 실수 5: 부모 기준으로만

**문제:**
"이 선생님 경력이 좋아"

**현실:**
- 아이가 싫어함
- 효과 없음
- 관계 나쁨

**올바른 방법:**
```
✓ 아이 의견 중요
✓ 체험 후 결정
✓ 관계 > 경력
✓ 편안해야 효과
```

## 예산별 추천 전략

### 예산이 여유로운 경우

**전략:**
```
✓ 1:1 과외
✓ 경력/전문 선생님
✓ 2-3과목 가능
✓ 주 2-3회
```

**주의:**
- 과하게 하지 말기
- 아이 부담 고려
- 선택과 집중

### 예산이 보통인 경우

**전략:**
```
✓ 1-2과목
✓ 대학생 or 경력 선생님
✓ 주 1-2회
✓ 온라인 고려
```

**팁:**
- 가장 필요한 과목
- 효과 보면 확대
- 합리적 선택

### 예산이 빠듯한 경우

**전략:**
```
✓ 1과목 집중
✓ 그룹과외
✓ 온라인
✓ 시험 기간만
```

**대안:**
- 무료 인강 + 짧은 과외
- 방학만 집중
- 자기주도 병행

**중요:**
```
적은 돈이라도
효과적으로 쓰기
선택과 집중
```

## 가치 있는 투자인지 판단

### 체크리스트 (3개월마다)

**성적 변화:**
- [ ] 내신 올랐나?
- [ ] 모의고사 올랐나?
- [ ] 안정적인가?

**학습 태도:**
- [ ] 숙제 잘 하나?
- [ ] 복습 하나?
- [ ] 스스로 공부 시작하나?

**관계:**
- [ ] 선생님과 좋나?
- [ ] 과외 가고 싶어 하나?
- [ ] 편하게 질문하나?

**관리:**
- [ ] 선생님 성실한가?
- [ ] 관리 잘 되나?
- [ ] 소통 잘 되나?

**효과:**
- [ ] 모르던 것 알게 됐나?
- [ ] 자신감 생겼나?
- [ ] 습관 생겼나?

### 판단 기준

**12개 이상 YES:**
```
✅ 매우 효과적
✅ 계속 투자 가치
✅ 현 상태 유지
```

**8-11개 YES:**
```
⚠️ 보통 효과
⚠️ 개선 여지 있음
⚠️ 선생님과 소통
```

**7개 이하 YES:**
```
❌ 효과 부족
❌ 변경 고려
❌ 다른 방법 탐색
```

## 자주 묻는 질문

### Q1. 비싼 선생님이 무조건 좋나요?

A. **아닙니다. 아이와의 궁합이 더 중요합니다.**

이유:
- 비싼 선생님 = 경력 많음
- 하지만 내 아이와 안 맞을 수 있음
- 관계가 좋아야 효과 있음

**올바른 선택:**
1. 체험 수업 필수
2. 아이 의견 듣기
3. 편안한 관계인지
4. 그 다음 실력, 경력

**예시:**
```
A선생님: 20년 경력, 높은 비용
→ 아이가 무서워함
→ 효과 (X)

B선생님: 5년 경력, 적정 비용
→ 아이가 편해함
→ 효과 (O)

결론: B선생님이 더 좋음
```

### Q2. 그룹과외는 효과가 1:1의 절반인가요?

A. **아닙니다. 70-80% 효과를 볼 수 있습니다.**

이유:
- 경쟁 자극 효과
- 친구와 함께 공부
- 서로 가르쳐주며 이해 ↑
- 지속성 ↑

**조건:**
- 비슷한 실력 (등급 차이 1등급 이내)
- 좋은 관계
- 둘 다 성실

**결론:**
```
조건 맞으면:
비용: 60-70%
효과: 70-80%
→ 가성비 높음!
```

### Q3. 과외 효과는 언제 나타나나요?

A. **최소 3개월, 보통 6개월 후입니다.**

**1개월 후:**
- 아직 적응 중
- 효과 판단 어려움
- 관계 형성 중

**3개월 후:**
- 습관 생김
- 성적 조금 오르기 시작
- 첫 평가 시기

**6개월 후:**
- 확실한 효과
- 안정적 성적
- 자기주도 시작

**주의:**
```
❌ 1개월만에 판단 (X)
✅ 최소 3개월 기다리기
✅ 명백한 문제 아니면 인내
```

### Q4. 모든 과목 과외 해야 하나요?

A. **아닙니다. 1-2과목 집중이 더 효과적입니다.**

**문제:**
- 모든 과목 하면 과부하
- 소화 안 됨
- 스트레스만 ↑

**올바른 방법:**
```
1단계: 가장 약한 과목 1개
↓ (3-6개월)
2단계: 효과 있으면 1개 추가
↓
3단계: 안정되면 다른 과목
```

**예시:**
```
현재: 수학 4등급, 영어 3등급, 국어 3등급

선택: 수학만 집중
→ 3개월 후 수학 3등급 됨
→ 영어 추가
→ 수학 2등급 안정되면
→ 국어 고려
```

### Q5. 온라인과 방문 과외, 어떤 게 나을까요?

A. **학생 성향에 따라 다릅니다.**

**온라인이 좋은 경우:**
- 집중력 좋음
- 자기 방에서 편함
- 이동 시간 아까움
- 다양한 선생님 만나고 싶음
- 약간이라도 절약하고 싶음

**방문이 좋은 경우:**
- 집중력 낮음
- 대면이 편함
- 기기 다루기 어려움
- 직접 봐야 효과
- 관리 필요

**팁:**
```
✓ 둘 다 체험해보기
✓ 아이가 더 편한 쪽
✓ 과목별로 다르게 가능
  (수학: 온라인, 영어: 방문)
```

## 마무리하며

**과외 비용, 이것만 기억하세요:**

**핵심 원칙:**
1. 비싸다고 무조건 좋은 것 아님
2. 아이와의 궁합이 가장 중요
3. 1-2과목 집중이 효과적
4. 최소 3개월 후 효과 판단
5. 가성비가 핵심

**합리적 선택을 위해:**
- 체험 수업 필수
- 아이 의견 존중
- 예산 범위 내에서
- 장기적 관점으로
- 3개월마다 재평가

**가장 중요한 것:**
```
돈 > 효과 (X)
효과 > 돈 (O)

효과 있으면 가치 있는 투자
효과 없으면 아무리 싸도 손해
```
