"같은 과외를 받는데 왜 누구는 오르고 누구는 안 오를까?"

**차이는 활용법입니다.** 과외 효과를 10배 높이는 전략을 알려드립니다.

## 과외 효과가 다른 이유

**같은 선생님, 다른 결과:**
```
학생 A: 3개월 만에 2등급 상승
학생 B: 6개월 해도 제자리

차이: 활용법
```

**과외는 도구:**
- 도구를 잘 쓰면 → 큰 효과
- 도구를 잘못 쓰면 → 돈만 낭비
- 수동적으로 받기만 하면 → 효과 반감

**핵심 원리:**
```
과외 시간 (주 3시간) < 혼자 공부 시간 (주 10-20시간)

과외를 어떻게 활용하는가가
전체 학습 효율을 결정!
```

## 효과 극대화 10대 원칙

### 원칙 1: 예습으로 시간 2배 절약

**예습의 힘:**
```
예습 안 함: 과외 시간에 처음 배움 → 이해 50%
예습 함: 과외 시간에 복습+심화 → 이해 90%

같은 시간, 2배 효과!
```

**효과적인 예습 방법:**
```
10-15분만 투자:
1. 다음 수업 내용 교과서로 훑기
2. 이해 안 되는 부분 동그라미
3. 질문 3-5개 정리

이것만으로도 과외 효율 2배
```

**예습 vs 무예습 비교:**
```
[예습 없이]
과외: "오늘 함수 배울 거야"
학생: "함수가 뭐예요?" (처음 들음)
선생님: 기초부터 설명... (시간 많이 소모)
학생: 겨우 이해

[예습하고]
과외: "오늘 함수 배울 거야"
학생: "교과서 봤는데 ○○ 부분 모르겠어요"
선생님: 핵심만 설명 (시간 절약)
학생: 빠르게 이해 + 심화까지
```

**실천 팁:**
- 과외 전날 저녁 10-15분
- 교과서만 가볍게
- 완벽히 이해 안 해도 됨
- 모르는 것 표시만

### 원칙 2: 복습이 90% 결정한다

**복습의 중요성:**
```
과외 직후: 이해도 80-90%
1일 후 복습 안 하면: 50%
3일 후: 30%
1주 후: 10% (거의 까먹음)

복습 = 장기 기억으로 전환
```

**에빙하우스 망각 곡선 활용:**
```
당일 복습: 30분 (필수!)
→ 기억 유지 100%

다음날: 10분
→ 기억 강화

주말: 30분
→ 완전 내 것으로
```

**최적의 복습 타이밍:**
```
1차: 과외 끝나고 1시간 이내 (30분)
2차: 자기 전 (10분)
3차: 다음날 (10분)
4차: 주말 (30분)
5차: 다음 과외 전 (10분)
```

**복습 방법:**
```
✓ 과외 노트 다시 읽기
✓ 오늘 푼 문제 다시 풀기
✓ 선생님 설명 떠올리기
✓ 안 떠오르는 부분 체크
✓ 다음 과외 때 질문
```

### 원칙 3: 숙제는 100% 혼자 풀기

**숙제의 진짜 목적:**
```
목적 1: 복습
목적 2: 약점 발견
목적 3: 선생님이 학생 파악

숙제를 안 하면 → 이 모든 게 불가능
```

**잘못된 숙제 방법:**
```
❌ 답지 보고 베끼기
❌ 부모님/형제 도움
❌ 인터넷 검색해서 복사
❌ 대충 풀고 넘어가기

→ 과외 효과 0%
```

**올바른 숙제 방법:**
```
✅ 100% 혼자 풀기
✅ 모르는 문제도 시도
✅ 못 풀면 표시만 해두기
✅ 왜 못 풀었는지 생각
✅ 다음 과외 때 질문

→ 약점 정확히 파악
→ 선생님이 맞춤 설명 가능
```

**숙제 시간 관리:**
```
예상 시간: 30분-1시간
실제 더 걸려도 괜찮음

오래 걸린 문제 = 약점
→ 과외 때 집중 학습 대상
```

### 원칙 4: 질문을 두려워하지 마라

**질문의 힘:**
```
질문 많은 학생: 빠른 성장
질문 없는 학생: 느린 성장

1:1 과외의 가장 큰 장점 = 무제한 질문
활용 안 하면 = 돈 버리기
```

**좋은 질문 vs 나쁜 질문:**
```
[좋은 질문]
"이 문제에서 왜 이렇게 푸는지 모르겠어요"
"이 개념이 ○○와 어떻게 다른가요?"
"이 풀이를 ○○ 방법으로 풀 수 있나요?"
→ 구체적, 사고를 깊게 함

[나쁜 질문 (없음!)]
모든 질문이 다 좋은 질문!
사소해도, 기초적이어도 괜찮음
```

**질문 준비 방법:**
```
1. 노트에 질문 리스트 작성
2. 과외 시작할 때 먼저 물어보기
3. 수업 중에도 즉시 질문
4. "이해했어요" 거짓말 금지
5. 애매하면 다시 질문
```

**질문하기 어려운 학생을 위한 팁:**
```
방법 1: 질문 노트 만들기
→ 직접 말하기 어려우면 보여주기

방법 2: "여기까지는 이해했는데..."
→ 이해한 부분부터 설명

방법 3: "다시 한번 설명해주실 수 있나요?"
→ 이것만으로도 OK
```

### 원칙 5: 과외 노트는 보물

**과외 노트의 가치:**
```
과외 노트 = 나만의 약점 교과서
= 선생님의 핵심 정리
= 시험 전 최고의 무기
```

**효과적인 노트 필기:**
```
✓ 과외 전용 노트 별도
✓ 날짜, 배운 내용 제목
✓ 선생님 설명 핵심만
✓ 틀린 문제, 약점 표시
✓ 질문과 답변 기록
✓ 색깔 펜으로 구분
```

**노트 활용법:**
```
매일: 10분 훑어보기
시험 전: 과외 노트 집중 복습
다음 과외 전: 지난 내용 복습

이것만으로도
성적 1-2등급 차이
```

**노트 정리 예시:**
```
[2025.10.28 - 함수]

핵심:
- 함수 = 대응 관계
- 일대일 대응 조건

틀린 문제:
- 문제집 p.45 #3
- 왜 틀렸나: ○○ 개념 혼동

질문했던 것:
Q. 함수와 관계의 차이?
A. 함수는 1:1, 관계는 자유

복습 필요: ★★★
```

### 원칙 6: 과외 시간 = 약점 시간

**과외 시간 활용 원칙:**
```
과외에서 할 것:
✓ 모르는 것 (약점)
✓ 어려운 것 (심화)
✓ 질문

혼자 할 수 있는 것:
✓ 쉬운 문제 풀이
✓ 기본 복습
✓ 암기

→ 역할 구분이 효율
```

**비효율적 과외 시간 사용:**
```
❌ 쉬운 문제 풀이 (혼자 할 수 있음)
❌ 단순 암기 (혼자 할 수 있음)
❌ 숙제 검사만 (시간 낭비)

→ 과외 시간이 아까움
```

**효율적 과외 시간 사용:**
```
✅ "이 개념이 이해 안 돼요" → 집중 설명
✅ "이 문제 못 풀었어요" → 약점 보완
✅ "이런 유형 자주 틀려요" → 패턴 학습
✅ "심화 문제도 풀고 싶어요" → 레벨업

→ 과외 시간 200% 활용
```

### 원칙 7: 컨디션 관리가 집중력 결정

**최악의 타이밍:**
```
❌ 식사 직후 (졸림)
❌ 너무 늦은 시간 (피곤)
❌ 잠 부족 (집중 불가)
❌ 학교 시험 직후 (지침)

→ 과외 효과 30%
```

**최적의 컨디션:**
```
✅ 충분한 휴식 후
✅ 가벼운 간식 후 30분
✅ 집중할 수 있는 시간대
✅ 정신이 맑을 때

→ 과외 효과 100%
```

**컨디션 관리 팁:**
```
과외 1시간 전:
- 가벼운 간식 (너무 많이 X)
- 화장실 다녀오기
- 5분 스트레칭
- 휴대폰 멀리

과외 중간 (1시간 지나면):
- 5분 휴식
- 물 마시기
- 간단한 스트레칭
```

### 원칙 8: 선생님과 소통하라

**소통의 중요성:**
```
소통 잘하는 학생:
"요즘 학교에서 ○○ 배우는데 어려워요"
"다음 시험 일정이 ○○인데..."
"이번 주는 피곤해서 집중이 어려워요"
→ 선생님이 맞춤 조정 가능

소통 안 하는 학생:
침묵... 수업만 듣고...
→ 선생님이 파악 어려움
```

**소통할 내용:**
```
✓ 학교 진도, 시험 일정
✓ 어려운 단원, 약점
✓ 학습 스타일, 선호
✓ 과외 페이스 (빠름/느림)
✓ 컨디션, 집중도
```

**소통 방법:**
```
1. 과외 시작할 때:
"선생님, 이번 주 학교에서 ○○ 배웠어요"

2. 수업 중:
"이 설명 속도가 좀 빠른 것 같아요"

3. 수업 끝날 때:
"다음엔 ○○ 부분 더 하고 싶어요"

4. 필요시 메시지:
"다음 주 시험이라 이번 과외는 시험 대비 하고 싶어요"
```

### 원칙 9: 3개월 단위로 점검

**단기 vs 장기:**
```
1개월: 변화 시작
2개월: 확실한 개선
3개월: 성적 반영

→ 너무 빨리 판단 X
```

**3개월 체크리스트:**
```
□ 이해도가 올랐는가?
□ 학습 습관이 개선되었는가?
□ 질문이 늘었는가?
□ 숙제를 잘 하는가?
□ 과외 시간이 알차다고 느끼는가?
□ 작은 변화라도 있는가?

3개 이상 Yes → 계속
3개 미만 → 방법 조정 필요
```

**조정이 필요한 신호:**
```
❌ 3개월 해도 전혀 변화 없음
❌ 학생이 과외 싫어함
❌ 숙제를 계속 안 해옴
❌ 선생님이 자주 바뀜
❌ 과외 시간에 딴짓

→ 문제 파악 및 해결 필요
```

### 원칙 10: 목표를 구체적으로

**막연한 목표:**
```
❌ "수학 잘하고 싶어요"
❌ "성적 올리고 싶어요"
→ 측정 불가, 동기 약함
```

**구체적인 목표:**
```
✅ "3개월 내 수학 4등급 → 2등급"
✅ "다음 시험 함수 단원 90점 이상"
✅ "방학 동안 1학기 완벽 복습"

→ 측정 가능, 동기 강함
```

**목표 설정 프레임:**
```
SMART 목표:
S - Specific (구체적): 무엇을
M - Measurable (측정 가능): 어느 정도
A - Achievable (달성 가능): 현실적
R - Relevant (관련성): 의미 있는
T - Time-bound (기한): 언제까지

예: 3개월 내(T) 수학(S) 2등급 달성(M, A, R)
```

## 학생이 해야 할 것 vs 하지 말아야 할 것

### 반드시 해야 할 것 ✅

**Before 과외:**
```
1. 예습 10-15분
2. 질문 리스트 준비
3. 지난 숙제 확인
4. 컨디션 체크
5. 준비물 챙기기
```

**During 과외:**
```
1. 집중 (핸드폰 끄기)
2. 적극적 질문
3. 노트 꼼꼼히
4. 이해 안 되면 즉시 말하기
5. 선생님과 소통
```

**After 과외:**
```
1. 당일 복습 30분 (필수!)
2. 숙제 100% 혼자
3. 모르는 것 표시
4. 다음 과외 전 노트 복습
5. 질문 미리 준비
```

### 절대 하지 말아야 할 것 ❌

**학습 태도:**
```
❌ 과외만 의존 (혼자 공부 안 함)
❌ 수동적 수업 (질문 없음)
❌ 숙제 안 함 / 베끼기
❌ 복습 안 함 (까먹음)
❌ 선생님 말씀 안 듣기
```

**시간 관리:**
```
❌ 과외 시간에 딴짓
❌ 늦게 시작 (지각)
❌ 피곤한 상태로 수업
❌ 중간에 자주 쉬자고 하기
❌ 급하게 끝내려 함
```

**소통:**
```
❌ 모르는데 "네" 라고만 대답
❌ 어려운데 말 안 함
❌ 선생님과 대화 안 함
❌ 부모님께 과외 상황 숨김
❌ 문제 있어도 방치
```

## 부모님이 해야 할 것 vs 하지 말아야 할 것

### 부모님의 올바른 역할 ✅

**환경 조성:**
```
✓ 조용한 학습 공간 제공
✓ 방해 요소 제거
✓ 적절한 조명, 온도
✓ 필요한 학습 자료 지원
✓ 간식/물 준비
```

**정서적 지원:**
```
✓ 격려와 칭찬
✓ 작은 발전 인정
✓ 인내심 갖기 (3개월)
✓ 과정 중시
✓ 믿고 맡기기
```

**소통:**
```
✓ 월 1-2회 선생님 피드백
✓ 학생 상태 관찰
✓ 문제 발생 시 상의
✓ 목표 함께 설정
✓ 응원 메시지
```

### 부모님이 피해야 할 것 ❌

**과도한 개입:**
```
❌ 과외 시간에 드나듦
❌ 수업 방식 간섭
❌ "이것도 저것도" 요구
❌ 선생님 불신 표현
❌ 아이 앞에서 불만
```

**압박:**
```
❌ "왜 안 올라?" 재촉
❌ 단기 성과 압박
❌ 다른 아이와 비교
❌ 과외 시간 늘리자고 압박
❌ 과도한 기대
```

**방임:**
```
❌ 전혀 관심 없음
❌ 선생님과 소통 0
❌ 학생 상태 파악 안 함
❌ 숙제 하는지 안 하는지 모름
❌ 효과 있는지 확인 안 함
```

## 과목별 효과 극대화 전략

### 수학 과외 극대화

**Before 과외:**
```
1. 지난 시간 문제 다시 풀기
2. 안 풀리는 문제 표시
3. 개념 교과서로 복습
4. 질문 3-5개 준비
```

**During 과외:**
```
1. 틀린 문제 왜 틀렸는지 분석
2. 비슷한 유형 반복
3. 풀이 과정 소리내어 설명
4. 다른 방법도 물어보기
5. 실전 시간 재고 풀기
```

**After 과외:**
```
1. 당일: 오늘 푼 문제 다시
2. 다음날: 비슷한 문제 5개
3. 주말: 전체 복습
4. 시험 전: 과외 노트 집중
```

**효과 2배 팁:**
```
★ 풀이 암기 X, 원리 이해 O
★ 한 문제를 여러 방법으로
★ 왜 이 방법인지 항상 질문
★ 계산 실수 패턴 파악
```

### 영어 과외 극대화

**Before 과외:**
```
1. 단어 미리 암기
2. 지문 한번 읽어보기
3. 모르는 문법 체크
4. 질문 준비
```

**During 과외:**
```
1. 문법 이해 안 되면 예문 요청
2. 서술형 첨삭 받기
3. 발음 교정
4. 실전 독해 연습
5. 약점 문법 집중
```

**After 과외:**
```
1. 당일: 배운 문법 복습
2. 매일: 단어 암기 + 복습
3. 주말: 지문 다시 읽기
4. 틈틈이: 영어 문장 소리내어 읽기
```

**효과 2배 팁:**
```
★ 문법은 이해, 단어는 암기
★ 서술형은 첨삭 필수
★ 매일 10분 영어 노출
★ 과외 때 영어로 말해보기
```

### 국어 과외 극대화

**Before 과외:**
```
1. 작품/지문 미리 읽기
2. 이해 안 되는 부분 표시
3. 내 생각 정리
4. 질문 준비
```

**During 과외:**
```
1. 작품 배경 이해
2. 주제 파악 연습
3. 서술형 답안 작성 + 첨삭
4. 비문학 독해 전략
5. 어휘력 확장
```

**After 과외:**
```
1. 당일: 작품 다시 읽기
2. 작품 정리 노트
3. 비슷한 지문 혼자 풀기
4. 서술형 답안 다시 써보기
```

**효과 2배 팁:**
```
★ 많이 읽기 (평소에)
★ 배경지식 쌓기
★ 서술형은 피드백이 핵심
★ 어휘 노트 만들기
```

## 효과 없는 학생의 공통점

### 패턴 1: 수동적 학습

**특징:**
```
- 과외 시간에만 공부
- 질문 없음
- "네" 라고만 대답
- 숙제 대충
- 복습 안 함
```

**결과:**
```
6개월 해도 제자리
돈만 낭비
선생님도 답답
```

**해결:**
```
→ 적극적 태도로 전환
→ 질문 의무적으로 3개 이상
→ 숙제 100% 완성
→ 매일 복습 습관
```

### 패턴 2: 과외 의존

**특징:**
```
- 혼자 공부 시간 0
- 모든 걸 과외 때만
- "과외 때 배우면 되지" 마인드
- 복습 안 함
- 예습 안 함
```

**결과:**
```
과외 때는 이해
1주일 후 까먹음
시험 때 기억 안 남
성적 안 오름
```

**해결:**
```
→ 과외 : 혼자 공부 = 3 : 7
→ 과외는 가이드, 실력은 혼자
→ 매일 30분 이상 혼자 공부
```

### 패턴 3: 복습 제로

**특징:**
```
- 과외 끝나면 노트 덮음
- 다음 과외 때까지 안 봄
- 숙제만 하고 끝
- 시험 전에만 벼락치기
```

**결과:**
```
과외 때마다 다시 설명
진도 못 나감
깊이 못 들어감
장기 기억 안 됨
```

**해결:**
```
→ 당일 복습 30분 필수
→ 과외 노트 매일 10분 보기
→ 주말 1시간 총복습
```

### 패턴 4: 숙제 불성실

**특징:**
```
- 숙제 안 해감
- 답지 베낌
- 대충 풀고 넘어감
- "깜빡했어요" 자주
```

**결과:**
```
약점 파악 불가
선생님이 학생 모름
맞춤 수업 불가
효과 0%
```

**해결:**
```
→ 숙제 = 복습 + 약점 발견
→ 100% 혼자 풀기
→ 못 풀어도 표시만
→ 과외 때 질문
```

### 패턴 5: 컨디션 최악

**특징:**
```
- 항상 피곤
- 과외 시간에 졸음
- 집중 안 됨
- 늦게 자고 늦게 일어남
```

**결과:**
```
과외 시간 낭비
설명 귀에 안 들어옴
돈만 버리기
선생님도 힘듦
```

**해결:**
```
→ 충분한 수면
→ 과외 시간 조정
→ 과외 전 가벼운 휴식
→ 건강 관리
```

## 효과 높은 학생의 공통점

### A학생: 3개월 만에 2등급 상승

**습관:**
```
✓ 예습 15분 (매번)
✓ 복습 당일 30분 (필수)
✓ 숙제 100% 혼자
✓ 질문 매 수업 5개 이상
✓ 과외 노트 매일 복습
```

**결과:**
```
이해도 90% 이상
장기 기억 확실
약점 빠르게 보완
3개월 만에 4등급 → 2등급
```

### B학생: 꾸준함이 핵심

**습관:**
```
✓ 과외 주 2회 (1년간 결석 없음)
✓ 매일 30분 복습
✓ 주말 1시간 총복습
✓ 선생님과 소통 활발
✓ 목표 명확 (2등급)
```

**결과:**
```
6개월: 3등급
9개월: 2등급
1년: 안정적 2등급 유지
```

### C학생: 적극성이 답

**습관:**
```
✓ 질문 폭탄 (수업 시간 대부분 질문)
✓ 이해 안 되면 끝까지 물어봄
✓ 숙제 외 추가 문제도 요청
✓ 선생님께 학습 방법 자주 상의
✓ 피드백 적극 수용
```

**결과:**
```
1달: 이해도 급상승
2달: 학습 태도 완전 변화
3달: 5등급 → 3등급
6달: 2등급 달성
```

## 시기별 활용 전략

### 학기 초 (3월, 9월)

**초점:**
```
- 기초 다지기
- 개념 확실히
- 꼼꼼한 진도
- 약점 보완
```

**활용 전략:**
```
✓ 진도 천천히, 이해 확실히
✓ 기초 문제 반복
✓ 학교 진도 맞추기
✓ 학습 습관 형성
```

### 학기 중 (4-5월, 10-11월)

**초점:**
```
- 시험 준비
- 실전 연습
- 약점 집중
- 시간 관리
```

**활용 전략:**
```
✓ 시험 2주 전부터 시험 대비 모드
✓ 기출 문제 반복
✓ 약점 완전 정복
✓ 실전 시간 재고 풀기
```

### 방학 (여름, 겨울)

**초점:**
```
- 약점 완전 보완
- 선행 or 복습
- 집중 학습
- 체력 관리
```

**활용 전략:**
```
✓ 빈도 증가 (주 3-4회)
✓ 긴 시간 (2시간)
✓ 약점 집중 공략
✓ 다음 학기 예습
```

### 시험 2주 전

**초점:**
```
- 완전 시험 모드
- 약점만 집중
- 실전 연습
- 멘탈 관리
```

**활용 전략:**
```
✓ 새로운 것 배우지 않기
✓ 틀린 문제만 반복
✓ 시험 범위 총정리
✓ 실전 시뮬레이션
✓ 컨디션 관리
```

## 자주 묻는 질문

### Q1. 과외를 3개월 했는데 성적이 안 올라요. 포기해야 하나요?

A. **3개월은 짧습니다. 활용법을 점검하세요.**

**체크리스트:**
```
□ 예습을 하고 있나요?
□ 매일 복습을 하나요?
□ 숙제를 혼자 푸나요?
□ 질문을 많이 하나요?
□ 과외 노트를 복습하나요?
□ 학생이 적극적인가요?
```

**3개 이상 No:**
→ 활용법 문제, 개선하면 효과
**5개 이상 No:**
→ 과외가 문제가 아니라 학습 태도 문제

**판단 기준:**
```
3개월: 이해도 향상 여부
6개월: 성적 반영 기대
1년: 확실한 효과

3개월에 성적 안 올라도
이해도가 올랐다면 계속!
```

### Q2. 과외 효과를 가장 빨리 보는 방법은?

A. **복습입니다. 당일 복습 30분이 핵심입니다.**

**실험 결과:**
```
그룹 A: 과외만 (복습 X)
→ 3개월 후 변화 미미

그룹 B: 과외 + 당일 복습 30분
→ 3개월 후 1-2등급 상승

차이: 복습 30분
```

**복습 루틴:**
```
과외 끝나고 1시간 이내:
- 과외 노트 다시 읽기
- 오늘 푼 문제 다시 풀기
- 30분 투자

→ 이것만으로도
   장기 기억으로 전환
   효과 3배
```

### Q3. 과외 시간에 너무 많이 질문하면 선생님이 싫어하지 않을까요?

A. **절대 아닙니다! 질문 많은 학생을 선생님들이 좋아합니다.**

**선생님 입장:**
```
질문 많은 학생:
"적극적이다, 열심히다, 이해하려 한다"
→ 가르치는 보람, 더 열심히 가르침

질문 없는 학생:
"이해했나? 집중하나? 효과 있나?"
→ 파악 어려움, 답답함
```

**과외의 가장 큰 장점:**
```
= 무제한 질문

질문 안 하면
과외비의 50%를 버리는 것!
```

**팁:**
```
✓ 사소한 것도 질문
✓ 이해 안 되면 여러 번
✓ 왜? 를 항상 물어보기
✓ 다른 방법도 물어보기
```

### Q4. 학원과 과외를 같이 하는데, 과외 효과를 어떻게 높이나요?

A. **역할을 명확히 구분하고, 과외는 학원 내용 완벽 소화에 집중하세요.**

**Best 조합:**
```
학원: 진도, 새 내용, 기본 문제
과외: 학원 복습, 약점 보완, 질문 해결

→ 학원 내용 100% 소화
→ 빈틈 없는 관리
```

**과외 시간 활용:**
```
1. 학원에서 이해 안 된 부분 질문
2. 학원 숙제 중 틀린 문제 분석
3. 학원 진도 복습
4. 약점 집중 보완

→ 학원 효과도 2배
```

**소통:**
```
과외 선생님께:
"학원에서 이번 주 ○○ 배웠는데
 이 부분이 이해 안 돼요"

→ 맞춤 설명 가능
→ 시너지 효과
```

### Q5. 과외를 더 자주 하면 효과가 더 좋을까요?

A. **적정 빈도가 있습니다. 무조건 많다고 좋은 게 아닙니다.**

**적정 빈도:**
```
중1-2: 주 1-2회 (충분)
중3: 주 2-3회 (적절)
시험 기간: 주 3-4회 (단기)

주 5회 이상: 과부하 위험!
```

**왜 많다고 좋지 않나:**
```
문제 1: 혼자 공부 시간 부족
→ 복습 못 함
→ 숙제 못 함
→ 소화 안 됨

문제 2: 과외 의존
→ 자기주도 학습 능력 저하

문제 3: 번아웃
→ 집중력 저하
→ 효율 급감
```

**효율적 조합:**
```
과외 주 2회 (3시간)
+ 혼자 공부 매일 1시간 (7시간)
= 총 10시간

이게
과외 주 5회 (7.5시간)
보다 효과적!
```

**판단 기준:**
```
✅ 혼자 공부 시간 확보되는가?
✅ 복습할 시간 있는가?
✅ 숙제 여유 있게 하는가?
✅ 과부하 아닌가?

→ 빈도보다 활용도가 중요
```

## 마무리

**과외 효과 극대화, 핵심 정리:**

**학생이 해야 할 것:**
```
1. 예습 15분 (과외 전)
2. 복습 30분 (과외 당일, 필수!)
3. 숙제 100% 혼자
4. 질문 매 수업 5개 이상
5. 과외 노트 매일 복습
```

**효과 10배 비결:**
```
과외 시간만으로는 부족
과외 전후가 진짜 승부처

Before: 예습 + 질문 준비
During: 집중 + 적극성
After: 복습 + 숙제 + 복습

이 사이클 = 성공
```

**기억할 것:**
```
✓ 과외 = 도구 (활용법이 핵심)
✓ 수동적 ≠ 효과
✓ 복습 = 90% 결정
✓ 질문 = 과외의 꽃
✓ 3개월 = 최소 기간
```

**최종 체크:**
```
효과 있는 과외:
□ 매번 배우는 게 명확
□ 약점이 보완되는 느낌
□ 이해도가 올라감
□ 질문이 줄어듦 (이해했으니)
□ 혼자 풀 수 있는 문제 증가

→ 3개 이상이면 잘하고 있습니다!
```

**행동 계획:**
```
오늘부터:
1. 과외 노트 다시 보기
2. 당일 복습 30분 실천
3. 질문 리스트 작성

이것만으로도
효과 2배!
```
