"서술형만 나오면 손이 안 가요!"

**서술형은 기술입니다.** 배우면 점수가 오릅니다.

**서술형 완벽 대비 전략을 알려드립니다.**

## 서술형의 중요성

### 내신에서 차지하는 비중

**증가하는 비중:**
- 과거: 10-20%
- 현재: 30-50%
- 과목에 따라 더 높음

**왜 늘어날까?**
```
객관식: 찍기 가능
서술형: 진짜 실력 평가
→ 변별력 높음
→ 비중 증가
```

**과목별 비중:**
- 수학: 40-50%
- 영어: 30-40%
- 국어: 30-40%
- 과학/사회: 30-40%

**등급 영향:**
```
객관식 다 맞아도
서술형 틀리면
1-2등급 불가능
```

### 서술형이 어려운 이유

**1. 뭘 써야 할지 모름**
```
문제: 원리를 설명하시오
학생: "원리가 뭐지?"
     "어디서부터 써야 하지?"
```

**2. 알아도 표현 못 함**
```
머릿속: 알고 있음
종이 위: 빈칸
→ 언어화의 어려움
```

**3. 키워드를 모름**
```
내 답: "온도가 올라가서"
채점 기준: "열에너지 증가로"
→ 0점 (키워드 없음)
```

**4. 시간 부족**
```
객관식: 빠르게 풀기
서술형: 생각하고 쓰기
→ 시간 관리 실패
```

**5. 부분점수 전략 모름**
```
전체 답 모르면 백지
→ 0점
부분만이라도 쓰면
→ 50% 점수
```

### 서술형 실패의 영향

**성적 하락:**
```
객관식 90점 + 서술형 30점
= 총 65점 (4등급)

객관식 80점 + 서술형 70점
= 총 75점 (2등급)

서술형이 등급 결정!
```

**자신감 상실:**
- "나는 서술형 안 돼"
- 포기 심리
- 악순환

## 서술형 과외의 필요성

### 혼자 준비의 한계

**문제 1: 첨삭 불가**
```
혼자: 답안 씀
     → 맞는지 모름
     → 개선 안 됨
```

**문제 2: 채점 기준 모름**
```
내 답: 길게 씀
채점 기준: 키워드 3개만
→ 방향 틀림
```

**문제 3: 피드백 없음**
```
계속 틀린 방식으로 연습
→ 시간 낭비
→ 실력 안 늘음
```

**문제 4: 부분점수 전략 모름**
```
답 확실히 모르면
→ 아무것도 안 씀
→ 0점
```

### 1:1 과외의 효과

**즉시 첨삭:**
```
학생: 답안 작성
↓
선생님: 바로 첨삭
↓
학생: 수정
↓
반복 → 실력 향상
```

**채점 기준 파악:**
```
선생님: "이 문제는 키워드 3개"
학생: 명확한 방향
효율적 연습
```

**맞춤 피드백:**
```
학생 A: 키워드는 아는데 문장 구성 약함
→ 문장 연습 집중

학생 B: 개념은 아는데 키워드 모름
→ 용어 암기 집중
```

**전략 지도:**
```
✓ 부분점수 따는 법
✓ 시간 배분
✓ 답안 구조화
✓ 실전 팁
```

### 언제 서술형 과외가 필요한가?

**필수인 경우:**
- [ ] 서술형 점수가 30% 이하
- [ ] 알아도 표현 못 함
- [ ] 키워드를 모름
- [ ] 백지로 내는 경우 많음
- [ ] 서술형 때문에 등급 떨어짐

**2개 이상: 서술형 과외 강력 추천**

**선택적인 경우:**
- 서술형 60-70% 맞음
- 조금만 더 올리면 1등급
- 특정 과목만 약함

**불필요한 경우:**
- 서술형 80% 이상 맞음
- 답안 작성 능숙
- 기본 실력 부족 (먼저 개념)

## 과목별 서술형 전략

### 수학 서술형

**출제 유형:**

**유형 1: 풀이 과정**
```
문제: 다음을 풀고 풀이 과정을 쓰시오.
     2x + 3 = 7

요구: 각 단계별 식
```

**유형 2: 증명**
```
문제: 삼각형의 내각의 합이 180도임을 증명하시오.
요구: 논리적 증명 과정
```

**유형 3: 설명**
```
문제: 이차방정식의 해가 2개인 이유를 설명하시오.
요구: 원리 설명
```

**답안 작성 원칙:**

**1. 식을 순서대로**
```
Bad:
답: x = 2

Good:
2x + 3 = 7
2x = 7 - 3
2x = 4
x = 2
∴ x = 2
```

**2. 근거 명시**
```
Bad:
x = 2

Good:
양변에서 3을 빼면
2x = 4
양변을 2로 나누면
x = 2
```

**3. 단위와 답 명확히**
```
✓ 단위 쓰기 (cm, m 등)
✓ 답에 밑줄 또는 ∴
✓ 최종 답 강조
```

**4. 중간 과정 생략 금지**
```
Bad:
2x + 3 = 7
x = 2 (과정 생략)
→ 부분점수 못 받음

Good:
2x + 3 = 7
2x = 4
x = 2
→ 각 단계 점수
```

**부분점수 전략:**
```
전체 못 풀어도:
✓ 식 세우기까지: 30%
✓ 중간 계산: 30%
✓ 답 도출: 40%

→ 아는 만큼 쓰기!
```

**연습 방법:**
1. 문제집 풀이 과정 따라 쓰기
2. 선생님께 첨삭 받기
3. 표준 풀이 형식 익히기
4. 시간 재며 연습

### 영어 서술형

**출제 유형:**

**유형 1: 영작**
```
문제: 다음 우리말을 영어로 쓰시오.
     "나는 어제 도서관에 갔다."

요구: 문법 정확한 영작
```

**유형 2: 본문 이해**
```
문제: 다음 글의 주제를 영어로 쓰시오.
요구: 본문 이해 + 영작
```

**유형 3: 문법 설명**
```
문제: 다음 문장의 시제를 설명하시오.
요구: 문법 용어 사용
```

**답안 작성 원칙:**

**1. 문법 정확성**
```
Bad:
I go to library yesterday.
→ 시제, 관사 오류

Good:
I went to the library yesterday.
→ 과거형, 관사 정확
```

**2. 키워드 포함**
```
문제: 본문의 main idea는?

Bad:
It's about health.

Good:
The main idea is the importance of regular exercise for health.
→ "main idea", "importance" 키워드
```

**3. 철자와 띄어쓰기**
```
틀린 철자 = 감점
✓ 자주 틀리는 단어 암기
✓ 시간 남으면 철자 확인
```

**4. 문장 완성**
```
Bad:
Because I was tired.
→ 불완전한 문장

Good:
I went home early because I was tired.
→ 완전한 문장
```

**부분점수 전략:**
```
영작 문제 (10점):
✓ 주어-동사: 3점
✓ 목적어: 2점
✓ 시제: 2점
✓ 기타: 3점

→ 완벽 못 해도 아는 부분 쓰기
```

**연습 방법:**
1. 교과서 핵심 문장 암기
2. 문법 용어 정리
3. 빈출 표현 암기
4. 첨삭 반복

### 국어 서술형

**출제 유형:**

**유형 1: 감상**

```
문제: 이 시의 분위기를 서술하시오.
요구: 작품 감상 + 근거
```

**유형 2: 분석**
```
문제: 이 글의 논증 구조를 분석하시오.

```

**유형 3: 설명**
```
문제: 속담의 의미를 현대적으로 해석하시오.
요구: 이해 + 적용
```

**답안 작성 원칙:**

**1. 키워드 필수**
```
문제: 이 시의 화자의 정서는?

Bad:
슬픈 느낌

Good:
화자는 '그리움'과 '애틋함'의 정서를 느끼고 있다.
→ 문학 용어 사용
```

**2. 근거 제시**
```
Bad:
화자는 슬프다.

Good:
화자는 "눈물"과 "한숨" 등의 표현을 통해 슬픔의 정서를 드러낸다.
→ 본문 근거 제시
```

**3. 문장 완결**
```
Bad:
그리움의 정서. 슬픔.
→ 명사형 나열

Good:
화자는 그리움의 정서를 느끼며, 이는 슬픔으로 이어진다.
→ 완전한 문장
```

**4. 구조화**
```
① 결론 먼저 (핵심 답)
② 근거 제시 (본문 인용)
③ 부연 설명 (필요시)
```

**부분점수 전략:**
```
감상 문제 (10점):
✓ 정서 단어: 3점
✓ 본문 근거: 4점
✓ 설명: 3점

→ 정서 단어라도 정확히!
```

**연습 방법:**
1. 문학 용어 암기 (정서, 표현법 등)
2. 교과서 지문 분석 연습
3. 답안 구조화 연습
4. 모범 답안 필사

### 과학 서술형

**출제 유형:**

**유형 1: 실험 설명**
```
문제: 이 실험 결과가 나타난 이유를 설명하시오.
요구: 원리 + 인과관계
```

**유형 2: 원리 서술**
```
문제: 광합성 과정을 서술하시오.
요구: 단계별 설명
```

**유형 3: 현상 해석**
```
문제: 이 현상을 과학 원리로 설명하시오.
요구: 개념 적용
```

**답안 작성 원칙:**

**1. 과학 용어 정확히**
```
Bad:
물이 끓어서 수증기가 된다.

Good:
물이 끓는점(100℃)에 도달하면 상태 변화(기화)가 일어나 수증기가 된다.
→ 용어: 끓는점, 상태변화, 기화
```

**2. 인과관계 명확히**
```
Bad:
온도가 오르고 부피가 커졌다.

Good:
온도가 상승하면 기체 분자의 운동이 활발해지므로 부피가 증가한다.
→ 원인 → 결과 명확
```

**3. 순서 명확히**
```
광합성 과정:
① 빛 에너지 흡수
② 물 분해
③ 이산화탄소 결합
④ 포도당 생성
→ 번호 또는 화살표로 순서
```

**4. 단위 표기**
```
✓ 온도: ℃
✓ 거리: m, cm
✓ 질량: g, kg
→ 단위 빠뜨리면 감점
```

**부분점수 전략:**
```
원리 설명 (10점):
✓ 핵심 용어: 3점
✓ 인과관계: 4점
✓ 부가 설명: 3점

→ 용어라도 정확히!
```

**연습 방법:**
1. 과학 용어 정리
2. 원리 인과관계로 정리
3. 그림과 함께 설명 연습
4. 교과서 탐구 활동 복습

## 서술형 답안 작성 기술

### 1. 키워드 파악법

**채점 기준의 핵심:**
```
채점 기준:
"열에너지", "전도", "분자 운동"
→ 이 3개가 있으면 만점
→ 없으면 0점
```

**키워드 찾는 법:**
1. 교과서 굵은 글씨
2. 선생님이 강조한 용어
3. 문제에서 요구하는 개념

**키워드 암기:**
```
과목별로 정리:
수학: 증명, 근, 해, 공식명 등
영어: 문법 용어, 주제 표현
국어: 정서, 표현법, 주제
과학: 과학 용어, 원리명
사회: 역사 용어, 개념
```

### 2. 답안 구조화

**기본 구조:**
```
① 결론 (핵심 답)
② 근거 (이유, 원리)
③ 부연 (예시, 추가 설명)
```

**예시:**
```
문제: 물의 끓는점이 100℃인 이유는?

답안:
① 물은 100℃에서 기화한다. (결론)
② 이 온도에서 수증기압이 대기압과 같아지기 때문이다. (근거)
③ 따라서 물 분자가 기체 상태로 변하게 된다. (부연)
```

### 3. 문장 쓰기 기술

**문장 구성:**
```
주어 + 서술어 명확히
원인 + 결과 명시
접속사 활용

예:
"온도가 상승하면(원인) 부피가 증가한다(결과).
왜냐하면(접속사) 분자 운동이 활발해지기 때문이다."
```

**피해야 할 표현:**
```
❌ "~것 같다" (추측)
❌ "아마도" (불확실)
❌ 명사 나열만
❌ 너무 짧은 답

✅ 단정적 표현
✅ 확실한 근거
✅ 완전한 문장
✅ 적절한 길이
```

### 4. 시간 관리

**서술형 시간 배분:**
```
50분 시험 기준:
객관식 20분 (20문제)
서술형 30분 (5문제)
→ 문제당 6분
```

**문제당 시간 배분:**
```
1분: 문제 이해, 키워드 체크
3분: 답안 작성
1분: 검토, 수정
1분: 예비
```

**시간 부족 시 전략:**
```
✓ 완벽보다 부분 점수
✓ 키워드만이라도 쓰기
✓ 구조 유지 (개조식이라도)
✓ 마지막 문제 포기 고려
```

## 서술형 과외 진행 방법

### 1단계: 채점 기준 학습 (1-2주)

**내용:**
```
✓ 과목별 채점 기준 파악
✓ 키워드 정리
✓ 배점 비율 이해
✓ 부분점수 기준
```

**방법:**
- 기출 문제 채점 기준 분석
- 선생님 설명
- 키워드 목록 작성

### 2단계: 답안 작성 연습 (2-4주)

**내용:**
```
✓ 구조화된 답안 쓰기
✓ 키워드 포함 연습
✓ 문장 구성 연습
```

**방법:**
1. 학생: 답안 작성
2. 선생님: 즉시 첨삭
3. 학생: 수정 반복
4. 모범 답안 비교

**첨삭 포인트:**
```
✓ 키워드 포함 여부
✓ 문장 정확성
✓ 구조화
✓ 길이 적절성
```

### 3단계: 실전 연습 (2-4주)

**내용:**
```
✓ 시간 재며 풀기
✓ 다양한 유형
✓ 실전 모의고사
```

**방법:**
- 실제 시험처럼 시간 제한
- 채점 후 피드백
- 틀린 문제 다시 쓰기
- 반복 훈련

### 4단계: 약점 보완 (지속)

**개별 맞춤:**
```
학생 A: 키워드는 아는데 문장 구성 약함
→ 문장 쓰기 집중 훈련

학생 B: 개념은 아는데 용어 모름
→ 용어 암기 + 적용

학생 C: 시간 관리 실패
→ 시간 단축 훈련
```

## 서술형 과외의 효과

### 단기 효과 (1-2개월)

**답안 작성 익숙:**
- 구조화 자동으로
- 키워드 자연스럽게
- 쓰기 속도 ↑

**자신감 향상:**
- "쓸 수 있다"
- 백지 두려움 ↓
- 적극적 시도

### 중기 효과 (3-4개월)

**점수 상승:**
```
Before: 서술형 30-40%
After: 서술형 60-70%
→ 총점 10-15점 ↑
```

**전략적 접근:**
- 부분점수 따기
- 시간 배분 능숙
- 문제 유형별 대응

### 장기 효과 (6개월 이상)

**완전한 숙달:**
- 서술형 80% 이상
- 자동화된 작성
- 안정적 고득점

**고등학교 대비:**
- 서술형 기초 탄탄
- 논술 준비 유리
- 수능 서술형 대비

## 서술형 과외 성공 사례

### 사례 1: 수학 서술형 극복

**학생 프로필:**
- 중2, 수학 3등급
- 객관식 잘하는데 서술형 0-20%
- 과정 쓰기 어려워함

**과외 내용:**
- 풀이 과정 표준 형식 학습
- 단계별 서술 연습
- 첨삭 반복

**결과 (3개월 후):**
```
Before: 서술형 15% → 총 72점 (3등급)
After: 서술형 70% → 총 88점 (1등급)
```

### 사례 2: 영어 영작 정복

**학생 프로필:**
- 중3, 영어 2등급
- 독해는 잘하는데 영작 약함
- 문법 실수 많음

**과외 내용:**
- 핵심 문장 패턴 암기
- 문법 정리
- 반복 첨삭

**결과 (2개월 후):**
```
Before: 영작 30% → 총 80점 (2등급)
After: 영작 80% → 총 93점 (1등급)
```

### 사례 3: 국어 감상 문제

**학생 프로필:**
- 중1, 국어 4등급
- 감상 문제 표현 못 함
- 백지로 내는 경우 많음

**과외 내용:**
- 문학 용어 암기
- 감상 구조 학습
- 모범 답안 필사

**결과 (4개월 후):**
```
Before: 서술형 백지 → 총 65점 (4등급)
After: 서술형 60% → 총 78점 (2등급)
```

## 단기 vs 장기 준비

### 단기 집중 (시험 2-4주 전)

**장점:**
- 시험 직전 마무리
- 실전 감각
- 빠른 효과

**내용:**
```
✓ 예상 문제 풀이
✓ 학교 기출 분석
✓ 채점 기준 파악
✓ 시간 훈련
```

**효과:**
- 10-20점 향상 가능
- 단기 목표 달성

**한계:**
- 근본적 실력 향상 어려움
- 다음 시험 다시 준비 필요

### 장기 준비 (3-6개월)

**장점:**
- 근본 실력 향상
- 안정적 고득점
- 지속 가능

**내용:**
```
✓ 기초부터 체계적
✓ 과목별 전략
✓ 반복 훈련
✓ 약점 보완
```

**효과:**
- 20-30점 향상
- 안정적 유지
- 고등학교까지 도움

**추천:**
- 근본적 개선 원할 때
- 장기적 관점
- 1-2등급 목표

## 효과적인 서술형 학습법

### 과외 시간 (주 1-2회)

**선생님과:**
1. 답안 작성
2. 즉시 첨삭
3. 수정 연습
4. 새로운 전략 배우기

### 혼자 할 일 (매일 30분)

**복습:**
- 첨삭받은 답안 다시 쓰기
- 키워드 암기
- 모범 답안 필사

**연습:**
- 새 문제 풀어보기
- 시간 재며 쓰기
- 자가 체크

**정리:**
- 키워드 노트 만들기
- 자주 틀리는 유형 정리
- 답안 템플릿 만들기

## 자주 묻는 질문

### Q1. 서술형 과외는 얼마나 해야 효과가 있나요?

A. **최소 2-3개월은 필요합니다.**

**기간별 효과:**
```
1개월: 구조 익히기 시작
2개월: 답안 작성 익숙
3개월: 점수 상승 시작
4-6개월: 완전히 숙달
```

**추천:**
- 시험 대비: 2-4주
- 근본 실력: 3-6개월
- 완전 숙달: 6개월 이상

**팁:**
```
단기: 빠른 효과, 임시방편
장기: 근본 해결, 안정적
→ 가능하면 장기 추천
```

### Q2. 개념 과외와 서술형 과외, 뭘 먼저?

A. **개념이 먼저입니다.**

**판단 기준:**
```
객관식 70% 이상: 개념 있음
→ 서술형 과외 가능

객관식 70% 미만: 개념 부족
→ 개념 과외 먼저
```

**이유:**
- 개념 없이 서술형 불가능
- 알아야 쓸 수 있음
- 순서 중요

**병행 가능:**
```
개념 과외 + 서술형 훈련
하나의 과외에서 함께
효율적
```

### Q3. 어떤 과목을 서술형 과외 해야 하나요?

A. **가장 비중 크고 약한 과목부터입니다.**

**우선순위:**
```
1순위: 수학 (비중 40-50%)
2순위: 영어 (비중 30-40%)
3순위: 국어 (비중 30-40%)
4순위: 과학/사회
```

**판단:**
- 서술형 비중이 큰 과목
- 현재 가장 약한 과목
- 등급에 영향 큰 과목

**팁:**
```
한 과목 완전히 숙달 후
다른 과목 추가
→ 한 번에 모든 과목 (X)
```

### Q4. 서술형 과외로 몇 점이나 오르나요?

A. **현재 실력에 따라 다릅니다.**

**현재 서술형 0-30%:**
```
→ 60-70%까지 가능
→ 총점 15-20점 향상
→ 1-2등급 상승 가능
```

**현재 서술형 50-60%:**
```
→ 80-90%까지 가능
→ 총점 10점 향상
→ 1등급 진입 가능
```

**현재 서술형 70% 이상:**
```
→ 90% 이상
→ 총점 5-7점 향상
→ 1등급 안정화
```

**조건:**
- 꾸준한 연습
- 3개월 이상
- 복습 철저

### Q5. 서술형 잘하는 법, 따로 비법 있나요?

A. **비법은 없습니다. 연습이 답입니다.**

**진실:**
```
서술형 = 기술
기술 = 반복 연습으로 습득
```

**효과적인 방법:**
1. 채점 기준 파악
2. 키워드 암기
3. 구조화 연습
4. 첨삭 받기
5. 반복 쓰기

**실수:**
```
❌ 문제만 많이 풀기
❌ 읽기만 하기
❌ 첨삭 없이 혼자

✅ 쓰고 첨삭 받기
✅ 반복 수정
✅ 선생님 피드백
```

**핵심:**
```
쓰기 → 첨삭 → 수정 → 반복
이 사이클이 비법!
```

## 마무리하며

**서술형, 이것만 기억하세요:**

**핵심 원칙:**
1. 서술형은 기술 - 연습하면 늘음
2. 키워드가 가장 중요
3. 구조화된 답안 쓰기
4. 부분점수 전략 활용
5. 첨삭 받고 반복 연습

**성공의 비결:**
```
알기 (개념 학습)
↓
쓰기 (답안 작성)
↓
첨삭 (선생님 피드백)
↓
수정 (개선)
↓
반복 (자동화)
```

**서술형 과외가 필요한 이유:**
- 혼자는 첨삭 불가능
- 채점 기준 모름
- 효율적 연습 어려움
- 빠른 개선 필요

**효과:**
```
2-3개월 후:
서술형 30% → 70%
총점 15-20점 ↑
1-2등급 상승 가능
```
