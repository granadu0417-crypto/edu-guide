"기초가 없어서 아무것도 모르겠어요!"

**기초부터 다시 시작하면 됩니다.** 늦지 않았습니다.

**4-5등급에서 3등급으로 가는 완벽한 로드맵을 알려드립니다.**

## 기초 과외란?

### 정의

**밑바닥부터 다시:**
- 초등 or 중1 기초부터
- 빠진 구멍 메우기
- 개념 하나하나 확실히
- 천천히, 확실하게

**일반 과외와의 차이:**
```
일반 과외: 현행 진도 → 문제 풀이
기초 과외: 기초부터 → 천천히 → 현행 연결
```

### 누가 필요한가?

**필수 대상:**
- [ ] 내신 4-5등급
- [ ] 기본 개념 모름
- [ ] "모르는 게 뭔지도 모름"
- [ ] 수업 들어도 이해 안 됨
- [ ] 포기 상태

**2개 이상: 기초 과외 필수**

**체크 방법:**
```
수학: 중1 수학 시험 봤을 때?
     60점 이하 → 기초 필요

영어: be동사, 일반동사 차이?
     모름 → 기초 필요

국어: 글 읽고 내용 이해?
     어려움 → 기초 필요
```

## 기초가 부족한 이유

### 1. 과거에 놓침

**원인:**
```
초등: 놀기만 함
중1: 시작부터 놓침
중2: 누적됨
중3: 포기
```

**결과:**
- 구멍이 많음
- 연결 안 됨
- 악순환

### 2. 학습 방법 모름

**문제:**
```
수업: 듣기만
집: 복습 안 함
시험: 벼락치기
→ 안 늘음
```

**해결:**
- 학습 방법 배우기
- 복습 습관
- 꾸준히

### 3. 자신감 없음

**악순환:**
```
안 됨 → 자신감 ↓
      → 안 함
      → 더 안 됨
      → 포기
```

**해결:**
- 작은 성공 경험
- 칭찬과 격려
- 성취감

### 4. 학원만 다님

**문제:**
```
학원: 진도 빨리
     개인 맞춤 없음
     못 따라가면 방치
→ 효과 없음
```

**해결:**
- 1:1 과외
- 속도 조절
- 맞춤 지도

## 기초 과외의 목표

### 단기 목표 (3-6개월)

**1. 구멍 메우기**
```
Before: 중1부터 모름
After: 중1-2 개념 완벽
```

**2. 기본 개념 이해**
- 핵심 개념 습득
- 원리 이해
- 적용 가능

**3. 학습 습관 형성**
- 매일 공부
- 복습 루틴
- 숙제 완수

**4. 자신감 회복**
```
Before: "나는 못해"
After: "조금씩 되네?"
→ 희망 생김
```

### 중기 목표 (6개월-1년)

**1. 3등급 진입**
```
Before: 5등급 (50점대)
After: 3등급 (70점대)
```

**2. 현행 따라가기**
- 학교 진도 이해
- 수업 참여
- 과제 완수

**3. 자기주도 시작**
- 스스로 공부 시작
- 계획 세우기
- 복습 자발적

### 장기 목표 (1년 이상)

**1. 안정적 2-3등급**
- 꾸준한 성적
- 자신감 확립
- 학습 즐거움

**2. 완전한 자기주도**
- 과외 의존 ↓
- 스스로 학습
- 목표 관리

## 기초 과외 진행 방법

### 1단계: 철저한 진단 (1-2주)

**진단 내용:**

**어디서부터 모르는지:**
```
수학 예시:
중3 현재 → 중2 문제 풀기
        → 중1 문제 풀기
        → 초등 문제 풀기
→ 어디부터 막히는지 파악
```

**구멍 목록 작성:**
```
✗ 중1 일차방정식
✗ 중2 연립방정식
✗ 중2 일차함수
✓ 기본 연산 OK
```

**학습 스타일 파악:**
- 시각적? 청각적?
- 빠르게? 천천히?
- 예시 많이? 원리 깊이?

**학습 태도 파악:**
```
✓ 의욕: 있음 / 없음
✓ 집중: 30분 / 1시간
✓ 복습: 함 / 안 함
✓ 숙제: 완수 / 대충
```

**진단 결과 공유:**
- 학생, 부모와 공유
- 현재 위치 명확히
- 목표 설정
- 로드맵 제시

### 2단계: 기초부터 차근차근 (3-6개월)

**원칙:**

**1. 모르는 학년부터**
```
현재 중3인데
중1부터 모름
→ 중1부터 시작!

자존심? 상관없음
실력이 중요
```

**2. 천천히, 확실하게**
```
빨리 빨리 (X)
하나씩 완벽히 (O)

진도보다 이해
```

**3. 반복 학습**
```
1회: 설명 듣기
2회: 문제 풀기
3회: 다시 설명 들어보기
4회: 응용 문제
→ 완전히 내 것으로
```

**수업 구성 (60분):**

**10분: 지난 시간 복습**
- 뭐 배웠지?
- 설명해볼까?
- 문제 풀어보기

**30분: 새 개념**
- 하나의 개념만
- 천천히 설명
- 이해될 때까지
- 예시 많이

**15분: 문제 풀이**
- 쉬운 문제부터
- 틀려도 괜찮아
- 왜 틀렸는지 분석
- 다시 풀기

**5분: 오늘 배운 것 정리**
- 핵심 개념 리마인드
- 숙제 설명
- 다음 시간 예고

**숙제:**
```
양: 적게 (30분 이내)
난이도: 쉽게 (70-80% 풀 수 있는)
목표: 완수 (성취감)
```

### 3단계: 현행 연결 (3-6개월 후)

**목표:**
```
기초 (중1-2) 완벽
↓
현행 (중3) 연결
↓
학교 수업 이해
```

**진행:**

**점진적 난이도 상승:**
```
Week 1-2: 쉬운 문제
Week 3-4: 중간 문제
Week 5-6: 학교 수준 문제
```

**학교 진도 맞춤:**
- 학교 시험 범위 확인
- 해당 부분 집중
- 내신 대비

**실전 연습:**
- 학교 기출 문제
- 시간 재며 풀기
- 실전 감각

### 4단계: 습관 정립 (지속)

**목표: 자기주도 학습**

**매일 공부 습관:**
```
과외 있는 날: 과외 + 복습 30분
과외 없는 날: 복습 1시간
주말: 총복습 2시간
```

**복습 루틴:**
```
당일 밤: 오늘 배운 것 노트 정리
다음 날: 다시 읽어보기
주말: 이번 주 내용 정리
```

**스스로 학습:**
- 선생님 의존 ↓
- 문제 혼자 풀어보기
- 모르면 찾아보기
- 정 모르면 질문

## 과목별 기초 전략

### 수학 기초 다지기

**왜 수학이 제일 중요한가?**
```
수학 = 누적 과목
기초 없으면 → 절대 못 함
기초 있으면 → 올라갈 수 있음
```

**중1 필수 기초:**

**1. 정수와 유리수**
```
✓ 음수 개념
✓ 사칙연산
✓ 절댓값
→ 이게 안 되면 아무것도 안 됨
```

**2. 문자와 식**
```
✓ x, y가 뭔지
✓ 2x + 3 계산
✓ 식 세우기
```

**3. 일차방정식**
```
✓ x 구하기
✓ 풀이 과정
✓ 검산
→ 중2 연립방정식 기초
```

**4. 좌표평면**
```
✓ x축, y축
✓ 점 찍기
✓ 거리 구하기
→ 중2 함수 기초
```

**중2 필수 기초:**

**1. 식의 계산**
```
✓ (x+2)(x+3) 전개
✓ 인수분해
→ 고등 필수
```

**2. 연립방정식**
```
✓ 두 개 방정식 풀기
✓ 대입법, 가감법
→ 응용 문제 기초
```

**3. 일차함수**
```
✓ y = ax + b
✓ 그래프 그리기
✓ 기울기, y절편
→ 중3, 고등 연결
```

**학습 순서:**
```
1단계: 연산 완벽
2단계: 방정식 마스터
3단계: 함수 이해
4단계: 응용 문제
```

**효과적 학습법:**
```
✓ 계산 매일 10문제 (5분)
✓ 공식 암기보다 이해
✓ 풀이 과정 반드시 쓰기
✓ 틀린 문제 3번 반복
```

### 영어 기초 다지기

**왜 영어가 어려운가?**
```
단어 모름 + 문법 모름 + 자신감 없음
= 포기
```

**해결 3단계:**

**1단계: 기본 단어 (1000개)**
```
목표: 중학 필수 단어 1000개
방법: 매일 30개
기간: 1개월
```

**암기법:**
```
아침: 새 단어 30개 쓰기
점심: 다시 보기
저녁: 테스트
주말: 이번 주 총정리
```

**2단계: 핵심 문법**

**필수 문법:**

**1) 8품사**
```
명사, 동사, 형용사, 부사...
→ 문장 구조 이해 기초
```

**2) 동사의 종류**
```
be동사 vs 일반동사
→ 제일 중요!
혼동하면 아무것도 안 됨
```

**3) 시제**
```
현재/과거/미래
현재진행/과거진행
→ 문장 만들기 기초
```

**4) 문장 구조**
```
S + V
S + V + O
S + V + C
S + V + O + O
S + V + O + C
→ 5형식 이해
```

**학습 방법:**
```
✓ 문법 하나씩
✓ 예문 10개씩 암기
✓ 직접 문장 만들기
✓ 반복 훈련
```

**3단계: 독해 기초**

**단계별 독해:**
```
1단계: 짧은 문장 (5-10단어)
2단계: 여러 문장 (2-3문장)
3단계: 짧은 지문 (50단어)
4단계: 중간 지문 (100단어)
5단계: 긴 지문 (150단어+)
```

**독해 방법:**
```
✓ 모르는 단어 체크
✓ 문장 구조 파악
✓ 해석 써보기
✓ 답 체크
✓ 틀린 부분 다시
```

### 국어 기초 다지기

**국어도 기초가 필요한가?**
```
Yes!
독해력 = 모든 과목의 기초
```

**문제:**
```
글을 읽어도:
- 무슨 말인지 모름
- 핵심 파악 못 함
- 문제 이해 못 함
```

**해결:**

**1. 문장 구조 이해**
```
주어 + 서술어 찾기
→ 누가 + 어떻게 했다

예:
"철수가 학교에 갔다"
주어: 철수가
서술어: 갔다
```

**연습:**
- 매일 1-2문장 분석
- 주어, 서술어 찾기
- 수식어 찾기

**2. 핵심 파악**
```
긴 글:
1) 문단별 핵심 찾기
2) 전체 주제 파악
3) 한 문장으로 요약
```

**연습:**
- 짧은 글부터
- 한 문장 요약 연습
- 점점 긴 글로

**3. 천천히 읽기**
```
빨리 읽기 (X)
천천히 정확히 (O)

한 번에 이해하며 읽기
```

**4. 어휘력 향상**
```
목표: 중학 필수 어휘 500개
방법: 독서, 어휘 노트
기간: 3개월
```

### 과학/사회 기초

**공통 전략:**

**1. 용어 정리**
```
모르는 용어 = 이해 불가
→ 용어 노트 만들기
```

**2. 그림/도표 활용**
```
글만 읽기 어려움
→ 그림 보며 이해
→ 시각적 학습
```

**3. 이야기로 기억**
```
암기보다:
원리 이해 → 이야기로 연결
→ 기억 오래
```

**과학:**
- 실험 이해
- 원리 암기보다 이해
- 그림 그리며 학습

**사회:**
- 흐름 중심
- 암기는 최소한
- 인과관계 파악

## 기초 과외 선생님 선택

### 필요한 자질

**1. 인내심 (최우선)**
```
✓ 천천히 설명
✓ 여러 번 반복
✓ 화내지 않음
✓ 포기하지 않음
```

**2. 낮은 눈높이**
```
✓ 학생 수준에 맞춰
✓ 쉬운 말로 설명
✓ "이것도 몰라?" (X)
✓ "괜찮아, 천천히" (O)
```

**3. 격려와 칭찬**
```
✓ 작은 발전 인정
✓ 과정 칭찬
✓ 자신감 심어주기
✓ 희망 주기
```

**4. 기초 지도 경험**
```
✓ 하위권 학생 지도 경험
✓ 동기 부여 능력
✓ 장기적 관점
```

### 피해야 할 선생님

**❌ 조급한 선생님**
```
"이것도 몰라?"
"빨리 따라와"
→ 학생 위축
→ 역효과
```

**❌ 진도에 집착**
```
이해 안 돼도 진도
→ 구멍 더 생김
→ 효과 없음
```

**❌ 1-2등급만 가르친 선생님**
```
하위권 이해 못 함
눈높이 안 맞음
인내심 부족
```

### 체험 수업 체크리스트

**확인할 것:**
- [ ] 설명이 쉬운가?
- [ ] 천천히 설명하는가?
- [ ] 학생이 이해할 때까지 반복하는가?
- [ ] 격려하고 칭찬하는가?
- [ ] 학생이 편안해하는가?

**3개 이상 YES: 적합한 선생님**

## 부모의 역할

### 해야 할 것

**1. 인내심**
```
빠른 변화 기대 (X)
장기적 관점 (O)

최소 6개월~1년
```

**2. 격려**
```
✓ "많이 늘었어"
✓ "꾸준히 하네"
✓ "잘하고 있어"
```

**3. 환경 조성**
- 조용한 공부 공간
- 적절한 휴식
- 스트레스 관리

**4. 선생님 신뢰**
```
✓ 선생님께 맡기기
✓ 간섭 최소화
✓ 소통은 정기적으로
```

### 하지 말아야 할 것

**1. 다른 애와 비교**
```
❌ "친구는 잘하는데..."
❌ "형은 너 때 1등급이었어"

→ 자존감 하락
→ 역효과
```

**2. 조급함**
```
❌ "왜 안 올라?"
❌ "이번에는 70점 받아야 해"

→ 압박감
→ 역효과
```

**3. 포기 암시**
```
❌ "너는 공부 안 맞나봐"
❌ "그냥 예체능 해라"

→ 자포자기
→ 완전 포기
```

**4. 과도한 기대**
```
현실적 목표:
6개월: 4등급 → 3등급 후반
1년: 3등급 안정화

단기간 1등급 (X)
```

## 동기 부여와 자신감

### 작은 성취 축하

**방법:**
```
✓ 숙제 완수 → 칭찬
✓ 5점 올랐어 → 축하
✓ 개념 이해 → 인정
✓ 꾸준히 공부 → 격려
```

**효과:**
- 자신감 ↑
- 동기 부여
- 선순환

### 과정 칭찬

**결과보다 과정:**
```
❌ "1등급 아니면 의미 없어"
✅ "매일 공부하는 게 대단해"

❌ "아직 멀었어"
✅ "많이 늘었어"
```

### 포기하지 않기

**힘들 때:**
```
✓ 휴식 필요
✓ 목표 재조정
✓ 방법 바꾸기
✓ 선생님과 상담

하지만 포기는 (X)
```

## 성공 사례

### 사례 1: 수학 5등급 → 3등급 (6개월)

**학생 프로필:**
- 중2, 수학 5등급 (평균 55점)
- 중1부터 막힘
- 자신감 없음
- "수학 포기"

**과외 내용:**
```
1-2개월: 중1 수학 전체 복습
         정수, 방정식 완벽히
3-4개월: 중2 수학 기초
         식의 계산, 연립방정식
5-6개월: 현행 연결
         학교 진도 따라잡기
```

**학습 방식:**
- 하루 1시간 과외
- 매일 30분 복습
- 주말 2시간 총정리

**결과:**
```
Before: 55점 (5등급)
3개월: 65점 (4등급) → "조금 되네?"
6개월: 75점 (3등급) → 자신감 회복!
```

**학생 변화:**
```
Before: "나는 안 돼"
After: "조금씩 되네"
      → 다른 과목도 시작
```

### 사례 2: 영어 4등급 → 2등급 (8개월)

**학생 프로필:**
- 중3, 영어 4등급 (평균 68점)
- 단어 모름
- 문법 엉망
- 독해 포기

**과외 내용:**
```
1-2개월: 단어 1000개 암기
         매일 30개씩
3-4개월: 문법 기초
         be동사, 시제, 5형식
5-6개월: 독해 연습
         짧은 지문부터
7-8개월: 실전 대비
         학교 시험 준비
```

**결과:**
```
Before: 68점 (4등급)
2개월: 단어 실력 ↑
4개월: 75점 (3등급)
8개월: 85점 (2등급)!
```

**핵심:**
```
단어 → 문법 → 독해
순서가 중요!
```

### 사례 3: 전과목 상승 (1년)

**학생 프로필:**
- 중2, 전과목 4-5등급
- 학습 습관 없음
- 게임만 함
- 부모와 갈등

**과외 내용:**
```
1-3개월: 수학 집중
         기초부터 다시
4-6개월: 영어 추가
         단어+문법
7-9개월: 학습 습관 정립
         매일 공부 루틴
10-12개월: 자기주도 시작
           과외 의존 ↓
```

**결과:**
```
Before: 수학 55점, 영어 60점 (4-5등급)
6개월: 수학 70점, 영어 70점 (3등급)
1년: 수학 80점, 영어 78점 (2-3등급)
```

**가장 큰 변화:**
```
학습 습관!
Before: 게임만, 공부 0시간
After: 매일 2시간, 주말 4시간
      자발적으로 공부
```

## 진행 과정과 기대 효과

### 1개월 차

**변화:**
```
✓ 수업 이해되기 시작
✓ 숙제 완수
✓ "아, 이거구나" 순간들
```

**아직:**
- 성적은 그대로
- 시험은 어려움
- 자신감 아직

**이 시기:**
- 포기하지 말기
- 과정 칭찬
- 꾸준히

### 3개월 차

**변화:**
```
✓ 개념 이해 늘어남
✓ 쉬운 문제 풀림
✓ 5-10점 상승
✓ 자신감 조금 생김
```

**효과:**
```
Before: 아무것도 모름
After: 기본은 알겠음
```

**평가 시점:**
- 계속 vs 조정
- 선생님과 상담
- 목표 재설정

### 6개월 차

**변화:**
```
✓ 10-20점 상승
✓ 등급 1단계 ↑
✓ 학습 습관 생김
✓ 자신감 확실히 ↑
```

**목표:**
```
4-5등급 → 3등급 진입
```

**이 시기:**
- 성취 축하
- 목표 상향
- 계속 진행

### 1년 차

**변화:**
```
✓ 20-30점 상승
✓ 안정적 3등급 or 2등급 진입
✓ 자기주도 학습 시작
✓ 과외 의존 ↓
```

**목표 달성:**
```
기초 탄탄 → 자기주도
과외 필요성 감소
```

**다음 단계:**
- 과외 빈도 ↓
- 자기주도 ↑
- 독립적 학습

## 자주 묻는 질문

### Q1. 중3인데 중1부터 다시 해야 하나요?

A. **네, 필요하면 중1부터 해야 합니다.**

**이유:**
```
수학 = 누적 과목
중1 모르면 → 중2 절대 못 함
중2 모르면 → 중3 당연히 못 함

영어도 마찬가지
기초 없으면 → 쌓을 수 없음
```

**걱정:**
```
"시간이 없는데..."
"지금 진도는?"
```

**답:**
```
지금 진도 따라가봤자:
→ 이해 못 함
→ 시간 낭비

차라리:
→ 기초부터 빠르게
→ 확실히 쌓기
→ 그게 더 빠름
```

**실제:**
```
중1-2 복습: 2-3개월
중3 연결: 1-2개월
총 4-5개월

vs

현행만 억지로: 1년 내내 효과 없음
```

### Q2. 기초 과외로 얼마나 오를 수 있나요?

A. **6개월에 1등급, 1년에 2등급 상승 가능합니다.**

**현실적 기대:**
```
3개월: 5-10점 ↑
6개월: 10-20점 ↑ (1등급 ↑)
1년: 20-30점 ↑ (2등급 ↑)
```

**예시:**
```
Before: 5등급 (50점)
6개월: 3-4등급 (70점)
1년: 2-3등급 (80점)
```

**조건:**
```
✓ 꾸준히 (주 2-3회)
✓ 복습 철저
✓ 숙제 완수
✓ 포기하지 않기
```

**개인차:**
- 빠른 학생: 6개월에 2등급 ↑
- 보통: 1년에 2등급 ↑
- 느린 학생: 1.5년에 2등급 ↑

### Q3. 얼마나 오래 해야 하나요?

A. **최소 6개월, 보통 1년 이상 필요합니다.**

**단계별:**
```
1-3개월: 적응, 기초 다지기
3-6개월: 실력 향상 시작
6-12개월: 등급 상승, 안정화
1년 이상: 자기주도, 독립
```

**중단 시점:**
```
✓ 목표 등급 달성
✓ 자기주도 가능
✓ 학습 습관 확립
✓ 과외 없이도 유지 가능
```

**주의:**
```
❌ 너무 빨리 중단
   → 다시 떨어짐

✅ 안정화 될 때까지
   → 그 후 점진적 독립
```

### Q4. 학원과 과외, 뭐가 나을까요?

A. **기초가 부족하면 1:1 과외가 훨씬 낫습니다.**

**학원의 문제:**
```
❌ 진도 빠름
   → 못 따라감

❌ 여러 학생
   → 개인 맞춤 없음

❌ 눈치
   → 질문 못 함

❌ 방치
   → 효과 없음
```

**1:1 과외의 장점:**
```
✓ 속도 조절
  → 이해할 때까지

✓ 완전 맞춤
  → 내 수준에 딱

✓ 질문 자유
  → 모르면 바로

✓ 집중 관리
  → 방치 없음
```

**결론:**
```
4-5등급: 1:1 과외 강력 추천
3등급 안정: 학원 or 과외 선택
1-2등급: 학원도 OK
```

### Q5. 부모가 할 수 있는 게 뭔가요?

A. **인내심 + 격려 + 환경 조성이 전부입니다.**

**가장 중요: 인내심**
```
빠른 변화 기대 (X)
6개월~1년 기다리기 (O)

"왜 안 올라?" → 압박
"천천히 가자" → 격려
```

**둘째: 격려**
```
✓ 작은 발전 인정
✓ 과정 칭찬
✓ 비교하지 않기
✓ 희망 주기
```

**셋째: 환경 조성**
```
✓ 조용한 공간
✓ 적절한 휴식
✓ 균형 잡힌 생활
✓ 스트레스 관리
```

**하지 말아야 할 것:**
```
❌ 다른 애와 비교
❌ 조급하게 재촉
❌ "너는 안 돼" 같은 말
❌ 과도한 압박
```

**기억:**
```
부모의 역할 = 지원
공부는 = 선생님 + 아이

믿고 기다리기
```

성공
