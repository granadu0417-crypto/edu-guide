# 고등 국어 과외 전략 - 독서·문학·문법 완벽 마스터

수능 국어는 독서·문학·화법과 작문(또는 언어와 매체)의 세 영역으로 구성되며, 상위권 대학 진학의 가장 중요한 과목 중 하나입니다. 체계적인 학습과 실전 감각으로 안정적인 고득점을 확보해야 합니다.

## 수능 국어의 구조와 특징

### 시험 구조 (2022학년도 이후)

**공통 과목 (75점, 34문항)**
- 독서: 15~17문항 (인문·사회·과학·기술·예술 등)
- 문학: 15~17문항 (현대시·고전시가·현대소설·고전소설 등)

**선택 과목 (25점, 11문항) - 택 1**
- 화법과 작문: 화법 5~6문항, 작문 5~6문항
- 언어와 매체: 언어(문법) 5~6문항, 매체 5~6문항

**총 45문항, 시험 시간 80분**

### 국어의 중요성

**정시에서의 비중**
- 대부분의 대학: 국어 반영 비율 30~35%
- 서울대·연세대·고려대: 국어 비중 매우 큼
- 의치한약대: 국어 1등급 거의 필수
- 자연계열도 국어 중요 (교차지원 시)

**국어가 어려운 이유**
- 시간 부족 (80분에 45문항, 지문량 많음)
- 독서 지문 난이도 높음 (철학·과학 등)
- 문학 배경지식 필요
- 선택과목 유불리
- 안정적 고득점 어려움

## 영역별 학습 전략

### 1. 독서 (Reading)

**특징**
- 15~17문항, 배점 높음 (3점 문항 많음)
- 인문·사회·과학·기술·예술 등 다양한 제재
- 시간 가장 많이 소요
- 고득점과 저득점 갈림길

**독서 지문 유형**

*인문*
- 철학, 역사, 미학, 종교 등
- 추상적 개념 이해 필요
- 논리적 사고력 요구
- 동서양 사상 비교 등

*사회*
- 경제, 법, 정치, 사회학 등
- 제도·이론 이해
- 그래프·도표 해석
- 실생활 연결

*과학·기술*
- 물리, 화학, 생물, 공학 등
- 과학적 원리 설명
- 실험·연구 과정
- 도식·그림 활용
- 가장 어려운 유형

*예술*
- 문학 이론, 음악, 미술 등
- 예술 작품 분석
- 창작 원리 설명

**독서 문제 유형**

*세부 정보 파악*
- 지문 내용 사실 확인
- 꼼꼼한 독해 필요
- 선택지와 지문 대조

*추론*
- 지문 내용 근거로 추론
- 논리적 사고 필요
- 과도한 추론 주의

*적용*
- 지문 내용을 사례에 적용
- <보기> 문제
- 이해도 확인

*비판*
- 관점 비교
- 한계 파악
- 비판적 사고

**독서 학습 전략**

*1단계: 정독 훈련*
- 시간 제한 없이 완벽히 이해
- 문단별 중심 내용 파악
- 문단 간 관계 이해
- 모르는 개념 찾아보기
- 구조도 그리기

*2단계: 속독 훈련*
- 시간 재며 읽기
- 핵심만 빠르게 파악
- 불필요한 부분 빠르게 넘김
- 문제 먼저 보고 지문 읽기
- 1지문 10~12분 목표

*3단계: 배경지식 쌓기*
- 다양한 분야 글 읽기
- 과학·경제·철학 개념 학습
- 교양서적 읽기
- 다큐멘터리 시청
- 호기심 가지고 탐구

*4단계: 실전 연습*
- 기출 문제 반복
- 실전 시간 내 풀이
- 오답 분석 철저히
- 약점 유형 집중 공략

**고난도 독서 지문 대처법**

*과학 지문 공략*
- 원리와 과정 중심으로 이해
- 도식·그림 먼저 확인
- 인과 관계 파악
- 비교·대조 정리
- 생소한 용어에 당황하지 않기

*철학 지문 공략*
- 핵심 개념 정확히 이해
- 학자별 입장 비교
- 논리 전개 과정 파악
- 추상적 내용 구체화
- 예시와 연결

### 2. 문학 (Literature)

**특징**
- 15~17문항
- 현대시·고전시가·현대소설·고전소설·극·수필 등
- 작품 감상 능력과 배경지식 필요
- 비교적 안정적 점수

**문학 갈래별 특징**

*현대시*
- 주제·화자·표현법 파악
- 심상·분위기 이해
- 시어의 함축적 의미
- 구조와 전개 방식
- 문학적 표현 기법

*고전시가*
- 시조·가사·한시 등
- 고어·한자어 이해 필수
- 시대적 배경 이해
- 작가와 작품 연결
- 주제 의식 파악

*현대소설*
- 인물·사건·배경 파악
- 서술자와 시점
- 갈등 구조 이해
- 주제 의식
- 서사 전개 방식

*고전소설*
- 고전 소설 문법 이해
- 인물 유형 파악 (영웅·기생 등)
- 시대적 가치관 반영
- 교훈적 주제
- 고어 해석

*극·수필*
- 갈래 특성 이해
- 극: 대화·지시문·무대 설정
- 수필: 필자의 생각과 경험

**문학 문제 유형**

*작품 이해*
- 주제, 화자, 인물 등
- 표현법, 서술 방식
- 작품 전개 과정

*감상·비평*
- 작품의 특징 파악
- 문학적 효과
- 작가의 의도

*작품 비교*
- 공통점·차이점
- 주제·표현법 비교
- <보기>와 비교

**문학 학습 전략**

*필수 개념 학습*
- 문학 용어 완벽 숙지
- 표현법 (은유, 상징, 역설, 반어 등)
- 시점 (1인칭, 3인칭 등)
- 갈래별 특징
- 서사 구조

*작품 읽기*
- 교과서 수록 작품 정독
- EBS 연계 작품 학습
- 기출 작품 정리
- 작품 해설 읽기
- 작품 배경 이해

*문학사 정리*
- 시대별 문학 흐름
- 주요 작가와 작품
- 문학 사조 (낭만주의, 사실주의 등)
- 대표 작품 암기

*실전 적용*
- 기출 문제 풀이
- 작품 해석 연습
- <보기> 활용 훈련
- 시간 내 풀이 연습

**문학 고득점 팁**

*시 해석 요령*
- 시어의 함축적 의미 파악
- 화자의 정서·태도 이해
- 시의 흐름과 구조 파악
- 표현법 찾기
- <보기>와 연결

*소설 해석 요령*
- 인물 관계와 갈등 파악
- 사건 전개 과정 이해
- 서술자 시점 확인
- 주제 의식 파악
- 시대적 배경 고려 (고전소설)

### 3. 화법과 작문 (선택과목)

**특징**
- 11문항 (화법 5~6문항, 작문 5~6문항)
- 비교적 쉬운 편
- 안정적 점수 확보 가능
- 문과 학생 선택 많음

**화법 영역**

*주요 개념*
- 의사소통의 원리
- 화법의 유형 (대화, 토론, 협상, 발표, 면접 등)
- 듣기와 말하기 전략
- 담화의 구성과 전개
- 언어 예절

*출제 유형*
- 발표·토론·면접 상황 제시
- 의사소통 전략 파악
- 담화의 특징 이해
- 적절한 말하기 방법
- <보기> 적용

**작문 영역**

*주요 개념*
- 작문의 과정 (계획-작성-고쳐쓰기)
- 글의 유형 (설명문, 논설문, 건의문 등)
- 내용 구성 전략
- 표현 전략
- 고쳐쓰기

*출제 유형*
- 작문 상황과 조건 제시
- 글쓰기 전략 파악
- 내용 생성·조직 방법
- 고쳐쓰기 (맞춤법, 문장, 내용 등)
- <보기> 적용

**학습 전략**
- 개념 정확히 숙지
- 기출 문제 반복
- 유형별 전략 익히기
- 실수 줄이기
- 시간 절약 (문항당 2분)

### 4. 언어와 매체 (선택과목)

**특징**
- 11문항 (언어 5~6문항, 매체 5~6문항)
- 언어(문법) 영역 어려움
- 매체 영역 비교적 쉬움
- 이과 학생 선택 많음 (표준점수 유리)

**언어(문법) 영역**

*주요 개념*
- 음운론 (음운 체계, 음운 변동)
- 형태론 (형태소, 단어 형성)
- 통사론 (문장 성분, 문장 구조)
- 의미론 (의미 관계, 중의성)
- 국어사 (훈민정음, 옛말)

*출제 유형*
- 개념 이해 문제
- 적용 문제 (음운 변동, 문장 분석 등)
- 옛말 해석
- <보기>와 연결

**매체 영역**

*주요 개념*
- 매체의 특성 (TV, 인터넷, SNS 등)
- 매체 언어의 특징
- 정보 구성 방식
- 매체 자료 해석

*출제 유형*
- 매체 자료 제시 (뉴스, 광고, SNS 등)
- 매체 언어 특성 파악
- 정보 전달 방식
- 비판적 수용

**학습 전략**
- 문법 개념 완벽 이해
- 기출 문제 반복 (특히 음운·형태·통사)
- 옛말 자주 나오는 것 암기
- 매체 유형 익히기
- 시간 관리 (언어는 시간 많이 소요)

## 선택과목 선택 전략

### 화법과 작문 vs 언어와 매체

**화법과 작문 추천 대상**
- 문법이 어려운 학생
- 안정적인 점수 원하는 학생
- 문과 학생
- 언어감각보다 상식적 사고 강한 학생

**장점**
- 비교적 쉬움
- 안정적 점수
- 실수만 안 하면 고득점

**단점**
- 표준점수 불리 (쉬워서 1~2문제 실수 시 등급 하락)
- 변별력 낮음

**언어와 매체 추천 대상**
- 문법에 자신 있는 학생
- 표준점수 유리한 과목 원하는 학생
- 이과 학생
- 논리적·체계적 사고 강한 학생

**장점**
- 표준점수 유리 (어려워서 원점수 낮아도 유리)
- 고난도 문제 맞히면 큰 이득

**단점**
- 문법 어려움 (특히 음운, 형태, 통사)
- 개념 학습 필요
- 변별 문제 있으면 어려움

**선택 시기와 변경**
- 고2 때 두 과목 모두 경험 후 선택
- 고3 초 확정 (늦어도 6월 모평 전)
- 고3 여름방학 이후 변경은 위험

## 학습 로드맵

### 고등학교 1학년

**국어 기초 다지기**

*1학기*
- 내신 위주 학습
- 교과서 작품 정독
- 문학 용어 학습
- 기본 문법 개념
- 독서 습관 들이기

*여름방학*
- 교과서 외 작품 읽기
- 문학 작품 30편 이상 읽기
- 다양한 분야 독서 (과학, 인문 등)
- 문법 기초 개념 정리

*2학기*
- 내신 대비
- 문학 작품 계속 읽기
- 수능 유형 맛보기
- 기출 문제 경험

*겨울방학*
- 문학 필수 작품 정리
- 문법 개념 심화
- 독서 다양한 제재 경험
- 수능 기출 5개년 풀어보기

### 고등학교 2학년

**본격적인 수능 대비**

*1학기*
- 선택과목 결정 준비 (화작 vs 언매)
- 독서 유형별 연습
- 문학 작품 확장 (50편 이상)
- 문법 개념 완성
- 모의고사 응시 시작

*여름방학 (중요!)*
- 선택과목 확정
- 기출 문제 10개년 풀이
- 독서 속독 훈련
- 문학 EBS 연계 작품 학습
- 약점 영역 집중 보완

*2학기*
- 실전 모의고사 주 1회
- 시간 관리 연습
- 오답 노트 작성
- 내신 대비

*겨울방학*
- 기출 문제 재풀이
- 고난도 독서 집중
- 문학 작품 정리
- 선택과목 완성

### 고등학교 3학년

**실전 완성**

*3월~5월*
- 모의고사 매회 응시
- 기출 문제 3회독
- 신유형 대비
- 시간 단축 훈련
- 실수 줄이기

*6월 모의평가 이후*
- 약점 유형 재점검
- EBS 연계 교재 정리
- 고난도 문제 대비
- 실전 감각 유지

*여름방학*
- EBS 교재 완성
- 기출 변형 문제
- 실전 모의고사 집중
- 시간 관리 완성

## 자주 묻는 질문 (FAQ)

### Q1. 국어는 타고난 실력이 중요한가요?

아닙니다! 국어도 공부하면 실력이 향상됩니다. 독서는 배경지식과 독해 훈련으로, 문학은 작품 학습과 개념 이해로, 문법은 체계적 학습으로 충분히 극복 가능합니다. 꾸준히 노력하면 반드시 성적이 오릅니다.

### Q2. 독서 지문을 빨리 읽으려면 어떻게 해야 하나요?

독해 속도는 많이 읽을수록 빨라집니다. 매일 다양한 분야의 글을 읽고, 시간을 재며 연습하세요. 또한 불필요한 부분을 빠르게 넘기고, 핵심만 파악하는 연습을 하세요. 배경지식이 쌓이면 자연스럽게 빨라집니다.

### Q3. 과학 지문이 너무 어려운데 어떻게 하나요?

과학 지문은 원리와 과정을 중심으로 이해하세요. 도식·그림을 먼저 확인하고, 인과 관계를 파악하세요. 생소한 용어에 당황하지 말고, 지문 안에서 설명을 찾으세요. 과학 관련 배경지식을 쌓으면 도움이 됩니다.

### Q4. 문학 작품은 몇 편이나 읽어야 하나요?

교과서 수록 작품과 EBS 연계 작품은 필수이며, 기출에 나온 작품도 모두 읽어야 합니다. 대략 50~70편 이상 읽는 것을 권장하며, 작품을 많이 읽을수록 감상 능력이 향상됩니다.

### Q5. 화법과 작문 vs 언어와 매체 중 어떤 것을 선택해야 하나요?

문법에 자신 있고 표준점수를 고려한다면 언어와 매체, 안정적인 점수를 원하고 문법이 어렵다면 화법과 작문을 선택하세요. 고2 때 두 과목을 모두 경험해보고 결정하는 것이 좋습니다.

### Q6. 문법은 어떻게 공부해야 하나요?

개념을 정확히 이해하는 것이 우선입니다. 음운·형태·통사·의미 영역을 체계적으로 학습하고, 기출 문제를 반복하며 적용 연습을 하세요. 특히 음운 변동, 단어 형성, 문장 분석은 자주 출제되므로 집중하세요.

### Q7. 시간이 항상 부족한데 어떻게 하나요?

영역별 시간 배분을 정하고 지키는 연습을 하세요. 독서에서 시간을 절약하는 것이 관건이며, 선택과목은 빠르게 처리하세요. 어려운 문제는 표시 후 넘기고, 나중에 풀어도 됩니다. 실전 모의고사로 시간 관리 감각을 익히세요.

### Q8. EBS 연계 교재는 꼭 봐야 하나요?

수능에서 EBS 연계율이 50% 정도이므로, 수능특강과 수능완성은 필수입니다. 특히 문학 작품은 반드시 읽어야 하며, 독서도 주제와 소재가 비슷하게 출제될 수 있으므로 학습하세요. 단, 기출이 우선입니다.

### Q9. 국어 1등급을 받으려면 어느 정도 점수가 필요한가요?

해마다 다르지만 대략 90점 이상이 필요합니다. 표준점수로는 130점 전후입니다. 1~2문제 실수로 2등급이 될 수 있으므로, 안정적으로 95점 이상을 목표로 하세요.

### Q10. 국어를 포기하고 싶은데 어떻게 하나요?

포기하면 안 됩니다! 국어는 정시에서 가장 중요한 과목 중 하나입니다. 지금 당장은 어렵더라도, 기초부터 차근차근 쌓으면 반드시 향상됩니다. 작은 목표를 세우고 매일 조금씩 실천하세요. 과외나 학원의 도움을 받는 것도 좋습니다.

특히 시간 관리가 매우 중요하며, 80분 안에 45문항을 정확하게 풀어내는 연습이 필요합니다. 꾸준한 기출 분석과 실전 모의고사로 실력을 쌓아가세요.

**핵심 요약**
- 독서: 배경지식, 독해 훈련, 시간 관리
- 문학: 작품 학습, 개념 이해, 감상 능력
- 선택과목: 개념 숙지, 실수 방지, 시간 절약
- 실전: 기출 10개년, 시간 배분, 오답 분석

체계적인 계획과 꾸준한 실천으로 수능 국어를 완벽하게 정복하시기 바랍니다!
