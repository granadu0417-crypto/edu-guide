# 고등학생 수능 전략 완벽 가이드

수능이 막막한가요? 개념부터 당일 전략까지, 체계적인 수능 대비로 목표 점수를 달성하세요.

## 수능의 이해

### 수능이란?

대학수학능력시험(수능)은 대학 교육을 받는 데 필요한 수학 능력을 측정하는 국가 단위 시험입니다.

**시행 시기:**
- 매년 11월 셋째 목요일
- 2025학년도 수능: 2024년 11월 14일

**시험 시간:**
- 1교시: 국어 (08:40~10:00, 80분)
- 2교시: 수학 (10:30~12:10, 100분)
- 3교시: 영어 (13:10~14:20, 70분)
- 4교시: 한국사+탐구 (14:50~17:40)
- 5교시: 제2외국어/한문 (17:50~18:40, 40분, 선택)

### 수능 영역 구성

**국어 영역 (45문항)**
- 독서 (인문, 사회, 과학, 기술 등)
- 문학 (현대시, 고전시가, 현대소설, 고전소설)
- 화법/작문/언어

**수학 영역 (30문항)**
- 공통: 수학I, 수학II (22문항)
- 선택: 확률과 통계, 미적분, 기하 중 1개 (8문항)

**영어 영역 (45문항)**
- 듣기 (17문항)
- 독해 (28문항)
- 절대평가 (90점 이상 1등급)

**탐구 영역 (과목당 20문항)**
- 사회탐구: 9과목 중 최대 2과목
- 과학탐구: 8과목 중 최대 2과목

**한국사 영역 (20문항)**
- 필수 영역
- 절대평가

**제2외국어/한문 영역 (30문항)**
- 선택 영역
- 절대평가

### 수능 점수 체계

**원점수:**
- 맞힌 개수에 따른 점수

**표준점수:**
- 원점수를 표준화한 점수
- 난이도에 따라 변동
- 정시 주요 활용 지표

**백분위:**
- 상위 몇 % 인지 나타내는 점수
- 0~100 사이

**등급:**
- 백분위를 9등급으로 구분
- 1등급: 상위 4%
- 2등급: 상위 11%
- 3등급: 상위 23%

## 시기별 수능 대비 전략

### 고등학교 1학년

**목표:**
- 기초 개념 완성
- 수능 유형 익히기
- 학습 습관 형성

**학습 전략:**
- 교과서 개념 완벽 이해
- 쉬운 기출 문제 풀이
- 내신과 수능 병행
- EBS 교재 가볍게 접근

**주요 과목:**
- 국어, 수학, 영어 중심
- 탐구는 2학년부터 본격

**시간 배분:**
- 내신 70%, 수능 30%
- 주 5~7시간 수능 대비

### 고등학교 2학년

**목표:**
- 수능 유형 완전 숙달
- 기출 10개년 1회독
- 약점 영역 파악

**학습 전략:**
- 기출 문제 본격 시작
- 유형별 문제 정리
- 탐구 과목 확정
- 모의고사 적응

**주요 과목:**
- 국수영 + 탐구 2과목
- 제2외국어/한문 선택 고민

**시간 배분:**
- 내신 50%, 수능 50%
- 주 10~12시간 수능 대비

### 고등학교 3학년

**목표:**
- 수능 완성
- 약점 완전 제거
- 실전 감각 완성

**학습 전략 (시기별):**

**3~5월: 개념 재정비**
- 기출 10개년 1회독 완성
- 개념 빈틈 메우기
- 약점 영역 파악

**6~8월: 유형 완성**
- 기출 2회독 (유형별)
- 약점 집중 보완
- 실전 모의고사 시작

**9~10월: 실전 대비**
- 기출 3회독 (약점 중심)
- 주 2회 실전 모의고사
- 시간 배분 완성

**11월 (수능 전): 마무리**
- 컨디션 관리
- 가벼운 복습만
- 멘탈 관리

**시간 배분:**
- 수능 90%, 내신 10% (1학기)
- 수능 100% (2학기)

## 영역별 수능 전략

### 국어 영역 전략

**기본 원칙:**
- 지문을 정확히 읽는 것이 최우선
- 선지를 꼼꼼히 분석
- 시간 배분이 매우 중요

**독서 영역:**

**지문 읽기 전략:**
- 지문 종류 먼저 파악 (인문/사회/과학/기술)
- 중심 문장 표시하며 읽기
- 문단별 핵심 내용 파악
- 전체 구조 이해

**문제 풀이 전략:**
- 세부 정보 문제: 지문에서 해당 부분 찾기
- 추론 문제: 지문 내용 바탕으로 논리적 추론
- 비판/평가 문제: 다른 관점 적용

**시간 배분:**
- 지문당 8~10분
- 어려운 지문은 과감히 넘기기
- 독서에 총 35~40분

**문학 영역:**

**현대시:**
- 시어의 의미와 상징 파악
- 화자의 정서와 태도
- 표현법 이해

**고전시가:**
- 현대어 풀이 정확히
- 시대 배경 이해
- 주제와 정서

**현대소설:**
- 인물 관계 파악
- 사건 전개 이해
- 서술자 시점

**고전소설:**
- 이야기 흐름 파악
- 인물 성격과 관계
- 시대적 배경

**시간 배분:**
- 문학에 총 25~30분
- 아는 작품이면 빠르게

**화법/작문/언어:**
- 문법 개념 완벽 암기
- 화법/작문 상황 파악
- 빠르게 풀고 다른 영역에 시간 확보
- 10~15분 목표

### 수학 영역 전략

**기본 원칙:**
- 개념 이해가 가장 중요
- 계산 실수 최소화
- 시간 배분 전략 필수

**난이도별 전략:**

**1~15번 (쉬운 문제):**
- 빠르고 정확하게
- 문제당 2~3분 목표
- 총 30~40분

**16~21번 (중간 문제):**
- 유형별 풀이 패턴 활용
- 문제당 3~5분
- 총 20~30분

**22~30번 (어려운 문제):**
- 준킬러/킬러 문제
- 시간 배분 유연하게
- 전략적 포기도 고려
- 총 30~40분

**풀이 순서 전략:**

**순차 풀이:**
- 1번부터 30번까지 순서대로
- 막히면 과감히 넘기기
- 안정적이고 무난

**난이도별 풀이:**
- 쉬운 문제부터 확실히
- 1~15번 → 16~21번 → 22~30번
- 시간 효율적

**개념별 풀이:**
- 자신 있는 단원부터
- 개인 맞춤 전략
- 연습 많이 필요

**주의사항:**
- 계산 실수 주의 (검산)
- 문제 조건 꼼꼼히 읽기
- 시간 부족 시 찍기 전략

### 영어 영역 전략

**기본 원칙:**
- 절대평가이므로 90점 이상 목표
- 어휘력이 기본
- 속독 능력 필수

**듣기 영역 (1~17번):**
- 집중력 최고조 유지
- 선택지 미리 읽기
- 메모 활용
- 17분 고정

**독해 영역 (18~45번):**

**빈칸 추론 (18~21번):**
- 가장 어려운 유형
- 지문 전체 흐름 파악
- 논리적 흐름 이해
- 유형당 3~4분

**순서 배열 (22~24번):**
- 연결사/지시어 주목
- 논리적 흐름
- 유형당 2~3분

**문장 삽입 (25~26번):**
- 삽입 문장의 역할 파악
- 앞뒤 문맥 확인
- 유형당 2~3분

**요약문 완성 (27~28번):**
- 지문의 핵심 내용 파악
- 선택지 의미 정확히
- 유형당 2~3분

**장문 독해 (29~45번):**
- 지문 빠르게 읽기
- 핵심 내용 파악
- 문제당 1~2분

**어휘 학습:**
- 기출 빈출 어휘 암기
- 하루 30~50개
- 동의어/반의어
- 맥락적 의미

**시간 배분:**
- 듣기: 17분 (고정)
- 18~28번: 25분
- 29~45번: 20분
- 검토: 8분

### 탐구 영역 전략

**과목 선택:**
- 흥미와 적성 고려
- 표준점수와 백분위 유리한 과목
- 진로 연계 (선택)

**학습 전략:**
- 개념 완벽 암기
- 기출 7~10개년 필수
- N회독 (최소 3회)
- 자료 해석 훈련

**시험 당일 전략:**
- 과목 순서 미리 정하기
- 빠른 과목 먼저 or 어려운 과목 먼저
- 과목당 30분 엄수
- 쉬는 시간 5분 활용

**사회탐구:**
- 개념 + 시사
- 통계/그래프 자료 해석
- 사례 적용 문제

**과학탐구:**
- 개념 + 계산
- 실험/그래프 분석
- 수식 적용 문제

## 수능 전략 과목 선택

### 수학 선택과목

**확률과 통계:**
- 상대적으로 쉬움
- 인문/상경계열 추천
- 1등급 커트라인 높음

**미적분:**
- 난이도 높음
- 이공계열 필수
- 표준점수 높음

**기하:**
- 중간 난이도
- 이공계/건축 관련
- 표준점수 유리

**선택 전략:**
- 진로와 직결
- 대학별 요구 사항 확인
- 자신의 강점 고려

### 탐구 과목 선택

**선택 기준:**

**1. 흥미와 적성**
- 가장 중요
- 지속적 학습 가능

**2. 유불리**
- 표준점수 차이
- 응시 인원
- 최근 추세

**3. 진로 연계**
- 필수는 아니지만 고려
- 대학별 권장 과목

**사회탐구 조합 예시:**
- 생활과 윤리 + 사회문화
- 한국지리 + 세계지리
- 경제 + 정치와 법

**과학탐구 조합 예시:**
- 물리I + 화학I
- 생명과학I + 지구과학I
- 화학I + 지구과학I

## 수능 D-100일 전략

**목표:**
- 기출 완벽 마스터
- 약점 제거
- 실전 감각 완성

**학습 계획:**
- 기출 3회독 시작
- 주 2회 실전 모의고사
- 약점 영역 집중 학습
- 새로운 개념 X

**과목별 전략:**
- 국어: 기출 지문 복습, 선지 분석
- 수학: 틀린 문제 재풀이, 계산 연습
- 영어: 어휘 마무리, 독해 속도 향상
- 탐구: 개념 최종 정리, 자료 해석 훈련

**주의사항:**
- 번아웃 주의
- 새로운 교재 X
- 컨디션 관리 시작

## 수능 D-30일 전략

**목표:**
- 실전 감각 유지
- 실수 최소화
- 컨디션 조절

**학습 계획:**
- 매일 실전 모의고사 or 복습
- 오답 노트 최종 점검
- 약점만 가볍게 복습
- 새로운 문제 지양

**시간 배분:**
- 모의고사: 주 3~4회
- 복습: 매일 2~3시간
- 휴식: 충분히

**멘탈 관리:**
- 불안감 정상
- 작은 성취에 집중
- 긍정적 마인드
- 충분한 수면

## 수능 일주일 전 전략

**목표:**
- 컨디션 최상
- 자신감 유지
- 실수 방지 점검

**D-7~D-4:**
- 가벼운 복습만
- 오답 노트 훑어보기
- 주요 개념 확인
- 실전 모의고사 1회

**D-3~D-1:**
- 과도한 학습 X
- 가벼운 산책, 스트레칭
- 수능 당일 일정 체크
- 준비물 챙기기

**준비물 체크리스트:**
- [ ] 수험표
- [ ] 신분증
- [ ] 검은색 컴퓨터용 사인펜
- [ ] 흰색 수정테이프
- [ ] 시계 (아날로그 권장)
- [ ] 샤프, 연필, 지우개
- [ ] 간단한 간식 (초콜릿, 사탕)
- [ ] 물
- [ ] 마스크 (선택)

**수면 관리:**
- D-3부터 일찍 자기
- 최소 7시간 수면
- 수면 패턴 수능 시간에 맞추기

## 수능 당일 전략

### 전날 (D-1)

**학습:**
- 절대 과도한 학습 X
- 가벼운 개념 확인만
- 오후 8시 이후 공부 중단

**준비:**
- 준비물 최종 점검
- 수험표 인쇄 (2~3장)
- 등교 경로 확인
- 날씨 확인

**수면:**
- 오후 10시 취침 목표
- 잠 안 오면 억지로 자지 말기
- 릴랙스, 명상

### 당일 아침

**기상:**
- 시험 3시간 전 기상 (오전 5~6시)
- 가벼운 스트레칭
- 샤워로 정신 맑게

**식사:**
- 평소와 같은 식사
- 과식 X, 소화 잘 되는 음식
- 카페인 적당히 (과하면 X)

**이동:**
- 여유 있게 출발
- 7시 30분까지 도착 권장
- 교통 체증 고려

### 시험장에서

**대기 시간:**
- 가벼운 개념 확인
- 과도한 학습 X
- 긴장 풀기
- 화장실 미리

**시험 중:**
- 긴장하되 흥분하지 말기
- 문제 끝까지 읽기
- 마킹 실수 주의

## FAQ: 자주 묻는 질문

**Q1. 수능은 언제부터 준비해야 하나요?**

A: 고1 때부터 기초를 다지고, 고2 때 본격적으로 시작하는 것이 좋습니다. 고3이 되면 늦다고 생각할 수 있지만, 집중해서 준비하면 충분히 좋은 결과를 얻을 수 있습니다.

**Q2. 내신과 수능 중 무엇이 더 중요한가요?**

A: 목표 전형에 따라 다릅니다. 수시 학생부 전형은 내신이 더 중요하고, 정시는 수능이 중요합니다. 하지만 대부분의 학생은 둘 다 필요하므로 균형 있게 준비하세요.

**Q3. 수능 전략 과목은 어떻게 선택하나요?**

A: 진로, 흥미, 유불리를 종합적으로 고려하세요. 대학별 요구 사항도 확인이 필요합니다. 수학은 진로와 직결되므로 신중히 선택하고, 탐구는 자신 있는 과목을 선택하세요.

**Q4. 기출 문제는 몇 년치를 풀어야 하나요?**

A: 최소 10개년을 권장합니다. 국어는 15개년 이상도 좋고, 수학 킬러는 20개년까지 풀어도 됩니다. 3회 이상 반복하는 것이 중요합니다.

**Q5. 수능 D-100일 전에는 무엇을 해야 하나요?**

A: 기출 완성과 약점 보완에 집중하세요. 새로운 개념 학습은 피하고, 기출 3회독과 실전 모의고사로 실력을 완성하세요.

**Q6. 수능 당일 떨리는데 어떻게 하나요?**

A: 적절한 긴장은 정상이고 오히려 도움이 됩니다. 심호흡하고, 평소처럼 문제를 푼다는 마인드를 가지세요. 첫 문제가 어려워도 당황하지 마세요.

**Q7. 수능 전날 공부를 해야 하나요?**

A: 절대 과도한 학습은 금물입니다. 가벼운 개념 확인 정도만 하고, 일찍 자는 것이 훨씬 중요합니다. 컨디션 관리가 최우선입니다.

**Q8. 수능 점수가 낮게 나오면 어떻게 하나요?**

A: 재수나 다른 전형을 고려할 수 있습니다. 하지만 최선을 다한 결과라면 받아들이고 다음 단계를 준비하세요. 한 번의 시험이 인생을 결정하지 않습니다.

**Q9. 수능 날 아프면 어떻게 하나요?**

A: 가능하면 응시하세요. 약간의 컨디션 난조는 감안됩니다. 정말 불가능할 정도면 병원 진단서를 받아 추가 시험 기회를 받을 수 있습니다.

**Q10. 수능 후에는 무엇을 해야 하나요?**

A: 우선 푹 쉬세요. 그 후 정시 지원 전략을 수립하고, 면접이 있다면 준비하세요. 결과가 나오기 전까지 자기 관리를 계속하세요.

## 마무리: 수능 성공의 비결

**수능 성공은 전략과 실행, 그리고 멘탈입니다.**

**기억하세요:**

1. **개념이 기본**: 개념 없이 문제만 푸는 것은 무의미
2. **기출이 핵심**: 기출 문제를 완벽히 마스터
3. **실전 감각**: 주기적인 모의고사로 실전 대비
4. **약점 보완**: 자주 틀리는 유형 집중 학습
5. **컨디션 관리**: 몸과 마음의 건강이 최우선
6. **멘탈 관리**: 불안을 다스리고 자신감 유지
7. **전략적 포기**: 모든 문제를 풀 필요 없음
8. **끝까지 최선**: 마지막 순간까지 포기하지 않기

수능은 단순한 시험이 아니라, 여러분의 노력과 전략이 만나는 순간입니다. 체계적으로 준비하고, 자신감을 가지고, 최선을 다하세요. 여러분의 성공을 응원합니다!
