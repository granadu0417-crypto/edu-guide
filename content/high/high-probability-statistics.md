확률과 통계는 쉽게 봤다가 생각보다 어려워서 당황하는 학생들이 많습니다. 개념 자체는 직관적으로 이해할 수 있지만, 실제로 문제를 풀 때는 체계적인 논리가 필요합니다. 빠짐없이 중복 없이 세는 방법, 조건에 따른 확률 계산, 통계적 추정까지 단계별로 정복해 봅시다.

확률과 통계는 불확실성을 다루는 수학입니다. 확률은 미래를 예측하고, 통계는 과거 데이터를 분석합니다. 경우의 수는 가능한 모든 경우를 세는 기초가 됩니다. 이 세 영역이 서로 연결되어 있으므로 순서대로 탄탄하게 학습하는 것이 중요합니다.

## 경우의 수의 핵심 원리

합의 법칙은 "또는"에 해당합니다. A 또는 B가 일어나는 경우의 수는 각각의 경우의 수를 더하되, 겹치는 부분을 빼야 합니다. 예를 들어 영어 또는 수학 1등급인 학생 수를 세려면, 영어 1등급 5명에 수학 1등급 7명을 더하고, 둘 다 1등급인 2명을 빼서 10명이 됩니다.

곱의 법칙은 "그리고"에 해당합니다. A와 B가 모두 일어나는 경우의 수는 각각의 경우의 수를 곱합니다. 주사위 2개를 던질 때 가능한 모든 경우는 첫 번째 주사위 6가지와 두 번째 주사위 6가지를 곱해서 36가지가 됩니다.

순열은 n개 중 r개를 뽑아 순서대로 배열하는 경우의 수입니다. 5명 중 3명을 뽑아 회장, 부회장, 총무를 정하면 5×4×3 = 60가지입니다. 원형으로 배열하는 원순열은 회전 대칭을 고려하여 (n-1)!로 계산합니다. 5명이 원탁에 앉는 경우는 4! = 24가지입니다.

조합은 순서를 고려하지 않고 뽑기만 하는 경우입니다. 5명 중 3명을 그냥 뽑으면 5C3 = 10가지입니다. 순열에서 r!을 나누면 조합이 됩니다. 뽑은 것들의 순서가 의미 없기 때문입니다.

중복을 허용하면 계산이 달라집니다. 중복순열 nΠr은 n^r로 계산합니다. 주사위를 3번 던지면 6³ = 216가지입니다. 중복조합 nHr은 (n+r-1)Cr로 계산합니다. 3종류 과일 중 5개를 중복해서 고르면 7C5 = 21가지입니다.

## 확률의 기본 개념

수학적 확률 P(A)는 사건 A의 경우의 수를 전체 경우의 수로 나눈 것입니다. 주사위에서 짝수가 나올 확률은 3/6 = 1/2입니다. 확률의 기본 성질로 모든 확률은 0과 1 사이이며, 전체 사건의 확률은 1, 공사건의 확률은 0입니다. 여사건의 확률 P(A')은 1 - P(A)입니다.

덧셈정리 P(A∪B) = P(A) + P(B) - P(A∩B)는 합의 법칙과 같은 논리입니다. 두 사건이 동시에 일어날 수 없는 배반사건이면 겹치는 부분이 없으므로 P(A∪B) = P(A) + P(B)가 됩니다.

## 조건부 확률 정복하기

조건부 확률 P(A|B)는 B가 일어났을 때 A가 일어날 확률입니다. 계산은 P(A∩B)/P(B)로 합니다. 핵심은 조건으로 인해 표본공간이 축소된다는 것입니다. B가 이미 일어났다고 가정하면 B만 고려하면 됩니다.

곱셈정리 P(A∩B) = P(B)·P(A|B) = P(A)·P(B|A)는 조건부 확률의 정의를 변형한 것입니다. 두 사건이 동시에 일어날 확률을 구할 때 유용합니다.

독립사건은 한 사건이 다른 사건에 영향을 주지 않는 경우입니다. 동전을 두 번 던질 때 첫 번째 결과가 두 번째에 영향을 주지 않습니다. 독립사건이면 P(A∩B) = P(A)·P(B)이고, P(A|B) = P(A)입니다.

베이즈 정리 P(A|B) = P(B|A)·P(A)/P(B)는 원인과 결과를 뒤집어 생각할 때 사용합니다. 결과를 보고 원인의 확률을 구하는 문제에서 활용됩니다. 예를 들어 질병 검사에서 양성이 나왔을 때 실제로 질병일 확률을 구하는 문제가 대표적입니다.

## 확률분포 이해하기

확률변수 X는 표본공간의 각 원소에 실수를 대응시키는 함수입니다. 주사위 2개를 던질 때 합을 X라 하면 X = 2, 3, ..., 12의 값을 가집니다. 확률분포는 각 값의 확률 P(X=x)의 분포이며, 모든 확률의 합은 1입니다.

기댓값 E(X)는 확률변수의 평균값으로, 각 값에 확률을 곱해서 더합니다. E(aX+b) = aE(X) + b의 성질이 있습니다. 분산 V(X)는 평균으로부터 얼마나 퍼져있는지를 나타내며, E(X²) - [E(X)]²로 계산합니다. 표준편차 σ는 분산의 제곱근입니다.

이항분포 B(n, p)는 성공 확률이 p인 시행을 n번 반복할 때 성공 횟수의 분포입니다. 동전을 10번 던져 앞면이 나오는 횟수는 B(10, 0.5)를 따릅니다. 기댓값은 np, 분산은 np(1-p)입니다.

정규분포 N(μ, σ²)는 좌우 대칭의 종 모양 곡선입니다. μ가 중심이고 σ가 퍼짐을 나타냅니다. 표준정규분포 N(0, 1)로 표준화하면 Z = (X-μ)/σ로 변환됩니다. 정규분포표를 활용하여 확률을 구합니다.

## 통계적 추정

모집단은 조사 대상 전체이고, 표본은 모집단의 일부입니다. 모수(μ, σ)는 모집단의 특성이고, 통계량(x̄, s)은 표본에서 계산한 값입니다. 목적은 표본으로 모집단을 추정하는 것입니다.

표본평균 X̄의 분포는 모집단이 N(μ, σ²)이고 표본크기가 n일 때 N(μ, σ²/n)을 따릅니다. 표준화하면 Z = (X̄-μ)/(σ/√n)입니다. 중심극한정리에 따르면 n이 충분히 크면(보통 30 이상) 모집단 분포에 관계없이 표본평균은 정규분포를 따릅니다.

모평균의 점추정은 표본평균 x̄로 모평균 μ를 추정하는 것입니다. 구간추정은 신뢰구간을 제시하는 것으로, 신뢰도 95%에서 μ는 x̄ ± 1.96σ/√n 구간에 있을 것으로 추정합니다.

## 효과적인 학습 전략

경우의 수 학습에서는 체계적으로 세는 연습이 핵심입니다. 수형도를 그리거나 표로 정리하거나 작은 경우를 나열하며 패턴을 찾습니다. 중복은 없는지, 빠진 것은 없는지, 순서를 고려했는지, 조건을 만족하는지 항상 확인합니다.

확률 학습에서는 표본공간을 정확히 파악하는 것이 첫걸음입니다. 전체 경우의 수를 구하고, 사건을 정의하며, 확률을 계산합니다. 결과가 0과 1 사이인지, 상식적으로 맞는지 검증합니다.

## 수능 확률과 통계 대비

수능에서 영역별 비중은 경우의 수 30%, 확률 40%, 통계 30% 정도입니다. 난이도는 쉬운 문제 40%, 보통 40%, 어려운 문제 20%로 구성됩니다.

시간 관리가 중요합니다. 21번부터 26번까지는 25분 안에 기본 계산을 빠르게 끝내고, 27번부터 29번까지는 35분 동안 정확히 풀며, 30번 킬러는 15분을 배정합니다. 검토 시간 25분도 확보하세요.

유형별로 전략이 다릅니다. 경우의 수 문제는 조건을 정리하고 순열인지 조합인지 판단하여 체계적으로 세고 검산합니다. 조건부 확률 문제는 조건을 파악하고 표본공간을 축소하여 사건을 재정의한 후 확률을 계산합니다. 이항분포 문제는 n과 p를 확인하고 구하는 확률에 공식을 적용합니다. 정규분포 문제는 표준화하고 표를 활용하며 대칭성을 활용합니다.

킬러 30번 전략은 3분 동안 문제를 파악하고, 2분 동안 n=1, 2, 3의 작은 경우로 패턴을 찾습니다. 5분 동안 규칙을 일반화하여 식을 세우고, 4분 동안 정확히 계산하며 1분 동안 상식적으로 검증합니다.

## 확률과 통계 1등급을 위한 마무리

확률과 통계는 논리적 사고와 체계적 접근이 필요한 과목입니다. 경우의 수는 빠짐없이 중복 없이 체계적으로 세는 연습을 하고, 확률 개념은 표본공간을 정확히 파악하는 것에서 출발합니다. 조건부 확률은 표본공간 축소 개념을 정복하고, 이항분포와 정규분포의 특성을 이해합니다. 계산 실수를 최소화하고 항상 결과를 검증하는 습관을 들이세요.
